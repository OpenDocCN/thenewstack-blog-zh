# 为什么您应该关注多元化和包容性

> 原文：<https://thenewstack.io/diversity-inclusion-care/>

[多样性&包容性:你为什么要在乎](https://thenewstack.simplecast.com/episodes/diversity-inclusion-why-you-should-care)

[Atlassian](https://www.linkedin.com/in/adblanche/)的多元化和包容性全球负责人 Aubrey Blanche 参加了我们最新一期的 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客，谈论多元化和包容性之间的区别，以及为什么每个人都应该关注这一点。

“多元化被邀请参加聚会，”她说。“包容就是很高兴你在那里。”

布兰奇解释说，当你创造一种包容的文化时，商业就会繁荣。那些愿意把真实的自己带到工作中的员工表现得更好，工作时更快乐，这导致更少的流动，从而带来更大的利润。

在这段对话中，Blanche 详细介绍了 Atlassian 为创建包容性文化所采取的具体步骤。她出身于政治学背景，关注的是可衡量的结果。在公司倡导包容性的工作中，她说她意识到团队是衡量的单位，因为团队是所有工作发生的地方。一旦她将精力集中在创造团队多样性上，她就开始做出改变并衡量结果，这最终导致 Atlassian 做生意和招聘方式的改变。

她说，同情心是 Atlassian 招聘的最重要标准。反正技术技能很可能过几年就过时了。
