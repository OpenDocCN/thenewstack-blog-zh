# 区块链的用例，真实的和假设的

> 原文：<https://thenewstack.io/the-use-cases-for-blockchain-real-and-hypothetical/>

[#165:与 Wired、Hyperledger、IBM](https://thenewstack.simplecast.com/episodes/165-discussing-blockchain-with-wired-hyperledger-and-ibm) 讨论区块链

区块链终于度过了华尔街的难关。既然比特币和以太坊本质上已经是旧闻，这些商品背后的实际技术开始渗透到现实世界的企业应用中。区块链似乎在商业世界中有许多有用的用例，在 Linux 基金会和 IBM 的帮助下，企业现在可以利用区块链技术的开源 Hyperledger 实现。

在今天这一集的[中，新栈分析师](https://thenewstack.io/podcasts/analysts)播客[在 2018 年 OSCON 大会上现场录制了](https://www.periscope.tv/thenewstack/1YpJkLjoeyjxj) [Klint Finley](https://klintfinley.com/) ，他是 [Wired](http://www.wired.com) 的记者，也是“ [Wired Guide to Blockchain](https://www.wired.com/story/guide-blockchain/) 的作者，他说他几个月前写的那篇文章中有一些真实和假设的用例。“区块链感觉是最难从炒作中解析出价值的东西。对于云计算，这显然是一个被大肆宣传的概念，但是有使用案例和好处。大数据是从谷歌和雅虎等公司的实际使用案例中产生的。就物联网而言，这可能是目前最不完善的，但仍有既定的使用案例。和区块链在一起，感觉真的很初步，”芬利说。

然而，Linux 基金会的 Hyperledger 项目的社区架构师 Tracy Kuhrt 指出了 Hyperledger 的一些当前用例。“我最近谈论的一个话题是他们在不列颠哥伦比亚省所做的事情。不列颠哥伦比亚政府正致力于为所有在不列颠哥伦比亚注册的企业提供数字身份。那是 5000 种不同的生意。从那里你可以基本上确认某人确实已经在不列颠哥伦比亚政府注册，然后继续完成在不列颠哥伦比亚获得营业执照所需的其他事情。

IBM[的杰出工程师兼开放技术首席技术官 Christopher Ferris](http://www.ibm.com) 说,“最初做的许多实验和用例本质上是非常专有的，人们试图设想区块链用例，他们可以破坏你们所有人，并取得胜利。但实际上根本不是这样。在区块链，这是一项团队运动。如果我们想解决供应链物流领域、贸易融资领域或食品信任和可追溯性用例中的问题，你实际上必须为此建立一个几乎无处不在的业务案例，并让每个人都接受它。这不是技术问题，这是社会问题。我如何让我最激烈的竞争对手与我合作，创造一个对我们大家都有利的解决方案，”Ferris 说。
