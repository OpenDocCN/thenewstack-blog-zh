# Neo4J 图表数据库如何帮助破译巴拿马文件

> 原文：<https://thenewstack.io/neo4j-biggest-break-in-data-journalism/>

在本期[新堆栈制造商](https://thenewstack.io/podcasts/makers)播客中，我们采访了 Neo Technology 产品副总裁 [Phillip Rathle](https://www.linkedin.com/in/prathle) 。去年， [Neo4J](https://neo4j.com/) 图表数据库帮助[国际调查新闻联盟](https://panamapapers.icij.org/)理解了现在臭名昭著的[巴拿马文件](https://panamapapers.icij.org/)，允许该组织将 320，000 个离岸账户的非法活动联系起来，[这项工作导致冰岛总理辞职](https://thenewstack.io/neo4j-defends-turf-graph-database-field/)。

在本期播客中，我们将探索图形数据库之间的差异，了解更多关于 Neo4J 在解密巴拿马文件中的作用，并探索 2017 年 Neo 技术的下一步。
