# code 公证人云保护软件供应链和开发人员

> 原文：<https://thenewstack.io/codenotary-cloud-secures-software-supply-chain-and-devsecops/>

软件供应链安全一直是一个问题。直到最近，由于 [Solarwinds 软件供应链的惨败，](https://www.zdnet.com/article/solarwinds-the-more-we-learn-the-worse-it-looks/)我们才意识到这一点。事情发生了变化，我们现在知道这是一件大事。因此，当[code 公证人宣布其](https://codenotary.com/)[code 公证人云](https://codenotary.com/category/codenotary-cloud/)中的新功能时，这是一项值得关注的顶级端到端软件供应链安全服务。

这建立在[code 公证人社区证明服务](http://cas.codenotary.com/)的基础上，这是一个开源的公证和验证服务。这可以与[持续集成/持续交付(CI/CD)](https://practical-tech.com/2018/07/10/continuous-integration-and-delivery-tool-basics/) 服务一起产生可靠的软件物料清单(SBOM)。

## 漏洞扫描

在这个最新版本中，code 公证人的漏洞扫描被纳入服务。您还可以将 code 公证人云与其他漏洞扫描器集成。这使您能够为您的 SBOMs 向客户提供安全、防篡改的软件供应链。其他 more 公证人云信任增强功能使其保护范围内的程序更易于在 DevOps 和 Kubernetes 部署中使用。

这个防篡改的 SBOM 用于开发工件的源代码、构建、存储库、Docker 容器映像等等。每个这样的工件都保留了一个加密的强身份，存储在开源的不可变数据库 [immudb](https://github.com/codenotary/immudb) 中。所有这些都可以以每秒数百万次检查的速度进行验证

## 更低的成本，更高的合规性

cost 公证人声称，这种新的和改进的 cost 公证人云大大降低了快速识别和删除不想要的工件的成本，高达 80%。该公司还表示，它可以满足那些要求遵守美国关于改善国家网络安全的行政命令的企业的需求。

code 公证人联合创始人兼首席执行官 [Moshe Bar](https://www.linkedin.com/in/moshebar/) 表示:“我们的云服务将简单性与业界最先进的功能相结合，以确保安全的软件供应链。“它是不可变的、防篡改的，让用户完全信任他们赖以开展业务的基本软件。在当今世界，像 [log4j](https://thenewstack.io/log4shell-we-are-in-so-much-trouble/) 和网络安全管理软件产品这样的软件供应链攻击已经产生了深远的影响，这不是小事。”

因此，今天，code 公证人云可以与许多流行的 CI/CD 系统集成。DevOps 证明服务作为托管服务运行在任何云或主机上，客户也可以自行托管。由 10 名开发人员组成的工作组的起价为 5500 美元。
