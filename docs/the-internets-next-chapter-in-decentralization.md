# 互联网去中心化的下一章

> 原文：<https://thenewstack.io/the-internets-next-chapter-in-decentralization/>

互联网的传奇历史包括分散化的里程碑，如[国防高级研究计划局(DARPA)](https://en.wikipedia.org/wiki/DARPA) 分组交换的最初发展 [(ARPANET)](https://en.wikipedia.org/wiki/ARPANET) ，以及 20 世纪 90 年代初[蒂姆·伯纳斯·李的](https://en.wikipedia.org/wiki/Tim_Berners-Lee) [万维网](https://en.wikipedia.org/wiki/World_Wide_Web)，它使每个人都可以用电脑共享数据。今天，随着微服务、Kubernetes 和分布式环境变得越来越普遍，互联网的使用将分散式通信带到了下一个层次。

在本期由 TNS 创始人兼发行人 [Alex Williams](/author/alex/) 主持的 New Stack Makers 播客中，Storj Labs 董事长兼临时首席执行官 Ben Golub 和 Storj 合规负责人 [Katherine Johnson](https://www.linkedin.com/in/katherine-johnson-73201058/) 讨论了互联网如何以去中心化为中心，以及更重要的是，点对点通信将如何继续发展。

[去中心化让互联网回归本源](https://thenewstack.simplecast.com/episodes/decentralization-returns-the-internet-to-its-roots)

DARPA 认为互联网是一个强大的网络，即使大部分被摧毁(比如说，被核攻击)，它也能存活下来。为此，它必须在没有中央组织中心的情况下运行，这实际上是一个单点故障。今天，有“许许多多的路由器分布在许许多多的地方，其中任何一个都可能发生故障，但这没关系，因为信息仍然会通过，”Golub 说。“所以，这是我第一次接触互联网和它的去中心化版本，也就是说，如果你利用去中心化，你实际上可能比让一架飞机不停地在空中飞来飞去或在大掩体中(包含基础设施)更安全。”

Golub 说，今天，在许多方面，对更多去中心化的一致推动“正在回归互联网的本来面目”。“如果你想想互联网是如何在你我交谈时工作的——当人们在观看猫的视频或其他东西时放大——所有的信息都被分解成大量可区分的信息包，这些信息被分解并通过由许多不同的人运行的路由器设置，而你在很大程度上并不关心。我的意思是，我不知道谁在运行路由器，这些路由器转发你和我正在交谈的数据包，我也不在乎其中一半是否会中断，因为消息只会转到下一个，”Golub 说。

“这才是去中心化的真正意义所在:说‘嘿，我们能不能采用那种推动了互联网的概念，它显然大大降低了成本，并促成了我们所看到的所有创新，我们能不能把它也应用到云计算中，你能不能以一种不集中的方式进行存储，或者你能不能以一种不集中的方式进行计算？’因为现在，四分之三的云计算是由这个星球上最大的三家公司之一完成的。"

实际上，这就是 Storj 的神奇之处，它提供了一种分布式或分散式的数据存储，这种存储建立在全球的备用存储空间上。文件被加密、分割成小块，并存储在地理位置不同的节点上。

如今，许多观察家认为，计算机产业正处于重复过去几十年的巨大进步的风口浪尖，其结果是经济和文化方面的重大影响。“有一种感觉，这里发生的事情不是每个人都理解的——有些人理解，有些人理解其中的一部分——但如果我们把所有的东西放在一起，理解不同部分的不同人把他们的知识放在一起，那么我们真的可以创造一些东西，”约翰逊说。“我总是能感觉到它的灵魂，这就是我的印象。在 Storj，我们有同样的想法:我们是一家开源公司，每个人都参与其中，每个人都可以产生影响，这也真正融入了我们的文化。”
