# 上下文:Dynatrace 向 Kubernetes 介绍 AIOps

> 原文：<https://thenewstack.io/context-dynatrace-introduces-aiops-to-kubernetes/>

欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)，在这个播客中，我们将讨论来自云原生社区的最新新闻和观点。本周，我们采访了 [Dynatrace](https://www.dynatrace.com/) 的 DevOps 积极分子 [Andreas Grabner](https://www.linkedin.com/in/grabnerandi/) ，探讨为什么 [Kubernetes](/category/kubernetes/) 和[人工智能](/category/machine-learning/) (AI)是天生一对。

Grabner 上周为我们写了一篇投稿文章，“[人工智能如何解决 Kubernetes 的复杂性难题](https://thenewstack.io/how-ai-solves-the-kubernetes-complexity-conundrum/)”，其中他认为，将 Kubernetes 与人工智能和 AIOps 文化相集成可以使 Kubernetes 的部署更易于 IT 和 DevOps 团队管理。Dynatrace 是新堆栈的赞助商，它为动态、网络规模的混合云生态系统构建了人工智能辅助的全堆栈和完全自动化的监控软件。

“Kubernetes 为企业提供了一个宝贵的新工具，将数字化转型和上市时间表推向新的阶段。但为了充分利用它，it 和 DevOps 需要推动向人工智能和自动化的文化转变；Grabner 在博客中写道:“大多数其他现有的 IT 方法根本无法在这个新世界中充分工作或扩展。
