在本周的 The New Stack Context 播客中，我们与NetApp云基础架构业务部门的业务总监戴尔·德根在本周于拉斯韦加斯举行的年度用户大会、 NetApp Insight 2019 上讨论了Kubernetes和云原生存储。

自从去年收购StackPointCloud 以来，存储巨头 NetApp 已经成为Kubernetes管理领域的领导者。 NetApp Kubernetes服务为运行任何商业甚至本土的Kubernetes发行版提供了一个控制平面。用户只需三次点击就可以部署一个完整的Kubernetes发行版。当与NetApp on tap存储操作系统和云卷结合使用时，用户可以设置多云Kubernetes部署。

我们与Degen讨论了NetApp如何支持Kubernetes，以及建立覆盖整个企业的数据结构的重要性。我们还聊到了一款全新的NetApp对象存储设备，它由高速闪存驱动器构建而成，据Degen称，这在存储行业尚属首次。
