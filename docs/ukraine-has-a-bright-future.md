# 乌克兰有着光明的未来

> 原文：<https://thenewstack.io/ukraine-has-a-bright-future/>

乌克兰有一个光明的未来。很快就是重建的时候了。但是重建需要的不仅仅是建造一座水电站或一所医院所需的资源。它包括软件和对如何使用软件的理解。

[乌克兰前途光明](https://thenewstack.simplecast.com/episodes/ukraine-has-a-bright-future)

云原生计算基金会(CNCF)的开发者倡导者伊霍尔·德沃雷茨基(Ihor Dvoretskyi)和 [Razom for Ukraine](https://www.razomforukraine.org/) 的董事会成员马頔·扎哈尔亚夫科来到底特律的 KubeCon，讨论在乌克兰从俄罗斯入侵造成的破坏中重建之际，为其提供培训材料的事宜。

Razom 是一家非营利组织，它放大了在美国的乌克兰人的声音，并帮助开展人道主义工作和信息技术培训。Zakhalyavko 说，Razom 是在俄罗斯 2014 年入侵乌克兰克里米亚半岛之前成立的。自今年早些时候全面入侵以来，Razom 的捐款和志愿者帮助他们的努力有了可以理解的增加。

Razom 为士兵提供个人急救包，止血带和医疗用品，但它也是培训，培训下一代 IT 的材料，翻译成乌克兰语。

Linux 基金会与云计算原生计算基金会(CNCF)共同参与 Razom for Ukraine 的[项目 Veteranius](https://www.razomforukraine.org/projects/veteranius/) ，为乌克兰退伍军人、他们的家人以及有需要的乌克兰人提供技术教育。

“我们已经意识到，基本上，我们可以从 Linux 基础培训组合中受益，包括最受欢迎的课程，如 Linux 入门或 Kubernetes 入门，这些课程可以很容易地翻译成乌克兰语，”Dvoretskyi 说。“通过这种方式，我们将能够以他们的母语提供教育材料。”

乌克兰有着非常光明的未来。

“我们只是需要度过这些困难时期，”德沃雷茨基说。“但从未来来看，乌克兰的科技产业显然正在增长。是啊。这就需要人。”

Dvoretskyi 说，每一次努力都很重要。

“一个强大、民主的乌克兰——这基本上是一个愿景——一个欧洲国家，一个真正的欧洲国家，在领土完整方面是完整的，”扎哈尔亚夫科说。“未来在于技术。如果我们能帮助实现这一目标，无论如何，我认为这对乌克兰和全世界都是一个胜利。技术可以让世界变得更美好。”
