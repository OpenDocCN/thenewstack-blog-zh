# 微软与 Anaconda 合作共享 Python 和 R 库

> 原文：<https://thenewstack.io/microsoft-anaconda-python-r/>

Python 数据科学平台提供商 [Anaconda，Inc.](http://www.anaconda.com/) 与[微软](https://www.microsoft.com/)联手，为微软提供 [Anaconda，这是 Anaconda 发行版的一个子集，可在 Windows、MacOS 和 Linux 上使用。在本周纽约举行的](https://www.microsoft.com/en-us/store/movies/anaconda/8d6kgwzl5xk3) [Strata Data Conference](https://conferences.oreilly.com/strata/strata-ny) 期间，两家公司还宣布，Anaconda 将被嵌入 [Azure Machine Learning](https://azure.microsoft.com/en-us/services/machine-learning-studio/) 、 [Azure Machine Learning](https://azure.microsoft.com/en-us/services/machine-learning-studio/) 、 [Visual Studio](https://www.visualstudio.com/free-developer-offers/) 和 [SQL Server](https://www.microsoft.com/en-us/sql-server/sql-server-2016) ，为这些微软实体提供实时数据洞察能力。

“这对 Anaconda 来说是一个非常令人兴奋的消息，”Anaconda 产品和营销高级副总裁 Mathew Lodge 说。“微软凭借 Azure 以及 SQL Server 和 Visual Studio 等非常成功的软件，拥有巨大的企业影响力和影响力。因此，让 Anaconda for Microsoft 在这些知名平台上可用，可以让数据科学家更快、更容易地将他们的工作部署到成千上万个组织的各种企业中。”

Python 是领先的数据科学语言，Anaconda 的 450 万活跃用户使其成为最受欢迎的 Python 数据科学发行版。Lodge 断言，Anaconda 与微软的合作将是企业数据科学实际应用的巨大进步。

例如，假设一名数据科学家想要对 SQL Server 中包含敏感个人信息的大型数据集运行预测分析。

“在这种合作之前，数据科学家不可能直接在数据库中运行他们的 Python 程序来获得最大的性能和安全性，因为他们无法访问数据库中所需的 Python 环境和库，”Lodge 解释道。相反，他们必须使用 Python 环境和包来设置远程服务器，编写代码来运行对数据库的查询，通过网络从数据库传输所有敏感数据，将其加载到内存中，然后运行他们的分析

展望未来，Anaconda 将确保在 SQL Server 中提供完全正确的 Python 环境和包，以便直接在数据上运行数据科学程序。洛奇补充说，同样的原理可以应用于其他数据密集型系统，如 Azure 机器学习。

### 你有经验吗？

合作的另一个方面集中在 Anaconda 用户身上，他们使用 R 语言和 T2 自由软件环境进行统计计算和制图。新协议将使[微软 R 开放包](https://mran.microsoft.com/rro/installed/)默认提供给 [Anaconda 的 R Essentials](https://anaconda.org/r/r-essentials) 包的用户。此外，Visual Studio 代码、R Client 和 R Tools for [Visual Studio 代码](https://code.visualstudio.com/)将被添加到 [Anaconda Navigator](https://anaconda.org/anaconda/anaconda-navigator) 中，它“帮助数据科学家了解 Anaconda 生态系统。”

> 根据合作条款，Anaconda 将成为嵌入 Azure 机器学习、Visual Studio 和 SQL Server 的事实上的 Python 发行版。Microsoft R Open 将成为 Anaconda 数据科学平台中可用的事实上的 R 发行版。

微软公司数据集团公司副总裁 Joseph Sirosh 在一份新闻稿中表示:“随着 Anaconda 发布 Microsoft R，微软在 Microsoft SQL Server、Microsoft Azure Machine Learning Services 和 Microsoft Visual Studio 中包括 Anaconda 发布，我们的数据平台和云客户可以利用一些最佳的开源和专有框架进行高性能分析和机器学习。“通过在 SQL Server 中结合 R 和 Python，我们正在促进数据科学在整个企业中的扩展。”

微软是新堆栈的赞助商。
