# 现成的黑客:来自游戏轨迹的在线更新

> 原文：<https://thenewstack.io/off-the-shelf-hacker-online-updates-from-the-game-trail/>

现在是八月初，狩猎季节即将来临。我更喜欢钓鱼和“享受在树林里”的类型。不管怎样，作为一个虔诚的杂食者，我从来没有拒绝过放在我面前盘子里的鹿肉汉堡、野猪或鳄鱼。

猎矢如何才能[现成的黑客](/tag/Off-The-Shelf-Hacker)？

想象一下，当我在每月的贝斯专卖店传单上的狩猎部分发现一个“游戏摄像头调制解调器”时，我有多惊讶。

没错，你可以购买现成的电池供电调制解调器，将你的游戏摄像头连接到蜂窝网络。游戏摄像头有运动传感器，当四条(或两条)腿的动物在镜头前移动时，它会拍照。图片和视频存储在 SD 卡上。传统上，您必须长途跋涉才能取回卡并将内容上传到您的笔记本电脑。

不再是了。调制解调器连接到您现有的游戏摄像头，您只需启动智能手机或 PC 上的在线应用程序，就可以准确地看到您最喜欢的游戏轨迹上正在发生的事情。

这是下一件大事吗？不，实际上有点过时了。有一大批游戏摄像头已经内置了调制解调器。游戏摄像头是便携式自供电摄像头，可以放置在小径、森林或任何其他偏远的地方。这些设备拥有 1200 万至 2000 万像素及以上的摄像头，以及 1080P 视频和音频以及夜视功能。甚至有带外部远程无线红外摄像头的型号，这些摄像头连接回主蜂窝功能的彩色白天/红外夜晚游戏摄像头。这些应用程序甚至可以识别它看到的动物。

他们接下来会想到什么？

## 一条与众不同的路

在绝对最新的尖端技术上，现成的黑客同时站在最前沿，而且经常落后于潮流。我可以告诉你，跟上所有最新的技术是非常困难的，我已经在这一行待了很长时间。大多数黑客觉得他们在群体中遥遥领先。

很多时候我们是，有时不是。一个游戏摄像头调制解调器直到几天前才出现在我的雷达上。没关系。保持谦逊，对“不在此发明”(NIH)的想法感到惊奇，这让我们保持积极的心态，对新的机会保持开放。只是把这一切都考虑进去，然后加入你自己的想法。这是现成黑客的基础。也许调制解调器 99 美元的价格标签可以使它成为一种可行的方式，将你现成的黑客项目之一放到物联网上。不仅如此，它们还是迷彩色的。

仔细想想，手机游戏摄像头技术并不那么先进。这是一个手机调制解调器，连接到一个微控制器，有一个 SD 卡和一个红外运动传感器。读者也许可以通过添加自己的调制解调器板和设置手机应用程序，从 2016 年开始在现有的 [Peak Nature 项目](https://peaknature.co.uk/blog/how-to-build-a-raspberry-pi-trail-cam--part-1-introduction)的基础上进行建设。Digi 有一个 [XBee 外形调制解调器](https://www.digi.com/products/embedded-systems/cellular-modems/digi-xbee-cellular#partnumbers)可以用于此目的。更进一步，你可能会连接上 [IFTTT](https://ifttt.com/) 或[亚马逊 Alexa](https://developer.amazon.com/alexa) 来获得某种漂亮的界面。虽然游戏摄像头已经存在了几年，但手机连接摄像头最近才出现。

我不得不笑自己，没有看到一个手机连接的游戏摄像头的到来。科技发展非常迅速，很高兴看到它出现在意想不到的地方。类似的进步也发生在鱼类探测器上。

离开人迹罕至的道路，用不同的眼光看待事物，探索“与众不同”的事物，帮助我对新的小工具和项目保持开放的态度，即使在我这个年龄也是如此。

## 好奇

全盘接受并加入自己的观点是现成黑客的基础。当然，这是基于个人高度的好奇心。我天生对几乎所有东西都好奇。我从蹒跚学步时就这样了。有时候会让人很烦。

总之，过去一周的一次互动证实了科技对我来说是一个闪亮、光明的未来。

我正在我的一个非现场咖啡馆里制作我的机器人头骨[，我称之为赫德利](https://thenewstack.io/off-the-shelf-hacker-conversations-with-hedley-the-robotic-skull/)。一个穿着医用工作服的年轻人正在邻桌吃东西。赫德利正坐在我的桌子上看着窗外。

年轻人吃完了饭，一直看着赫德利。我知道会发生什么。没多久，他走过来，礼貌地问是什么事。我说这是一个机器人头骨，带有智能机器视觉传感器，平移和下颚伺服系统，还有一个树莓派的大脑。他显然很兴奋，我们坐着聊了几分钟。

原来他是一名机械工程专业的学生，来自一所州立大学，在当地一家医疗机构实习。他的兴趣是制造外科设备，他说今年秋天他将学习机电一体化课程。我用 Linux 笔记本上的 [guvcview viewer](http://guvcview.sourceforge.net/) 演示了视觉传感器如何识别附近的椅子。他不太熟悉树莓派，而 [JeVois 传感器](http://jevois.org/)对他来说绝对是新事物。他显然对传感器 50 美元的价格感到惊讶，而且这实际上是两年前的技术。

赫德利和我鼓励他保持联系，我建议他去看看“[现成的黑客](/tag/Off-The-Shelf-Hacker)”专栏。当然，他已经在他的智能手机上调出来了。

## 科技圈

新想法、项目和社交的灵感有多种形式。有时候，你可能会在一家大型户外连锁店的狩猎部里走来走去。其他时候，你可能会在餐厅遇到一个默默无闻、有点古怪的科技作家，带着一些明显“与众不同”的小工具。

作为现成的黑客，我们可以推广这项技术，甚至同时影响未来。年长的高级黑客可以把这些点联系起来，给人灵感，并帮助指导新的黑客。年轻的和新的黑客可以保持好奇心，无畏地探索即将到来和即将到来的事情。

这是一个科技行业的大好时机。你的下一件大事是什么？
