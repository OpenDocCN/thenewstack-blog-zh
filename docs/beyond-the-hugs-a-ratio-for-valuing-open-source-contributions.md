# 拥抱之外:评估开源贡献的比率

> 原文：<https://thenewstack.io/beyond-the-hugs-a-ratio-for-valuing-open-source-contributions/>

[拥抱之外:一个重视开源贡献的比率](https://thenewstack.simplecast.com/episodes/beyond-the-hugs-a-ratio-for-valuing-open-source-contributions)

除了拥抱，开源贡献还有相当大的好处。回报在于开源贡献的价值。确实有一个可以使用的比率。Sensu 的 [Matthew Broberg](https://twitter.com/mbbroberg) 在本期 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客中说道，这是从开源项目中发现价值的人数与开源公司从社区互动中获得的收入和整体价值的对比中得出的，该播客是在本月早些时候在加利福尼亚州索诺玛谷举行的 Linux 基金会[开源领导力峰会](https://events.linuxfoundation.org/events/open-source-leadership-summit-2018/)上录制的。

Sensu 的社区副总裁 Broberg 从人文主义和商业的角度阐述了他对社区管理如何为商业和使用开源软件的个人提供价值的看法。发现平台价值的总体贡献和开源用户并不总是意味着这些人会使用 Sensu 提供的监控平台的商业版本。但是如果有一定比例的开源用户在使用 Sensu 产品，那就是一个健康的信号。

这就是为什么当布罗伯格与社区中的人会面时，互动很重要。纯粹从个人和商业层面来看，这些互动是有价值的。
