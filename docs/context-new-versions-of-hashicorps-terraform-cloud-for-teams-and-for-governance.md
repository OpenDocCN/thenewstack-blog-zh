# 背景:HashiCorp 面向团队和治理的 TerraForm 云的新版本

> 原文：<https://thenewstack.io/context-new-versions-of-hashicorps-terraform-cloud-for-teams-and-for-governance/>

欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)，在这个播客中，我们将讨论来自云原生社区的最新新闻和观点。本周，我们在西雅图的 HashiConf 录制了我们的播客，采访了 Hashicorp 的联合创始人兼首席技术官[艾蒙·达德加尔](https://twitter.com/armon?lang=en)。

[hashi corp 面向团队和治理的 TerraForm 云的新版本](https://thenewstack.simplecast.com/episodes/new-versions-of-hashicorps-terraform-cloud-for-teams-and-for-governance)

今年在西雅图举行的年度用户大会上，HashiCorp 为其 Terraform 云服务引入了两个新的层次，一个是付费版本，不是完整的企业版本，而是针对较小团队的，另一个涵盖治理和成本预测。 [Terraform Cloud](https://www.hashicorp.com/resources/what-is-terraform-cloud) 就像是 GitHub 对 Terraform 的 git。Terraform 是一个基础设施管理工具，可以从笔记本电脑上使用，HashiCorp 在 Terraform Cloud 上提供了一个免费层，在这里 CLI 命令被转换为 API 调用。新的团队层提供了管理团队的能力，具有完全的 RBAC 控制。治理版也是付费的，它有一个代码框架，用于针对基础设施执行粒度规则，甚至是成本估计，这提供了对提议的基础设施在云中运行的成本的预测。

在我们的谈话中，Dadgar 解释说，该层解决了这家快速增长的公司迄今为止服务不足的部分:已经发展成为需要可预测基础设施的全面应用程序的项目。它们不像企业(也称为“财富 2000 强”)那样需要治理控制，但是它们需要协调。该公司认为，随着地形改造项目的增长，将会有更多的参与者，一切都需要跟踪和控制。

我们还谈到了 HashiCorp 最受欢迎的软件[consult](https://www.consul.io/)，这是一个服务网格，它是在“服务网格”成为一个东西之前几年创建的。今年，在微软的帮助下，[公司在 Azure](https://thenewstack.io/hashicorp-consul-service-on-azure-the-first-fully-managed-service-mesh/) 上发布了该软件的托管版本。
