# Priyanka Sharma 担任云计算原生计算基金会的领导

> 原文：<https://thenewstack.io/priyanka-sharma-assumes-leadership-of-the-cloud-native-computing-foundation/>

云原生专家 [Priyanka Sharma](https://www.linkedin.com/in/pritianka/) 现在将领导[云原生计算基金会](http://cncf.io)，接任总经理一职，由前执行董事 [Dan Kohn](https://www.dankohn.com/) 担任。

Sharma“将与代表整个技术行业的 CNCF 商业和技术领导层密切合作，继续建设云原生生态系统，”Linux 基金会执行董事 Jim Zemlin 在一份声明中说。CNCF 是 Linux 基金会的一部分。

Sharma 之前是 git lab[的云原生联盟总监，他为这一领导职位带来了丰富的经验，包括参与了几个 CNCF 项目，并且是 CNCF 董事会的成员。她还担任各社区最终用户的联络人。](https://about.gitlab.com/)

Sharma 是 CNCF [OpenTracing 标准](https://thenewstack.io/opentracing-opencensus-merge-into-a-single-new-project-opentelemetry/)的创始团队成员，并为其他几个 CNCF 项目做出了贡献，如 [Jaeger](https://www.jaegertracing.io/) 监控工具和 [Vitess](https://vitess.io/) 数据库集群工具。她还发表了许多关于 CNCF 和云原生技术的演讲。在 [GitLab](https://about.gitlab.com/) ，她组建了开发者福音团队，将公司的 DevOps 应用生命周期服务引入云原生社区。

Kohn 将启动一个新的 Linux 基金会计划，为公共卫生机构提供更多的开源软件和支持，尽管他将继续参与 CNCF 的一些活动，并继续担任顾问角色。

CNCF 现在支持近 50 个不同的项目，包括 [Kubernetes](https://kubernetes.io/) 容器编排引擎、[普罗米修斯](https://prometheus.io/)监控软件和[特使服务网](https://www.envoyproxy.io/)。所有这些项目都专注于提供开源技术，以帮助组织构建云原生计算系统。超过 90，000 名开发人员为这些项目做出了贡献。

Sharma 指出，Kohn 成功地吸引了 500 多个组织成员加入 CNCF，其中包括 142 个最终用户参与者，这是所有 Linux 基金会组织中比例最高的。但是她承认前面还有许多挑战。

“我相信开发人员的参与可以更强，我们可以做很多事情来教育社区关于 DevOps 故事的细节，”她说。“在教育、社区和培训方面，我们可以做得更多，我认为这只是基金会和整个社区另一段旅程的开始。”

云计算原生计算基金会、GitLab 和 Linux 基金会是新堆栈的赞助商。
