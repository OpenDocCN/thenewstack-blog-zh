# 背景:数据的未来在于流式传输

> 原文：<https://thenewstack.io/context-the-future-of-data-is-in-streaming/>

[未来的数据正在流式传输](https://thenewstack.simplecast.com/episodes/the-future-of-data-is-in-streaming)

在本周的 [The New Stack Context](https://thenewstack.io/podcasts/context) 播客中，我们采访了 [Lightbend](https://www.lightbend.com/) 的首席执行官 [Mark Brewer](https://twitter.com/mabrewer7) ，以及 New Stack 的研究主管兼专栏作家 [Lawrence Hecht](https://thenewstack.io/author/lawrence-hecht/) ，探讨数据流技术的现状和未来。

新堆栈最近与我们的赞助商 Lightbend 合作，开展了一项关于数据流的调查。Lightbend 上周发布了该调查的结果，该报告的标题为“[流数据和未来的技术堆栈，](https://www.lightbend.com/white-papers-and-reports/survey-streaming-data-future-tech-stack)”，它完全是关于开发人员和软件架构师如何在他们的应用程序中使用数据流。在播客中，布鲁尔分享了他从调查中获得的最大收获，赫克特讨论了他是如何分析这些数据的，以及他的结论是什么。我们还谈到了 Lightbend 的 [Akka 消息驱动运行时](https://www.lightbend.com/akka)。

我们还讨论了本周的头条新闻，包括 [VMware 收购 Bitnami](https://thenewstack.io/vmware-to-acquire-bitnami-the-app-marketplace-platform-and-container-packager/) ，英特尔针对云原生工作负载的新专用 Linux 虚拟机管理程序，以及 New Relic 面向未来的平台 New Relic One。

[Libby Clark](https://twitter.com/LibbyMClark) 是《新书库》的编辑和营销总监，他主持了这一集，并得到了《新书库》的创始人兼发行人 Alex Williams 和 TNS 执行主编 Joab Jackson 的一贯支持。

Lightbend、New Relic 和 VMware 是新堆栈的赞助商。
