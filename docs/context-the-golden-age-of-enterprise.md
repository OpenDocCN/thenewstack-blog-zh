# 背景:“企业的黄金时代”

> 原文：<https://thenewstack.io/context-the-golden-age-of-enterprise/>

欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)，在这个播客中，我们将讨论来自云原生社区的最新新闻和观点。

[企业的黄金时代](https://thenewstack.simplecast.com/episodes/the-golden-age-of-enterprise)

本周，我们正在录制 9 月 5 日在旧金山举行的[TechCrunch Sessions:Enterprise](https://techcrunch.com/events/tc-sessions-enterprise-2019/)活动。我们的采访嘉宾是种子期风险投资基金[异常风险投资](https://unusual.vc)的高级助理 Haley Daiber。

在过去的几个月里，作为媒体合作伙伴和参与者，新的团队一直在谈论和撰写大量关于 TechCrunch Sessions: Enterprise 的文章。我们在上午举行了煎饼早餐和播客小组，TNS 创始人亚历克斯·威廉姆斯在下午主持了一场会议。我们对这一事件感到兴奋，因为它标志着企业软件领域出现了新的热潮。这在我们今天听到的主题演讲中已经很明显了:例如，来自新兴资本的[杰森·格林称之为“企业的黄金时代”。因此，我们采访了 Daiber，从投资者的角度来谈谈企业软件是什么样子的。](https://www.emcap.com/people/jason-green/)

然后，在稍后的播客中，我们讨论了本周的热门故事和播客，包括关于为 Kubernetes 采用基于角色的访问控制[和迁移到 Python 3](/three-realistic-approaches-to-kubernetes-rbac/) 的优秀帖子。我们还讨论了新的服务网络，以及 20 世纪 70 年代的 T21“朋克摇滚”音乐和今天的开源音乐是如何相似的。

TNS 的编辑和营销总监 Libby Clark 在 TNS 创始人兼出版商 Alex Williams 和 New Stack 的执行主编 Joab Jackson 的帮助下主持了这个播客。
