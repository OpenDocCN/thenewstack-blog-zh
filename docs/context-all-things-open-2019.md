# 背景:万物开放 2019

> 原文：<https://thenewstack.io/context-all-things-open-2019/>

[VMware](https://www.vmware.com/company.html) 赞助了这次播客。

在本周的 [The New Stack Context](https://thenewstack.io/podcasts/context) 播客中，我们将讨论我们最喜欢的年度活动之一，即本月早些时候在北卡罗来纳州罗利举行的 [All Things Open](https://allthingsopen.org/) 开源会议。

我们本周的嘉宾有 [Matthew Broberg](https://www.linkedin.com/in/mbbroberg/) ，红帽[OpenSource.com](https://opensource.com/)的技术编辑，以及[Anais dot is-乔治乌](https://www.linkedin.com/in/anais-jackie-dotis-029623113/)，他是 [InfluxData](https://www.influxdata.com/) 的开发者倡导者，也是会议的发言人。我们讨论了让所有事情开放如此令人愉快的草根独立性，以及我们在今年的展会上捕捉到的开源趋势。

[上下文:万物开启 2019](https://thenewstack.simplecast.com/episodes/context-all-things-open-2019)

然后，在节目的稍后部分，我们讨论了本周的一些热门帖子和播客，包括[对 Pivotal 的 Cornelia Davis 的广泛采访](/cornelia-davis-patterns-javas-big-break-and-beyond-rest-apis/)，关于如何充分利用 Ubuntu 上的[ZFS 文件系统的教程](/how-to-create-and-destroy-zfs-snapshots-on-ubuntu-19-10/)，微软[如何让开发人员更容易使用 Kubernetes](/what-does-the-open-application-model-oam-and-rudr-mean-for-kubernetes-developers/) ，以及在最近一波基于 CPU 的漏洞之后 Linux 内核开源的状态[。](https://thenewstack.io/linux-kernel-security-in-the-age-of-spectre-meltdown/)

TNS 的编辑和营销总监 Libby Clark 主持了这次播客，一同主持的还有 TNS 创始人兼总编辑 Alex Williams 和 TNS 执行主编 Joab Jackson。

InfluxData、Linux 基金会、Pivotal 和 Red Hat 是新堆栈的赞助商。
