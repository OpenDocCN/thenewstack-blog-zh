# 2018 年 Sensu 峰会，凯尔西·海塔尔和凯勒·海利

> 原文：<https://thenewstack.io/the-new-stack-context-sensu-summit-2018/>

[新栈语境:Sensu 峰会 2018](https://thenewstack.simplecast.com/episodes/the-new-stack-context-sensu-summit-2018)

本周，我们在波特兰的[感官峰会](https://sensu.io/summit)现场录制了播客。谷歌技术专家[凯尔西·海托华](https://twitter.com/kelseyhightower)和 Sensu 首席执行官[凯莱布·海利](https://twitter.com/calebhailey)加入我们，讨论微服务监控的挑战。该小组同意，许多传统的监控工具，如 Nagios，甚至亚马逊网络服务的 CloudWatch，都无法捕捉到系统内部正在发生的事情的全貌。

本周，Hailey 为 TNS 写了一篇关于 Sensu 如何自动化从多种工具和多种格式收集传感器数据的过程的帖子。“当一切都是‘工作流’时，混合云监控变得更加容易推理。”这不仅更容易，而且你可以监控什么(以及如何监控)的可能性几乎是无限的，”他写道。

TNS 编辑部主任 Libby Clark 主持了这个播客，TNS 创始人 Alex Williams 和 TNS 执行主编 Joab Jackson 提供了帮助。
