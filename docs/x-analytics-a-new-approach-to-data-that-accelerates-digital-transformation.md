x 分析(X Analytics)是最近由研究公司 Gartner[创造的一个术语，指的是对组织的所有结构化和非结构化数据运行任何类型的分析的能力，无论这些数据位于何处或以何种格式存储。今天，这种能力比以往任何时候都更加重要。疫情极大地改变了客户和组织的行为，迫使各个行业加速数字化转型。](https://www.gartner.com/en)

微软首席执行官塞特亚·纳德拉表示，他的公司在短短两个月内见证了为期两年的数字化转型，毫无疑问，这是今年大多数商业领袖都能体会到的。在大多数情况下，这是新冠肺炎时代之前的数字趋势的加速，现在是商业成功和生存的关键。

商业银行、娱乐和零售等行业都经历了在线流量和移动应用使用的大幅增长。在追踪全球数据和趋势方面，这种巨大的行为转变造成了一个洞察力缺口。

## 洞察力差距是一个数据访问问题

在疫情之前收集的数据不再被视为对未来行为的良好预测。世界从根本上变了，我们需要适应。根据《哈佛商业评论》最近的一篇文章,组织需要认识到他们现有的预测模型、预测和仪表板可能都不可靠，甚至过时，他们的分析工具需要重新校准。我非常同意这一点。

推动过去洞察的数据现在并非完全无关紧要，但公司需要能够结合过去六个月收集的数据以及现在流入仓库或云数据湖的数据进行分析。这是 X Analytics 的承诺，它将帮助组织在今年及以后保持运转。它让商业智能(BI)分析师和数据科学家有机会从所有企业数据中提取价值，将新数据与核心基础数据相匹配，以了解行为如何发生了变化，哪些模式仍然存在，以及如何利用这些转变来推动新业务。

## 数据分析将定义企业如何适应

那些找到挖掘所有数据(尤其是最新数据)以获得洞察力的企业，更有可能适应甚至发展壮大。数据分析推动了全球各行业战略和财务决策的变化，实现了更高效的方式来管理使用趋势、客户需求、供应链库存等事项。

理想情况下，企业领导人将利用这些更新的数据，并制定短期和长期的战略和运营计划。您希望让您的数据分析师有机会分析您的所有数据，无论这些数据位于何处，是基础数据还是全新数据。更好的数据访问将带来更深刻的见解和更好的业务决策。许多组织将在 2021 年寻求解决的挑战是如何以安全、高效和经济的方式访问分布式数据。

## 当今企业如何利用 X 分析

组织已经从 X 分析方法中受益。这种变革性的数据访问和新的思维方式正在各行各业得到应用。例如，零售商更快地预测新客户行为，制造商提高生产和供应链决策的透明度，政府机构将地理空间数据和联系人追踪信息结合起来评估新冠肺炎暴露风险，网络安全公司正在识别不熟悉的黑客模式并训练新模型来检测它们。

所有这些例子都依赖于在多个地方与多个数据集进行交互的能力。通过融入 X 分析的理念，组织可以解决由来已久的数据孤岛问题。更好的是，它还是一个面向未来的解决方案。如果您将所有数据迁移到云中，这些后端存储系统将会改变，但您的数据消费层将保持不变。这意味着您的终端用户不会受到影响，您的 BI 和数据科学团队可以像往常一样继续工作。这是一个前所未有的时代，正如高德纳(Gartner)、麦肯锡(McKinsey)和其他许多人所指出的那样，要在不断变化的经济中繁荣发展，需要非凡的创造性努力。
