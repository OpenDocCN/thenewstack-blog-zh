# 监控提供商 Raygun 推出的全新 APM 平台

> 原文：<https://thenewstack.io/a-new-apm-platform-from-raygun-the-monitoring-provider/>

[ray gun 的新 APM 平台，监控提供商](https://thenewstack.simplecast.com/episodes/a-new-apm-platform-from-raygun-the-monitoring-provider)

John Daniel "JD" Trask 做软件已经 25 年了。一路走来，他学到了很多关于软件架构的知识，以及如果你真的想让它工作，什么是最重要的。Raygun 是他共同创立的公司，该公司因其错误监控功能而引起了软件开发社区的兴趣。它现在推出了 Raygun APM 作为应用性能监控市场的新成员，与 New Relic 等巨头竞争。

在本期 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客中，Trask 讲述了开发优秀软件的要素，并演示了该公司的应用性能管理(APM)服务，这是一个为软件开发人员设计的用于检测、诊断和解决错误的平台。根据一篇关于这项新服务的 [Raygun 帖子](https://raygun.com/blog/announcing-raygun-apm/)，它提供了一个集中的工具来检测、诊断和解决错误和性能问题，无论它们发生在哪里或原因是什么。

Raygun 在其服务中进行了性能改进，允许通过集成图形数据库、Elasticsearch 和 Postgres 来扩展要监控的数据库和收集的不同类型的数据。Raygun 提供了不同部分的视图，让用户在一个地方就能看到所有部分。Raygun 团队利用自己的架构构建了自己的堆栈，包括一个时间序列数据库和一个定制的存储环境来收集原始数据。
