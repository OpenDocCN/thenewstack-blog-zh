# 在大厨，乐队会一直演奏

> 原文：<https://thenewstack.io/chef-band-will-always-play/>

上周在得克萨斯州奥斯丁的 ChefConf 的主题演讲中，首席技术官 Adam Jacob 借用了传奇嘻哈乐队[The Roots](http://www.theroots.com/)——美国人知道这是 NBC 由吉米·法伦主演的[T4 今夜秀]的家庭乐队——来展示体验的承诺是如何保持不变的，但发展实际上可能会发生变化。

雅各布说，粉丝们对看根乐队表演有着不变的期待，但那些看过乐队几次的人很快就会意识到，他们从来没有看过同样的表演两次。所以乐队必须练习，他继续说，以确保他们的表演恰到好处，并且他们的行为超出了粉丝的期望。

是的，有一点是肯定的:“我们划定 API 边界的方式确实有问题，”CTO 告诉我们。“我们通过应用程序的生命周期来考虑它，现在，可能有八到十种不同的东西构成了应用程序的生命周期。我认为这是一个非常大的错误，当你在根的上下文中考虑它时，因为如果我换出，比方说，管道的构建部分有 8 或 10 或 12 或 20 步长，它会在管道的所有其余部分产生级联副作用。它们都是相互关联的。”

这是一个有趣的困境，不像用乔恩·巴蒂斯特取代 Questlove 那样——你可能会丢掉整个表演。(也许还会接哥伦比亚广播公司的官司。)Chef 首席执行官 Barry Crist 解释说，Chef 思考其自身架构演变的方式还有其他音乐相似之处，他为我们描述了他的公司在一个结果最重要的世界中的核心关注点。

现在在 Chef 听**，乐队将一直演奏**，最新版本的*新堆栈制造者*。
