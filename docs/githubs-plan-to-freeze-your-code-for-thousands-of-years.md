# GitHub 计划冻结你的代码几千年

> 原文：<https://thenewstack.io/githubs-plan-to-freeze-your-code-for-thousands-of-years/>

最近，我发现我写的一些计算机代码会比我活得久——几个世纪。它的副本被保存在北极圈一个寒冷的洞穴里。

这是 GitHub 的一个引人入胜的项目“北极代码库”计划的一部分，该计划将现代技术带入了一个令人惊讶的原始环境中，为 GitHub 服务器上目前托管的 1 亿个代码库中的一大部分提供了意想不到的荣誉，方法是将所有这些材料永久存档在挪威的一个异国群岛上，靠近世界上最北的城镇。

GitHub 负责特殊项目的副总裁托马斯·多姆克告诉 news.com.au， [GitHub 在档案方面有着独特的定位](https://www.news.com.au/technology/science/archaeology/microsoft-storing-worlds-open-source-github-code-in-norwegian-archive-vault-for-the-next-millennium/news-story/14eaaf98ac382ca491468e715327c1c3)，并且“有责任保护和保存全球数百万开发者的合作成果”在项目的[网页上，GitHub 打出了类似的高调，称开源软件是“现代文明隐藏的基石，是全人类的共同遗产”](https://archiveprogram.github.com/)

“我们将通过跨各种数据格式和位置持续存储多个副本来保护这些无价的知识，”他说。

根据最近彭博的一篇文章，在一次访问中，GitHub 的首席执行官 Nat Friedman 将这个废弃的煤矿描述为“比我想象的更加多矿，更加质朴，更加原始”。新闻服务继续指出，“对弗里德曼来说，这是一个自然的下一步。在他看来，开源软件是人类最伟大的成就之一，与文学和艺术杰作齐名。”

这并不是存储在这个偏远地方的唯一无价的知识。根据彭博的说法，矿井中的其他书架包括梵蒂冈档案、意大利电影、巴西土地登记记录，“以及某家汉堡连锁店的特殊酱料配方。”

## 拉取请求

但是这一巨大努力的基本原理是什么呢？该项目的页面列举了代码被“抛弃、遗忘或丢失”的威胁更糟糕的是，万一发生全球性灾难，该如何保存代码呢？

“工作的现代计算机存在着一系列可能的未来，但是它们的软件已经在很大程度上由于比特腐烂而丢失了。GitHub 档案计划将包括更长期的媒体，以解决随着时间的推移数据丢失的风险，”该网站指出。

当然，代码库服务也考虑了“未来”可能会如何使用我们的代码。项目网页上的[解释道:“世界本可以从中受益的失落技术有很长的历史，被遗弃的技术也有意想不到的新用途。”。“很容易想象，在未来，今天的软件会被视为古怪的、早已被遗忘的无关紧要的东西，直到出现对它的意外需求。”](https://archiveprogram.github.com/)

未来的历史学家可能会看到“我们这个开源无处不在、志愿者社区和摩尔定律的时代”的意义。

哪些代码块成功了？根据 GitHub 的说法:“该档案将包括 11 月 13 日至 2020 年 2 月 2 日在 GitHub Universe 上发布的任何提交的每个回购，至少有 1 颗星的每个回购和快照前一年(2019 年 2 月 2 日–2020 年 2 月 2 日)的任何提交，以及至少有 250 颗星的每个回购。另外，符合上述标准的任何存储库的 gh-pages。”

挪威数据存储公司 Piql 的定制胶片和归档技术将允许该项目存储 1000 多年的万亿字节数据，该公司夸口说，代码现在正走向“长期数据存储的黄金标准”

但除了提供“保险库存储服务”，Piql 还提供一种独特的数据数字化形式。Piql 将代码存储在数百卷由聚酯和卤化银制成的胶片上。彭博指出，它们涂有氧化铁粉末，“以增加对世界末日的抵抗力”。“它的每一个像缩微胶卷一样的框架能容纳超过 880 万个像素。Piql 解释说，它的方法包括将 1 和 0 转换成二维码。不需要电力或其他人工干预，因为北极的气候条件非常适合电影的长期存档，”Piql 网页解释道。

“通过使用一种独立的技术存储介质，未来的几代人将能够读回这些信息，”Piql 说。该项目还包括关于如何解包和读取代码的[指令](https://github.com/github/archive-program?utm_campaign=1579895123&utm_medium=social&utm_source=facebook,linkedin,twitter&utm_content=1579895123)。

彭博甚至提到有一个条约在战争时期保持斯瓦尔巴德群岛的中立。因为都是存储在离线胶片卷轴上，所以 GitHub 不用担心停电。它的远程位置增加了一层安全性。一个 [GitHub 视频](https://www.youtube.com/watch?v=fzI9FNjXQ0o&feature=youtu.be)指出斯瓦尔巴群岛是世界上最北部城镇的家园——也是数千只北极熊的家园。该视频的描述指出，虽然它被称为 GitHub“北极代码库”，但实际上它比北极圈更靠近北极的*。*

### **响应时间**

看人们对 GitHub 视频的反应很有趣。“未来将会被我的 JavaScript 计算器震惊，”一条评论开玩笑说。

其他人忍不住评论北极的位置。(“现在我的代码可能在运行前就冻结了……”)另一个唱反调的人甚至调侃道，“当你的代码糟糕到你需要把它埋在永久冻土下面的时候……”

GitHub 的常见问题称，该公司计划每五年对该程序(及其存储介质)进行重新评估——届时将决定是否拍摄另一张快照。

如果你想知道斯瓦尔巴特群岛的煤矿是什么样的，附近的煤矿会提供旅游服务。“斯瓦尔巴德群岛的大部分老的挪威和俄罗斯煤矿都已经关闭了，”彭博解释道，“所以当地人把他们的大片永久冻土重新命名为对科学家、末日准备者和科学家末日准备者的吸引力。”
