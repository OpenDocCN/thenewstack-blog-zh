# Docker 换了新 CEO 会有什么结果？

> 原文：<https://thenewstack.io/comes-docker-new-ceo/>

Docker 决定雇佣 Steve Singh 作为其新的首席执行官，这一决定暴露了在向基于容器的架构转变的过程中，可以说仍然是最具象征性和最重要的公司的未来问题。

Docker 在这一深刻转变中的作用一直是一个问题，特别是当竞争来自 Kubernetes 生态系统中的同行时，它处理 Docker Swarm 捆绑的方式以及它与整个开源生态系统合作的方式。但在过去的一年里，该公司将重点放在了为企业开发产品上，这为其发展方向提供了更多的明确性。它开源了 [containerd](https://thenewstack.io/dockers-quest-simplicity-evolution-containerd/) ，这是一项重要的技术贡献，有助于创建跨生态系统的连接，特别是与 Kubernetes 社区的连接。

由辛格担任首席执行官，这一信息是明确的。Docker 正在大举进军企业市场。在第 138 集的 [The New Stack Analysts](https://thenewstack.io/podcasts/analysts) 播客中，我们向 [RedMonk](http://redmonk.com) 的分析师 [Fintan Ryan](http://redmonk.com/team/fintan-ryan/) 询问了 Singh 的任命，这说明了 Docker 的重点，它如何影响他们的工程战略以及生态系统合作伙伴、云服务提供商和整个开发人员社区的更大背景。

[# 138:Docker 有了新 CEO 后会怎么样](https://thenewstack.simplecast.com/episodes/138-what-comes-of-docker-with-a-new-ceo)

总的来说，生态系统正在巩固，蛋糕很大，Docker 与甲骨文等公司合作的原因显而易见。这是 Docker 必须处理的二元性。一方面，开发者承诺反映了 Solomon Hyke 自身的根源。他的首要角色将使 Docker 成为一家以开发者为中心的公司。另一方面，辛格在那里带来销售。如何管理这种二元性将成为 Docker 在未来几个月和几年的最大挑战。
