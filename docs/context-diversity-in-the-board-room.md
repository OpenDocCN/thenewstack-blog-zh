# 背景:董事会会议室的多样性

> 原文：<https://thenewstack.io/context-diversity-in-the-board-room/>

欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)，在这个播客中，我们将讨论云计算领域的最新新闻和观点。本周，我们与[伊冯娜·瓦塞纳尔](https://puppet.com/people/yvonne-wassenaar/)交谈，她是木偶公司的首席执行官，关于高盛宣布不再让没有女性或少数民族董事会成员代表不足的公司上市。

[多样性](https://thenewstack.simplecast.com/episodes/diversity-in-the-board-room) 在板房里

上周晚些时候，高盛首席执行官大卫·所罗门宣布，投资公司[不会承销董事会成员不多元化的公司的 IPO](https://nypost.com/2020/01/23/goldman-sachs-will-no-longer-do-ipos-for-companies-with-all-male-boards/)，该政策将特别关注女性。这可能是我们行业的一个重要转折点，因为它可能会改变未来科技公司的运营和建议方式。就在这份声明发布之前，加州刚刚通过了一项法律，对公开全男性董事会的公司罚款 10 万美元。

瓦塞纳尔不仅是木偶委员会的成员，也是许多其他委员会的董事，包括 Forrester Research、Anaplan 和 Harvey Mudd College。我们问她新法将在公司和投资者的政策中扮演什么角色。我们还询问了在大公司中增加多样性的挑战。

然后，在这一集的后面，我们讨论了网站上的一些热门播客和新闻帖子，包括[与 Zeit 创始人 Guillermo Rauch 关于分布式系统的讨论](https://thenewstack.io/distributed-systems-and-the-butterfly-effect/)，一个名为 TriggerMesh 的[新的无服务器集成提供商](/triggermesh-anywhere-serverless-integration/)，SaltStack 帮助开发人员[最大限度地减少他们需要编写的 YAML](/saltstack-pop-code-more-and-forget-about-those-pesky-yaml-files/) 的计划，以及为什么 IBM 转向 [Humio 来扩大其 ELK 部署](/ibm-opts-for-humios-scalable-logging-to-boost-struggling-elk-cloud-deployments/)。

TNS 的编辑和营销总监 Libby Clark 与 TNS 出版商 Alex Williams 和 TNS 执行主编 Joab Jackson 一起主持了这一集。
