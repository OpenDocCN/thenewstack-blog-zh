新云铸造容器运行时:刚刚事实

Cloud Foundry容器运行时是Kubo的新名称，它是在BOSH上运行的Kubernetes。在今天这一集的《新栈制造者》中，TNS创始人Alex Williams采访了 Cloud Foundry Foundation 首席技术官 Chip Childers 以了解Cloud Foundry对这一新运行时的计划，Childers强调了BOSH如何满足当今开发人员的需求。

Childers继续指出，Cloud Foundry容器运行时和应用程序运行时将彼此相邻，允许Kubernetes和Cloud Foundry应用程序运行时之间共享身份。

“我认为现在最重要的是为开发者着想。企业中的开发人员需要什么？在我们看来，开发者体验是最重要的事情。Childers说，“其他一切都是从这一点出发的，”他补充说，“Cloud Foundry社区将其视为:开发人员团队、开发人员体验，然后向下。”
