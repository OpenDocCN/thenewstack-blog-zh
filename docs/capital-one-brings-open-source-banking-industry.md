# Capital One 如何为银行业带来开源

> 原文：<https://thenewstack.io/capital-one-brings-open-source-banking-industry/>

到目前为止，甚至大多数大型组织都已经意识到了使用开源软件所带来的好处——集体智慧以任何商业软件公司都无法比拟的速度修复错误和开发新功能。然而，在高度管制的行业中的公司，如银行业，在使用开源软件时面临相当大的监管障碍。在这个 [New Stack Makers](https://thenewstack.io/podcasts/makers) 的采访中，[Capital One 开源社区外联主管 Jared Smith](https://www.linkedin.com/in/jaredsmith/) 讲述了该公司如何克服这些挑战，享受开源的好处。
