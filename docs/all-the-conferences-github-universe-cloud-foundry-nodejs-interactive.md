# 所有会议:GitHub Universe、Cloud Foundry 峰会、Node+JS Interactive

> 原文：<https://thenewstack.io/all-the-conferences-github-universe-cloud-foundry-nodejs-interactive/>

[所有发布会:GitHub Universe、Cloud Foundry、Node+JS 互动](https://thenewstack.simplecast.com/episodes/all-the-conferences-github-universe-cloud-foundry-node-js-interactive)

本周在我们的每周新闻总结播客 [TNS Context](https://thenewstack.io/podcasts/context) 中，编辑团队介绍了我们最近参加的一些活动。TNS 通讯员 [TC Currie](https://thenewstack.io/author/tc/) 来说说本周举行的 [GitHub Universe](https://githubuniverse.com/) 。我们还有 TNS 创始人兼主编 [Alex Williams](https://twitter.com/alexwilliams/) 来谈谈[Cloud Foundry Summit Europe](https://www.cloudfoundry.org/event/eusummit2018/)和 TNS 执行主编 [Joab Jackson](http://www.joabj.com) 来聊聊温哥华的 [Node+JS Interactive](https://events.linuxfoundation.org/events/node-js-interactive-2018/) 。这两项活动都是上周举行的。

上周[在播客](https://thenewstack.io/the-new-stack-context-cloud-foundry-summit-europe-and-a-kafka-alternative/)中，我们与 IBM 的[杰森·麦基](https://twitter.com/jrmcgee)谈论了云铸造大会上的一些云新闻。但我们没听到太多关于其他事情的消息。威廉姆斯向我们展示了展会上发生的其他一些亮点，包括一些关于 [Cloud Foundry BOSH](https://bosh.io/docs/) 生命周期管理工具的精彩对话。Currie 讨论了自 6 月份微软宣布收购以来 GitHub 的首次大型用户会议，包括围绕该存储库的新工作流工具 [GitHub Actions](https://github.com/features/actions) 的所有讨论。接下来，杰克逊将向我们讲述他从 Node+JS Interactive 收集的一些 TNS 故事。

你可以在这里听更多集的新栈上下文[。此外，如果你喜欢你所听到的，请在 iTunes 上评价和评论我们，就像在](https://thenewstack.io/podcasts/context) [YouTube](https://www.youtube.com/channel/UCWea_KfCnTMD39uQuA5TKOQ) 上评价我们一样，并在 [Simplecast](https://thenewstack.simplecast.com/) 上关注我们。感谢收听，下次见。
