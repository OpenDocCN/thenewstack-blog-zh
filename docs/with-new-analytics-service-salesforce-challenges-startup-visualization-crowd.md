# 借助新的分析服务，Salesforce 挑战初创企业可视化群体

> 原文：<https://thenewstack.io/with-new-analytics-service-salesforce-challenges-startup-visualization-crowd/>

到目前为止，萌芽中的数据可视化市场一直是初创公司的领地。像 Chartio 和 ClearStory 这样的公司，以及 GoodData 和 Birst 这样的云商业智能提供商一直在推动这个市场。如今，随着 Salesforce 发布 Wave，这种情况发生了变化。

Wave 是一种新的数据分析云服务，Salesforce 表示它已经研究了大约两年。与其他公司一样，该公司的目标是从本质上实现大数据访问的民主化。“它是为所有人设计的，而不仅仅是数据分析师，”Salesforce 的产品营销总监安娜·罗森曼说。

凭借其强大的市场影响力，Salesforce 肯定会吸引用户使用这项服务。然而，Wave 似乎有一些缺点，这可能会给其竞争对手带来一些优势。

例如，Wave 没有提供用于集成常见企业数据源(如 SAP 和 Oracle 产品)的现成连接器。相反，Salesforce 引入了 Dell Boomi、Informatica、Jitterbit 和 MuleSoft 来为 Wave 用户处理结构化和非结构化数据的集成。许多竞争对手的数据可视化服务在其产品中处理数据接收和集成。

自然地，Wave 吸收现成的 Salesforce 数据。

也不清楚 Salesforce 有多认真地理解管理员为这种数据可视化服务的个人用户设置参数的愿望。虽然企业对 Wave 这样的服务感兴趣，因为它们旨在让企业中更多的人检查数据，以便做出更明智的决策，但他们也想努力确保[用户在使用这些工具时不会出错。如果没有太多处理数据的经验，用户很容易犯错误。许多新的数据可视化公司强调其产品中内置的控件，这些控件允许管理员设置护栏，以防止用户犯错。](http://www.itworld.com/big-data/369226/data-visualization-beneficial-perilous "ITworld")

罗森曼说，Wave 继承了管理员在 Salesforce.com 发布的设置，尽管这些设置不适用于人们可以在 Wave 中使用的许多功能。她还说，管理员也可以设置其他权限，包括限制某些用户只能查看某些数据段。然而，她强调，Salesforce 专注于让更多员工能够访问更多类型的数据。

从可用性的角度来看，Wave 似乎比市场上的其他产品更有竞争力。罗森曼说，该产品从一开始就被设计成可以在智能手机、平板电脑和全尺寸显示器上很好地工作。与传统 BI 工具提供的静态仪表板不同，Wave 的仪表板可以由用户轻松重新配置，如果他们对趋势有所了解并希望探索它，他们也可以轻松地深入了解。

在过去的几年里，数据可视化市场经历了巨大的变化。随着越来越多的企业收集更多的数据，他们希望能够快速方便地查询这些数据。然而，传统的 BI 产品需要大量的前期投资，而且使用起来非常困难，通常一个组织中只有一两个人接受过使用培训。

到目前为止，传统的 BI 平台供应商还没有真正对易于使用且通常基于云的可视化服务的新兴市场做出反应。有了 Wave，Salesforce 超越了那些传统供应商，但可能要赶上独立的可视化开发人员。

Wave 将于 10 月 20 日正式上市，不过一款预装了样本数据的 iPhone 和 iPad 应用预计将于今天在 App Store 上架，以便任何人都可以开始玩它。

Wave 将分为两层定价。大多数最终用户都将享受每月 125 美元的许可，这将允许他们查看、发现、个性化和共享可视化和仪表板。每月 250 美元的订阅，听起来是针对管理员和数据分析师的，允许用户创建、部署和管理数据集、数据连接和用户访问。
