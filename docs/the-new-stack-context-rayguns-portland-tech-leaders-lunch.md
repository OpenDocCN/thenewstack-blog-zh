# 新的堆栈环境:Raygun 波特兰技术领导午餐

> 原文：<https://thenewstack.io/the-new-stack-context-rayguns-portland-tech-leaders-lunch/>

CloudBees 赞助了这个播客。

[创建代码向客户反馈循环](https://thenewstack.simplecast.com/episodes/creating-a-code-to-customer-feedback-loop)

本周，[新的 Stack Context](https://thenewstack.io/podcasts/context) 播客从波特兰为您直播，在那里我们录制了来自 [Raygun Tech Leaders 午餐会](https://www.eventbrite.co.nz/e/portland-tech-leaders-lunch-tickets-80347683083)的要点，这是一场关于监控、指标以及 IT 系统如何帮助提高客户满意度的讨论。在午餐时间的小组讨论中，我们听到了 Chef、Nike、Microsoft 和 The Standard 的软件团队如何在他们的产品开发工作流程中通过关闭反馈回路和监控重要事项来优先考虑用户体验和性能。

[微软合作伙伴项目经理 Scott Hanselman](https://www.hanselman.com/) 担任主持人。小组成员是:

*   [Brian Brewer](https://www.brianbrewer.net/home) ，Nike 产品质量管理、测试工程和开发运营专家
*   Chef Software 的解决方案架构师经理 Niamh Cahill
*   [Jon Edwards](https://www.linkedin.com/in/edwardsjon1/) ，标准组织基础设施和运营部 IT 总监
*   微软高级应用开发经理戴夫·哈里森
*   [凯西·李](https://www.linkedin.com/in/kathy-lee-she-her-2235a41/)，微软的软件工程师。

在今天播客的第一部分，TNS 编辑和营销总监 [Libby Clark](https://twitter.com/LibbyMClark) 与 Harrison 和 [Raygun](https://raygun.com/about) 联合创始人兼首席执行官 [John-Daniel Trask](https://twitter.com/traskjd) 谈论开发者如何帮助客户并改善他们的软件开发生命周期反馈循环。

然后我们听到了节目中的一些亮点…

然后在节目的后期，TNS 发行人 [Alex Williams](https://twitter.com/alexwilliams) 和 TNS 执行主编 [Joab Jackson](http://joabj.com/) 讨论了本周的一些热门新闻和播客，包括 [Kubernetes 1.17](/kubernetes-1-17-brings-volume-snapshots-easier-plugin-management/) 、[CI/CD 的连续文档](/continuous-documentation-in-a-ci-cd-world/)、Packet 的 [Tinkerbell 裸机部署技术](https://thenewstack.io/why-open-source-matters-more-for-bare-metal/)以及[云原生操作中数据保护的重要性](https://thenewstack.io/why-cloud-native-data-protection-can-no-longer-be-ignored/)。

云计算原生计算基金会、Packet 和 Raygun 是新堆栈的赞助商。
