# 开源协作和创造力来到硬件

> 原文：<https://thenewstack.io/open-source-collaboration-and-creativity-comes-to-hardware/>

开放硬件，有时被称为开源硬件，实际上并不是全新的，尽管你认为它是也情有可原。开放计算项目(Open Compute Project)已经有 10 多年的历史了，这是一个全球性的工程师社区，旨在为硬件带来与开源软件相同的创造力和协作。但是对于那些沉浸在软件世界的人来说，很容易忽略开源原则改变硬件世界的方式。

## **什么是开放硬件？**

如果不是显而易见的话，从澄清什么是开放硬件开始是有意义的——它指的是开放硬件组件的设计，这样任何人都可以根据开放的设计制造硬件。就像开放源码软件一样，个人或公司可以对开放硬件设计进行修改，他们可以将它们合并到更大的项目中，并重新发布修改后的设计。

这确实以开源软件不受限制的方式限制了大众想象中的开放硬件——任何人，甚至是非技术人员，都可以成功下载和使用许多开源项目。然而，开放硬件需要有能力在设计的基础上创造出实物，然后才能为大众所用。

虽然开放计算项目不是昨天才开始的，但它只有 Linux 基金会的 1/3 历史。虽然开放硬件不太为人所知，尤其是在超大规模社区之外，但它最近一直在呈指数级增长。

## **为什么开放硬件很重要？**

对于开放硬件，“最大的优势是成本和灵活性，”阿里·芬恩(Ali Fenn)说，他是专门帮助公司重用硬件的公司 ITRenew 的总裁。"没有锁定，不依赖于特定于 OEM 的固件版本."

与此相关，Fenn 说，“一个巨大的(优势)是开放支持重新认证，支持 second lives，支持更丰富的生命周期管理方式，以及我们如何最大化它们的总体价值。这是专有技术永远做不到的。”重复使用硬件的能力不仅有助于可持续发展，还有助于降低成本——重复使用和重新调整硬件用途可显著降低总拥有成本。然而，使用专利设计是不可能的。

鉴于芯片短缺和供应链挑战，规避锁定的能力具有特别重要的意义。有了开放的硬件，公司可以很容易地调整设备设计，以纳入可用的组件，而不是几乎依赖于单一供应商。

## **公司正在留意**

RISC-V 是一种免费开源的计算机芯片标准指令集架构(is a)，其背后的基础 RISC-V International 自 2015 年以来一直存在。正如 RISC-V 的首席执行官卡莉斯塔·雷德蒙在 Linux 基金会成员峰会的一次演讲中所说，“这是关于创造商业成功，改变游戏规则，打破半导体现状。”

RISC-V 的采用和 RISC-V 国际基金会成员的增长最近都呈指数级增长。该基金会的成员在 2021 年的前六个月翻了一番，包括芯片制造商、软件和固件公司以及联网设备制造商。与此同时，近四分之一的硬件设计采用了 RISC-V open ISA，包括 40%的 CPU 设计。

对开放硬件需求的增长部分与硬件需求的增长有关。随着越来越多的联网设备进入市场，有更多的地方可以整合开放硬件。“让我们想想到 2030 年有 500 亿台联网设备，”雷德蒙说。"这个飞速增长的机会意味着我们正在把微处理器放在我们从未想象过的角落里."

## **创造力和协作**

开放硬件不仅仅提供硬商业价值——开放硬件运动的主要驱动力之一是将创造力和协作的冲刺带到硬件世界，这是开源软件开发的核心部分。芬恩说:“开源软件运动促成了一种心态和精神，这在很大程度上推动了开放硬件的机遇。”。
