# 现在是一个 Web3 的世界——宣传与 Web 2.0 相比如何

> 原文：<https://thenewstack.io/its-a-web3-world-now-how-the-hype-compares-to-web-2-0/>

不管你喜不喜欢，术语“Web3”已经迅速成为硅谷的最新流行语。今年，加密货币爱好者和投资者一直在大肆宣传加密货币，尤其是风险投资公司安德森·霍洛维茨(Andreessen Horowitz)。Web3 的主要代言人是 16z 的合伙人克里斯·迪克森。通过一篇[有影响力的文章](https://future.a16z.com/why-web3-matters/)和源源不断的 Twitter 帖子，迪克森已经把自己变成了新的[蒂姆·奥赖利](https://twitter.com/timoreilly)——这位技术出版商在 21 世纪初共同发明并推广了术语“Web 2.0”。

我发现网络 3 的宣传既令人厌恶又吸引人。排斥是因为围绕 Web3 的大部分讨论与技术架构几乎没有关系，几乎所有内容都与加密货币和 NFT(不可替代的代币)的投机有关。但 Web3 也很有吸引力，因为像脸书和谷歌这样的 Web 2.0 平台已经变得过于强大，甚至有潜在的危险(隐私问题、虚假信息等)。)Web3 号称是 Web 2.0 的解毒剂。

## 但是……那个技术架构

Richard 是 New Stack 的高级编辑，每周撰写一篇关于 web 和应用程序开发趋势的专栏文章。此前，他在 2003 年创立了读写网，并将其打造为全球最具影响力的科技新闻和分析网站之一。在 Twitter @ricmac 上关注他。](https://twitter.com/ricmac) 

当 Web3 的粉丝们谈论技术属性时，通常是在争论 Web3 是 Web 的一个去中心化版本，它将以某种方式取代前面提到的今天的技术巨头。“集中”这个词已经被 Web3 粉丝武器化了，就像“婴儿潮”这个词被千禧一代武器化了一样。但是 Web3 去中心化依赖于区块链的采用这一事实是一个严重的技术弱点，原因我在以前的专栏文章中概述过。此外，在后续的专栏中，我展示了今天 Web3 中的大部分基础设施与 Web 2.0 中的[完全相同——服务器归企业所有。](https://thenewstack.io/web3-isnt-fully-decentralized-a-look-at-alchemy-bitclout/)

除了 Web3 的技术缺陷，加密迷们对它的有效性也存在分歧。Web3 已经成为一个热门趋势，因为它被定位为一种开发 Web 应用程序的新方式，通过使用区块链——特别是[以太坊](https://ethereum.org/en/)区块链。然而，比特币区块链(最初的区块链)的支持者并不看好 Web3。事实上，比特币“最大化主义者”公开嘲笑它，因为他们认为以太坊是加密货币宝座的觊觎者。

如今，比特币通常被视为黄金的虚拟形式，而以太坊项目将自己标榜为一个全功能的计算平台。因此，很难说比特币区块链会出现被称为“Web3”的东西。也就是说，以太坊的竞争对手正在出现，例如[Solana](https://solana.com/)——它没有以太坊那么分散，但却是一个更快、更高效的技术平台。因此，争夺 Web3 关键平台的战斗还没有结束。

顺便说一下，A16z 已经在以太坊(通过公司建立在以太坊之上的[和索拉纳(通过](https://techcrunch.com/2021/06/24/andreessen-horowitz-triples-down-on-blockchain-startups-with-massive-2-2-billion-crypto-fund-iii/)[对索拉纳实验室](https://www.coindesk.com/business/2021/06/09/solana-labs-raises-314m-in-token-sale-led-by-a16z-polychain/)的大笔投资)上下了赌注。所以每个方向都有一个 bob。

## Web3 的文化影响

尽管我对 Web3 持技术上的怀疑态度，但从文化的角度来看，我觉得这场运动很有趣。Web3 有一种真正的社区感和兴奋感。它压倒一切的使命——去平台化中央平台——对年轻一代特别有吸引力，他们认为自己被“大技术”所控制，买不起房子，因为婴儿潮一代和 x 一代拥有所有的房子。Web3 既提供了一条获得房屋所有权的潜在金融途径(通过加密货币投机)*又提供了一条坚持到底的途径(大型科技)。*

此外，Web3 无疑很酷——卡通猿头像和社交媒体迷因提供了一种视觉美感，在很大程度上是有趣和娱乐性的。正如我在 Twitter 上开玩笑说的，我们在 Web 2.0 中所拥有的只是圆角！

Web3 中的一些新的文化规范有点令人畏缩——比如早上发“gm ”,晚上发“gn ”(早上好，晚安)。但也许这就是我愤世嫉俗的本性。我可以看到每天早上在你开始交易密码和寻找区块链局限性的集中解决办法之前问候你的信徒同胞的吸引力。

## Web3 会做 Web 2.0 承诺的事情吗？

你们中的一些人可能还记得“Web 2.0”何时成为一个无所不包的流行语。它开始于 2004 年左右，当时 O'Reilly Media 创造了这个术语，并在 Web 2.0 的旗帜下召开了一次会议，然后从 2005 年起获得了主要的关注。

我在 2003 年创建的博客 ReadWriteWeb 恰好赶上了 Web 2.0 的浪潮(还有其他一些，如 TechCrunch、GigaOm 和 Mashable)。所以我随波逐流，一方面作为技术博客记录 Web 2.0 的兴起，另一方面——我在这里坦率地说——相信 Web 2.0 真的会把我们带到一个开放的、符合 Web 标准的、基于 AJAX 的 Web 应用程序天堂。

当我们被推销的开放网络愿景没有成为现实时，我和其他人一样失望。还记得 Twitter 的 API 吗，一开始是开放的，后来被严格限制了？(具有讽刺意味的是，Twitter 刚刚[宣布](https://twitter.com/ashevat/status/1460321868985085961)再次“开放”其 API 是的，没错)。所以是的，Web 2.0 中犯了很多错误；或者从更仁慈的角度来看，有很多意想不到的后果。

所以我理解创建新版本网络的渴望。坦白地说，我自己在 2018 年期间写的一篇博客中试图对定义“Web 3.0”的提出主张，以探索当时新生的区块链场景。我刚刚重读了那篇文章，里面没有什么让我感到羞耻的，因为我真的为区块链的承诺感到兴奋。不幸的是，到 2018 年底[，我对区块链的所有技术问题和猖獗的市场投机感到失望](https://ricmac.org/2018/12/13/top-5-technology-trends-of-2018/)。所以我关闭了博客，最终回去写关于 web 开发的文章。

我是否希望在 2018 年继续运营我的区块链博客，从而也赶上 Web3 浪潮？不，因为我仍然看到区块链与我在 2018 年底看到的相同的基本技术问题-此外，市场投机变得更加糟糕。

## 未来

那么从这里去哪里呢？尽管我批评了 Web3 的技术架构，但我仍然认为这是一个值得关注的趋势。目前，对我们许多人来说，NFTs 可能是一个笑话，但我看到了 NFT 技术在虚拟世界([meta vs](https://thenewstack.io/metaverse-developers/))和未来其他可能的网络平台中的应用潜力。

我的最佳猜测是，在未来一两年内，加密货币将出现类似网络泡沫的崩溃，这将大幅重置预期。在即将到来的加密崩溃的废墟中，将出现新的网络技术——很可能它们将是分散的，并包括像令牌这样的东西。请记住，Web 2.0 是从互联网泡沫破裂的废墟中诞生的。

我并不自称是未来主义者，但我是网络历史的敏锐观察者。所以我还是等 Web3 崩溃后的版本再太较真吧。
