# Cloud Foundry Summit Pancake 播客:云原生时代的通信

> 原文：<https://thenewstack.io/cloud-foundry-summit-pancake-podcast-communications-cloud-native-era/>

Stackie 是新 Stack 的[旅行煎饼机器人](https://thenewstack.io/off-shelf-hacker-stackie-pancake-bot-tours-berlin/)，本周冒险前往加州圣克拉拉，为 2017 年硅谷云铸造峰会举办煎饼早餐和播客录制。当蒸汽朋克黑客[Torq](https://thenewstack.io/author/rob-reilly/)博士倾向于挑剔的 3-D 煎饼制造商时，TNS 创始人 Alex Williams 主持了一场关于 [Cloud Foundry](https://www.cloudfoundry.org/) 、开放服务代理 API[Kubo](https://thenewstack.io/living-multi-cloud-world/)的生动的小组讨论，Kubo 是 Pivotal 和 Google 正在开发的一个新的开源项目，它使用 [Google Bosh](https://thenewstack.io/bosh-hooks-cloud-foundry-google-cloud-platform-offers-large-scale-vm-lifecycle-management/) 来[打包和部署 Kubernetes](https://thenewstack.io/kubo-two-platforms-orchestrating-orchestrator/) 。小组还讨论了地理上分散的开发团队最喜欢的沟通形式，提到了 Twitter、Slack、telepathy 和 conferences。

本期[新堆栈分析播客](https://thenewstack.io/podcasts/analysts)的主讲人是:

*   [Dieu Cao](https://twitter.com/dieuthicao) :产品管理关键总监，
*   [霍尔格·穆勒](https://twitter.com/holgermu):星座研究副总裁兼首席分析师，
*   [艾比·卡恩斯](https://twitter.com/ab415) : Cloud Foundry 执行董事，
*   莎拉·诺沃特尼:谷歌 Kubernetes 社区的项目经理。

查看本播客视频的结尾，可以看到 Torq 博士解释硬件黑客方法并展示他的[蒸汽朋克眼球](https://thenewstack.io/off-shelf-hacker-behold-steampunk-eyeball-complete-part-5/)和他的[电子会议个性识别设备](https://thenewstack.io/hacking-hardware-the-never-ending-saga-of-steampunk-name-badge-development/)的特别片段
