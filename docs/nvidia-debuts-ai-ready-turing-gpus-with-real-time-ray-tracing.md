# 英伟达推出人工智能就绪图灵图形处理器，具有实时光线跟踪功能

> 原文：<https://thenewstack.io/nvidia-debuts-ai-ready-turing-gpus-with-real-time-ray-tracing/>

GPU 制造商[英伟达](https://www.nvidia.com)推出了专门为人工智能深度学习工作设计的新芯片架构。被称为[图灵](https://nvidianews.nvidia.com/news/nvidia-unveils-quadro-rtx-worlds-first-ray-tracing-gpu)的 GPU 也将能够进行实时光线追踪，这是价值 2500 亿美元的视觉效果行业长期以来一直追求的功能。

“这从根本上改变了计算机图形的制作方式，”Nvidia 首席执行官[黄仁勋](https://nvidianews.nvidia.com/bios/jensen-huang)在周一温哥华专业图形会议的主题演讲中说道。

黄称这种新的第八代 GPU 架构是自 2006 年推出的 CUDA 架构以来最重大的进步。该公司预览了一系列基于这种设计的新 GPU 卡，即 [Quadro RTX 5000、6000 和 8000](https://nvidianews.nvidia.com/news/nvidia-unveils-quadro-rtx-worlds-first-ray-tracing-gpu) ，将于今年年底上市。它还为视觉特效行业展示了一个参考架构，即 [Quadro RTX 服务器](https://www.nvidia.com/en-us/design-visualization/quadro-servers/#source=pr)，并发布了一个开源的[材质定义语言](https://developer.nvidia.com/mdl-sdk)软件开发包，用于将物理对象映射到渲染应用程序中。

据该公司称，图灵架构是 10，000 工程师年开发的结果。它具有一组“RT 内核”，专门设计用于加速[光线跟踪](https://www.tomshardware.com/news/ray-tracing-definition,37600.html)——或者模拟光波和声波相对于观众实际视角的轨迹的能力。

该公司估计，与上一代 GPU 相比，新架构最初将实时光线跟踪操作的速度提高了 25 倍，比 CPU 的速度提高了 30 倍。为了演示这种架构，黄展示了如何在单个图灵 GPU 上运行星球大战主题的反射光线跟踪演示，而早期的演示需要一个配备四个 Volta GPUs 的 70，000 美元的站。

黄吹嘘说，4 个 8 GPU 的服务器应该可以做 240 个双核服务器的渲染工作，消耗 1/11 的功率。这可以将制作一个动画镜头的时间从五六个小时减少到一个小时。

“这将彻底改变人们拍电影的方式，”他说。

图灵还包括一组“张量核”，用于人工智能推理工作。单个 GPU 每秒可以提供高达 500 万亿次张量运算。对深度学习工作很重要的是，张量是可以计算的相关数字的数据结构(很像矩阵和向量)。

在主题演讲中，黄解释了这种肌肉如何用于为视觉效果行业的深度学习操作提供动力:“在某些时候，你可以使用人工智能或一些启发式方法来找出哪些是缺失的点，以及我们应该如何填充它们，这使我们能够以比其他方式更快的速度完成帧。”

Nvidia 已经从戴尔 EMC、惠普公司、惠普企业、联想、富士通和超微等系统制造商那里找到了对图灵的支持。视觉特效行业的 24 家独立软件供应商(ISV)也承诺提供支持。
