Raygun 的技术领袖系列正在巡回演出，现在，由于新的堆栈，你可以现场观看整个活动。

由 [Scott Hanselman](https://www.linkedin.com/in/scotthanselman) 主持，并有一个全明星小组参加，波特兰技术领导者的活动将讨论为了交付世界一流的软件，监控什么是重要的。

为了保持领先地位，领先的软件团队必须让他们的最终用户跟上他们所做的一切的前沿速度，例如找到并使用最好的工具、流程和工作流，以前所未有的速度进行构建、学习和迭代。

在本次互动小组讨论中，您将了解 Nike、Oracle、Microsoft 和 Chef 的软件团队如何通过关闭反馈回路和监控重要事项，在其产品开发工作流程中优先考虑用户体验和性能。

他们将讨论:

*   如何以一种推动真正学习和行动的方式让客户更接近你的开发工作流程。
*   监控用户体验和性能的关键成功指标。
*   如何平衡交货的质量和速度，让您团队中的每个人都能更放心、更频繁地发货。
