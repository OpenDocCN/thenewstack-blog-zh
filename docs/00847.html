与Node.js基金会执行董事马克·辛克尔一起探索node . js

尽管 JavaScript 已经存在了20多年，但它正在成为开发企业应用程序的一等公民。这项技术背后有一个庞大的开发者社区。

更有趣的是，有了Node.js，JavaScript可以在服务器上运行，因此开发人员可以用JavaScript编写端到端运行的应用程序。Node.js非常适合服务应用，因为服务器应用越来越成为单一功能事件驱动的微服务。

这还不是全部。“Node.js是无服务器基础架构中的头号工作负载。这是第一个运行在Amazon Lambda上的解决方案，Google functions。这是在IBM Bluemix上运行的最大的工作负载，” Node.js Foundation 的执行董事 Mark Hinkle 在接受我们最新一集 The New Stack Makers 播客的采访时说，该播客是上个月在奥斯汀举行的KubeCon 2017上录制的。

当一项技术变得如此流行时，第一个受害者就是老技术的人才库。因此，认证开发人员需要保持更新。为了满足这种需求，Node.js基金会正在制定一个认证计划，以便企业可以放心地雇佣至少具备Node.js最低开发能力的开发人员。
