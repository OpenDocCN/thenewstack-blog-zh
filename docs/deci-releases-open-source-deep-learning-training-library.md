# Deci 发布开源深度学习训练库

> 原文：<https://thenewstack.io/deci-releases-open-source-deep-learning-training-library/>

寻求实现深度学习的开发人员通常会花费太多时间来整理各种开源存储库——试图集成工具、复制结果和提高准确性。作为回应，深度学习开发平台 [Deci](https://deci.ai/) 利用其在这些任务中的专业知识，发布了[超梯度](https://www.supergradients.com/)。这是一个用于计算机视觉模型的开源深度学习训练库，“使开发人员能够通过一个训练脚本来训练基于 PyTorch 的模型，以完成最常见的计算机视觉任务，包括对象检测，图像分类和语义分割，”根据一份新闻稿。

[Deci 的联合创始人兼首席执行官 Yonatan Geifman](https://il.linkedin.com/in/yonatan-geifman-8a449a3) 向新堆栈解释说，开源深度学习的状态“非常分散”，SuperGradients 正在试图解决这种分散。

“我们对超级梯度所做的是，我们收集了所有这些储存库，所有技术，并将它们集成到一个库中，可以将开源社区的所有价值集中到一个地方，”Geifman 说。“通过这种方式，您可以非常轻松地重现最先进的结果，并使它们适应您的用例、数据集等。”

自 2019 年开始以来，Deci 一直致力于简化深度学习的工作-无论是解决硬件或数据集的限制，还是建立和训练模型本身。Geifman 表示，SuperGradients 是该公司“训练模型的基础组件”的开源产品，旨在分享“我们收集的关于如何有效建立模型和训练模型的所有知识，我们在内部工具中集成的所有新架构，以及为了获得更好的准确性、更快的训练和跨多个 GPU 扩展训练的技巧。”

更具体地说，SuperGradients 专注于深度学习的三个方面——图像分类、对象检测和语义分割——为每个方面提供了库，如 YOLOv5、DDRNet、EfficientNet、RegNet、ResNet、MobileNet。该公司表示，与现有的训练库相比，它经常优化这些库，以提供更高的准确性。在发布时，SuperGradients 包括 20 多个计算机视觉模型(更多模型正在开发中)，为开发人员提供了一个清晰的代码结构，他们可以使用它将这些模型集成到他们的代码库中。Geifman 声称，超梯度不仅可以帮助开发人员处理样板代码，还可以实现最先进的结果。

“我们正在努力将训练深度学习的所有技巧和最佳实践纳入知识库，以提高结果，”Geifman 说。“例如，我们有一系列模型，与宣布它们的学术论文相比，我们看到精度提高了 1.5%。如果我们不相信我们给出的精确度，或者你可以从超梯度上的模型中获得的精确度，比不上你在另一个地方获得的精确度，我们就不会把这个模型放进去。我们希望超梯度成为获得最先进结果的一站式商店。”

使用超梯度，开发人员可以使用预先训练的模型，为他们的特定用例微调模型，或者自己构建模型。Geifman 说，这些任务中的每一项，他们可能会单独执行，但有了超梯度后变得更加容易和有效。他说，添加一个预先训练好的模型只需要几行代码，微调一个模型可能需要 10 到 20 行，从头开始构建可以将所需的代码减少一半。

展望未来，Geifman 表示，该公司不仅计划添加更多的模型，还计划“通过越来越多的技巧和最佳实践，将精确度水平推得更高。”之后，更好的集成和与云提供商的密切合作就在路线图上了。

“我们希望成为生态系统的一部分，与人们今天使用的其他工具相结合，”Geifman 说。“超梯度可能是公司今天正在建立的更广泛的人工智能平台的一个组成部分。”
