# 积云和网络规模网络的挑战

> 原文：<https://thenewstack.io/building-efficient-affordable-data-center-network-using-web-scale-principles/>

在《新堆栈制造商》这一集中，我们采访了首席执行官[乔希·莱斯利](https://www.linkedin.com/in/joleslie)和首席技术官[JR·里弗斯](https://www.linkedin.com/in/jr-rivers-3723b73/)以及网络规模网络软件提供商[积云网络](https://cumulusnetworks.com/)的联合创始人。我们讨论了如何最好地定义 web 级网络，以及 Cumulus Linux 与传统网络解决方案有何不同。他们谈到了新工作负载对系统间通信的需求，以及这如何影响软件定义的数据中心的构建。他们告诉我们他们公司最近发布的产品 [Cumulus Express](https://cumulusnetworks.com/products/cumulus-express/) ，以及围绕脸书[背包](https://code.facebook.com/posts/864213503715814/introducing-backpack-our-second-generation-modular-open-switch/)发布的公告，这是一种开放式网络机箱。我们的谈话包括关于超融合基础设施、客户障碍和积云网络路线图的要点。
