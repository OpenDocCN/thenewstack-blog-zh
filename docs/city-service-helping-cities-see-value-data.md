# 城市即服务:帮助城市看到数据的价值

> 原文：<https://thenewstack.io/city-service-helping-cities-see-value-data/>

在软件开发行业，收集、处理、存储大型数据集并使其具有可操作性是一种成熟的商业模式，但如果您将同样的技术应用于城市服务和基础设施，会怎么样呢？在这一集里，我们采访了 Stae.co[的联合创始人兼首席执行官 John Edgar](http://stae.co/) ，这是一个土木工程数据平台，为城市和地方政府带来了宝贵的数据能力。

[城市即服务:帮助城市看到数据的价值](https://thenewstack.simplecast.com/episodes/city-as-a-service-helping-cities-see-the-value-of-data)

Edgar 告诉我们，他是如何想到将数据平台的元素应用于城市和地方政府模型的。我们讨论了将相对高科技的计划引入那些通常陈旧且行动缓慢的机构的一些困难。正如 Edgar 所指出的，利用数据平台和传感器网络的城市的回报是巨大的。让数据对科技公司有价值的理念同样适用于市政当局，但这需要一些所有权和改变。Edgar 还向我们讲述了开源城市背后的理念，引领这一变革模式的城市，以及 Stae 的路线图。
