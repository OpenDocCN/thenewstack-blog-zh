有Node.js技能？很快，将会有一个证书

原文：https://thenewstack.io/got-node-js-theres-cert/#0001-01-01

我们都有过使用Node.js让我们觉得有点，嗯，可证明的时刻。来自 Node.js Foundation 的一个新的专业认证赛道将使其正式化。

每个月都有超过450万人使用Node.js平台来构建、调试、测试和维护 JavaScript 应用。各种规模和服务的组织(如NASA、优步和Capital One)都依赖Node.js进行全栈开发。

然而，在这一群体中有无数的熟练程度，并且直到现在，工程师没有简单的方法来表明他们在Node.js中的流利程度，或者组织验证能力。

一家公司如何知道一个潜在的雇员符合标准？自由开发者如何有效地宣传他们的mad Node skillz？显然，招聘过程的双方都可以从Node.js实力的通用标准中受益。

令人高兴的是，这一标准将很快建立:本周，Node.js基金会宣布了其全新的 Node.js认证开发者计划。预计将于2017年第二季度推出，自定进度的在线考试将验证(甚至更具体地说，认证)对平台的基本能力。

“无论是希望重新培训Java开发人员的大型企业，还是希望在市场中脱颖而出的承包商和顾问，该认证计划都证明了开发人员在获得Node.js项目后可以立即投入工作，”基金会教育社区经理Tracy Hinds 说。

她继续说，一旦它开始运行，这个认证“将是最大的供应商中立的编程语言认证项目之一，”Hinds说。受益者包括希望通过经验证的Node.js在市场中脱颖而出的开发人员，希望在内部人才库或求职者中提升或验证能力的公司，甚至是希望获得这些非常受欢迎的技能的技术职业生涯刚开始的人。

根据基金会的说法，cert的主要候选人将精通“Node.js平台的JavaScript，以构建、调试、测试和维护安全的独立于框架的应用程序和CLI工具”。根据该基金会的说法，他们应该了解如何有效地处理异步I/O，以及“使用HTTP、文件、流和多个进程有效地操纵、转换和持久存储各种数据”。他们还应该能够有效地使用第三方模块。

这种自定进度的在线考试预计需要大约3.5个小时，价格仍有待确定。考试涉及的主题将会公开，基金会将提供资源帮助考生学习和准备认证。

该计划的根源可以追溯到2016年7月，当时基金会组建了一个工作组，为Node.js认证所包含的内容勾勒出一个框架。这个核心小组通过虚拟会议建立核心技能、知识和能力，任何通过认证考试的人都应该展示这些技能、知识和能力。在Node Interactive 2016期间举办了现场工作任务分析(JTA)研讨会，以制定考试“蓝图”

考试的具体题目已经确定，基金会现在正在寻找志愿者来帮助编写实际的问题。但是每个志愿者将只写一个问题:为了防止任何人在认证考试中获得内幕消息，志愿者被限制写一个问题，并且不能访问大多数其他测试问题。通过这种方式，你可以为考试做贡献，并仍然有资格参加考试，并获得认证。

鼓励任何Node.js的中级到高级人员参与进来。根据基金会的说法，时间承诺包括“一个1.5小时的介绍性网络研讨会，该研讨会将审查格式和如何编写问题，以及2-5小时的核心项目审查团队编写和审查问题。”

有兴趣帮忙吗？申请 Node.js基础认证开发项目写作工作坊环节。
