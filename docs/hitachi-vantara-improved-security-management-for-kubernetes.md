# Hitachi Vantara:改善 Kubernetes 的安全性和管理

> 原文：<https://thenewstack.io/hitachi-vantara-improved-security-management-for-kubernetes/>

随着科技公司继续向 Kubernetes 靠拢，日立子公司 [Hitachi Vantara](http://www.hitachivantara.com/) 宣布增强其托管云服务，以提高其 K8s 功能。

本周在圣地亚哥举行的 Hitachi [NEXT 2018](http://www.hitachinext.com/en-us/home.html) 大会上，该公司推出了新的自动化安全功能、简化的多云管理以及更快的 Kubernetes 容器环境部署和管理。

日立在 2017 年剥离了 Vantara，包括收购商业智能工具 Pentaho、日立数据系统公司和日立洞察集团。除了托管云服务，Vantara 还提供物联网、大数据集成和分析、存储、数据保护和数据中心管理产品。

它与 [Aqua Security](https://www.aquasec.com/) 合作，为日立企业云(HEC)添加全面、受管和预先设计的安全功能，以提供对集装箱活动的全面可见性和控制。它们包括运行时保护、加密、漏洞扫描和微分段，以帮助客户实施安全策略并确保法规遵从性。

Aqua 专注于容器和云原生安全，最近发布了一个开源工具，名为 [Kube-hunter](https://thenewstack.io/kube-hunter-aquas-new-open-source-tool-for-hunting-kubernetes-security-issues/) ，帮助搜索 Kubernetes 集群中的安全问题。它还与 AWS 合作开发了新的 Fargate 服务，该服务允许你运行容器而无需管理服务器或集群。

第二个新特性是 HEC Container 平台的单一云管理界面，用于同时运行多个云实施。它提供了对私有云、混合云和公共云部署的可见性，在这些部署中，工作负载可以运行，而无需使应用适应专有的公共云 API 和方法。

它还增加了 Kubernetes 集群的一键式部署，增加了其自助式 DevOps 和数据服务产品的目录，如 Spark、Cassandra 和 Kafka。

7 月，Hitachi Vantara 宣布有意收购云系统集成商 REAN 云，Gartner 称其为亚马逊网络服务、微软 Azure 和谷歌云平台的[顶级托管服务提供商](https://www.gartner.com/doc/3870075/magic-quadrant-public-cloud-infrastructure)中的利基参与者。

REAN 云的产品包括物联网分析、预测分析和机器学习的定制解决方案。该公司表示，此次收购将增强 Hitachi Vantara 在多云环境中加快企业云采用以及管理和运营传统应用程序的能力。

根据 Grand View Research 的数据，到 2025 年，全球云管理服务市场预计将达到 825.1 亿美元，年复合增长率为 10%。

在另一个有趣的项目中，本月早些时候，该公司宣布了面向执法部门的日立数字证据管理(HDEM)，这是一个用于[从各种来源收集证据](https://globenewswire.com/news-release/2018/09/20/1573788/0/en/Hitachi-Vantara-Delivers-Digital-Evidence-Management-To-Enable-More-Effective-Policing.html)的集成管理解决方案，包括人体摄像头、车载摄像机、手机、文件和闭路电视(CCTV)。

Aqua 是新堆栈的赞助商。
