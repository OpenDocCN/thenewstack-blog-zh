# Automatic 面向联网汽车的技术堆栈

> 原文：<https://thenewstack.io/automatics-technology-stack-for-the-connected-car/>

自动实验室有一个连接到汽车的硬件设备。它的价格为 99.95 美元，本质上是作为一个汽车监控器，例如，帮助节省汽油价格。但它还有更多功能，包括定位功能，分析性能，发送信息和连接其他汽车的能力。

今天在旧金山的 API Days 上，我采访了 Automatic Labs 的创始工程师 Ram Jayaraman，了解其产品以及正在为联网汽车开发的新兴技术堆栈

[联网汽车的自动技术堆栈](https://thenewstack.simplecast.com/episodes/automatics-technology-stack-for-the-connected-car)

自动链接，其硬件技术，连接到汽车的数据端口。只要司机在路上，汽车和智能手机就会以无线方式自动连接。它有 iOS 应用和 Android 应用两种版本。

Automatic 是作为汽车助手设计的。它了解司机的风格，当司机浪费汽油时，它会给出微妙的音频提示。它通知司机突然刹车，超速或快速加速。该服务还为司机提供每周记分卡。分数越高，节省的汽油就越多。它还提供了一个时间表，记录了司机行驶了多少路程，去了哪里。

它还能检测加油情况，跟踪当地的油价，以显示司机的花费。

这项服务正在以几种方式扩展。Automatic 开发了一个与 ifttt 集成的 API，if TTT 是连接不同应用程序的服务。通过 ifttt，客户可以连接发送文本消息的通知等服务。例如，当一个人下班时，通过 ifttt 可以自动发送一条消息通知接收者他们正在回家的路上。

技术堆栈有很多未知，用例仍然没有定义。但是，优化一个人的驾驶带来的简单成本节约，体现了正在出现的使联网汽车成为现实的微服务星座。
