# 有 Node.js 技能？很快，将会有一个证书

> 原文：<https://thenewstack.io/got-node-js-theres-cert/>

我们都有过使用 Node.js 让我们觉得有点，嗯，可证明的时刻。来自 [Node.js Foundation](https://nodejs.org/en/foundation/) 的一个新的专业认证赛道将使其正式化。

每个月都有超过 450 万人使用 Node.js 平台来构建、调试、测试和维护 [JavaScript](/tag/javascript/) 应用。各种规模和服务的组织(如 NASA、优步和 Capital One)都依赖 Node.js 进行全栈开发。

然而，在这一群体中有无数的熟练程度，并且直到现在，工程师没有简单的方法来表明他们在 Node.js 中的流利程度，或者组织验证能力。

一家公司如何知道一个潜在的雇员符合标准？自由开发者如何有效地宣传他们的 mad Node skillz？显然，招聘过程的双方都可以从 Node.js 实力的通用标准中受益。

令人高兴的是，这一标准将很快建立:本周，Node.js 基金会宣布了其全新的 [Node.js 认证开发者计划](https://medium.com/@nodejs/volunteers-needed-for-the-next-phase-of-the-node-js-certified-developer-program-eeb631effc49#.8d5shyxws)。预计将于 2017 年第二季度推出，自定进度的在线考试将验证(甚至更具体地说，认证)对平台的基本能力。

“无论是希望重新培训 Java 开发人员的大型企业，还是希望在市场中脱颖而出的承包商和顾问，该认证计划都证明了开发人员在获得 Node.js 项目后可以立即投入工作，”基金会教育社区经理 Tracy Hinds 说。

她继续说，一旦它开始运行，这个认证“将是最大的供应商中立的编程语言认证项目之一，”Hinds 说。受益者包括希望通过经验证的 Node.js 在市场中脱颖而出的开发人员，希望在内部人才库或求职者中提升或验证能力的公司，甚至是希望获得这些非常受欢迎的技能的技术职业生涯刚开始的人。

根据基金会的说法，cert 的主要候选人将精通“Node.js 平台的 JavaScript，以构建、调试、测试和维护安全的独立于框架的应用程序和 CLI 工具”。根据该基金会的说法，他们应该了解如何有效地处理异步 I/O，以及“使用 HTTP、文件、流和多个进程有效地操纵、转换和持久存储各种数据”。他们还应该能够有效地使用第三方模块。

这种自定进度的在线考试预计需要大约 3.5 个小时，价格仍有待确定。考试涉及的主题将会公开，基金会将提供资源帮助考生学习和准备认证。

该计划的根源可以追溯到 2016 年 7 月，当时基金会组建了一个工作组，为 Node.js 认证所包含的内容勾勒出一个框架。这个核心小组通过虚拟会议建立核心技能、知识和能力，任何通过认证考试的人都应该展示这些技能、知识和能力。在 Node Interactive 2016 期间举办了现场工作任务分析(JTA)研讨会，以制定考试“蓝图”

考试的具体题目已经确定，基金会现在正在寻找志愿者来帮助编写实际的问题。但是每个志愿者将只写一个问题:为了防止任何人在认证考试中获得内幕消息，志愿者被限制写一个问题，并且不能访问大多数其他测试问题。通过这种方式，你可以为考试做贡献，并仍然有资格参加考试，并获得认证。

鼓励任何 Node.js 的中级到高级人员参与进来。根据基金会的说法，时间承诺包括“一个 1.5 小时的介绍性网络研讨会，该研讨会将审查格式和如何编写问题，以及 2-5 小时的核心项目审查团队编写和审查问题。”
