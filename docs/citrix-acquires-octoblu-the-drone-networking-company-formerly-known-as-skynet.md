# 思杰收购了无人机网络公司 Octoblu，其前身是天网

> 原文：<https://thenewstack.io/citrix-acquires-octoblu-the-drone-networking-company-formerly-known-as-skynet/>

Octoblu 是一个机器对机器的网络平台，已经被 Citrx 收购。官方消息尚未发布，但 Octoblu 网站称该公司“于 2014 年 12 月被 Citrix 收购。”

目前还不清楚交易的条款，也不清楚这是否是一笔收购租金。无论如何，该公司的联合创始人之一克里斯·马蒂厄(Chris Matthieu)已经将他的 LinkedIn 个人资料改成了头衔:思杰(Citrix)物联网工程总监。

Octoblu，原名天网，是无人机相互通话的系统。正如我们之前所写的，它被命名为“终结者电影系列中具有自我意识的人工智能系统。”

在范围上，Octoblu 旨在运行于单个网络或物联网网络网格上，这些网络共享一个公共 API 或通信协议。在这种情况下，设备可以发现、查询并向网络上的其他设备发送消息。

以下是 Matthieu 在上个月的 Defrag 会议上的一次讲话中概述的关于该公司的更多信息，我们记录了这些信息:

*   该 API 支持以下物联网协议:HTTP、REST、WebSockets、MQTT(消息队列遥测传输)和 CoAP(受限应用协议)。
*   天网是开源的，是一个多协议和跨协议的平台，任何说这种语言的设备都可以相互通信。
*   网状网络上的每个资源都有一个 UUID，这是一个 36 个字符的设备凭证，与网状网络相关联，网状网络有自己的 UUID，最终使其成为一个 uuid 网络。
*   该平台还需要能够支持 uuid 的发现和向其他 uuid 发送消息的能力，它应该能够根据给定的搜索标准搜索一组 uuid。

Gartner Research 表示，到 2020 年，物联网装机量将增长至 260 亿台。这意味着将会有大量的无人机出现在天空。但是基础设施几乎不存在。

对无人机公司的投资开始增加。3D Robotics 由前《连线》编辑克里斯·安德森(Chris Anderson)领导，在两轮融资中从风险投资家那里筹集了 3600 万美元。
