# 云集成商 Boomi 负责数据管理

> 原文：<https://thenewstack.io/boomi-cloud-integrator-takes-data-management/>

在本期 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客中，我们将了解云集成领域正在发生的事情。我们采访了[克里斯·麦克纳布](https://www.linkedin.com/in/chris-mcnabb-a555325/)，这一领域的先驱公司之一戴尔[博米](https://boomi.com/)的首席执行官。早在 2000 年，Rick Nucci 创立了这家公司，他预见到云集成将需要一种不同于当时中间件软件所提供的方法。十年后，戴尔收购了 Boomi。现在，该公司拥有超过 6，000 家企业客户，他们利用该服务将其系统与任何可用的公共云连接起来。

“个人可以非常迅速地融入亚马逊。但在规模上，它开始呈现非常不同的问题，”麦克纳布说。

该平台已经扩展到其他用途，如主数据管理、API 管理和低代码应用程序开发。销售助理可以更改云 CRN 系统中的账单地址，但是金融公司会希望这样吗？“这不是集成问题，而是数据管理问题。如果你批准了这些改变，你就必须围绕这些改变执行所有的治理和政策，我们也有这些能力，”麦克纳布说。
