# 牧场主将如何帮助 SUSE 的“优势”

> 原文：<https://thenewstack.io/how-rancher-will-help-give-suse-the-edge/>

SUSE 对 Kubernetes 分销商 Rancher 将如何支持其平台和服务一直守口如瓶，理由是在完成对 T2 的收购之前有一段平静期。然而，Linux 驱动的系统提供商已经给出了一些迹象，Rancher 将在 SUSE 将提供的 Kubernetes edge 部署中发挥关键作用。

在由 Cloud Foundry 主办的[Cloud Foundry Summit Europe 2020](https://www.cloudfoundry.org/events/summit/europe-2020/)、[SUSE](https://ch.linkedin.com/in/thomasdigiacomo)工程和创新总裁托马斯·迪·吉亚科莫召开的新闻发布会和分析师会议上，Rancher 证实将帮助 SUSE 扩展其边缘应用产品。

迪·吉亚科莫在回答新堆栈提出的一个问题时说，“从商业公司的角度来看，我真的不能分享，因为交易尚未完成，但我可以分享”关于开源相关技术主题的信息，这是“开源之谜的美丽之处”。

“Rancher 有几个关键的解决方案，”如用于轻量级边缘类型 Kubernetes 开发的 [K3S](https://k3s.io/) 和用于典型“典型私有数据中心内部部署”的[Rancher Kubernetes Engine(RKE)](https://rancher.com/docs/rke/latest/en/)，迪·吉亚科莫说。他还提到了 Rancher 如何提供其 Rancher Kubernetes 管理层，当然，该管理层与任何 Kubernetes 集群的云铸造平台相集成。

![](img/c71862bfa2d643c91804c1126e0b2d2b.png)

SUSE 工程与创新总裁托马斯·迪·吉亚科莫。

根据去年的一项调查，1197 名被调查的 Kubernetes 用户中有 16%说他们的组织使用 Rancher。该公司还承认软件巨头 Red Hat 和 VMware 是其主要竞争对手。SUSE 也被视为牧场主扩大客户规模的一种方式，从而更有效地与规模更大的竞争对手竞争。

SUSE 表示，第三季度 Rancher 还将帮助其扩大 Kubernetes 的整体覆盖范围。然而，Rancher 首席执行官兼联合创始人[盛亮](https://www.linkedin.com/in/shengliang/)在本月早些时候的一次采访中告诉 New Stack，Rancher 的 Kubernetes distro 代表了“我们业务的一小部分”,而此前其销售额超过了 90%。“仍然需要我们做 Kubernetes 发行版…让人们在安全、相当专业的环境中使用 Kubernetes…而 edge 代表着巨大的机会。”

作为 edge Kubernetes 部署以及云原生环境平台的早期提供商之一，[福乐鸡快餐店](https://www.chick-fil-a.com/)连锁餐厅在 2018 年采用了 Rancher 的 RKE 来管理安装在美国数千个单独位置的设备上的集群，以支持其基础设施。从一开始，餐厅经理就能够在不了解 Kubernetes 的情况下，通过简单地将现场节点连接到电源插座和不同的端口，在设备上安装和运行 Kubernetes 集群。

对于[边缘部署](https://edgeconference.io/)，Rancher 的以 GitOps 为中心的推送也涉及到自动更新边缘设备中的设备上的集群。“对于部署来说，你必须把更新推送到有时成千上万的设备上，”梁说星团遍布世界各地。这需要高度的控制和自动化。"
