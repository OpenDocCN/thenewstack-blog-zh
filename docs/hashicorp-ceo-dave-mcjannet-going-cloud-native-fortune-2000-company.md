# HashiCorp 首席执行官 Dave McJannet 谈企业基础设施的地形改造

> 原文：<https://thenewstack.io/hashicorp-ceo-dave-mcjannet-going-cloud-native-fortune-2000-company/>

HashiCorp 的新任首席执行官戴夫·麦克詹尼特(Dave McJannet)有着悠久的技术领导历史:他来自 GitHub，在此之前，他曾在 Hortonworks 和 VMware 的 SpringSource 担任高管职务，建立了后来成为 Pivotal 核心组成部分的业务。自从我们上次与 HashiCorp 接触以来，该公司一直忙于增强其全球 2000 强公司的投资组合，包括家得宝、塔吉特和威瑞森。

在今天这一集的 [The New Stack Makers](https://thenewstack.io/podcasts/) 中，TNS 编辑经理 [Joab Jackson](http://www.joabj.com) 与 [McJannet](https://twitter.com/davidmcj) 交谈，了解 [HashiCorp](https://www.hashicorp.com/) 如何着眼于统一其产品，以更好地帮助当今的大型组织向基于云的架构转变。

[Terraform](https://www.hashicorp.com/terraform.html) 是 HashiCorp 的旗舰产品:它用于跨公共云、私有云以及外部服务提供基础设施和应用资源。“财富 5000 强面临的挑战是异质性。这不是一个“我要去云吗？”的问题是的，它们都将转向云计算。Terraform 将是一个工具的例子，人们将使用它来提供基础设施，而不管环境如何。这是 Terraform 中非常常见的使用模式，”McJannet 说。

“在企业基础设施领域，我们已经在这个领域工作了很长时间。这对人们的生意至关重要。这不是一个应用程序，事实上这是他们基础设施的一部分，”McJannet 谈到 Terraform 时说。
