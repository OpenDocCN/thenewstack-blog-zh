# 人工智能控制的机器人需要更聪明

> 原文：<https://thenewstack.io/ai-controlled-robots-need-to-be-smarter/>

人工智能(AI)和[机器学习](https://thenewstack.io/category/machine-learning/) (ML)已经见证了 IT 应用程序的采用和进步，特别是数据库管理、CI/CD 支持和其他功能。与此同时，机器人技术在很大程度上被归入工厂车间自动化。在最新的 New Stack Makers 播客中，机器人“通用人工智能”供应商 [Covariant.ai](https://covariant.ai/) 的联合创始人、总裁兼首席科学家[彼得·阿比尔](https://www.linkedin.com/in/pieterabbeel/)讨论了由于信息技术的进步，机器人技术的潜力为何以及如何能够超越仅作为预编程设备。作为加州大学伯克利分校的教授和机器人大脑播客的主持人，Abbeel 还利用他的背景来提供他的观点。

《新书库》的创始人兼发行人亚历克斯·威廉姆斯主持了本期播客。

[为什么人工智能控制的机器人需要更智能](https://thenewstack.simplecast.com/episodes/why-ai-controlled-robots-need-to-be-smarter-for-it)

Abbeel 说，协变解决的挑战是如何将人工智能的现代进步应用于机器人，特别是“操纵机器人”。“机器人手不像今天的人手——它们是夹子，是吸盘。但操纵机器人赋予它们智能，这正是我们正在做的，”阿贝耳说。“你可能会想，这有什么大不了的？每部电影里的每个机器人都很聪明。但实际上，如果你看看现实世界中的机器人，它们在哪里？他们在汽车工厂和电子工厂，在那里你会发现他们中的大多数。”

阿比尔说，今天的机器人主要涉及预编程的动作。“这些机器人没有真正的智能。他们很有效率，也很有帮助。这没什么不对，”阿贝耳说。“但是，如果作为一个机器人，你所能做的只是预编程的动作，那就非常有限了。因此，我们正在开发人工智能，这将为机器人提供更多帮助的机会。”

人工智能的应用，特别是人工智能的应用，在十年前得到了巨大的推动，当时深度神经网络——以与人脑功能相似的方式处理信息的网络——开始比“任何以前手工设计的解决方案”更好地识别和处理图像，Abbeel 说。

“你拥有的数据越多，你根据这些数据训练的神经网络就越大——当然，随之而来的许多软件会将你的计算分布在许多机器上，等等，并管理你的所有实验，”Abbeel 说。

一个关键的挑战是如何确保为利用高级人工智能应用的机器人收集正确的数据。该系统“可以直接了解真实世界，并获得所需的可靠性，例如，确保仓库中的拾取和放置操作可以可靠地完成，”Abbeel 说。“你可以得到你订购的东西，而不是得到错误的东西或根本没有得到任何东西，因为没有东西运送给你。”
