# 经济不确定性和开源生态系统

> 原文：<https://thenewstack.io/economic-uncertainty-and-the-open-source-ecosystem/>

在这一集的开源峰会上, [Percona](https://www.percona.com) 的开源负责人 Matt Yonkovit 分享了他对经济不确定性如何影响开源生态系统的看法。

开源当然是免费的。那么经济在开源软件是否被贡献、下载和用于生产中扮演了什么角色呢？“一般来说，开源被认为是有点抗衰退的，”扬科维奇说。但这并不意味着事情不会改变。在过去的几年里，开源公司的数量急剧增加，在生态系统中流动的资金数额巨大。这可能会改变。

如果资金情况确实发生变化呢？“我认为对于开源领域的许多人来说，最大的区别在于社区，”扬科维奇说。当我们谈论拥有“支持”时，它通常是指金融投资者，但是在开源中，社区的支持同样重要。在缺乏资金的情况下，一群相信这个项目的人可以帮助它生存下来——并表明这个想法是真正可靠的。

[经济不确定性与开源生态系统](https://thenewstack.simplecast.com/episodes/economic-uncertainty-and-the-open-source-ecosystem)

如果你回顾开源的历史，扬科维奇说，它是关于人们有一个想法，激励其他人做出贡献，使之成为现实。有时候，这些想法在商业上是不可行的，即使在最好的时候——即使它们被广泛采用。现在唯一改变的是，金融投资者将变得更加挑剔，以确保他们资助的项目不仅仅是鼓舞人心的想法，而且具有商业可行性。
