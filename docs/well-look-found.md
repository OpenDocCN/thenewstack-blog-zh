*“幸运青睐有准备的人”——[路易·巴斯德](http://lauriebrunner.com/fortunefavorsthepreparedmind/)*

赢家，赢家，鸡肉晚餐！你在我们下一本电子书的早期发布阶段偶然发现了这篇文章，“用 Docker 和 Containers 监控和管理[这本书目前可以在网站上下载，但我们要到周一才正式推出。这使你成为比其他人更早一点阅读电子书的独家读者的一部分，我希望你喜欢它！](https://thenewstack.io/ebookseries/)

你可能已经看到我[在](https://thenewstack.io/hello-world-ebook-container-orchestration-automation-now-available/)之前说过这个电子书系列被证明是困难的——但是我可能没有足够强调这些挑战。容器生态系统是巨大的。当我们开始的时候，它很大，随着我们在过去的一年半时间里出版了这个系列，它变得越来越大。至少可以说，用任何一种全面的范围来涵盖它都是令人生畏的。它帮助我们确定并提炼了关注生态系统层面分析的意义。因为我们喜欢挑战，我们很快就会扩展到容器生态系统之外。那应该很有趣。
