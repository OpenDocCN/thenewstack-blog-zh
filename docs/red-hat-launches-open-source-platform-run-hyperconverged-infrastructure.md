# 红帽推出开源平台运行超融合基础设施

> 原文：<https://thenewstack.io/red-hat-launches-open-source-platform-run-hyperconverged-infrastructure/>

超融合系统市场变得非常拥挤。 [Red Hat](https://www.openshift.com/) 是最新一家推出超融合基础设施(HCI)产品的公司，该产品在同一台服务器上提供全部虚拟化的计算、存储和网络。

上周，红帽发布了它声称是第一个“[生产就绪的开源超融合基础设施](https://www.redhat.com/en/about/press-releases/red-hat-unveils-industry%E2%80%99s-first-production-ready-open-source-hyperconverged-infrastructure)，称为[红帽超融合基础设施](https://www.redhat.com/en/technologies/storage/use-cases/hyperconverged-infrastructure)。

超融合是一种大型系统设计的架构方法，它依靠软件将计算、存储和网络组件紧密集成到由单一供应商支持的服务器中。EMC/VMware、 [SimpliVity](https://www.simplivity.com/) 和 [Stratoscale](https://www.stratoscale.com/) 都是红帽在这个新兴市场将要面对的竞争对手[。](http://www.networkworld.com/article/3112622/hardware/12-most-powerful-hyperconverged-infrastructure-vendors.html)

该公司正进入一个欣欣向荣的时期。根据市场研究公司 Gartner 的数据，未来四年，超融合市场收入的复合年增长率将超过 60%。

Red Hat 产品宣传员 Ross Turk 说，Red Hat 能够超过其竞争对手有两个原因。“我们是唯一一家提供纯软件超级融合的公司，也是唯一一家提供完全开源解决方案的公司，”他说。

Red Hat HCI 基于该公司的产品组合:其 KVM 虚拟化平台、其软件定义的存储产品 Gluster 和 Red Hat Enterprise Linux。

他说，该公司已经对用户的举动做出了回应。“我们遇到了一些客户，他们已经从我们的组件中整合了超融合解决方案。”他欢迎这一举措。“很多时候，客户会将我们带入他们是开拓者的境地。我们已经做了大量的测试和验证工作，以确保它们能够协同工作。”

而且，尽管 Red Hat 的一些客户已经走上了这条路，但他表示，尽管超融合市场现在已经非常成熟，但对该产品的需求仍然很大。

“他们可能会从 Red Hat 购买产品，原因有很多，”他说。“他们可能不想依赖专有解决方案。因为我们只是软件，所以我们可以在现成的硬件上运行。许多客户不想要单点解决方案的所有花哨功能，”他补充道。

但是，特克沉思道，也许价格是一个决定性因素。“也许客户也不喜欢三节点解决方案 10 万美元的价格，”他说。

相比之下，红帽产品有两个级别的定价，红帽产品经理肖恩·墨菲说。“我们有入门级三节点解决方案，标准服务级别协议(SLA)价格为 2，000 美元，特优 SLA 价格为 18，000 美元。”

特克认为，鉴于市场的不确定性，从长远来看，红帽的发行将更适合企业市场。“肯定会出现整合。我们与之交谈的 IT 人员正在认真思考未来三到五年内哪种解决方案更适合他们，购买 Red Hat hyperconverged 解决方案可以减少他们被供应商锁定的可能性。”

该公司还在使用两年前收购的 IT 自动化软件 Ansible。特克说这是发射的关键部分。“对于第一个版本，部署由 Ansible 驱动。它让客户能够更轻松地构建软件，并让它正常运行。”

根据 Turk 的说法，Red Hat HCI 将特别吸引那些拥有分布式或远程分支办公室的组织，如金融部门、石油和天然气行业、零售商和政府。

[红帽](https://www.openshift.com/)是新栈的赞助商。
