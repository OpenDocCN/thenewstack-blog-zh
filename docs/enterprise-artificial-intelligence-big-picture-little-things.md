# 企业人工智能关乎大局和小事

> 原文：<https://thenewstack.io/enterprise-artificial-intelligence-big-picture-little-things/>

智能机器人即将到来。现在，在他们的末日赌注中，无论你是站在 [埃隆·马斯克还是马克·扎克伯格](https://www.theatlantic.com/technology/archive/2017/07/musk-vs-zuck/535077/) 一边，都无关紧要。随着成本的降低，技术也在加速发展，这意味着我们很快就会生活在一个人工智能的世界里。

伦敦新人工智能和机器学习在企业应用 Meetup 的首次聚会并没有对这场科技巨头的战争过于大惊小怪。微软的全球策略师 [奈杰尔·王绍博](https://www.linkedin.com/in/nigelwillson/) 来揭示即将到来的人工智能的迹象，然后毕马威的首席技术官 [普里蒂·帕迪希](https://www.linkedin.com/in/priti-padhy-b114b31/) 都是关于如何将其应用到企业空间。今天，我们将介绍他们认为我们目前可以获得的产品，以及企业将如何利用这些产品。比你想象的实用很多，科幻少很多。

## 转向 AI 主流

微软的威尔逊提供了人工智能成为主流的三种方式，并展示了三款微软支持的工具，可以免费试用。

> “为什么现在对艾？ 指数增长+指数缩减=不可能变成可能。” —微软全球战略家奈杰尔·王绍博

王绍博引用了 [加速收益定律](http://www.businessinsider.com/ray-kurzweil-law-of-accelerating-returns-2015-5)，其中 技术进步遇到了成本的快速降低。在这种情况下，法律就是将人工智能带给大众的东西。

有许多地方人工智能更便宜，因此更容易获得。这甚至包括 DNA 测序，他说这在世纪之交花费了大约 10 亿英镑，现在英国政府可以负担得起了，以至于可以利用它来促进预防性医疗保健。

微软自己也在研究一些基于人工智能的技术，以降低媒体处理的成本。Wilson 抓拍了一张观众的照片，并上传到了 [微软 Pix 智能相机 app](https://www.microsoft.com/en-us/research/product/microsoftpix/) 。该工具能够扫描面部，以识别性别、年龄、特定的面部特征，甚至情感。然后，通过两到三行代码，它可以调用一个 API 来返回结果和累积数据。

在另一个例子中，Wilson 展示了 2017 年微软 Build 大会主题演讲的视频如何被 [微软视频索引器](http://VideoIndexer.ai) 扫描。该工具能够理解视频中谁在什么时候说了什么，并适当地标记了这些信息。一个人只对一个关键词或一个演讲者感兴趣，他可以很容易地跳过。该系统可以确定知道那个人是正面还是负面地谈论它，并可以将演示翻译成许多不同的语言。

但是这种加速的变化不仅仅是关于削减成本，而是关于技术能力的快速增长。IBM 和微软继续在语音识别领域展开竞争，这两个公司都已经达到了与人耳相当的水平。仅在一年前，这种近乎成功的做法似乎是不可能的，部分原因是利用了对深度学习至关重要的流行词大数据。

“你拥有的数据越多，性能就越好，大数据性能的准确性就越好，”王绍博说。

## 自动化日常工作

毕马威(KPMG)的帕迪(Padhy)提醒我们，人工智能一直在以小的方式进行，就像我们的 Cortana、Alexa 或 Siri 虚拟助手一样。但是他说:

> “在企业中，我们被困在个人电脑互联网世界，只有很少的人工智能，除非你是谷歌、微软或亚马逊。”—普里蒂·帕迪，毕马威首席技术官

在与这家四大咨询公司的客户合作中，帕迪发现人工智能领域还没有得到太多关注。不过，他说，归根结底，任何人工智能都必须专注于为客户或员工实现某些目标。

但就目前而言，他表示“没有愿景，也没有真正以客户为基础。如果你想驱动 AI，你需要在背后有真正的激情和目的。”

他说，在企业领域，数据将成为一切的基础，这是一个额外的障碍，但当数据存储在内部时，利用这些数据将很困难。

帕迪认为，要让企业成为这种加速变化的一部分，首席执行官必须是拥抱人工智能业务的人。它不能局限于一个小的创新团队。T 首席执行官必须与首席技术官和首席信息官合作，利用企业级的多云人工智能系统，实现考虑架构因素的跨公司愿景。此外，他表示，人工智能解决方案不能依赖于一个供应商，而是必须利用许多来源，特别是开源。

“运营是数据的基础。你怎么动员人？”帕迪说。

从哪里开始？帕迪说，从人力资源、财务和设施等小事做起，比如今天需要多少餐饮服务。当然，这些没有什么闪亮的，所以他说你必须用数字让他们眼花缭乱，比如:

*   11%的时间花在一般的时间表、费用和会议安排上。AI 能解决这些生产力问题吗？
*   18%的时间花在寻找和联系他人上。AI 能回答谁做什么吗？

毕马威(KPMG)正在开发一款“费用机器人”(ExpenseBot)，员工只需拍下他们的收据，这款机器人就会自动对其进行分类。这家咨询公司还在为个性化税收开发认知审计工具和应用程序。

帕迪说:“这不是一个单一的系统，而是一系列的图表——你可以将整个业务整合在一起，但随后你必须与许多数据所有者和不同国家的所有权进行谈判。”

当围绕人工智能协调整个企业时，它必须有 C 级支持，你必须推销商业价值，这意味着节省时间和/或金钱。

> 人工智能要求“企业变革管理——它需要是一种业务变革，而不是技术问题。”—普里蒂·帕迪，首席技术官毕马威

最后，记住这一点很好，正如帕迪所说，“每个人都认为人工智能将做很多大事，但这些都是企业需要的简单事情。”
