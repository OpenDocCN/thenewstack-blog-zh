# 开源支撑着一家家居用品供应商的全球野心

> 原文：<https://thenewstack.io/open-source-underpins-a-home-furnishings-providers-global-ambitions/>

底特律——way fair 将自己描述为“所有家居用品的目的地:帮助任何地方的任何人创造家的感觉。”

它提供了一个在线平台来购买家居家具、户外装饰品和其他家具。根据 Wayfair 的全球领导、开源项目办公室(OSPO)和高级软件工程经理的说法， [Natali Vlatko，](https://de.linkedin.com/in/natalivlatko) 它还支持其供应商，以便他们可以使用该平台来销售他们的家庭商品。

[开源支撑着一家家居用品供应商的全球野心](https://thenewstack.simplecast.com/episodes/open-source-underspins-a-home-furnishings-providers-global-ambitions)

“这需要很多幕后的技术工作才能实现，”Vlatko 在 New Stack Makers 播客的“在路上”中说道，该播客是在 [KubeCon + CloudNativeCon 北美 2022 期间录制的。](https://www.cncf.io/kubecon-cloudnativecon-events/?utm_content=inline-mention)

这一集的对话由 TNS 经常撰稿人 [B. Cameron Gain 主持，并由 KubeCon + CloudNativeCon 赞助。](https://thenewstack.io/author/bruce-gain/)

## 全面转向云原生模式

随着该公司在全球范围内扩大业务规模，Wayfair 所依赖的技术工作变得越来越紧张。基础设施必须是高度分布式的，依靠容器化、微服务、[、Kubernetes、](https://thenewstack.io/docker-versus-kubernetes-start-here/)特别是[开源](https://thenewstack.io/ebay-betting-on-open-source-is-great-for-your-business/)来完成工作。

“我们在世界各地都有技术专家，在北美和欧洲都有，”弗拉特科说。“我们希望确保我们正在利用[原生云](https://thenewstack.io/how-boeing-uses-cloud-native/)和开源，不仅作为推动我们业务的技术，而且作为我们现在工作的最佳方式。”

弗拉特科说，开源已经成为创造和提供技术服务的“伟大途径”，为了实现这一点，她聚集了所需的人才。这个小型工程师团队专注于平台工作、宣传、社区管理以及内部的许可证合规性。

她说，大约五年前，当 Vlatko 加入 Wayfair 时，该公司尚未“完全转向云原生模式”。Wayfair 混合了内部和云基础架构。

在从整体架构分离到微服务架构之后，“这一旅程真正开始了，我们了解了微服务的真正巨大优势，并达到了我们认为的地步，*好吧，这种混合模式对我们来说实际上有利于我们的微服务完全在云中，*”vla tko 说。

【2020 年末，Wayfair 决定退出数据中心，将运营转移到云上，这一决定已于去年 10 月完成。

公司文化是这样的，工程师有空间通过在沙盒环境中做大量开发工作来进行实验，而不用担心失败。

弗拉特科说:“人们可以边学边做，而不用担心失败或错误。

“所以，我认为实验是我们自身学习和成长的一个非常重要的方面。此外，参加 KubeCon + CloudNativeCon 等大型活动[也很有帮助]。我们收到了其他公司的反馈，他们已经完成了相同的旅程和流程，并正在从用例中学习。”
