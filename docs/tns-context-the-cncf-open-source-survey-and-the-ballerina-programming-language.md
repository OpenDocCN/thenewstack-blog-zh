# TNS 背景:CNCF 开源调查和芭蕾舞演员编程语言

> 原文：<https://thenewstack.io/tns-context-the-cncf-open-source-survey-and-the-ballerina-programming-language/>

[TNS 上下文:CNCF 开源调查和芭蕾舞演员编程语言](https://thenewstack.simplecast.com/episodes/tns-context-the-cncf-open-source-survey-and-the-ballerina-programming-language)

今天在[新的堆栈环境](https://thenewstack.io/podcasts/context)播客中，我们与 [Chris Aniszczyk](https://www.linkedin.com/in/caniszczyk/) 交谈，他是 TODO 集团的联合创始人和[云本地计算基金会](https://www.cncf.io/) (CNCF)的首席技术官，讨论我们最近的开源项目管理调查的结果。我们还将讨论 [WS02](https://wso2.com/) 的新云原生编程语言 [Ballerina](https://ballerina.io/) 。

参与这一集的语境主持人 TNS 编辑总监 [Libby Clark](https://twitter.com/LibbyMClark) 是 TNS 创始人 [Alex Williams](https://twitter.com/alexwilliams) 和 TNS 执行主编 [Joab Jackson](https://twitter.com/Joab_Jackson) 。

本周，我们[发布了一项关于公司如何管理开源软件以及正式政策和项目的好处和挑战的调查结果。这项调查是由 Linux 基金会赞助的，具体来说是由 TODO Group 赞助的，它是一个开放源码管理人员的专业组织。](https://thenewstack.io/survey-open-source-programs-are-a-best-practice-among-large-companies/)

我们调查了 700 多名受访者，其中大部分是开发人员，发现:

*   所有行业中超过一半的受访者(53 %)表示他们的组织有开源软件项目或计划建立一个。
*   但是拥有一个正式的项目是大型科技公司的最佳实践
*   程序的数量正在增长:我们预计拥有开源程序的大公司的数量到 2020 年将增加两倍。

我们讨论了这些结果，并与 Chris 讨论了 TODO 组。

然后，在节目的后期，Joab 给了我们一些本周新闻报道的亮点，我们将讨论由 TNS 赞助商 WSO2 开发的云原生编程语言 Ballerina，该公司声称它最终可能取代 Java。
