# 新的 Cloud Foundry 容器运行时:事实

> 原文：<https://thenewstack.io/cloud-foundry-container-runtime-facts/>

[新云铸造容器运行时:刚刚事实](https://thenewstack.simplecast.com/episodes/the-new-cloud-foundry-container-runtime-just-the-facts)

Cloud Foundry 容器运行时是 Kubo 的新名称，它是在 BOSH 上运行的 Kubernetes。在今天这一集的《新栈制造者》中，TNS 创始人 Alex Williams 采访了 [Cloud Foundry Foundation](https://www.cloudfoundry.org/) 首席技术官 [Chip Childers](https://twitter.com/chipchilders) 以了解 Cloud Foundry 对这一新运行时的计划，Childers 强调了 BOSH 如何满足当今开发人员的需求。

Childers 继续指出，Cloud Foundry 容器运行时和应用程序运行时将彼此相邻，允许 Kubernetes 和 Cloud Foundry 应用程序运行时之间共享身份。

“我认为现在最重要的是为开发者着想。企业中的开发人员需要什么？在我们看来，开发者体验是最重要的事情。Childers 说，“其他一切都是从这一点出发的，”他补充说，“Cloud Foundry 社区将其视为:开发人员团队、开发人员体验，然后向下。”
