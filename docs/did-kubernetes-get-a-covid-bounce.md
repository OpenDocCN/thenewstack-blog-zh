# Kubernetes 得到了 COVID 反弹吗？

> 原文：<https://thenewstack.io/did-kubernetes-get-a-covid-bounce/>

超过三分之二(68%)的 IT 专业人员认为，由于疫情，他们 500 多名员工的公司对 Kubernetes 的使用增加了。没错，根据 Pure Storage 2021 年 Kubernetes 采用调查显示，已经在使用的企业看到了进一步的采用。在那些看到增长的人中，83%的人认为 Kubernetes 的努力只比计划提前了不到 6 个月。公司可能会将疫情作为一个契机，启动已经计划好的工作负载到容器化云环境的迁移。此外，大多数新应用程序都在容器中运行。这些变化导致[在容器和其他云原生技术上的支出](https://twitter.com/RachelDines/status/1352269671345623043)增加，但是大多数公司仍然只管理少量的 Kubernetes 集群。

当被问及为什么 Kubernetes 的使用因疫情而增加时，一半的调查对象声称这是疫情推动的更广泛的数字化转型计划的一部分。正如我们[去年](https://thenewstack.io/no-easy-fix-cloud-natives-role-in-digital-transformation/)所写的，云原生技术在数字化转型的道路上可能会产生自己的问题。许多接受调查的 IT 专业人员并不真正知道他们的组织为什么使用 Kubernetes。当然，他们一致认为更快的部署时间是最大的好处，但这不是该组织致力于 Kubernetes 的最大原因。至少在他们看来，这是由于业务战略的变化，可能是为了安抚一位高管，即 it 正在采取措施加速数字化转型。
