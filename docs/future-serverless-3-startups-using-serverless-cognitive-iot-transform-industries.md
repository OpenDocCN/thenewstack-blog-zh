云是应用程序开发的游戏规则改变者，而[无服务器计算](/category/serverless/)正在彻底改变开发者对云提供的一些最强大服务的访问，包括认知智能、数据分析和物联网。

在传统的云计算模式中，开发人员通常必须维护他们的基础设施，并担心何时以及以多快的速度扩展，如果他们在多个地区部署应用程序和功能，还会有潜在的弹性问题。即使在应用程序空闲时，他们也要为计算能力付费——对于资源有限、应用程序处于早期增长的初创公司和小公司来说，这是一个特别痛苦的问题。

无服务器计算被许多人誉为云计算的下一个时代，它通过抽象出基础设施、运行代码和按需扩展来减轻这些麻烦。对于开发人员来说，具有强大认知堆栈的无服务器平台，如 IBM Bluemix OpenWhisk，为他们提供了前所未有的强大服务，如沃森 API、沃森物联网平台和天气智能。

> 作为少数几个建立在开放标准上的无服务器平台之一，OpenWhisk 在这些应用程序中充当着无形的力量，将相关的事件、动作和触发器绑定在一起。

无服务器框架将应用程序分解成独立的功能，在可由云监控和扩展的并行容器中运行。这种模式被称为事件驱动编程，根据需要执行代码和服务，以适应使用高峰。开发人员只为他们使用的东西付费，并享受能够同时处理多段代码以创建更好更快的用户体验的额外好处。

随着来自社交媒体分析和物联网传感器等多种来源的数据爆炸式增长，管理海量情报、集成新的认知和物联网工作负载以及快速扩展的能力是保持竞争力的关键。因为数据可能来自广泛的系统和服务，所以开放无服务器平台以保持数据的快速可访问性也是必不可少的。

最近，许多初创公司转向了 IBM 在开放社区中构建的无服务器计算平台 [Bluemix OpenWhisk](https://www.ibm.com/cloud-computing/bluemix/openwhisk) 。例如，可持续发展初创公司 [GreenQ](https://greenq.gq/) 正在通过垃圾互联网(IoG)解决方案改变城市可持续收集垃圾的方式，该解决方案通过调度、路线和车队管理来跟踪和优化垃圾收集。这可以为城市节省高达 50%的垃圾支出。使用 OpenWhisk，安装在垃圾车上的 GreenQ 传感器利用认知服务来帮助识别优化这些操作的模式和相关性。

在营销界，[神经应用](https://www.neuroapplied.com/)有助于揭示消费者对品牌的想法。这家营销初创公司的平台通过快速、有趣和简单的在线游戏吸引用户，转向 OpenWhisk 来处理用户活动的爆发，并为每个用户创建不同的个性化游戏内容。

最后，作为一种有助于拯救生命的解决方案， [AbiliSense](https://www.abilisense.com/) 通过使用无服务器连接到物联网技术，触发声音到感觉的转换，帮助有听力障碍的人更加意识到他们周围的潜在危险。用户戴上带有传感器的手环，这些传感器可以检测和分析火警等紧急声音，并通过文本、图标、振动或闪光灯在用户的手机上传达这些声音。

作为少数几个建立在开放标准上的无服务器平台之一，OpenWhisk 在这些应用程序中充当着无形的力量，将相关的事件、动作和触发器绑定在一起。随着所有行业的数据持续增长和激增，无服务器无疑将成为资源、可扩展性和连接到云上运行的认知和物联网工具的标准。
