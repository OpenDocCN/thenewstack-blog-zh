# 区块链会改变商业运作方式吗？

> 原文：<https://thenewstack.io/will-blockchain-change-way-businesses-operate/>

在企业界，[区块链](/tag/blockchain/)分布式账本技术似乎被两大阵营追随。一个人呆在很远很远的地方，警惕着这是地下的加密货币。另一个阵营密切关注，寻找可以用区块链来简化(和扰乱)传统商业会计流程的方法。

有鉴于此，我们从上个月伦敦[区块链周](https://www.blockchainweek.com/)的一个小组中汲取了一些智慧，以了解区块链技术的进展和障碍。

## 主流区块链采用的障碍？

金融技术投资公司 Ripple Labs 的顾问 Anish Mohammed 说:“如果你想在许多实体间进行数百万笔交易，你需要达成共识。

让区块链局限于令牌用例的原因是法律用例并不明确。此外，在决定什么时候应该公开哪些交易，以及如何监控所谓的私人区块链方面，还存在额外的担忧。

[Paul Puey](https://twitter.com/paullinator),[air bitz](https://airbitz.co/)区块链钱包的创始人对此表示赞同，认为最大的障碍是它需要多方达成协议才能使用。"采用区块链的最大障碍是网络效应——创建一个标准."

毫无疑问，网络效应是区块链，尤其是比特币迅速走红的原因。但这也是阻碍它的原因，因为分散的权力使得制定一致同意的标准变得异常困难，其中一些标准可能会被采纳为法规。

英国电信的 Joshua Daniel 补充说，建立一个全自动的 API 系统或另一种普遍认可的技术来构建一切是至关重要的。

## 合作伙伴如何帮助区块链成长？

这个小组最大的共识似乎是，区块链繁荣的最佳方式是依靠政府的支持。现在，他们没有深入研究最好的方法，但正如穆罕默德所说，“最简单的方法是有一个 GDP 很高的国家来支持它——每个供应商都希望他们的东西成为标准，这是未来的游戏计划。”

他说，不一定非得是政府，但如果一个大型金融系统采用区块链，它可能会导致该机构乃至整个行业的创新。

Puey 补充说，它本身不一定是政府或金融机构，而是有大量资金支持一个标准或品牌，“类似于互联网初期的 AOL 效应。”

## 二等兵区块链的真实使用案例有哪些？

区块链仍然是一个闪亮的新事物，没有很多经过验证的用例——坦率地说，对于普通商务人士来说，这是一个复杂的概念，对于 it 专业人士来说，它还没有被证明是有用的。既然我们已经超越了开源实验者和早期采用者，看起来区块链的下一批采用者将会是私有的、内部使用的。

虽然这可能看起来违背了区块链的开源起源，但私有区块链作为一种替代[安全结算系统](https://stats.oecd.org/glossary/detail.asp?ID=6793)和一些合同需求的方式是有意义的。

私有区块链分发数据的方式与公共区块链相同，但它通过一个协议将计算机连接起来，该协议只允许这些计算机交换交易并维护一个公共账本，而不受外界的检查。

“一般来说，这是共识和隐私的问题。区块链是你不想透露给别人的数据，”穆罕默德指出。

普伊补充澄清说，“在私人区块链，有这种误解，一切仍然是公开的参与者。”

丹尼尔说，风险在于信任元素——“能够将多个区块链或创作的多个元素分层。这是谁有权访问机器本身的问题。”

除了这种额外的安全感，私有区块链的好处还在于它的可伸缩性，特别是对于事务日志记录。

私人区块链的问题随着如何管理成本而增长。这里的成本归结为存储需求:您真正想要保留多少交易历史。你想缓存一天吗？一年？还是整个交易？

Puey 建议你只需要回到某些人不同意的地方，这是一个很好的经验法则，尽管可能很难预测。
