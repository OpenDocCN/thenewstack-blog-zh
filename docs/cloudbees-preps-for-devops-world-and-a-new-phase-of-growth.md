# CloudBees 为 DevOps 世界和新的成长阶段做准备

> 原文：<https://thenewstack.io/cloudbees-preps-for-devops-world-and-a-new-phase-of-growth/>

随着持续集成和交付提供商 [CloudBees](https://www.cloudbees.com/?utm_content=inline-mention) 为其年度 [DevOps World](https://www.devopsworld.com/) 会议做准备，该公司也在为一个新的增长阶段做准备，更加关注安全性、人工智能和使 DevOps 更容易。

[DevOps 世界](https://www.cloudbees.com/devops-world-2021/registration)将于 9 月 28 日至 30 日举行。去年，该活动吸引了大约 30，000 名虚拟参与者。今年的活动又是虚拟的，也是免费的。凭借“共同构建软件交付的未来”的口号，DevOps World 的重点将是接触整个 [DevOps 生态系统](https://thenewstack.io/category/devops/)以分享关于当前使用的和未来预期的工具、技术和最佳实践的知识。

在最新一集的《新堆栈制造商》播客中，我们采访了 CloudBees 的联合创始人兼首席战略官 Sacha laboury，探讨了 DevOps World 和公司的未来。TNS 出版商亚历克斯·威廉姆斯在 TNS 新闻编辑达里尔·k·塔夫特的帮助下主持了这一集。

[CloudBees 为 DevOps 世界和新的成长阶段做准备](https://thenewstack.simplecast.com/episodes/cloudbees-preps-for-devops-world-and-a-new-phase-of-growth)

“我们谈论 devo PS——我们有多种途径，我们有更具体的途径，以及数字化转型意味着什么和与之相关的概念，而我们有更多关注实践者的其他途径，”Labourey 说。“我们也关心共享服务工程经理，因此，根据你是谁以及你每天关心什么，你会找到关注你需求的正确途径。”

苹果公司的联合创始人史蒂夫·沃兹尼亚克和劳瑞、CloudBees 公司的首席执行官 T21 以及其他一些人将会做主题演讲。

在播客中，Labourey 还谈到了他从首席执行官到首席战略官的转变，这发生在 2 月份，当时该公司任命 DeWitt 为首席执行官。

“斯蒂芬和我非常互补，我们天生不同，”劳瑞说。“这带来了好事情。我们不是你知道…他不只是一个 Sacha plus-plus 或 Sacha plus-one…我们相处得非常好，对我来说，这意味着我可以专注于我真正热爱的事情，我已经离开太久了(专注于)产品、战略和市场。我真的很喜欢(转向 CSO)需要我重新整理我的大脑，因为我习惯了不断地从一个主题切换到另一个主题。”

CloudBees 雄心勃勃，期待着一个新的增长期。该公司增加了一名新产品负责人、一名新的首席信息安全官(CISO)、一名新的首席营销官和一名新的人力关系主管。

“我们真的在为下一阶段的增长奠定基础，”Labourey 说。“未来几个月，你会看到 CloudBees 引擎迸发出大量火花。”

Labourey 说，CloudBees 聘请 Prakash Sethuraman 担任首席信息安全官，因为客户需要更安全的工具和实践来构建软件。尽管 CloudBees 本身不会开发安全产品，但其目标是将更多的安全性融入其产品中，以帮助开发者[向左转](https://thenewstack.io/what-does-shift-left-mean-if-every-process-is-a-circle/)。Labourey 还谈到了 CloudBees 将如何应用机器学习来为其工具集提供更多自动化。

此外，对于所有老派的 Java 掌门人，Labourey 分享了他担任 JBoss 首席技术官时的一些趣闻轶事，以及他在与 JBoss 创始人兼首席执行官[马克·弗勒里](https://en.wikipedia.org/wiki/Marc_Fleury)共事时学到的一些经验。有趣的，有益的东西。
