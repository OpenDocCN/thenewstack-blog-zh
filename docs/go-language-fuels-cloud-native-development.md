# Go 语言推动云本地开发

> 原文：<https://thenewstack.io/go-language-fuels-cloud-native-development/>

Go 于 2007 年由谷歌创立，旨在提高多核网络机器和大型代码库时代的编程效率。从那时起，谷歌和整个行业的工程团队都采用 Go 来大规模地构建产品和服务，包括云原生计算基金会，该基金会有超过 75%的项目是用该语言编写的。

在这一集的 New Stack Makers 播客中，[史蒂夫法兰克王国，](https://www.linkedin.com/in/stevefrancia/)产品负责人:Go 语言，谷歌和 MongoDB 的校友，Docker 和 Drupal 董事会成员讨论了编程语言，Go 1.18 中的新功能，以及为什么 Go 会继续被开发人员加速采用。[达里尔·塔夫脱，](https://www.linkedin.com/in/darryltaft/)新闻编辑的新堆栈主持了这个播客。

[走语抱拳云本土发展](https://thenewstack.simplecast.com/episodes/go-language-fuels-cloud-native-development)

在开发者生态系统 2021 的[状态中，Go 位列开发者计划采用的前五种语言，并且继续成为发展最快的语言之一。根据法兰克王国的说法，它的创建动机是看看一种新的系统编程语言是否能够以安全为首要焦点快速构建和编译。随着开发人员在 Google 进进出出，这种语言的简单性和可伸缩性使得许多人能够在任何给定的时间参与多个项目。](https://www.jetbrains.com/lp/devecosystem-2021/)

“将围棋从大多数语言中分离出来的影响是它背后的创造者的经验，他们都是用他们的集体经验来构建它的，”法兰克王国说。今天，围棋正在影响许多主流语言。它的元素可以在一个工具中找到，该工具将每个人的源代码格式化为相同的，更具可读性。从那以后，许多语言都采用了同样的做法，”法兰克王国说。“然后就生锈了。[去锈](/rust-vs-go-why-theyre-better-together/)在平行的轨道上，我们互相学习。还有一种叫做 V 的新语言最近被开源了，这是受围棋启发的第一种主要语言，”法兰克王国说。

最新发布的 [Go 1.18](/go-1-18-the-programming-languages-biggest-release-yet/) 是谷歌迄今为止最大的一款。“它包括四个主要特性，每一个你都可以围绕着它们开发一个版本，”法兰克王国说。在这个版本中，“泛型是已经工作了 10 年的 Go 语言的最大变化，”法兰克王国补充道。“因为我们知道泛型有可能让语言变得更复杂，所以我们花了很长时间研究不同的提议，”他说。模糊、工作空间和性能是 Go 上一版本中发布的另外三个功能。

“从改进我们的文档和学习——你可以去 go.dev/learn/[的](https://go.dev/learn/)获得最新的资源——我们真的专注于开发者体验的广阔视野，”法兰克王国说。“在未来，我们看到的不是我们的团队，而是社区以新的方式前进，”他补充道。
