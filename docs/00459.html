云代工厂:关注云的灵活性和选择性——原生世界

对于一些在硅谷等技术中心之外工作的组织来说，可供选择的开发人才严重短缺。因此，许多公司采取了主动学习的方法来帮助他们的开发人员跟上当今云原生软件平台的需求。

Cloud Foundry执行董事 Abby Kearns 在上个月的Cloud Foundry欧洲峰会上录制的 The New Stack Makers 播客中解释道, Cloud Foundry 正在发展其技术，以使这些用户的目标受益。

“对于每一家正在成为软件公司的公司，他们都在努力寻找，他们在哪里找到人才？”卡恩斯说，并补充说，“我们看到许多公司投资培训和再培训人员，教他们成为云原生开发人员，我认为这是所有这些中最迷人的一块。”

Kearns后来继续指出，Cloud Foundry使开发人员能够专注于简单地编写代码，而新的 Cloud Foundry容器运行时让他们可以在基于 Kubernetes 的容器编排或Cloud Foundry的应用运行时之间进行选择，完全在Cloud Foundry内。

“当你想到这种快速发展的技术生态系统时，它真的会让人不知所措。卡恩斯说:“我们试图通过Cloud Foundry实现这一目标。Cloud Foundry提供了“这种灵活性和选择，但给你足够强大的护栏和意见，使你作为一个组织很容易采用它并取得成功。”
