# 新的堆栈环境:Tanzu，面向开发人员的 VMware Kubernetes 发行版

> 原文：<https://thenewstack.io/the-new-stack-context-tanzu-the-vmware-kubernetes-distro-for-developers/>

欢迎来到[新的堆栈环境](/podcasts/context)，在这个播客中，我们将讨论云计算领域的最新新闻和观点。在本周的节目中，我们采访了 [Craig McLuckie](https://www.linkedin.com/in/craigmcluckie/) ，他是[VMware](https://www.vmware.com/company.html)Tanzu development 的主管，也是[Kubernetes](https://thenewstack.io/beda-burns-and-mcluckie-the-creators-of-kubernetes-look-back/)的创建者之一。我们向他询问了开发人员对现代商业的重要性、Kubernetes 为开发人员带来的价值以及 VMware 的 [Tanzu 产品组合](https://tanzu.vmware.com/tanzu)如何实现这一点。

TNS 编辑和营销总监[利比·克拉克](https://thenewstack.io/author/libby/)主持这一集，旁边还有 TNS 高级编辑[理查德·麦克马努斯](https://thenewstack.io/author/richard/)和 TNS 执行主编[约阿布·杰克逊](https://thenewstack.io/author/joab/)。

[第 124 集:Tanzu，VMware 面向开发者的 Kubernetes 发行版](https://thenewstack.simplecast.com/episodes/episode-124-tanzu-vmwares-kubernetes-distro-for-developers)

上周在 [Cloud Foundry Summit](/cloud-foundry-summit-developers-shouldnt-have-to-worry-about-kubernetes/) 上，VMware 发布了面向 Kubernetes 的 VMware Tanzu 应用服务测试版，这是一个通过使用 Cloud Foundry 开源 PaaS 为 Kubernetes 带来一致开发体验的包。我们向 McLuckie 询问了 Tanzu 的定义，K8s 的 TAS 如何帮助开发人员，如何使用 Kubernetes 的数据库，以及为什么开发人员对今天的企业如此重要。

“我认为把开发者看成是一个单一方面的东西是危险的。有很多很多不同的开发者…作为一个行业，我们的工作就是确保这不是一个放之四海而皆准的体验。这是关于在正确的时间为开发者创造正确的体验。“一个吉勒 团队 需要 敏捷 基础设施。Kubernetes 是一种更自然的敏捷基础设施表达方式。它提供了处理使应用程序适应环境的机制的能力。但我们有责任根据开发人员在自己的旅程中所处的位置，为他们精心打造合适的切入点和合适的体验。”
