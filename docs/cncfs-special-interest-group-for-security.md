# CNCF 安全特别利益集团

> 原文：<https://thenewstack.io/cncfs-special-interest-group-for-security/>

[Honeycomb](https://www.honeycomb.io/) 正在赞助新 Stack 对 Kubecon+CloudNativeCon 北美 2020 的报道。

曾经想知道云本地计算项目的安全状况吗？你应该。任何试图保护库伯内特斯的人[都知道这并不容易。安全性是保护系统和降低风险的永恒斗争，但随着云计算的快速变化和复杂的相互关系，没有简单、千篇一律的答案。试图弄清楚这一切正是](https://thenewstack.io/kubernetes-security-best-practices-to-keep-you-out-of-the-news/)[云本地计算基金会(CNCF)特别兴趣小组(SIG)安全](https://github.com/cncf/sig-security)的用武之地。

在 [2020 北美 KubeCon](https://events.linuxfoundation.org/kubecon-cloudnativecon-north-america/) 、 [Brandon Lum](https://www.linkedin.com/in/brandon-lum-a7b79418/) 、IBM[高级软件工程师](https://www.ibm.com)和美国[国家安全局](https://www.nsa.gov/) (NSA) DevOps 安全负责人 [Emily Fox](https://www.linkedin.com/in/themoxiefox/) 解释了 CNCF SIG 的工作、目标和发展方向。等等，你说什么？国家安全局不就是为了破解密码和安全吗？实际上，这只是其工作的一半，另一半是锁定和保护系统。

SIG 的最终目标是为运营商、管理员和开发者提供一个创建云原生应用所需的工具生态系统。这些工具还将让他们清楚地了解风险，并能够验证他们的安全策略决策。为此，需要:

1.  系统安全架构，了解并适应不断增长的系统异构性，并提供一个框架来保护资源和数据，同时为用户提供服务
2.  通用词汇和开源库，使开发人员能够轻松创建和部署满足系统安全要求的应用
3.  公共库和协议，使人们能够对系统的安全性进行推理，例如审计和可解释特性。

这并不容易。这一点也不容易。福克斯和卢姆知道这一点。

为了完成这些目标，福克斯表示他们需要更多的开发人员。福克斯补充说，“我们不要求你是一个安全专家，我们希望你倾听和学习。”当然，如果你已经是一名安全专家，他们会很乐意收到你的来信。

那么，你能为 SIG 做些什么呢？很多事情。福克斯继续说道，“我们鼓励你通读回购协议。熟悉我们是谁，我们在做什么，以及我们是如何运作的。”

完成了。福克斯说，他们希望你“加入并审查我们的 PR(拉请求)评论，或者甚至制作一个 PR 来结束一个开放的问题”。“我们有一些简单的。我们有需要帮助的标签，有时只需快速浏览即可。我们有演示文稿、提案和建议，甚至安全评估，这些都是促进和发展社区的绝佳方式。它只是从提交一个问题开始。”

两人特别谈到了对现有项目的安全评估。毕竟，除非你知道你已经运行的程序中有什么，否则很难构建安全工具。

卢姆说。“我们的核心活动之一是安全评估。”安全评估是我们经常做的事情，到目前为止，我们已经完成了五个 CNCF 项目的安全评估。其中包括 SPIFFE、Spire、in-toto 和 OPA(开放策略代理)。

那么什么是安全评估呢？Lum 解释说，“安全评估这个术语经常会被重复使用，所以我们想说明安全评估不是安全审计。”

对于 SIG 来说，安全评估实际上是一种“评估项目安全状况”的方式。这包括查看项目的目标和评估对其安全方面的考虑。因此，这包括项目的整体架构和设计模式，以及安全流程的评估。例如，项目如何处理安全问题和应对安全事件。”

当然，评估还包括攻击者可能发起的攻击类型。例如，如果一个项目使用一个插件或 XML 数据库，它应该解决如果数据库或组件受到损害会发生什么。

安全评估的结果是一份文件。这有助于向项目的技术监督委员会提供建议。一旦评估的结果在代码中得到处理。Lum“建议该项目通过安全审计，作为 CNCF 生态系统中的一个关键安全组件。”

安全评估的另一个有价值的结果是，由于它提供了项目及其安全问题的概述，最终用户可以使用它作为安全使用程序的指南。Lum 总结道，理想情况下,“安全评估文档包含项目设计和架构的细节。因此，它很好地概括了你需要了解的关于项目的大部分细节，从头开始。”

你可能认为这是一项艰苦的工作。你是对的。但是，这是很好的工作，非常值得去做。如果你愿意帮忙。所罗门群岛政府是开放的，它的人民随时准备帮助你帮助 CNCF 社区。
