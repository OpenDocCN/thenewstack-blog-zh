# 罗格斯大学如何打造多元化创业之路

> 原文：<https://thenewstack.io/rutgers-university-revolutionizing-urban-entrepreneurship/>

对于任何人来说，开始一个新的项目、业务或应用想法都是一个挑战，但是如果一个企业家是黑人、 [Latinx](http://www.huffingtonpost.com/entry/why-people-are-using-the-term-latinx_us_57753328e4b0cc0fa136a159) 或女性，这种挑战可能会成倍增加。许多第一代移民企业家错过了常春藤联盟白人男性同行可能获得的“家人和朋友”种子资金。其他人面临的任务是获得商业潜力不是数十亿美元的资本。

如果少数民族企业家无法获得这些资源，技术生态系统和整个世界都会发现自己缺乏这些资源。“这意味着重要的、有利可图的业务没有启动，没有扩大规模，这真的意味着我们的世界正在失去我们都想要的、我们都想促进和鼓励的创新，”罗格斯大学城市创业和经济发展中心教授 [Lyneir Richardson](http://www.business.rutgers.edu/faculty-research/directory/lyneir-richardson) 解释道。

[罗格斯大学如何打造多元化创业之路](https://thenewstack.simplecast.com/episodes/how-rutgers-university-forges-a-path-for-diverse-entrepreneurship)

CUEED 旨在帮助少数技术企业家获得资金、会议场所、商业建议等资源，并提供如何申请技术加速器计划或政府资助的种子资金的指导。

Richardson 解释说，他每天都会接到少数族裔社区成员的电话，询问如何发展他们的公司。通常，他们对金融技术、安全和国防项目以及医疗保健技术的进步有自己的想法。“他们没有获得牵引力，所以他们没有获得推动公司增长所需的收入。真正的问题是，你如何让人们更近距离地观察？我们这里有一位教员，他围绕无意识偏见做了很多工作。人们如何真正看到机会，而不是真正看到种族或性别？你如何寻找和倾听机会，而不会有这样的障碍:“嗯，我只是不确定产品是否会扩大规模？””理查森说。

最终，这一挑战的答案来自科技加速器及其管道。让被边缘化的个人作为投资者、董事会成员或公司评估者参与进来，将有助于使资本多样化，并向那些以前可能无法获得资本的人开放获得资本的渠道。直到 2016 年，我们才看到 [Monique Woodward](http://500.co/team/monique-woodard/) 任命第一位黑人女性[成为 500 家初创公司的风险合伙人](http://atlantablackstar.com/2016/02/11/as-first-black-venture-partner-of-500-startups-monique-woodard-could-help-fund-more-african-american-businesses/)。理查森解释说，加速器可以通过赞助不同人群提出想法、参加他们社区的会议以及将不同投资者加入他们的团队来扩大他们的影响。理查森说:“加速器的决策层需要更多不同背景的人。

罗格斯大学最近看到越来越多的科学家、工程师、发明家和科技企业家在寻找创业的途径。理查森指出，资本不再仅仅由应用程序开发者获得，而是由那些在医疗保健和国防等领域从事新创新的人获得。

CUEED 在 2017 年的重点是帮助这些创新者申请和完成资本申请，如美国政府资助的[小企业创新研究](https://www.sbir.gov/)计划(SBIR)。“对于第一代移民企业家来说，他们缺乏朋友和家庭资助。他们甚至无法获得早期资本来度过 MVP 阶段。像 SBIR 项目、孵化器、能力建设加速器这样的项目；这不是一大笔钱，但这可以帮助公司真正成长，并为他们提供起飞所需的初始火箭燃料。”
