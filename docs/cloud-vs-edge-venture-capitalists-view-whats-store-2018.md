# 云与边缘:风险投资家对 2018 年的看法

> 原文：<https://thenewstack.io/cloud-vs-edge-venture-capitalists-view-whats-store-2018/>

[云 vs. Edge 以及一位风投对 2018](https://thenewstack.simplecast.com/episodes/cloud-vs-edge-and-a-vcs-view-on-whats-in-store-for-2018) 的看法

在这一集的《新烟囱制造者》中，我们采访了[史蒂夫·赫罗德](http://generalcatalyst.com/team/dr-steve-herrod/)，他是 [General Catalyst](http://generalcatalyst.com/) 的风险投资家，专注于基础设施建设。他最近在 LinkedIn 上发表了一篇名为“[Cloud vs . Edge——这不是笼子里的比赛](https://www.linkedin.com/pulse/cloud-vs-edge-cage-match-steve-herrod/)”的文章

“这归结于用例，”Herrod 谈到云和边缘计算之间的差异，“你不希望汽车的物联网芯片在决定转向之前必须进入云端，但进入云端处理来自数以千计的物联网设备的传感器数据会更快。”

请收听现有公司如何像大象一样，Herrod 在哪里为他的初创公司寻找工程师，对希望更新其堆栈的现有公司的建议，他认为 2018 年的趋势是什么，以及关于无服务器的哪两件事让他着迷。
