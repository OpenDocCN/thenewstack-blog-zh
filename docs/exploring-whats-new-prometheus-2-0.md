# 普罗米修斯 2.0 的新功能

> 原文：<https://thenewstack.io/exploring-whats-new-prometheus-2-0/>

[#150:探索普罗米修斯 2.0](https://thenewstack.simplecast.com/episodes/150-exploring-whats-new-in-prometheus-2-0) 的新功能

在最新一集的 New Stack Analysts 播客中，我们采访了 Robust Perception 创始人 [Brian Brazil](https://www.linkedin.com/in/brianbrazil) 和 [Frederic Branczyk](https://www.linkedin.com/in/frederic-branczyk-3b88299b/) ，CoreOS 的工程师，以获得关于最近发布的 [Prometheus](https://prometheus.io/) 2.0 的最新见解。随着存储领域的新发展和新时序数据库(TSDB)的加入，由[云本地计算基金会](https://www.cncf.io/)管理的 Prometheus 2.0 为用户带来了即时备份等新功能。

“真正有帮助的是，我们现在完全控制了存储，”巴西说。“以前，我们的一些事情依赖于 level db[……]，而现在，因为我们完全控制了代码，我们可以说‘好了，停止编写一点，转储当前状态的快照。真正有帮助的是我们自己必须控制代码，并且完全了解它如何工作的语义，以便产生正确的备份。"

Branczyk 继续解释说，随着其用例的不断发展，Prometheus 仍然易于使用，这是其在生产环境中使用的众多优势之一。“人们喜爱普罗米修斯的原因之一是它的简单。这是一个单一的静态构建二进制文件，你只需把它放在服务器上，运行它，它就会工作。所以这是用户真正欣赏的东西。”
