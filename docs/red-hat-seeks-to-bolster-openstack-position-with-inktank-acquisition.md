# Red Hat 寻求通过收购 Inktank 巩固 OpenStack 的地位

> 原文：<https://thenewstack.io/red-hat-seeks-to-bolster-openstack-position-with-inktank-acquisition/>

编者按:Red Hat Storage group 是咨询公司 Cloud Pulse Strategies 的客户，该公司由 Sam Charrington 所有。啊

红帽宣布将收购广受欢迎的 Ceph 存储系统提供商 [Inktank](http://www.inktank.com/) ，这一举动令许多人瞠目结舌。红帽将支付约 1.75 亿美元现金。

此举距离 OpenStack 社区半年一度的峰会不到两周，旨在巩固该公司在该市场的地位。Ceph 被称为 OpenStack 的[事实上的存储后端，是一个非常一致的分布式存储系统，提供块和对象存储。](https://www.openstack.org/summit/openstack-summit-hong-kong-2013/session-videos/presentation/ceph-the-de-facto-storage-backend-for-openstack)

Ceph 在 OpenStack 领域与该项目的原生对象存储 Swift 以及 Red Hat 自己的 Red Hat 存储系统(RHSS)展开竞争，后者是该公司在 2011 年秋季以 1.36 亿美元收购 Gluster 后获得的。RHSS 也是一个支持对象和块语义的分布式存储系统，它一直在 OpenStack 社区中争夺关注，但却是这个游戏的后来者。Red Hat 存储团队对 Ceph 的最新挑战是在去年秋季的 OpenStack 峰会上发起的一项活动，该活动强调了[相比 Ceph](http://redhatstorage.redhat.com/2013/11/07/red-hat-storage-outperforms-ceph-by-more-than-3x-for-openstack-cloud-environments/) 的 3 倍性能优势。

事实上，Red Hat“已经拥有了其中的一个”,这凸显了该公司对 OpenStack 的承诺，它将 Open stack 视为其所谓的开放混合云的基础技术。该公司希望，通过在企业云计算领域占据积极地位，它可以避免威胁其核心操作系统业务的无关性，因为云和虚拟化等技术使用户越来越远离操作系统。

托管公司 Dreamhost 的子公司 Ceph 获得了来自 Canonical 创始人马克·舒托沃尔斯的第一笔 100 万美元投资，Canonical 是另一家基于 Ubuntu Linux 争夺 OpenStack 主导地位的公司。该公司 1440 万美元投资的其余部分来自新梦想网络，可能是一群 Inktank 和 Dreamhost 内部人士。

红帽首席技术官布莱恩·史蒂文斯和墨水罐创始人兼首席技术官塞奇·韦尔将在今天晚些时候的网络直播中分享更多交易背后的想法。我们会继续关注。
