# 谷歌、脸书、优步加入 Linux 基金会的 OpenChain 项目

> 原文：<https://thenewstack.io/google-facebook-uber-join-linux-foundations-openchain-project/>

谷歌、脸书和优步正在加入 Linux 基金会的 [OpenChain](https://www.openchainproject.org/) 开源许可项目，取代其他 13 家公司，包括 GitHub、Adobe、思科、Comcast 和其他公司，该基金会本周在横滨举行的[开放合规峰会](https://events.linuxfoundation.org/events/open-compliance-summit-2018/)上宣布。

作为白金会员，这些公司将获得 OpenChain 管理委员会的席位，在该项目迈向正式成为行业标准的时候提供额外的见解。

OpenChain 提供了一个标准，以确保公司遵守其供应链中众多不同的开源许可证。根据 OpenChain 总经理 [Shane Coughlan](https://www.linkedin.com/in/shanecoughlan/) 的说法，OpenChain 不是通过一种指令性的方法，而是通过一种经济的方法来做到这一点，在这种方法中，客户选择能够证明其合规性的供应商。考夫兰说，如果 OpenChain 过于规范，它将无法满足需求，并且该标准将不适用于不同规模和行业的公司。

“OpenChain 是为每个市场中任何规模的公司设计的，”Coughlan 说。“这个想法是，客户可以对供应商说，‘我看到你是合规的，你能给我看看你的合规工件吗？’从这个意义上来说，作为一个标准，OpenChain 通过供应链本身的经济性拥有了它的听觉能力。它允许不同规模的公司以适合其市场和情况的方式解决开源合规性问题。OpenChain 确定了关键需求，但为公司确定细节留下了足够的灵活性。"

新公司将进一步增加其当前董事会的多样性，董事会成员已经横跨各个行业，如丰田、西门子、索尼和东芝。

“你会注意到 Adobe 是我们唯一的云优先公司，直到今天。“OpenChain 旨在适用于所有市场领域，这为我们提供了多个领域和市场的大量知识，”考夫兰说。“脸书、谷歌和优步是在标准化方面拥有丰富经验的组织，我们将进入下一个阶段，进一步发展 OpenChain。”

除了扩大该项目的董事会，脸书、谷歌和优步还帮助扩大 OpenChain 在开源世界的影响力，因为这三家公司都是开源社区的广泛参与者，并运营着一些由大量开源软件和硬件支持的世界上最大的数据中心、平台和云基础设施。

在宣布这些新成员的同时，该项目还宣布了其与 Linux 基金会的[开放兼容计划](https://compliance.linuxfoundation.org/)，该计划作为 Linux 基金会项目的门户，为寻求以兼容方式使用开源软件的公司和个人提供参考资料和帮助。类似地，OpenChain 提供了一个关于开源合规的培训和参考材料的[课程](https://www.openchainproject.org/curriculum)，以及一个免费的在线[自我认证](https://www.openchainproject.org/conformance)。

最后，考夫兰解释说，开放链规范“在概念上类似于 ISO 9000 等标准”，并保证公司有适当的流程，但不是这些流程采取的具体形式。他说，事实上，今天的大部分公告都是该项目在 2019 年成为正式标准的前奏，下一次更新将于 4 月份进行。

Linux 基金会是新堆栈的赞助商。
