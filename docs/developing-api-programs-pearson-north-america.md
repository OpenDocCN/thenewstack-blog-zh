# 开发 API 程序:培生北美公司

> 原文：<https://thenewstack.io/developing-api-programs-pearson-north-america/>

Pearson North America 一直在从多个传统平台过渡到单一的开放式 API 平台。内部软件工程团队使用 Pearson 的 API 来构建同类最佳的学习应用。因为致力于学习，培生的 API 也是公开的，允许外部开发者基于他们的平台创建学习应用。

**Alex Williams** 来自 *The New Stack* 在参加 [Apigee 的](http://apigee.com) [我爱 API](http://iloveapis2014.com/)2014 年旧金山会议时，采访了**培生的开发者网络和 API 项目总监 Allen Rodgers** 。他们讨论了 API 的未来，以及 Pearson 不断发展的产品平台将如何支持一系列体验平台。培生已经积累了大量关于其资产表现的数据，未来的挑战将包括将其最有效的产品与 API 开发相匹配。Pearson 也意识到需要平衡消费者的长期需求和快速发展的 API 技术，以保持产品的稳定性。
