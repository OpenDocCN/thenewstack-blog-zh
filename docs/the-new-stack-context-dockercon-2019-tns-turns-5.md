# 新的堆栈上下文:DockerCon 2019，TNS Turns 5

> 原文：<https://thenewstack.io/the-new-stack-context-dockercon-2019-tns-turns-5/>

[DockerCon 2019，TNS 转 5](https://thenewstack.simplecast.com/episodes/dockercon-2019-tns-turns-5)

欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)播客。本周的节目由《新书库》的编辑主任莉比·克拉克主持。今天加入我们的还有 Alex Williams，The New Stack 的创始人和发行人，以及 Joab Jackson，TNS 的执行主编。

在这一集里，我们来回顾一下本周在旧金山举行的 Docker 旗舰用户大会 [DockerCon 2019](https://www.docker.com/dockercon/) 。本周的大新闻是 Docker Enterprise 3.0 的发布，这是唯一一个用于构建、共享和运行基于容器的应用程序的端到端平台。这个完全集成的包包括一个开发人员的桌面界面和用于管理整个应用程序生命周期每个阶段的所有工具，无论是在云中还是在防火墙后面。Docker 为任何应用程序(从传统应用程序到棕色地带到绿色地带)提供最快的生产时间，并能够在任何地方安全地运行它们(从混合云到边缘)。

然后，在节目的稍后部分，我们[庆祝新堆栈](https://thenewstack.io/the-new-stack-turns-five-heralding-a-new-era-of-tech-journalism/)的五周年纪念日，威廉姆斯沿着记忆之路重新讲述这个非常网站的起源故事，一个涉及码头工人的起源故事！

播客中提到的 Pulumi 和 Puppet 是新堆栈的赞助商。
