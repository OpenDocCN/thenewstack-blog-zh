# Gupshup 以及为什么您的企业需要聊天机器人平台

> 原文：<https://thenewstack.io/expanding-chatbot-usecases-gupshup/>

在本期[新堆栈制造商](https://thenewstack.io/podcasts/makers)播客中，我们将深入探讨聊天机器人的新兴世界和对话界面。本月早些时候在奥斯丁举行的博特尼斯企业会议上，我们采访了一家名为 [Gupshup](https://www.gupshup.io/developer/home) 的公司的首席执行官[比鲁德·谢思](https://twitter.com/beerud)。“Gupshup”在印度语中是闲聊的意思，Gupshup 称其服务是世界上最先进的机器人和信息平台。它使开发人员能够快速轻松地构建、测试、部署和管理所有消息传递渠道中的聊天机器人。我们想了解更多关于如何使用 GupShup 运行聊天机器人的信息，以及该公司对技术发展方向的看法。
