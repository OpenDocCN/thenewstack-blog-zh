# Capital One 如何利用 API 简化汽车融资

> 原文：<https://thenewstack.io/capital-one-using-apis-streamline-auto-financing/>

Capital One 像对待产品一样对待 API。其与轰鸣合作的新汽车融资 API 允许轰鸣客户在线预先获得汽车融资资格，而不是必须亲自与某人一起完成这一过程。轰鸣目前正在利用该 API 作为他们汽车购买流程的一部分。这是单一合作伙伴集成。

API 使用 Amazon Web Services (AWS)上的 Tomcat“Dockerized”容器运行，具有蓝/绿部署，通过 API 网关进行调节，允许速率限制、节流、计量和其他功能。这是一个按钮式 DevOps 自动化，设计简单。目标是:使用复杂的后端架构为最终用户创建简单的体验。

在 SXSW 的这个赞助播客中，我们采访了 Capital One 负责汽车金融的副总裁 [Shishir Singhania](https://www.linkedin.com/in/shishir-singhania-1ab5212/) 和 Capital One Tech 的副总裁 [Ray Bajaj](https://www.linkedin.com/in/raman-ray-bajaj-2111a35/) ，讨论了构建这个 API 和通过使用 API 扩展 Capital One 的服务所带来的挑战和成功。
