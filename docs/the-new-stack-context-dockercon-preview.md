# 新的堆栈上下文:DockerCon 预览

> 原文：<https://thenewstack.io/the-new-stack-context-dockercon-preview/>

[新栈上下文:DockerCon 预览](https://thenewstack.simplecast.com/episodes/the-new-stack-context-dockercon-preview)

欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)播客。今天我们采访的是 [Joe Duffy](https://www.linkedin.com/in/joejduffy) ，他是 [Pulumi](https://www.pulumi.com/) 的联合创始人兼首席执行官，他和我们谈论了是什么推动了他的公司的发展，以及为什么他们会在下周出席 [DockerCon](https://www.docker.com/dockercon/) 。今天加入我们的还有 Alex Williams，The New Stack 的创始人和发行人，以及 Joab Jackson，TNS 的执行主编。我是《新书库》的编辑主任莉比·克拉克。

Pulumi 是新 Stack 的赞助商，总部设在西雅图——离波特兰的 TNS 总部不远！—我们很高兴在下周旧金山的 DockerCon 之前邀请到 Joe 参加今天的节目。基于基础设施即代码的理念， [Pulumi](https://www.pulumi.com/) 致力于使开发人员和 DevOps 团队能够使用流行的编程语言和工具来部署到任何公共或私有基础设施，以及跨基础设施部署。联合创始人达菲和埃里克·鲁德尔来自微软，首席技术官卢克·霍班来自亚马逊。

然后在下半部分，我们将讨论本周的热门播客和故事，并讨论下周我们将在 DockerCon 上讨论的内容。我们将在现场，为我们新推出的 [@Scale](/podcasts/at-scale) 播客的新一集录制采访，并在展示厅与公司谈论容器管理工具和应用程序架构的最新进展。
