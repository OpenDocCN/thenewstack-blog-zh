# AWS 和 Mendix 合作开发智能自动化解决方案

> 原文：<https://thenewstack.io/aws-and-mendix-partner-on-intelligent-automation-solutions/>

亚马逊网络服务公司(Amazon Web Services)正在与低代码提供商 T2 合作，以简化其智能自动化解决方案，这是一种融合了人工智能和人工智能等技术的自动化。从 Mendix 的角度来看，它希望将支持亚马逊的组件(如语音到文本)引入其低代码解决方案。但这也意味着客户可以通过 Mendix 或 AWS 市场购买应用和服务组件。

“智能自动化”是一个新的理念，可以帮助创建解决方案，带来更高水平的效率和更可靠的业务运营。为了将智能自动化构建到您当前的工作流程中，您需要关注三种不同的技术:

*   人工智能:这是智能自动化的核心。你需要利用人工智能和机器学习，以及能够分析来自众多来源的海量数据的复杂算法。
*   业务流程管理:这是一个使用各种方法来改进、优化和自动化业务工作流的范例。在智能自动化领域，BPM 用于为您的业务流程提供更多的灵活性和一致性，并简化交互和参与。
*   机器人流程自动化(RPA):这是自动化和机器人技术的实现，专注于执行后台任务(例如从大量来源提取数据)。

智能自动化提供了许多重要优势，例如降低成本、提高决策和重复性任务的准确性、改善客户体验以及保持企业合规性。任何需要分析历史和实时工作负载和计算数据的任务都是智能自动化应用的完美候选。

## 谁受益？

但是哪些行业可以从智能自动化的应用中受益呢？虽然信息架构可以应用于几乎所有类型的业务，但某些行业可能会发现应用这种技术会立即受益。任何消除浪费和管理非常精确的仪器仪表的行业都会发现 IA 非常有效。汽车、生命科学、医疗保健和保险等行业显然是智能自动化的目标。

我和技术联盟的 Mendix 主管 David Bailey 坐在一起，讨论这种合作关系。一开始，贝利就明确表示，该公司知道他们擅长低代码，但意识到这涉及到许多其他技术，如人工智能和机器学习。由于 Mendix 没有能力做到所有这些，为了保持自我，他们需要利用其他人和公司将更高层次的技术和服务融入其中。

“有许多其他技术正在我们周围发展，如机器学习、数据和集成、人工智能和用户体验，”贝利说。“我们不可能做所有的事情。我们并不真的想做所有的事情。我们想成为我们自己。所以我们需要能够利用我们周围的人来帮助我们。当我们看 AWS 时，考虑到我们与他们的合作，以及我们的母公司西门子与他们的合作，我们意识到现在实际上有一个非常强大的联盟。”

Mendix 在保险、银行和制造业有许多客户。但亚马逊处于完全不同的水平，拥有成千上万的客户。

“他们也有大量的数据，”贝利指出。“他们正在帮助我们大规模了解这些客户问题，然后能够与我们合作。”

从 2022 年 2 月 15 日开始，Mendix 和 AWS 的客户都可以从任何一个市场获得 Mendix 的许可，客户可以在那里找到整个平台。对于那些不熟悉低代码的人来说，Mendix 确实提供了支持，帮助企业快速使用这些工具，以便他们最终能够完全自给自足。
