# AWS 开源战略

> 原文：<https://thenewstack.io/the-aws-open-source-strategy/>

没有开源，亚马逊网络服务就不会有今天。

[AWS 开源策略](https://thenewstack.simplecast.com/episodes/the-aws-open-source-strategy)

“我认为这从可持续性开始，”AWS 开源战略和营销负责人 David Nalley 在都柏林举行的新堆栈制造商开源峰会上接受采访时说。“这真的可以追溯到亚马逊网络服务的起源。没有开源，AWS 就不会有今天。”

对开源的长期支持是该组织开源战略的三大支柱之一。AWS 在开源的基础上构建和创新，并将为其创新、客户和更大的数字经济保持这种方法。

Nalley 说:“这意味着我们从开源中受益和投资开源有很长的历史。“但最终，我们会长期坚持下去。我们将继续投资。我们将增加对开源软件的投资。”

客户对开源的兴趣是 AWS 开源战略的第二个支柱。

“我们觉得我们必须代表我们的客户进行投资，”Nalley 说。“但事实是，我们的客户选择开源来运行他们的工作负载。”

第三个支柱侧重于在更大的数字经济中倡导开源。

值得注意的是，AWS 在市场中的存在在保罗·维谢决定加入该公司时起到了多大的作用。互联网先驱 Vixie 现在是安全副总裁，也是 AWS 杰出的工程师，他还在开源峰会上接受了 [New Stack Makers 播客的采访。](https://thenewstack.io/paul-vixie-story-of-an-internet-hero/)

纳利在社区中有他公认的重要性。Nalley 是 Apache 软件基金会的主席，这是世界上最重要的开源基金会之一。

其三大支柱战略的重要性体现在 AWS 支持的许多项目中。AWS 最近向作为 Linux 基金会一部分的[开源软件供应链基金会](https://openssf.org/) (OpenSSF)捐赠了 1000 万美元。

AWS 是 [Rust 基金会](https://foundation.rust-lang.org/)的重要支持者，该基金会支持 [Rust 编程语言](https://thenewstack.io/rust-whats-next-for-the-fast-growing-programming-language/)和生态系统。它特别关注管理项目的维护者。

上个月，脸书公布了 Linux 基金会将管理的 PyTorch 基金会。AWS 是理事会成员。
