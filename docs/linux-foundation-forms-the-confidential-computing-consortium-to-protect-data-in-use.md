# Linux 基金会成立了保密计算联盟来保护使用中的数据

> 原文：<https://thenewstack.io/linux-foundation-forms-the-confidential-computing-consortium-to-protect-data-in-use/>

谈到数据安全，有两种应用加密的标准状态—静态和动态。然而，当数据被使用时，它通常是不加密的。[Linux 基金会](https://www.linuxfoundation.org/)已经组织了[保密计算联盟](https://confidentialcomputing.io/)来定义和加速加密数据的使用，或“保密计算”

该联盟将由硬件供应商、云提供商(包括阿里巴巴、Arm、百度、谷歌云、IBM、英特尔、微软、红帽、瑞士电信和腾讯)以及开发人员和学者组成。该组织计划帮助推动机密计算市场，以及权衡技术和监管标准，并为可信执行环境(TEE)开发开发工具开源工具。

除了众多公司加入该财团外，还有几家公司也捐赠了项目。英特尔捐赠了[软件卫士扩展 SDK](https://01.org/intel-softwareguard-extensions) ，该软件旨在“使用受保护的飞地保护特定代码和数据在硬件层不被泄露或修改。”微软捐赠了 [Open Enclave SDK](https://openenclave.io/) ，这是一个帮助开发者构建 TEE 应用的开源框架。而红帽捐赠的 [Enarx](http://enarx.io/) ，为 TEEs 提供了一个平台抽象。微软杰出的工程师 John Gossman 说，所有这些都是为了一个共同的目标，即让开发者更容易接触到 t 恤——这是机密计算联盟的一个关键功能焦点。

“这是一项相当早期的技术。芯片组本身仍在不断涌现，下一代将更大更快，这将使构建应用程序变得更容易，但因为它还处于早期，我们往往不知道最佳实践，”戈斯曼说。“是指令集，所以真的很低级。它不是普通开发人员可以轻松访问的东西，因此 Open Enclave SDK 和其他由各种供应商提供的代码都是为了让应用程序开发人员能够广泛访问这种非常低级的非常硬的技术，以便创建一个应用程序和平台的生态系统。”

对于 Linux 基金会的执行董事吉姆·泽姆林(Jim Zemlin)来说，实现这种技术解决方案的唯一途径是让所有这些不同的实体聚集在一个地方。

Zemlin 在接受 New Stack 采访时说:“每次你有一个重大举措来推进如此复杂的技术，行业都会意识到这需要一个完整的生态系统。”“你需要硬件供应商与软件供应商合作，与云提供商合作，与学术界合作，来加速这样的事情。对于所有这些人来说，走到一起并为之努力本身就是一项成就，但这只是让我们站在了起跑线上。”

“这里的关键是，你有一个跨越技术供应链不同组成部分的广泛的利益相关者集合在一起。本质上，最终目标是通过保护使用中的数据来实现更好的数据安全性，”Zemlin 说。要做到这一点，你需要让你用来创造这些环境的技术变得易于使用，易于理解，并以一种标准化的方式构建，这就是为什么整个行业像这样走到一起是非常重要的。"

该联盟是一个基金会，目前正在组建各种委员会，其中包括一个管理委员会，一个技术咨询委员会，以及一个针对每个项目的独立技术监督委员会，项目将在麻省理工学院开源许可证下提供。该联盟将通过会费提供资金，并将接受会员申请。

本周在圣地亚哥举行的[北美开源峰会](https://events.linuxfoundation.org/events/open-source-summit-north-america-2019/)上，Linux 基金会公布了这个新的联盟。
