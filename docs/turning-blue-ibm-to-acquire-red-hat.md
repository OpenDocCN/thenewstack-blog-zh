# 变蓝:IBM 收购红帽

> 原文：<https://thenewstack.io/turning-blue-ibm-to-acquire-red-hat/>

为了成为世界领先的开源云软件提供商，IBM 周日宣布将收购 Red Hat，交易价值约 334 亿美元。

Red Hat 一直是开源软件赚钱的典范，它为企业 Linux 提供集成服务，建立合作伙伴关系，如与前对手微软的联盟，以及最近对 T2 容器和 Kubernetes 的推动。

在发布[公告](https://newsroom.ibm.com/2018-10-28-IBM-To-Acquire-Red-Hat-Completely-Changing-The-Cloud-Landscape-And-Becoming-Worlds-1-Hybrid-Cloud-Provider)时，IBM 首席执行官 Ginni Rometty 说:“今天大多数公司只完成了 20%的云之旅，租用计算能力来削减成本。接下来的 80%是释放真正的商业价值和推动增长。这是云的下一章。这需要将业务应用转移到混合云，提取更多数据并优化业务的每个部分，从供应链到销售。”

IBM 承诺继续 Red Hat 对开放治理和参与开源社区的承诺。

根据公告，红帽将加入 IBM 的混合云团队，作为一个独立的单位，拥有其当前的产品组合，走向市场的战略和独特的开发文化。红帽公司首席执行官吉姆怀特赫斯特将继续领导他目前的管理团队。

怀特赫斯特在声明中说:“开源是现代 IT 解决方案的默认选择，我为红帽在企业中实现这一点所发挥的作用感到无比自豪。”。“与 IBM 联手将为我们提供更大的规模、资源和能力，以加速开源作为数字化转型基础的影响，并将 Red Hat 带给更广泛的受众，同时保留我们独特的文化和对开源创新的坚定承诺。”

它还将继续建立合作伙伴关系，如亚马逊网络服务、微软 Azure、谷歌云、阿里巴巴等。

IBM 和 Red Hat 已经合作了 20 年，在企业级 Linux 上进行合作，最近还为客户带来了企业 Kubernetes 和混合云解决方案。这些创新已经成为 IBM 价值 190 亿美元的混合云业务的核心技术。两者都是主要的开源贡献者。

蓝色巨人将以每股 190 美元的价格收购这家总部位于北卡罗来纳州罗利的软件公司，比红帽公司周五的收盘价高出 60%以上。

Red Hat 在 1 月[收购了 CoreOS](https://thenewstack.io/red-hat-will-acquire-coreos-greater-kubernetes-presence/) ，这是一个竞争对手，提供精简的、以容器为中心的 Linux 发行版。五月初，在哥本哈根的 [KubeCon + CloudNativeCon，](https://events.linuxfoundation.org/events/kubecon-cloudnativecon-europe-2018/)[Red Hat](https://www.openshift.com/)s[Brandon Philips，](https://www.linkedin.com/in/brandonphilips/)在 [The New Stack Makers](https://thenewstack.io/all-things-enterprise-cloud-native-at-red-hat/) 的一集中，讲述了合并 CoreOS 和 Red Hat 团队的感受。

在收购 CoreOS 之前,[收购了 CodeEnvy](https://thenewstack.io/red-hat-acquire-codenvy-integrate-ide-openshift-io/) ，CodeEnvy 是第一个全功能协作式基于云的集成开发环境的制造商。红帽已经将 CodeEnvy 的 IDE [Eclipse Che](https://www.eclipse.org/che/) 集成到了 [OpenShift](https://openshift.io) 中，这是红帽历时数年开发的平台。OpenShift 的核心是一种基于容器的技术，与用于多云部署的编排技术平台 Kubernetes 深度集成。Red Hat 将其 OpenShift 描述为一个用于规划、创建和部署混合云服务的在线开发环境，这表明了 IBM 在收购公告中看到的与 Red Hat 的契合。

CodeEnvy 的首席执行官泰勒·朱厄尔现在是 WSO2 的首席执行官。

红帽公司和 WSO2 是新堆栈的赞助商。
