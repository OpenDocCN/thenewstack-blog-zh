# 描述微服务采用的状态

> 原文：<https://thenewstack.io/characterizing-the-state-of-microservices-adoption/>

今天，在新一期的 Stack Analysts 中，我们将我最近对丹尼尔·布莱恩特和 T2·马特·乔丁的采访组合在一起。布莱恩特是新一批人期待他的深刻分析的人之一。他是一名独立顾问、演说家和作家。Chotin 是一位备受尊敬的技术专家，在 [AppDynamics](https://www.appdynamics.com/) 担任开发人员计划高级总监，负责领导 AppDynamics“产品更好地支持开发人员提高性能和实现业务成果”我们本期播客的讨论主题是:在讨论微服务时，你会如何描述市场的现状？

[表征微服务采用状态](https://thenewstack.simplecast.com/episodes/characterizing-the-state-of-microservices-adoption)

随着金融投资的增长，对容器的开发兴趣也在增长。我们首先与 Bryant 讨论了市场状况，然后 Chotin 谈到了正在开发微服务的用户的整体成熟度。

更多的微服务意味着在开发、部署和管理的所有阶段都有更多的容器。这对开发人员工具社区来说是个好消息。人们正在构建基于 DevOps 实践的连续交付模型，该模型在任何可以想到的基础设施上交付。正是这些 DevOps 实践将在很大程度上保持文化性，因为董事会专注于数字化转型，并使用这种范式作为长期支出的基础。

业务驱动着应用程序架构转换的方向。在这个关于新堆栈分析师的两部分采访中，我们讨论的基础是市场在这个转型弧中的位置。
