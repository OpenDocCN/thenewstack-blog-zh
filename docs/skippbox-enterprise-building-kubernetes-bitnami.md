# 收购 Skippbox 为 Bitnami 带来了 Kubernetes 的专业知识

> 原文：<https://thenewstack.io/skippbox-enterprise-building-kubernetes-bitnami/>

库伯内特斯的景观已经有了轻微的改变，所以我们与改变背后的推动者进行了交谈。在这一集的[新堆栈制造商](https://thenewstack.io/podcasts/makers)播客中，我们采访了 [Sebastien Goasguen](https://twitter.com/sebgoa) ，他是[skip box](http://www.skippbox.com/about/)的创始人，这是一家专注于管理 Kubernetes 容器编排引擎的应用程序的初创公司。今天，云软件包经理 Bitnami 宣布它将收购 Skippbox，因此我们的讨论范围从收购到还包括 Sebastien 与使用 Kubernetes 的客户的经验。

在这一集里，我们将谈论 Bitnami 的收购动机，以及收购将会给他们处理 Kubernetes 生态系统带来什么变化。我们还询问了 Sebastien 关于他在 Kubernetes 之上构建的经验，包括 Kompose 和 Kubeless 等解决方案。他分享了一些客户的经历，以及他认为 Kubernetes 社区即将关注的话题:

[skip box 将基于 Kubernetes 的专业知识构建引入 Bitnami](https://thenewstack.simplecast.com/episodes/skippbox-brings-expertise-building-on-top-of-kubernetes-to-bitnami)
