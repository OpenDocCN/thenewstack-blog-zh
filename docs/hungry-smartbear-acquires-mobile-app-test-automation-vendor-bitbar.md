# 饿了么智能熊收购移动应用测试自动化厂商 Bitbar

> 原文：<https://thenewstack.io/hungry-smartbear-acquires-mobile-app-test-automation-vendor-bitbar/>

测试自动化厂商 [SmartBear](https://smartbear.com/) ，表现出对收购的兴趣，通过增加移动应用测试工具初创公司 [Bitbar](http://bitbar.com/) 再次做到了这一点。

总部位于芬兰的 Bitbar 声称提供了世界上最大的基于云的应用测试设备集合，能够并行测试，并且跨原生测试自动化框架和流行语言(如 Java、Ruby、Python、JavaScript 等)不可知。它有公共云、私有云和内部产品。

[smart bear 产品副总裁 Anand Sundaram](https://smartbear.com/blog/test-and-monitor/just-announced-smartbear-acquires-bitbar/) 在一篇[博客文章](https://smartbear.com/blog/test-and-monitor/just-announced-smartbear-acquires-bitbar/)中解释说，大约三年前收购了 [CrossBrowserTesting](https://crossbrowsertesting.com/) (CBT)，它为功能测试客户提供了跨浏览器、版本、操作系统和设备类型测试 web 应用程序的能力，使用按需设备云进行实时测试和自动化。

Sundaram 说，像 Mozilla、WeWork、CBS Interactive 和 Square 这样的大型供应商已经在 Bitbar 上进行了大量投资。

几个月前，SmartBear 购买了商业版的 Cucumber，这是一个专注于行为驱动开发(BDD)的开源项目。它在 2018 年收购了测试管理软件[泽法](https://www.getzephyr.com/)和 [HipTest](https://hiptest.com/) ，这是一个 BDD 的合作平台。

[SmartBear 本身](https://xconomy.com/boston/2017/05/03/source-smartbear-software-sold-to-francisco-partners-for-410m/)于 2017 年被私募股权公司 Francisco Partners 收购。

Gartner 在其 2018 年软件测试自动化魔力象限报告[中将 SmartBear 评为领导者之一，并称赞该公司在 UI 层和 API 层创建功能测试的工具。](https://smartbear.com/resources/white-papers/gartner-magic-quadrant-2018/)

ReadyAPI 为 API 层提供了功能、性能和安全测试能力。Hiptest 增加了对 BDD 的支持，并提供了生成和同步自然语言产品文档的能力。它指出，交叉浏览器测试云提供了在云中大规模运行基于硒和 Appium 的测试的能力。

它还提到了该公司在开源社区的积极参与。

它指出，SmartBear 已经开始通过增加人工智能和机器学习能力来解决自动测试设计和测试用例生成方面的限制。

SmartBear 与 Tricentis、T2、茄子和 T4、Parasoft 等公司竞争，在报告中的表现优于微软、IBM 和 CA Technologies 等大公司。有趣的是，Bitbar 一直在为 IBM 提供移动测试服务。

几周前，SmartBear 发布了用于在持续集成/持续交付(CI/CD)环境中自动化测试执行的 [TestEngine](https://smartbear.com/product/ready-api/testengine/overview/) ，以及 ReadyAPI 版本。

TestEngine 使用户能够自动化 SoapUI 和 ReadyAPI 测试，并同时执行它们。它可以安装在任何云或内部环境中，并与集中式或分布式模型一起使用。

ReadyAPI 2.8 使得在 SoapUI 中创建数据驱动测试变得更加容易，并提高了与泽法和 Azure DevOps 的集成。
