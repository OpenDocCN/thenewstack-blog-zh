# TrumpScript 希望让 Python 再次变得伟大

> 原文：<https://thenewstack.io/trumpscript-satirical-programming-language-will-make-python-great/>

一个大学程序员团队本周发布了一种新的编程语言，模仿房地产企业家和美国总统候选人唐纳德·特朗普，引起了轰动。

“我们发现，当前的编程语言领域不包括任何特朗普的金冠会认可的语言，”他们在 TrumpsScript 的官方网站[上开玩笑说。](https://devpost.com/software/trumpscript)

他们的大多数笑话都融入了他们语言的语法。

例如，TrumpScript 不接受浮点数，只接受整数，因为“美国从不半途而废。”

所有数字必须大于 100 万。"这些小事对我们来说无关紧要。"

为了认可特朗普在移民问题上的立场，该语言不允许任何进口声明。“所有代码都必须是本土的、美国制造的。”

该语言的大多数错误信息都是引用唐纳德本人的话。这种语言的官方网站警告说“很多时候你的代码会失败，而且会悄无声息地失败……”

他们的理由？“特朗普不喜欢谈论他的失败……”

可以说，TrumpScript 是基于 Python 的。“丹和我试图找出 Python 的抽象语法树类，他试图破坏它，只有一个流利的 TrumpScript 用户才能做到，”[萨姆·沙德威尔](https://github.com/samshadwell)在网站上解释道。

在特朗普竞选口号的一个片段中，该语言的口号是“让 Python 再次伟大”

TrumpScript 是为莱斯大学一年一度的[黑客大会](http://hack.rice.edu/)创作的，这是一个“为夜猫子和梦想家”而设的编码活动上周末，从周五晚上到周日，参与者在休斯顿学院的学生中心通宵工作，以不超过五人的团队工作，试图从零开始创建一个获奖的编码项目。该活动吸引了 200 多名参与者，今年的许多赞助商包括谷歌、微软和脸书，他们也派代表参加了该活动的 13 名评委。

四名程序员从事 TrumpScript 的工作，在周一，其中一人有了将它[提交给*黑客新闻*](https://news.ycombinator.com/item?id=10924416) 的想法。很快，这个链接吸引了近 100 张选票——在 Reddit 上获得了更多关注。

“我很惊讶这得到了如此多的关注，”程序员 Sam Shadwell [告诉 Inverse](https://www.inverse.com/article/10448-coders-assimilate-donald-trump-to-a-programming-language) ，分享了他对他们开发过程的回忆。“实际上，我们第一天晚上回去喝醉了……没有写任何代码。

“但是，我们集思广益……”
