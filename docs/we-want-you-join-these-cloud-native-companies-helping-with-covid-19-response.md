# 我们希望您:加入这些云原生公司，帮助新冠肺炎响应

> 原文：<https://thenewstack.io/we-want-you-join-these-cloud-native-companies-helping-with-covid-19-response/>

Snyk 赞助了这篇文章。

 [彼得·麦凯

彼得是 Snyk 的首席执行官。](https://www.linkedin.com/in/pemckay/) 

新冠肺炎疫情的影响给每个人都带来了很大的不确定性。随着这一史无前例的重大事件的展开，预算、优先事项和运营政策每天都在发生巨大的变化。

受到疫情直接影响的组织最能感受到这种焦虑，无论是通过业务模式中断还是通过资源的突然重新分配，因为他们处于应对措施的第一线。与其他组织不同，这些组织所处的环境要求他们在规模和功能上做出重大转变。几周前被调整为交付一组用例的交付系统现在正面临以计划外的规模交付计划外服务的压力。

> 许多用于远程学习、电子商务、协作和在家工作的服务都依赖于微服务和云原生架构。

我们都需要这些组织能够可靠地提供服务。作为回应，许多组织需要访问我们在云原生生态系统中每天构建和部署的软件。对于其中许多组织来说，采购流程停滞不前，预算也不确定。与此同时，基础设施运营商和开发商正在尽最大努力提供客户需要的服务。

 [大卫·萨法伊

大卫是 Trilio 数据公司的首席执行官。](https://www.linkedin.com/in/safaii/) 

许多用于远程学习、电子商务、协作和在家工作的服务都依赖于微服务和云原生架构。提供这些基本服务的公司需要云计算原生社区的帮助。因此，作为回应，越来越多的向依赖云原生架构的行业提供软件的公司宣布了一些计划，通过这些计划，响应新冠肺炎疫情的组织可以免费获得所需的软件。

每家公司的细节略有不同(例如，我们的公司 Snyk 和 Trilio 将专注于医疗保健、娱乐、酒店和旅游)，但共同点是云原生软件公司正在携手消除许可费，并简化收购和部署流程，以将技术推向前线。

在一段时间内免费提供我们的软件和专业知识是我们都可以做的事情，以帮助那些帮助我们应对疫情的人。我们已经开始[一份云原生软件供应商的名单](https://docs.google.com/document/d/14JoIh9RbU-2HNATZPX1f1YQyW1No5Ht0wNL3I0GaZjI/)，在疫情回应期内向受影响的行业提供免费许可。

如果你的公司有这样一个项目，并且你想上市，**请将你的信息添加到[这个谷歌文档](https://docs.google.com/document/d/14JoIh9RbU-2HNATZPX1f1YQyW1No5Ht0wNL3I0GaZjI/)。**

我们云原生社区的所有人都应该尽我们所能。我们呼吁社区中的每一个人清除障碍，让有用的软件更快地到达那些构建关键支持服务的人手中；通过将您的免费软件和服务添加到此列表中，帮助加速这些服务并支持那些受疫情影响最严重的人。谢谢你。
