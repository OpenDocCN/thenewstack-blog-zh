# 英特尔衡量开源社区多样性和包容性的方法

> 原文：<https://thenewstack.io/intels-approach-measuring-diversity-and-inclusivity-in-open-source-communities/>

[英特尔如何在开源社区培养多样性](https://thenewstack.simplecast.com/episodes/how-intel-is-fostering-diversity-in-open-source-communities)

在本期 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客中，TNS 创始人 [Alex Williams](https://twitter.com/alexwilliams) 与英特尔高级全球营销经理兼开源倡导者 [Nicole Huesman](https://www.linkedin.com/in/nicolehuesman) 坐在一起，讨论公司鼓励多元化的流程。

休斯曼指出，英特尔一直在与软件分析软件提供商 [Bitergia](https://bitergia.com/) 合作进行性别多样性研究。但她解释说，对于英特尔来说，这只是冰山一角:“作为社区的一员，我们看了看(并问)我们如何工作，我们可以在建立包容性多元化和跨社区包容性方面发挥什么作用？”

休斯曼还强调，在今年 3 月的开源领袖峰会上，英特尔还与 Linux 基金会合作开展了 [CHAOSS 项目](https://www.linuxfoundation.org/blog/chaoss-project-creates-tools-to-analyze-software-development-and-measure-open-source-community-health/)，这是一套旨在衡量开源项目社区健康状况的工具。
