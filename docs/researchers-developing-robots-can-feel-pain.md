# 研究人员正在开发能感觉你疼痛的机器人

> 原文：<https://thenewstack.io/researchers-developing-robots-can-feel-pain/>

面对现实吧，痛苦太糟糕了。不管你是不小心踢到了脚趾还是碰到了热炉子，由此造成的伤害都不是一件令人愉快的事情。但是我们与生俱来的感受痛苦的能力，以及本能地远离给我们带来痛苦的事物的能力，保护我们免受进一步的伤害。

但是这种感觉疼痛的能力也能保护机器人吗？来自汉诺威莱布尼茨大学的两名德国研究人员认为，疼痛(或者至少是对疼痛的人工模拟)对于机器人来说是无价的，尤其是对于将与人类并肩工作的协作机器人来说。这两个人正在开发他们所谓的“人工机器人神经系统，以教会机器人如何感受疼痛”——或者更准确地说，“人工机器人神经系统(aRNS)”——这将提醒机器人潜在的危险，让它们及时做出反应，以避免损坏它们的机械或电子系统。

本月早些时候，在瑞典斯德哥尔摩举行的 [IEEE 机器人和自动化国际会议](http://www.icra2016.org/) (ICRA)上，研究人员 Johannes Kuehn 和 Sami Haddadin 教授提交了他们的[论文](http://ieeexplore.ieee.org/xpl/articleDetails.jsp?url=http%3A%2F%2Fieeexplore.ieee.org%2Fstamp%2Fstamp.jsp%3Ftp%3D%26arnumber%3D7422729%26userType%3Dinst&denyReason=-133&arnumber=7422729&productsMatched=null&userType=inst)，描述了他们的原型，其特点是配备了触觉指尖传感器的 [KUKA](http://www.kuka-robotics.com/) LWR4+机械臂，可以检测温度和压力的变化。在实验过程中，机器人手臂经历了物理和模拟碰撞，以测试其反应。通过 [IEEE Spectrum](http://spectrum.ieee.org/automaton/robotics/robotics-software/researchers-teaching-robots-to-feel-and-react-to-pain) 观看此视频中的操作:

## 感受你的痛苦

当然，[疼痛是主观的](https://en.wikipedia.org/wiki/Pain)，不容易定义。研究人员解释说，机器人感受到的所谓“疼痛”不同于该词的生物学意义:“机器人需要能够检测和分类无法预见的物理状态和干扰，评估它们可能对它造成的潜在损害，并启动适当的对策，即反应，”他们写道。“为了解决这一苛刻的要求，人类原型将作为我们的灵感，这意味着人类的疼痛反射运动用于根据机器人疼痛感觉模型和反应控制进行设计。我们专注于机器人疼痛的形式化，基于人类疼痛研究的见解，作为触觉的解释。”

这种用于“触觉”的人工机器人神经系统可能类似于嵌入人类皮肤的感觉神经元网络，能够感受力、压力和温度。如果任何外部刺激超过预编程的阈值，则在激活合适的反应之前，将其分类为不同的类别。

“在[轻度]疼痛类别中，这种接触可能会伤害机器人或阻止它执行任务。研究人员解释说:“机器人‘感觉’不舒服，会平稳地缩回，直到接触结束，然后再回来。”。“强烈碰撞属于(中度)疼痛范畴。机器人“感觉”到中度疼痛，应迅速缩回，并远离，直到接触事件结束。然后，它可能会向后移动。[严重]疼痛等级包括机器人可能受损并因此需要某种“帮助”的所有接触。为了防止损坏变得更严重，机器人切换到重力补偿，具有额外的阻尼用于耗散，通过其严格的被动行为提高了机器人和环境的安全性。”

## 工作和家庭用更安全的机器人

随着能够[阅读情绪并对情绪做出反应的机器](https://thenewstack.io/affective-computing-emotionally-intelligent-machines/)、[经营酒店](https://thenewstack.io/hotel-run-entirely-robots-opens-japan/)和医院的服务机器人的出现，以及越来越多的[协作机器人](https://thenewstack.io/collaborative-robots-will-help-human-workers-not-replace/)出现在工作场所，设计能够对有害刺激做出适当反应的机器人是有意义的。这是机器人可以配备的一种额外的安全预防措施，除此之外还有无数其他传感器，例如，当检测到人类存在时，会让它们停下来或减速的传感器。

由于这些受生物启发的机器人机制将被设计成保护机器人免受伤害，因此，通过扩展，它们还可以防止严重和昂贵的维修，或防止工作场所发生意外事故，如果机器人的设备在人类看不见的地方或附近受到损坏。最终，在这个即将到来的机器人时代，能够感觉到自己疼痛的机器人很可能是一件好事，我们的工厂和仓库，甚至我们的超市、T2 的办公室和家庭都将实现更多的自动化。
