# 代码 n00b:这不是数学，天才

> 原文：<https://thenewstack.io/code-n00b-its-not-math-genius/>

正如之前 [Code n00b](/tag/code-n00b/) 帖子的读者所知，我是在从事新闻工作后进入 web 开发的。如今，这种特殊的职业已经成为一种冒险的谋生方式，我有很多朋友/前同事问我是如何进行职业转变的。然而，几乎所有的人都认为这是他们永远做不到的事情——他们确信编程太难了。我们所做的事情就像是某种神秘的知识，超出了像他们这样的普通人的知识范围。

我们这些真正以编码为生的人知道没有什么特别的魔力。(也不是什么数学！)所以我一直在琢磨这个问题:是什么阻碍了人们学习编程？过去两年的春季、夏季和秋季，我一直在给十几个学生教授前端 web 开发；他们中的大多数人来的时候可能有点担心，但肯定愿意尝试一下。然而，这些学生是自主选择的:他们进来时已经决定要学习 web 开发。其他聪明、能干、勤奋的人被困在夕阳职业中怎么办？

例如，我知道一些极其聪明和无畏的记者，他们自学了各种疯狂和复杂的东西，以便根据知识来写这些东西。更不用说，他们还以新闻业的名义(沉着地)面临严重的个人危险——然而，当面对非 GUI 计算机环境时，他们几乎会因压力而昏倒。或者我认识的另一个女人，她在看了一个 YouTube 视频后，刚刚更换了自己汽车的刹车鼓，但她拒绝相信自己能够掌握 JavaScript。

为什么让计算机和编程变得不同(而且，很明显，很可怕)？我自己也成为了一名程序员，我过去常常想，我们如何才能打开这个行业的大门，让更多的人进来。现在我意识到，他们中的大多数人不会来，即使我们把门完全从铰链上拆下来，并发出刻有承诺的亚马逊礼品卡的邀请，让他们只需偷看一眼。注意:我甚至不是在谈论严肃的技术，比如建立一个 Docker 容器集群，甚至是像 React 或 Angular 这样的预打包技术。只是常规的旧 HTML 和 CSS，在命令行上有一些 Git。

所以我开始直接问别人:是什么让你觉得你学不会编程？

> 编程绝不要求你是某种天才。欢迎任何人和任何人申请。

到目前为止，排名第一的回答是，“因为这是一项对于在数学方面有天赋的非常聪明的人来说的技能。我可能在其他方面足够聪明，但我数学很差。”

啊！！这是从哪里来的？即使当我告诉他们:( 1)编程通常很少涉及数学,( 2)即使涉及数学，我们也有库和插件来处理——他们坚决拒绝相信。

我希望我能告诉他们，我也不擅长数学，嘿，看看我，我是一名专业的网站开发人员，但事实是我一直喜欢/擅长数学，所以我不能打这张牌。

相反，我指出成为一名程序员所需要的最重要的智力资产是逻辑地思考一个问题，将它分解成步骤，然后有条不紊地完成它们的能力。例如，我的朋友丹妮尔 DIY 刹车替代品，也是一名律师。这是一个需要与编写或调试计算机代码完全相同的解决问题、逻辑和分析技能的职业。然而，当我告诉她她可以完全、完全、彻底地做到这一点时，她的眼神里也充满了恐惧。

我不知道还能做什么。不管我告诉人们多少种方法，你可以做到这一点，请看注释的原因 1 到 197，他们有一个答案。“不，我不能。”所以。如果你说你不能，那么你肯定永远也做不到。“这是一个完美的自我实现的预言。

编程绝不要求你是某种天才。欢迎任何人和任何人申请。让你潜在地擅长于此的事情与智力火箭力量或缺乏智力火箭力量无关。我的意思是，如果你是那种冲动地抓住一个结论，甚至在你知道它是正确的之前就开始工作的人，那么你可能不太适合编程。但是如果你是那种在遇到你不能或不知道做的事情时不会自动放弃的人。相反，谁来解决这个问题，把它拆开，找出它的工作原理，然后疯狂地搜索，直到你找到答案……你。你有资格成为我们电脑法师超级秘密联盟的一员。

不管你的智商恰巧是什么。
