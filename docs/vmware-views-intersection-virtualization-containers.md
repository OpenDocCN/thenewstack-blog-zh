# VMware 如何看待虚拟化和容器的交集

> 原文：<https://thenewstack.io/vmware-views-intersection-virtualization-containers/>

VMware 正在对容器原生生态系统进行更深入的投资。它通过诸如 [Lightwave](https://vmware.github.io/lightwave/) 这样的努力增加了对开源社区的参与。但是当考虑虚拟化和容器技术的交叉时，它的方法是什么呢？

[Paul Fazzone](https://www.linkedin.com/in/paulfazzone/) ，在我们从 Dockercon 2017 对他的采访中，为 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客指出，现在有几个可用的容器框架，每个都有一个关于底层基础设施的特定视图。每个云提供商都有自己的基础设施，这些基础设施有自己的特定要求。

VMware 将其技术视为位于这些云框架和基础架构本身之间。这种方法如何发挥作用是本次对话的主题，其中涉及 Docker、不同的框架、Cloud Foundry 以及 VMware 对云原生架构的看法。
