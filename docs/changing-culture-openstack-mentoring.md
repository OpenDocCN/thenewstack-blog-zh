# 指导是扩大 IT 多样性的关键

> 原文：<https://thenewstack.io/changing-culture-openstack-mentoring/>

在今天的 [The New Stack Makers](https://thenewstack.io/podcasts/Makers) 播客中，我们听到两位女性帮助公司成功增加其多样性，她们是 HPE 开发商倡导者和社区架构师 [Lisa-Marie Namphy](https://www.linkedin.com/in/lnamphy) 和 OpenStack 生态系统技术负责人 [Ildiko Vancsa](https://www.linkedin.com/in/ildiko-vancsa-a963499) 。Vancsa 和 Namphy 在巴塞罗那 OpenStack 峰会上与 TNS 创始人 Alex Williams 一起探讨如何创建更加多样化和开放的社区。

虽然行业内的多样性[仍处于低谷](http://spectrum.ieee.org/tech-talk/at-work/tech-careers/computer-vision-leader-feifei-li-on-why-ai-needs-diversity)，但 OpenStack 社区已经经历了多年的发展。Bitergia 的 Daniel Izquierdo [报告](https://blog.bitergia.com/2016/04/25/openstack-genderdiversity-technical-contributions-analysis/)称，2011 年女性仅占 OpenStack 总人数的 10.5%，尽管 2015 年女性人数略有增加，达到 11%。

Nampy 解释说，解决持续存在的多样性问题不完全是一个管道问题。特别是，这是 OpenStack 和 tech 整体文化的问题。“文化是一件大事。坦率地说，如果你是女性，在 IRC 上进行的对话并不令人愉快。这是一个由男性主导的世界，谈话可以很好地反映这一点。人们忘记了会议中也有女性。我希望看到的一件事是文化上的清理。”

在基础层面，OpenStack 专注于指导和社区建设，将其作为成功的支柱，希望在社区中培养多样性。“我们支持 OpenStack 团队的[女性，以及旅行和指导项目。我们正努力更多地关注打开大门，支持那些愿意加入的女性，以及那些致力于为女性创造更舒适环境的女性。在过去的两天里，我们几乎所有的技术演示都有女性登台，这非常令人惊讶，”Vancsa 说。](https://wiki.openstack.org/wiki/Women_of_OpenStack)

采访接近尾声时，Namphy 重申了一个事实，即指导不仅是增加多样性的关键因素，也是留住 OpenStack 社区中有才华的女性的关键因素。“在很多情况下，是那种接触，那种舒适度。如果你走进一个房间，没有人看起来像你，谈话与你无关，事实上这很令人不快，你会走出房间。女人来了又不留下，IRC 也一样。那么如何挽留呢？你建立了这种联系。这只是迈出了第一步。”
