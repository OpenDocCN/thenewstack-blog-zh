# 让欧洲的“浪漫”开源世界更加实用

> 原文：<https://thenewstack.io/making-europes-romantic-open-source-world-more-practical/>

都柏林——根据 Linux 基金会 9 月发布的首次调查，欧洲的开源贡献者比他们的美国同行更受理想主义的驱动。数据显示，欧洲人比美国人更经常引用为开源项目做贡献的社会原因，美国人更可能说他们参与开源是为了职业发展。

哥伦布是意大利人，也是 Fintech 开源基金会 FINOS 的负责人，他回忆了自己作为 Apache 项目个人贡献者的经历，并引用了他所谓的欧洲“开源的非常草根、热情、浪漫的一面”

相比之下，他指出，“美国的商业生态系统肯定要强大得多。但事实是，你知道，开源的这两个特性是不可替代的。”

Columbro 说，他看到了开源在理想主义和实践方面的优势，以及欧盟和该地区其他国家的观念，即互联网和支持互联网的软件作为共享资源具有价值。

“我真的很想把开源的这三种特性结合起来:个人浪漫的特性、商业动力和公共部门的集体价值，”他说。

## 区域项目的“跳板”

欧洲距离脸书、苹果、亚马逊、网飞和谷歌这些科技巨头的总部有数千英里之遥。(事实上，哥伦布仍在硅谷工作，尽管他说他计划在某个时候回到欧洲。)

他说，对于个人开发者来说，Linux 基金会欧洲分会有助于提高地区项目的可见性，让潜在的贡献者有更多的机会。他说，向欧洲 Linux 基金会捐赠一个项目是“一个潜在推动你的项目的强有力的方法。”

他补充说，“我认为任何开发者都应该把这看作是该技术的一个潜在跳板平台，不仅仅是在欧洲可见，而且有希望超越欧洲。”

欧洲组织的第一个主要项目，OpenWallet 基金会，旨在帮助开发者创建一个模板来构建数字钱包。“我发现它不仅非常符合 Linux 基金会的愿景，即不仅要创建成功的开源项目，还要围绕这些开源项目定义新的市场和新的商业生态系统。”

科伦布罗补充说，这也“非常符合欧洲创建数字共享的愿景，基于开源，他们可以实现某种数字独立。”

## 欧洲的动荡可能会激发创新

随着地缘政治和经济动荡搅乱欧洲的几个国家，Columbro 建议，如果该地区的公司开始削减成本，开源可以看到一个繁荣。

他寄希望于开源合作来帮助调和一些分歧。“当然，我相信开源有潜力把各方聚集在一起，”哥伦布说。

他还指出，“一般来说，我们认为开放源码和对开放源码的投资与专有软件的投资趋势是反周期的。…换句话说，当有更多的压力时，当有更多的压力来降低成本或减少劳动力时。”

“那时，人们被迫更加认真地考虑如何在保持吞吐量和效率的同时进行实际协作。我认为开源是实现这一目标的最佳途径。”
