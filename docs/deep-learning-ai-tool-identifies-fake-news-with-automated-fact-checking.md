# 深度学习人工智能工具通过自动事实检查识别“假新闻”

> 原文：<https://thenewstack.io/deep-learning-ai-tool-identifies-fake-news-with-automated-fact-checking/>

由于网上“假新闻”的泛滥，我们已经目睹了过去几年中日益严重的政治动荡——其中大部分是[假信息](https://en.wikipedia.org/wiki/Disinformation),即[为了政治目的而故意设计的病毒式传播](/frauds-behind-fake-facebook-news-feeds/)和误导公众舆论。研究表明，平均而言，虚假故事向 1500 人传播的速度[是真实故事的六倍](https://www.state.gov/wp-content/uploads/2019/05/Weapons-of-Mass-Distraction-Foreign-State-Sponsored-Disinformation-in-the-Digital-Age.pdf)，尤其是如果它与政治有关，尤其是当[机器人被用来进一步自动化假新闻的传播](https://www.cits.ucsb.edu/fake-news/spread)时。

这也难怪，许多专家正在寻找通过自动化事实核查过程来跟上虚假泛滥的方法。到目前为止，我们已经看到了可以根据外部来源评估一篇文章的[真实性的算法，而其他工具将评估](/mits-new-ai-tackles-loopholes-in-fake-news-detection-tools/)[整个新闻网站是否一致真实](/mit-algorithm-sniffs-out-sites-dedicated-to-fake-news/)，或者学习如何[通过首先生成假新闻来检测令人信服的假新闻](/grover-ai-detects-machine-generated-neural-fake-news-by-also-generating-it/)。滑铁卢大学的加拿大研究人员现在正在用假新闻检测工具为这个难题添加另一个部分，该工具使用深度学习人工智能算法来验证一篇新闻文章中的说法是否得到同一主题的其他文章的支持。如果这些说法没有得到其他文章的证实——尤其是那些来自知名新闻来源的文章——那么它很可能是虚假信息。

“每天都有大量的内容被放到网上，人类不可能筛选所有的内容来识别假新闻和虚假信息，”系统设计工程教授亚历山大·王(Alexander Wong)和克里斯·杜汉蒂(Chris Dulhanty)、杰森·德格林特(Jason Deglint)以及易卜拉欣·本·达雅(Ibrahim Ben Daya)说。“通过建立一个可以自动检查帖子或故事中的声明与其他帖子和故事的人工智能，它可以成为一个强大的工具，通过标记潜在的假新闻和虚假信息来加强人类的事实检查，从而让他们在打击虚假信息传播的斗争中更快更可靠地进行事实检查。

## 自动化姿态检测

正如该小组所指出的，一个自动化的事实核查系统可能包括几个不同的子任务，如从各种来源检索可能支持或反对某一主张的文档；检测每篇文章关于声称的权利要求的立场；评估每个商品的声誉和可信度；声明验证，评估文章的立场和来源的声誉，以确定声明的真实性。

特别是，该团队的工作集中在一个关键的研究领域，称为姿态检测。给定一篇新闻报道中的一个主张，以及同一主题的其他报道，该团队的系统可以确定这些外部来源是支持还是反驳主要主张，准确率高达 90%。通过预测其他文章是否支持目标主张，人类事实检查员可以快速而容易地推断出该文章是否是虚假报道。相比之下，以前的方法依赖于手动调整，这既耗时又不足以结合识别虚假信息的更复杂方面。

该系统是使用一个大规模的神经语言网络 AI 开发的，该网络基于一个预先训练好的、开源的、深度双向转换器语言模型，用于自然语言处理(NLP)，称为 T2 罗伯塔(RoBERTa)和 T3(稳健优化的 BERT 方法)。至关重要的是，这意味着该系统能够从左到右和从右到左分析文本序列，与单向模型相比，它可以更深入地理解语言上下文和流程。此外，该团队的系统使用从[假新闻挑战数据集](http://www.fakenewschallenge.org/)中收集的基准进行训练。

“我们惊讶于[我们的模型]能够如此好地学习主张和文章之间的细微差别和语言关系，以可靠地确定主张是否确实得到文章的支持，”Wong 告诉我们。“这正好说明了大规模神经网络语言人工智能最近取得的重大进展及其捕捉语言的能力。”

目前，该团队正在努力改善其模型的训练，为其提供大量的声明和故事，特别关注英语和西方世界以外的媒体，以便更好地了解世界各地新闻报道中的语言和文化差异。

正如人们可以想象的那样，开发更有效的人工智能工具对于正在进行的全球反虚假信息斗争至关重要，Wong 说:“这项工作表明，可以建立强大的自动筛选工具来遏制互联网和社交媒体上不断增加的假新闻和虚假信息，这些信息可能导致世界各地真正的社会，道德甚至身体伤害。让记者和事实核查人员掌握这些工具，可以让他们能够让自己以及公众了解所有声明和头条新闻背后的真相。”
