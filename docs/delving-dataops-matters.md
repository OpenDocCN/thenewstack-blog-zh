# 什么是“数据运营”,为什么它如此重要

> 原文：<https://thenewstack.io/delving-dataops-matters/>

“DevOps”、“SecOps”、“DevSecOps”、“ChatOps”、“NoOps”——这些术语层出不穷。在这一集的[新堆栈分析师](https://thenewstack.io/podcasts/analysts)播客中，[蓝山研究](http://bluehillresearch.com/)的首席分析师[托弗·惠特莫尔](https://twitter.com/tophw47)谈论数据操作，或“数据操作”他将其描述为以整体的方式看待数据生产管道，将数据管理目标与数据消费理想结合起来，以最大化数据衍生的价值。

数据操作和开发操作都需要多个团队之间的协调。上图描述了数据生产价值链。DevOps/CI/CD 管道的鉴赏家们，在收听我们的播客时，请随意与您最喜爱的型号进行对比:
