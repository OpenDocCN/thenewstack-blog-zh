# 企业如何使用软件和 DevOps 在颠覆性的创业世界中保持领先

> 原文：<https://thenewstack.io/how-enterprises-use-software-and-devops-to-keep-up-in-a-disruptive-startup-world/>

[TC 专题讲座预告:为什么软件是企业新的五年增长计划](https://thenewstack.simplecast.com/episodes/tc-sessions-preview-why-software-is-the-new-5-year-growth-plan-for-the-enterprise)

New Stack 与 TechCrunch 建立了令人兴奋的新伙伴关系，以帮助推广其将于 9 月 5 日在旧金山举行的最新活动 [TC Sessions: Enterprise](https://tcrn.ch/2Y3wJGI) 。新 Stack 创始人兼主编亚历克斯·威廉姆斯将主持此次活动的一场会议，直播报道并举行煎饼早餐和播客讨论。

为了预览这一事件，威廉姆斯与 [TechCrunch](https://techcrunch.com/) 的企业记者[罗恩·米勒](https://www.linkedin.com/in/ronsmiller)共同主持了这期创客播客。两人采访了两位企业软件高管，探讨他们的组织和客户如何在新的强大开源工具激增的情况下努力跟上这种中断:

瓦塞纳尔说，随着大规模系统变得更加普遍，特别是在云原生平台上，DevOps 中的开发人员和运营角色也在改变他们的工作方式，特别是通过使用开源组件加快部署速度。

瓦塞纳尔说:“从运营的角度来看，坦率地说，从公司的角度来看，如果你不开始重新思考和转变你如何利用、开发和运行技术，你将很快变得无关紧要。”

瓦森纳说，虽然绿地系统“总是新的、简单的”，但遗留系统“超级、超级难”。与此同时，“我认为我们开始接受一个正确的现实，至少对于较大的公司来说，混合和混合的环境代表了我们必须在其中工作的新的复杂性和机会。”

沃姆克说，重要的第一步是“了解你可以做什么新的，你需要从你的旧系统中利用什么”。“因为企业是非常庞大、复杂的机器，他们面临着许多组织挑战、许多法规遵从性挑战，以及来自所有不同组织的各种不同策略，他们需要应用这些策略，并确保他们的应用程序以不被报道的方式运行。”

在即将到来的 [TC Sessions: Enterprise](https://tcrn.ch/2Y3wJGI) 活动中，一些主要主题将包括企业如何充分利用其遗留系统，同时利用当今的开源和新技术(如云原生工具)完成新的数字化之旅。

“任何对企业感兴趣的人都应该参加，包括投资者、企业创始人和 IT 高管，”米勒说。“任何对这类主题和解决其中一些问题感兴趣的人”如何让传统玩家适应今天的 IT 挑战。"
