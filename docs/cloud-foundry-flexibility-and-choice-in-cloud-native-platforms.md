# Cloud Foundry:关注云原生世界的灵活性和选择

> 原文：<https://thenewstack.io/cloud-foundry-flexibility-and-choice-in-cloud-native-platforms/>

[云代工厂:关注云的灵活性和选择性——原生世界](https://thenewstack.simplecast.com/episodes/cloud-foundry-focusing-on-flexibility-and-choice-for-a-cloud-native-world)

对于一些在硅谷等技术中心之外工作的组织来说，可供选择的开发人才严重短缺。因此，许多公司采取了主动学习的方法来帮助他们的开发人员跟上当今云原生软件平台的需求。

Cloud Foundry 执行董事 [Abby Kearns](https://www.linkedin.com/in/abbykearns) 在上个月的 Cloud Foundry 欧洲峰会上录制的 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客中解释道, [Cloud Foundry](https://www.cloudfoundry.org/) 正在发展其技术，以使这些用户的目标受益。

“对于每一家正在成为软件公司的公司，他们都在努力寻找，他们在哪里找到人才？”卡恩斯说，并补充说，“我们看到许多公司投资培训和再培训人员，教他们成为云原生开发人员，我认为这是所有这些中最迷人的一块。”

Kearns 后来继续指出，Cloud Foundry 使开发人员能够专注于简单地编写代码，而新的 [Cloud Foundry 容器运行时](https://thenewstack.io/cloud-foundry-container-runtime-facts/)让他们可以在基于 [Kubernetes](/category/kubernetes/) 的容器编排或 Cloud Foundry 的应用运行时之间进行选择，完全在 Cloud Foundry 内。

“当你想到这种快速发展的技术生态系统时，它真的会让人不知所措。卡恩斯说:“我们试图通过 Cloud Foundry 实现这一目标。Cloud Foundry 提供了“这种灵活性和选择，但给你足够强大的护栏和意见，使你作为一个组织很容易采用它并取得成功。”
