# 为 IT 行业设定核心价值

> 原文：<https://thenewstack.io/end-year-2017-values-industry/>

[#156:为 IT 行业树立核心价值观](https://thenewstack.simplecast.com/episodes/156-setting-core-values-for-the-it-industry)

在我们关于价值观及其如何影响职场文化的系列的第二部分[中，我们采访了](https://thenewstack.io/how-do-values-affect-companies/)[谷歌](https://cloud.google.com/kubernetes-engine)产品管理副总裁[萨姆·拉姆奇](https://www.linkedin.com/in/sramji/)；[丹尼尔·洛佩斯·里德鲁埃霍](https://www.linkedin.com/in/daniel-lopez-ridruejo-993120/)，[比特娜美](https://bitnami.com/)的首席执行官；[克里斯·布兰顿](https://www.linkedin.com/in/cebrando/)， [StorageOS](https://storageos.com/) 的 CEO 和[戴夫·麦克罗里](https://www.linkedin.com/in/davemccrory/),[通用电气数字](https://www.ge.com/digital/)软件工程副总裁。TNS 创始人 Alex Williams 和 TNS 驻旧金山记者 TC Currie 共同主持了这一集的新分析师播客。

他们的公司规模从几个员工到几千人不等，但无论公司规模大小，都普遍需要一致的价值观。“我们在新的堆栈中发现，”威廉姆斯说，“这一切都归结为信任、尊重和诚信。”

拉姆奇说，谷歌的使命是“组织全世界的信息，并使其普遍可用和普遍有用。”这一核心价值覆盖在工程层面，关注的焦点是“我们如何在下一个数量级做到这一点？”作为一家公司，它正在研究如何为工程师提供对单个工程师的同情。

“当我们专注于与他们的用户互动的遥测技术时，我们如何确保我们有一个内部同理心和广泛包容的文化？”他问道，并提到谷歌有七个不同的业务，每个都有超过 10 亿的用户。

Brando 说，为了确保 StorageOS 正确地划分优先级，他们建立了三个核心业务流。首先，关注团队，确保合适的人员到位，并确保他们拥有完成工作所需的资源。其次，是产品。第三，产品运行良好并融入客户环境。此外，公司确保每个人每天都在学习。

在 Bitnami，Ridruejo 说，价值观是“做到这一点，保持好奇，赋予用户权力，简单和团队合作。”他说，筛选价值观的方法是在招聘的时候。一个普遍的想法是，最好有一个擅长某事但很难共事的人，但该公司发现，真正聪明和善良并不相互排斥，善良聪明的人需要更少的管理时间，使团队运行更顺畅。

但是当声明的值与执行不匹配时怎么办？

麦克罗里说，当一个组织拥有强大的价值观时，你必须让管理层负责确保它们得到实践，否则它们对个人贡献者来说就变得毫无价值。他说，在这种情况下，“执行开始失败。”当你有一种信任度很低的文化时，他说，“你会进入一种辩论比实际执行更受重视的文化。”
