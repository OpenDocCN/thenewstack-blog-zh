# MapR:下一代应用程序将如何改变我们看待数据的方式

> 原文：<https://thenewstack.io/mapr-next-gen-applications-will-change-way-look-data/>

[下一代应用将如何改变我们看待数据的方式](https://thenewstack.simplecast.com/episodes/how-next-gen-applications-will-change-the-way-we-look-at-data)

数据格局正在我们眼前发生变化。我们看到数据总量的巨大增长；我们正在生成和消耗大量的视频、图像和各种传感器输入。

此外， [MapR Technologies](https://mapr.com/) 的技术战略副总裁 [Crystal Valentine](https://www.linkedin.com/in/crystal-valentine/) 在最新版的 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客中表示，“增长最快的数据类型不是我们过去认为的传统企业 IT 堆栈的一部分。”。

MapR 是一家总部位于硅谷的大数据公司。它的创始人意识到数据将变得越来越重要，而现有技术，包括开源的 Apache Hadoop，无法支持实时事务性操作应用程序等。因此，他们花了数年时间构建出了 MapR 产品的核心技术，包括旗舰产品[融合数据平台](https://mapr.com/products/mapr-converged-data-platform/)，这是一款为多云环境设计的平台无关软件。它甚至可以在嵌入式边缘设备上运行。

数据的这种增长导致了新一代应用程序的发展，这些应用程序与传统的企业应用程序有着本质的不同。瓦伦丁谈到了这些全新的应用程序如何推动新的创新和新的商业模式。
