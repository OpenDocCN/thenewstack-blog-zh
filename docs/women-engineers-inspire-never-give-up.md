# 鼓舞人心且永不放弃的女工程师

> 原文：<https://thenewstack.io/women-engineers-inspire-never-give-up/>

在一个如今被性别歧视和骚扰困扰的技术市场，鼓励女性永远不要放弃自己的工程职业的激励和以身作则的方法似乎是极其必要的。

在本期[新堆栈制造商](https://thenewstack.io/podcasts/makers)播客中，我们采访了[Capital One 的工程副总裁 Margaret Mayer](https://www.linkedin.com/in/margaret-mayer-153b06/) 和[Pivotal 的首席技术专家 Ashley McNamara](https://twitter.com/ashleymcnamara) 。这段对话是 4 月份在奥斯丁举行的 OSCON 开源会议上录制的，内容集中在对那些可能害羞或不想说话的女性的指导和建议上。
