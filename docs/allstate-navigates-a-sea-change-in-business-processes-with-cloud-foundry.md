# 好事达借助 Cloud Foundry 引领业务流程的巨变

> 原文：<https://thenewstack.io/allstate-navigates-a-sea-change-in-business-processes-with-cloud-foundry/>

曾经有一段时间，业务流程被认为是一家公司的基石——当时一个人做什么以及如何去做被认为定义了该公司此后几十年的性质。但是正如好事达公司负责技术和战略投资的副总裁奥珀尔·佩里告诉《新堆栈》的亚历克斯·威廉姆斯所说，当今商业竞争的中心是服务客户。这些客户不断变化的期望——经常被竞争对手改变——构成了现代企业的基础。因此，这些基础必须灵活，甚至具有可塑性，这一点至关重要。

“我们谈论了很多关于从以项目为中心的组织转变为以产品为中心的组织，”Perry 说。“我在某种程度上把‘产品’这个词和用于开发它的服务互换使用。让较小的团队共处一地—作为我们网络的一部分，我们是一家大型企业的多地点团队。我们在不同的地方都有团队。我们一直致力于为每个团队定义我们产品和服务的所有权，并让他们能够在构建需要[*]构建的内容时自我指导……我们需要整合备份，并使其成为整体的一部分。所有这些都是为企业服务的。”*

 *佩里同意威廉姆斯的观点，即今天的商业模式仍然类似于工厂车间，但现代自动化工具使这些工厂变得可行，如果你愿意，可以重新配置。分析实践可以通过在流程中的故障发生之前预测它们来提供帮助。

随着知识转移的进行，一家成立于 1931 年的机构如何继续为其业务流程带来巨大变化，同时强化其为面临生活突变的客户提供“稳定之手”的形象？现在请收听“好事达引领业务流程的巨变”，这是一个[新堆栈制造商](https://thenewstack.io/podcasts/makers)的播客，来自上次在圣克拉拉举行的云铸造峰会。
