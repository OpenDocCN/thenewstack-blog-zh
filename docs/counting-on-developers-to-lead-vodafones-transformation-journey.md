# 依靠开发者引领沃达丰的转型之旅

> 原文：<https://thenewstack.io/counting-on-developers-to-lead-vodafones-transformation-journey/>

英国电信提供商沃达丰(Vodafone)在 20 多个国家拥有和运营网络，并正在成为一家专注于数字服务的科技公司，该公司计划雇佣数千名软件工程师和开发人员，帮助公司进入云原生轨道，并通过 API 利用他们的网络。

在纽约市 [MongoDB World 2022](https://www.mongodb.com/world-2022) 的这一集 New Stack Makers 播客中，沃达丰全球产品经理 [Lloyd Woodroffe](https://www.linkedin.com/in/lloydwoodroffe/?originalSubdomain=uk) 分享了该公司如何与 MongoDB 合作开发电信即服务(TaaS)平台，以帮助他们的工程师提高软件开发速度，并推动在 DevSecOps 管道中采用最佳实践自动化。新堆栈的创始人亚历克斯·威廉姆斯主持了这次播客。

[指望开发者引领沃达丰的转型之旅](https://thenewstack.simplecast.com/episodes/counting-on-developers-to-lead-vodafones-transformation-journey)

沃达丰已经建立了一个主干来保持业务的弹性和可伸缩性。但是他们现在想做的一件事是创新，给他们的开发者自由和灵活性来进行创造性的开发。

“TaaS 平台——我们正在打造的产品——本质上是一个开发者优先的框架，允许开发者和沃达丰打造你认为有助于业务增长的东西。但是因为我们是一个企业，我们需要安全和财务保证，TaaS 是一个框架，它允许我们以一种给开发者他们需要的工具和我们需要的安全的方式来做到这一点。

作为内部采购模式的一部分，重用的想法是沃达丰规模的关键。该公司的关键举措“one source”使他们的开发人员能够采用这样的策略，“我们在所有市场和团队中都有一个单一的存储库，您可以在那里发布您的代码，来自其他国家的其他团队可以获取这些代码，重用它们，并将其实施到他们的应用程序中，”Woodroffe 说。“就外包给社区而言，我们的工程师希望开始将 API 产品化，并构建新的创新应用程序，我们很快就会看到这一点，”他补充道。

“我们用 MongoDB 构建的 TaaS 开发者平台充当了我们平台的服务注册中心。当您为开发人员提供工具时，我们会注册我们从安全和财务角度设置的组织、成本中心和护栏，”伍德罗夫说。"然后我们为开发者提供 MongoDB 作为他们选择的数据库."

“我们最终会看到，随着开发者更多地使用这些工具(TaaS)和产品，他们将能够构建新的创新，这些创新可以通过 API 在我们的网络中得到利用，”伍德罗夫说。
