# 准备好迎接新一轮云降价吧

> 原文：<https://thenewstack.io/get-ready-for-a-new-round-of-cloud-price-cuts/>

谷歌今天上午[表示](https://googlecloudplatform.blogspot.com/2014/10/Announcing-across-the-board-price-cuts-on-Compute-Engine.html?m=1 "Google")已经将所有地区所有实例类型的谷歌计算引擎价格下调了约 10%。准备好让它的竞争对手效仿。

谷歌表示，随着其数据中心效率的提高和硬件成本的持续下降，裁员正在进行。

过去，当三巨头之一——亚马逊网络服务、微软 Azure 和谷歌——降价时，其他公司也会跟着降价。事实上，今年早些时候，微软表示，它将为计算、存储和带宽等商品服务提供与 AWS 价格相当的[。如果 AWS 追随谷歌，微软也会。](http://azure.microsoft.com/blog/2014/01/24/storage-price-match/ "Microsoft")

这种连锁降价还将持续一段时间。它们对用户和竞争对手意味着什么？

对用户来说，低价是一件喜忧参半的事情。降价将继续发生在计算等商品服务领域，在这些领域，大型供应商由于规模经济和不断提高的效率而具有优势。但是相关服务的价格不会一直下降。大型提供商正在提供越来越多的服务——想想后端即服务和大数据功能——他们计划在这些领域锁定客户并带来收入。用户可能会发现，即使商品服务的价格下降，他们仍然会因为增加的服务而在云提供商那里花很多钱。

此外，即使三大巨头说他们在匹配彼此的价格，也很难比较他们的价格。这是因为它们都以不同的方式捆绑服务和硬件。当 AWS 和谷歌今年早些时候降低价格时，RightScale 发布了一个有用的图表，比较了服务和价格。这一次也可能如此。

对于一些竞争对手来说，持续的价格下跌可能会带来挑战。就价格和服务而言，中型供应商无法与大型供应商的产品相匹敌。这意味着他们必须尝试区分。

到目前为止，差异化战略已经取得了不同程度的成功。例如，Rackspace 正从“狂热支持”的角度出发。但该公司似乎正在苦苦挣扎，至少如果我们可以假设它的挑战刺激了最近一次不成功的寻找买家的努力。

IBM [今年早些时候告诉 ReCode](http://recode.net/2014/03/31/microsoft-joins-amazon-and-google-in-cloud-price-war/ "ReCode") ,它不打算配合三大巨头的降价。它在几个方面与众不同。一个是它的[裸机产品](https://thenewstack.io/softllayer-cto-big-data-containers-driving-interest-in-bare-metal/ "The New Stack")，随着对容器和大数据的兴趣日益增长，它可能会越来越受欢迎。

另一个原因仅仅是因为它是 IBM。像微软一样，IBM 有机会从与 IBM 合作了几十年的传统企业中赢得云客户。这些类型的客户并不总是像被 AWS 或谷歌吸引的初创公司那样对价格敏感。相反，企业想要一些帮助，他们愿意支付额外的费用。

敬请关注:可以肯定的是，AWS 和微软将在未来几天宣布他们自己的降价。
