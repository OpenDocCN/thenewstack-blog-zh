# 欢迎来到新的堆栈

> 原文：<https://thenewstack.io/welcome-to-the-new-stack/>

我们对推出新的堆栈“测试版”感到非常兴奋我要从头说一遍:感谢我们所有的赞助商以及在过去两个月里与我密切合作的人们。你真的做到了。还有所有其他人，朋友、家人和数百名给予支持的同事们(T2)，你们是我们站在这里的原因。

我称之为测试版有几个原因。我们有一些功能要添加和平滑，您将在接下来的几周内看到这些功能的推出。我们需要添加一些帖子，例如 Go 编程语言的简介。我们还在进行数据研究，预计将在未来几周内完成。

今年 1 月，我从未想过我会写一篇关于我自己的媒体公司成立的文章。那时候看起来不怎么样。我正要离开 TechCrunch，只是想弄清楚我到底要做什么。为另一个科技博客写稿？那会很有趣。蔻驰人喜欢写博客吗？我可以做到。但是开博客看起来更像是一个梦。

但是多亏了像希瑟·菲茨西蒙斯这样的人，新的堆栈现在是真实的了。我们正在直播。在这里，您将会看到一家媒体公司介绍这种新堆栈对世界各地的人们意味着什么。

在我们的核心，新的堆栈是开发者和其他人在复杂和快速的基础设施上构建服务的地方。

新的堆栈有五个核心类别:技术、文化、数据研究、面试和实践。在这些类别中，我们将从技术、文化和业务的角度来阐述新的堆栈。您将看到对新兴轻量级服务和系统的特别关注，这些服务和系统越来越受开发人员和他们的运营同行的欢迎。

通用技术媒体对市场上的新堆栈制造商报道不足。我们对组织如何实施他们的新堆栈知之甚少。我们将与开发人员、产品经理、IT 专业人员、首席执行官、首席财务官和各行各业的技术专家交谈，全面了解这些新的轻量级技术提供了什么以及如何使用它们。

数据研究对我们来说至关重要。我们现在正在研究 GitHub 和其他来源，以探索趋势，开源运动。

你还可以期待无人机报道，3D 打印焦点，以及关于农民、艺术家和音乐制作人的故事，这些问题你不会从科技博客中想到。领导新堆栈基础设施开发的 DevOps 专家、开发经理和高管将对我们的报告至关重要。这些讨论将以特写、简介、访谈的形式出现，并作为我们对市场总体观察的一部分。

然后，是我们计划要做的事情。在另一篇文章中会有更多的介绍。

我觉得很幸运有机会成为如此激动人心的事情的一部分。再次感谢你的支持。
