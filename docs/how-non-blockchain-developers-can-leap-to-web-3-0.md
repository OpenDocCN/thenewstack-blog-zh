# 非区块链开发者如何跨越到 Web 3.0

> 原文：<https://thenewstack.io/how-non-blockchain-developers-can-leap-to-web-3-0/>

今天，由于脸书和谷歌等少数大型科技公司控制着大量用户数据，一个极度集中的互联网影响着我们日常生活的许多方面。然而，随着[区块链技术](https://thenewstack.io/what-you-need-to-know-about-blockchain-before-making-the-leap/)的出现和对更加[去中心化的网络](https://thenewstack.io/the-people-pushing-for-a-decentralized-web/)的不断推动，情况可能会慢慢改变，在那里，用户数据的控制权将从少数追逐利润的看门人手中夺走。

一个新生的 Web 3.0 正在出现，取代了我们现在所知的社交互动 Web 2.0。它的特点是分散或分布式的网络架构，这将为真正开放的互联网奠定基础，在那里我们不必盲目地将我们的数据委托给垄断企业，或者为了参与而获得他们的许可。

对许多人来说，这种向真正自由和透明的互联网的逐渐转变无疑是一个令人兴奋的前景，但对于开发人员来说，从构建 Web 2.0 应用程序跳到学习创建[去中心化应用程序](https://ethereum.org/en/developers/docs/dapps/)或 dapps 所必需的技巧可能是一个陡峭的学习曲线。

例如，许多 dapps 建立在[以太坊](https://thenewstack.io/as-blockchain-hype-fades-developers-give-ethereum-a-serious-look/)区块链之上，允许使用[智能合约](https://www.computerworld.com/article/3412140/whats-a-smart-contract-and-how-does-it-work.html)，这消除了对像银行这样的集中中介的需要，并有助于简化任何相关的金融交易。

然而，在以太坊区块链上运行 dapps 有一些缺点，包括交易速度慢、交易成本高和不可持续的能源消耗。

## 建设“区块链互联网”

更广泛地说，一些观察家也指出了区块链技术目前的局限性，包括缺乏可扩展性、存储和不同区块链生态系统之间的 T2 互操作性。所有这一切促使越来越多的[备选方案](https://bdtechtalks.com/2021/05/21/ethereum-alternatives/)来解决这些问题，比如建设“区块链互联网”的提议，将分离的区块链连接成一个相互联系的整体。

“区块链的互联网是一个不断扩大的互联主权区块链的实时网络，”丹尼斯·法德耶夫解释道，他在一家为分散化应用程序构建和维护关键的最先进基础设施的初创公司 Tendermint 负责监督开发人员的体验，并且是一个旨在提高不同生态系统之间的互操作性的并行独立区块链网络的核心贡献者。

“该网络于 2019 年随着第一个区块链宇宙中心的发射而上线。自那以后，超过 250 个区块链应用和服务加入了该网络，许多最大的区块链项目，如 THORchain、币安 Chain、Terra 和 Crypto.org，都是使用宇宙技术建造的。该网络现在保护超过 1000 亿美元的数字资产。”

区块链互联网将解决用户被孤立在孤立的区块链生态系统中的问题，这有时会迫使人们使用集中服务或其他潜在的不安全解决方案来转移有价值的令牌和其他资产。Cosmos 网络在过去几年中有了相当大的发展，特别是今年发布了一个软件升级，可以在网络中的区块链之间传输令牌、资产和其他数据。

这一切都是通过使用[区块链间通信](https://ibcprotocol.org/) (IBC)来实现的，这是一种安全的通信协议，旨在在各种分布式区块链分类账之间中继消息，而不会损害它们的自主权，这是区块链行业的一个关键方面。

法德耶夫说:“这个网络由拥有主权的区块链组成:一个区块链可能有自己的验证器集、治理、费用，并独立于网络中的任何其他区块链。”。网络中的区块链通常是特定于应用的:每个区块链托管一个或几个应用。不是在单个区块链上构建所有应用，而是作为独立的区块链构建和启动应用，能够通过 IBC 在彼此之间发送数据。”

重要的是，新兴的区块链互联网不会与现有的互联网基础设施竞争，而是建立在基础设施之上。相反，区块链的互联网将作为一个更高的层次，使用户能够转移有价值的资产，而不需要依赖一个集中的组织。

“与过去可用的系统相比，这个网络既具有包容性，又基于任何区块链或应用程序都可以实施的开放协议，而且是分散的，不依赖于中央权威机构，因此使网络更具弹性和容错能力，”法德耶夫补充道。

## 互操作性和包容性

尽管主流玩家对区块链技术的投资越来越多，对区块链开发者的需求也越来越大，但仍然没有足够的拥有合适技能的开发者。开发人员在构建区块链应用时面临的最大挑战之一是学习另一种编程语言，并必须适应不同的库和工具集，这促使一些区块链项目采取更具包容性的方法。

例如，Cosmos 的栈是用 Go 编写的，Go 是目前最流行的通用编程语言之一。此外，还有其他易于使用的系统，如 Tendermint 的 [Starport Network](https://github.com/tendermint/starport) ，这是一个一体化平台，供开发人员在主权和安全的区块链上构建、启动和维护任何加密应用程序，计划于 2021 年推出。

“Starport 是 Tendermint 面向开发者的旗舰产品之一，”法德耶夫说。“Starport CLI 是一款用于创建主权区块链、生成代码、搭建附加功能、创建 web 和移动客户端以及运行开发中的区块链节点的工具。Starport 负责生成样板代码，并将无缝开发体验引入区块链世界，从而弥合了传统 web 和加密应用程序开发之间的差距。这种易用性和对开发人员的低准入门槛将大大加快向 Web 3.0 的过渡。”

从长远来看，Cosmos 背后的贡献者正在实施的那些提议是有意义的。用一个共同的通信协议将不同的区块链连接起来，同时仍然允许它们保持主权，这将最终允许区块链生态系统的多样化阵列进一步共同发展。

甚至其他顽固的孤立的生态系统，如比特币和以太坊，也可以通过使用分散的、安全的、不可信的桥梁被捆绑在一起。这种转变有助于提高整体可用性、便利性和降低成本。

“区块链互操作性是区块链技术可持续发展的关键，”法德耶夫解释道。“Cosmos 的愿景是通过将大多数区块链与 IBC 连接起来，弥合它们之间的差距，创造一个多链的未来，在这个未来，大多数应用程序都以主权链的形式构建，所有区块链都相互连接。用户完全保管自己的私钥，但用户体验却可以与传统的 web 应用相媲美。用户知道他们使用的应用程序是由区块链技术驱动的，但不必考虑个人区块链。对他们来说，所有的一切就是他们拥有的资产、他们使用的服务以及以他们希望的方式转移这些资产的能力。”

## 解决区块链存储

这是一个诱人的想法，也许有一天，像健康记录这样的敏感私人信息可能会全部存储在区块链上，如果有人试图修改它，它可以被安全地保存并容易地验证。但现实情况是，除了互操作性之外，由于缓慢的交易速率和访问费用，在公共区块链上存储大型数据文件会带来可扩展性和成本增加的问题。

一种可能性是使用分散存储解决方案，如由[协议实验室](https://protocol.ai/)提供的解决方案，这是一个分布式、开源的研究、开发和部署实验室，支持[星际文件系统](https://thenewstack.io/interplanetary-file-system-could-pave-the-way-for-a-distributed-permanent-web/) (IPFS)，这是一种新的超媒体协议和基于对等(p2p)网络架构的分散数据存储系统。

“我们团队已经推出了两款产品， [Web3。存储](https://web3.storage/)和 [NFT。Protocol Labs 的产品经理卡迪尔·托帕尔说:“存储](https://nft.storage/)帮助开发人员冗余地存储数据，并使其可以通过 IPFS 和 Filecoin 网络访问。“我们正在构建 API 和服务，为开发人员提供一个简单的、类似 Web 2.0 的抽象来存储任何数据，但使用 Web 3.0 原语，以 Web 3.0 原生方式与新的分散式对等网络进行互操作。”

根据协议实验室的说法，Web3 背后的巨大优势。存储和 NFT。相对于 Web 3.0 标准，存储更易于使用，而且不会损害[的 Web 3.0 原则](https://media.consensys.net/a-warm-welcome-to-web3-89d49e61a7c5)，如透明度、开放性和数据所有权回到个人手中，而不是锁在中央机构或公司内部。

“我们的 API 和服务是免费的，只需几分钟就可以开始使用，”Protocol Labs 产品经理 David Choi 说。“尽管数据是通过我们的 API 端点移动的，这在某种程度上是集中的，但用户可以使用数据在 IPFS 的唯一内容地址从任何地方访问这些数据。他们可以通过检查任何下载的数据是否与其地址匹配来毫无根据地验证数据是他们的，并通过告诉他们固定数据来轻松使用其他存储提供商来冗余存储数据。如果我们的服务不再适合他们的需求，他们甚至可以轻松地迁移出我们的服务，并在新服务中提供数据，而无需停机。”

最终，这些倡议都指向不同的区块链技术在不久的将来获得越来越多的关注，特别是随着各种生态系统的出现和现有生态系统继续找到共同点，同时仍然保持各自的独立性。

“跨链交易的用例是潜在无限的，区块链可以交换价值，扩展和容纳更大的用户群，”法德耶夫总结道。“区块链最大主义和“一个链条将胜利出现并统治所有链条”的时代已经结束。将会有成千上万的区块链，每个都服务于不同的特定应用用例。”
