# Go 如何对系统管理员有效

> 原文：<https://thenewstack.io/how-go-is-effective-for-sysadmins/>

编者按:从工具制造的角度看 Go。啊

有几件事我需要完成。一个主要任务是写一个关于谷歌主导的编程语言 [Go](https://golang.org/) 的简介。那么，为什么不同时叙述一下我在 Twitter 上看到的关于 Go 的消息，然后发布到这里呢？这就是我要做的，希望它能导致一个更长、更深入的关于围棋的帖子，以及我在最出色的 [Gophercon](http://gophercon.com) 上学到的东西，这是有史以来第一次围棋大会。

我一直在寻找 Go 在大会上的相关例子，但是随着所有演讲者演讲的发布，伟大的素材源源不断。最近的一个来自[新遗迹](http://newrelic.com) s [凯尔西·海托华](https://twitter.com/kelseyhightower)(同为波特兰人！)，他讨论了 Go 对系统管理员的好处。他的演讲是关于从工具制造的角度出发，部署应用程序。这是一个关于他如何学习 Go 和用于大规模服务器部署的新 Linux 发行版 CoreOs 的新鲜幽默的故事。

他从博文中摘录的内容概述如下:

*   Hightower 如何利用 Go 标准库完成日常系统管理任务
*   goroutines 和通道在系统实用程序和脚本中的实际使用方式
*   最佳实践
