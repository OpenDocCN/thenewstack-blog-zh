# Teleport，一个为开发者提供云资源的统一访问平面

> 原文：<https://thenewstack.io/teleport-a-unified-access-plane-that-brings-cloud-resources-to-the-developer/>

Teleport 以前被称为 Gravitational，直到最近[，它以其旗舰统一访问平面平台的名字重新命名为](https://thenewstack.io/gravitational-becomes-teleport-launches-a-unified-access-plane/)。Teleport 基于谷歌的加密技术构建，除其他外，它允许工程师绕过传统架构的各个层，以便通过互联网连接从世界各地安全地利用云资源。

在这个 [The New Stack Makers](/podcasts/makers/) 播客中，New Stack 的发行人兼创始人 [Alex Williams](/author/alex/) 与 Teleport 的联合创始人兼首席执行官 [Ev Kontsevoy](https://www.linkedin.com/in/kontsevoy) 谈论了向广泛分布式架构的转变对工程师和开发人员意味着什么，以及 Teleport 如何在这种新的动态中满足他们的需求。

[瞬移，统一接入平面，建立在 Google 的 Crypto](https://thenewstack.simplecast.com/episodes/teleport-a-unified-access-plane-built-on-googles-crypto) 之上

Kontsevoy 说:“如果你认为地球是一台巨型计算机，那么 Teleport 就是这种连接结构——所有这些计算资源和人类通过它相互通信。”“我们认为这是云技术发展过程中必要的下一步，因为目前，在管理和访问云基础设施和云应用程序时，我们很不幸地处于试图将客户端-服务器时代的旧技术拼凑在一起的混乱状态。”

Kontsevoy 说,“通用访问平面”的概念是“我们想出来解释传送是什么的一个术语”。他解释说，这个想法是为了让工程师能够即时访问他们需要的任何云计算资源，以提高工作效率。这些资源可能包括 [SSH 服务器](https://en.wikipedia.org/wiki/Comparison_of_SSH_servers)或者亚马逊网络服务(AWS)上的计算实例。当然，也可以访问 Kubernetes 和数据库平台——这在今天特别受欢迎。“它可能是您自己地下室中的一个 VMware 盒子；它可能是一辆自动驾驶卡车，可能是一颗卫星——绝对是任何东西。”

Teleport 的一个重要组成部分是，它允许工程师在云环境中将应用程序部署到生产中时，继续使用他们熟悉的 CI/CD 管道的现有工具。

“我们真的不希望任何人学习任何新东西，因为所有这些命令实际上都很好。Kontsevoy 说:“Teleport 为您提供了一个我们称之为统一的访问平面，它实际上是一个分布式端点。“正是这种分布在不同地理位置的代理，让您能够访问组织所需的所有计算资源。”

在许多方面，Teleport 帮助工程师利用云环境中可用的巨大资源池创建应用程序。Kontsevoy 说:“从我的角度来看，我认为 Teleport 是一个巨大的生产力工具”，它还“极大地提高了工程师的个人幸福度”。“[这是]因为你与这台云计算机有着近乎亲密的关系，你真的是在为它开发应用。”
