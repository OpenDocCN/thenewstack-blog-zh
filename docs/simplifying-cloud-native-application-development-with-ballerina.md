# 使用 Ballerina 简化云原生应用开发

> 原文：<https://thenewstack.io/simplifying-cloud-native-application-development-with-ballerina/>

在过去的六年中，WSO2 一直在开发 Ballerina，这是一种开源编程语言，可以简化新服务和 API 的编写。它旨在简化能够使用、组合和创建网络服务的过程，并使高度分布的应用程序一起工作以实现确定的结果。

在这一集的 New Stack Makers 播客中，WSO2 的首席技术官 Eric 新人讨论了该公司如何从头开始创建一种新的编程语言，并计划让它成为一种主流的云原生语言。 [Darryl Taft](/author/darryl-taft/) 《新书库》的新闻编辑主持了这次播客。

[用芭蕾舞演员](https://thenewstack.simplecast.com/episodes/simplifying-cloud-native-application-development-with-ballerina)
简化云原生应用开发

基于集成开发太难的想法，Ballerina 被创建来在高度分布式的环境中编程。[云计算](https://thenewstack.io/10-key-attributes-of-cloud-native-applications/)是集成分布式计算的一种进化。你说的是需要在云中相互对话的微服务和 API，”新人说。“芭蕾舞演员所做的，是思考节目之外的哪些功能需要交流，”他补充道。

使用 Ballerina，开发人员可以轻松地使用它来创建云应用程序。语言设计受到 TypeScript 和 JavaScript 的影响，但也有一些额外的功能，新人说。“开发人员可以为 JSON 有效负载创建记录和模式，以支持云移动或 web 应用程序的 API，它可以并发处理多个调用事务控制，但使用非常熟悉的语法，如 TypeScript 或 JavaScript。”

WSO2 正在该公司的[低代码产品 Choreo](/wso2s-choreo-offers-low-code-for-kubernetes/) 中使用 Ballerina，该产品包括创建图表的能力等功能。“行业内的长期挑战是如何以图形形式表示您的编程代码。[Sanjiva Weerawarana，WSO2 的创始人]通过在语言中加入可以用来创建图表的语法元素，解决了这个问题。他的做法是，你可以编辑图表并创建代码，”新人说。

新人说，云工程需要一种编程语言，可以重新设计应用程序，以实现自动扩展、弹性和独立敏捷性。WSO2 正在继续推进其工作以应对这一挑战。“我们认为 Choreo 将帮助我们，因为它利用芭蕾舞演员的魔力来帮助人们更快地完成工作。一旦他们看到这一点，他们就会看到芭蕾舞演员，并从中受益，”新人说。
