# 构建沃克网:软件开发中的可访问性和社会公正

> 原文：<https://thenewstack.io/building-the-woke-web-accessibility-and-social-justice-in-software-development/>

[构建沃克网:软件开发中的可访问性和社会公正](https://thenewstack.simplecast.com/episodes/building-the-woke-web-accessibility-and-social-justice-in-software-development)

当我们谈论网络上的可访问性时，我们经常会谈到有视觉或灵巧障碍的人，但这不是唯一的一点。虽然满足这些法律上的可访问性要求很重要，但我们不能忘记，网络已经系统地将一些群体抛在后面，包括老年人、LGBTQ+、妇女、仍未联网的人等等——基本上是那些不是独联体白人男性的人，他们最常建立和运营科技行业。简而言之，最脆弱的人群无法接触到网络的潜力。

这就是我们在今年的 Afrotech Fest 的演讲前，在 BBC 软件工程师[奥卢·伊妮-阿沃苏西](https://twitter.com/oluoluoxenfree)的播客中谈论的话题。

伊妮-阿沃苏西在 BBC 的工作有两个角色，BBC 的目标是成为世界上最容易访问的新闻网站。他们使用 React 构建文章网页，并且是可访问性冠军。后者是一个自愿但正式的角色，可能是 BBC 独有的，任务是让同事们把可访问性放在首位。

易访问性是必不可少的，因为我们开发的软件不会被特定人群使用。

仅在英国，就有 8%的人仍然缺乏数字技能。当重要的政府程序——比如申请英国退出欧盟事件后的欧盟定居身份——完全在线时，这些人就无法享受这些福利。这些人被排除在以下情况之外:

1.  **管理信息:**使用搜索引擎查找信息，查找网站，或者下载并保存在网上找到的照片。
2.  **沟通:**通过电子邮件或在线信息服务发送个人信息，或在线发表评论和分享信息。
3.  **交易:**从网站购买商品或服务，或在设备上购买和安装应用程序。
4.  **解决问题:**在线验证信息来源或使用在线帮助解决设备或数字服务的问题。
5.  **创作:**填写包括个人信息在内的在线申请表，或从现有的在线图片、音乐或视频中创作新的东西。

这就是[英国数字鸿沟](https://www.ons.gov.uk/peoplepopulationandcommunity/householdcharacteristics/homeinternetandsocialmediausage/articles/exploringtheuksdigitaldivide/2019-03-04)所基于的框架。因此，这些被排除在数字之外的人错过了增加 10%的工资，并被认为更不可雇佣，而不是省钱，因为网上购物便宜 13%，他们与亲人联系的能力低 40%。也许最重要的是，以模拟方式访问政府服务意味着平均要多花 30 分钟。

据估计，至少 90%的工作不久将需要数字技能。8%被数字技术排斥的人来自上述被剥夺权利的群体。他们接触不同信息来源的机会较少，申请工作的能力也明显较低。这使得这些群体只能扩大他们的数字鸿沟和机会。

“那些经常被排除在外的人认为互联网无法为他们提供任何东西。伊妮-阿沃苏西说:“即使他们上了年纪或有残疾，也不是因为他们不能使用互联网，而是因为互联网没有给他们任何东西。”。

科技行业习惯于创造甚至推广太困难的工具，这进一步排斥了弱势群体。

“想到一生的潜力下降是非常可怕的，随着你开始经历残疾和年龄的增长，这变得越来越难，”他们继续说道。

当人们，尤其是来自被排斥群体的人们，确实能够访问互联网时，他们面临着更高的被在线骚扰的风险，或者他们甚至无法获得信息，就像在 YouTube 这样的网站上，即使是 G 级 LGBTQ+信息也经常被[过滤为成人内容。](https://www.theguardian.com/technology/2017/mar/20/lgbt-community-anger-over-youtube-restrictions-which-make-their-videos-invisible)

这些都是开发人员在重写边缘案例和审查他们的应用程序时必须考虑的因素，以确保他们没有系统地将人们排除在网络潜力之外。无论如何，在英国，每六个人中就有一个残疾人，这个数字并不是一个极端的例子，不是吗？
