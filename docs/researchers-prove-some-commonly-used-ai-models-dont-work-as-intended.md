# 研究人员证明，一些常用的人工智能模型并不像预期的那样工作

> 原文：<https://thenewstack.io/researchers-prove-some-commonly-used-ai-models-dont-work-as-intended/>

我们的世界是由复杂的网络定义的——无论是通过生态系统中物种的相互联系，还是通过在线社交媒体网络。如今，用人工智能模型和算法分析这种复杂的网络形成了许多有用工具的主干，如搜索引擎和推荐系统。但是这些机器学习方法真的像它们的设计者预期的那样有效吗？答案似乎是否定的，根据最近的一项研究，该研究表明，这些广泛使用的人工智能模型中的一些无法准确捕捉此类网络中发现的真实世界的复杂性。

特别是，被称为[低维嵌入](https://towardsdatascience.com/neural-network-embeddings-explained-4d028e6f0526)的机器学习技术是最近发表在《美国国家科学院院刊》上的[研究论文](https://www.pnas.org/content/117/11/5631)的重点。嵌入和其他类似的机器学习技术被用于通过分析从生物学、流行病学和社交媒体等任何不同数量的复杂网络中发现的关系来生成预测。

“低维嵌入是一种流行的机器学习技术，它将‘节点’——或社交网络中的人——转化为一系列数字或‘向量’，”该论文的主要作者[c .“Sesh”Seshadhri](https://users.soe.ucsc.edu/~sesh/)解释道，他是加州大学圣克鲁斯[分校计算机科学与工程的副教授。](https://www.ucsc.edu/)

“这些数字然后被用来预测一个人可能会买什么，推荐什么内容，以及谁可能会成为谁的朋友。然而，我们的论文认为这些方法并不像人们想象的那样有效，”他说。

## 社区结构丢失

虽然更新更好的包埋方法正在不断开发，但研究人员认为这些技术都存在同样的缺陷。值得注意的是，它们在充分代表复杂网络的一些关键方面方面并不那么出色，例如一组紧密连接的节点，在网络科学术语中称为“社区结构”。在现实世界中，社区结构可以在拥有共同兴趣的人群中找到。

“我们发现低维嵌入通常会失去社交网络的社区结构，这可能是最重要的特征，”Seshadhri 说。这意味着在社交网络中，人们或节点被组织成大量紧密结合的社区。这些信息在许多低维嵌入方法中会丢失。我们从数学上证明，对于一些低维嵌入类，这样的损失是不可避免的。”

嵌入技术将一个人在社交网络中的位置转换为几何空间中一个点的一组坐标，然后可以生成一个“低维”或一系列数字，然后可以用作算法的输入。然而，正如研究人员在他们的论文中所显示的那样，重要的信息丢失了，因为这种几何方法在转换完成后忽略了更大的社会网络，并且只根据空间中这些点之间的关系进行预测。使用数学证明，并在各种网络上测试不同的嵌入技术，研究人员证明了这些网络的重要结构质量通过使用这些方法被省略。

例如，嵌入技术没有很好地表现的一个结构特征是三角形的密度，它体现了三个人之间的联系。社交网络中许多三角形的存在表明社区结构的丰富性。然而，许多这些数据丰富的三角形在嵌入过程中消失了。

“我们怀疑低维嵌入在捕捉社区结构方面表现不佳，但我们对它们的表现如此之差感到惊讶，”Seshadhri 说。

该团队的发现有效地淡化了这些嵌入是强大的机器学习方法的观点，因为这些技术生成的少量数字不可能完全表达社交网络的内在复杂性。当然，嵌入并不是唯一可以用于分析和预测的方法，该团队的工作表明，使用多种机器学习方法，而不是仅仅依赖于一种技术，并检查人们可能对某些机器学习方法的假设，可能是一个更好的主意。

“我认为这样的工作对从业者是有用的，他们将获得机器学习技术的更完整的图片，”Seshadhri 指出。“通常，这些技术是高度数学化的，涉及复杂的代码。一个非专家很难理解什么有效，什么无效。所以很自然地，人们求助于共同的研究趋势，并遵循流行的技术。最终，我们需要更多的工作来调查这些技术是否能够真正实现它们的承诺。”
