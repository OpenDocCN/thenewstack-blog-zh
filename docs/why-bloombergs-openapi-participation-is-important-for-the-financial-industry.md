# 为什么彭博加入 OpenAPI 对金融行业很重要

> 原文：<https://thenewstack.io/why-bloombergs-openapi-participation-is-important-for-the-financial-industry/>

金融媒体公司[彭博](https://www.bloomberg.com/)参与 [OpenAPI 计划](https://www.openapis.org/)和开源社区是基于更广泛的愿望，而不仅仅是选择合适的工具来发展业务。在这个动荡时期，风险尤其高，因为新冠肺炎的破坏仍在继续，已经摧毁了经济的大部分。

在最新一集的[新栈制造者](https://thenewstack.io/podcasts/makers)播客中，我们采访了两位来自彭博的开源领导者:

*   [理查德·诺顿](https://www.linkedin.com/in/richard11/)，彭博数据许可工程组负责人。
*   [凯文·弗莱明](https://www.linkedin.com/in/kpfleming/)，开源社区项目负责人，彭博首席技术官办公室成员。

[为什么彭博加入 OpenAPI 对金融行业很重要](https://thenewstack.simplecast.com/episodes/why-bloombergs-openapi-participation-is-important-for-the-financial-industry)

讨论了彭博对 OpenAPI 的合作和支持，它对开源的参与以及这对金融行业的意义。

从一开始，API 就被视为彭博通过其著名的彭博码头所提供的核心基础。彭博向其金融机构和其他客户提供的数据和分析(可能包括期货或大宗商品的数据源)有助于做出关键决策，这些决策往往会影响全球资本市场。

诺顿说:“我们还为企业客户提供了一种方式来访问所有这些信息，所有这些丰富的数据，作为丰富的分析信息，因此他们可以将这些信息纳入自己的应用程序、自己的分析、机器学习或他们希望使用这些信息的任何内容中。”。“我们向客户公开这些数据的策略已经随着时间的推移而发展和变化。这就是我们在 API 战略方面越来越深入的地方，以及向我们的客户展示这些信息的更多新的、简单而有趣的方法。”

开源社区也为彭博的 API 和 DevOps 战略发挥了关键作用。事实上，彭博参与 OpenAPI 计划和开源社区不仅对彭博，而且对整个全球金融行业都很重要。

“开源友好的另一个方面是，彭博和该领域的其他公司传统上提供接口来获取，比如说市场数据、金融数据或新闻数据，这些数据作为客户端库提供(我们实际上给了你一个库，你必须链接到你的软件才能使用我们的数据)。这不是一个非常详细的开发人员友好的模型，”弗莱明说。“人们希望使用不同的编程语言，在不同的环境中工作，诸如此类。所以，通过 OpenAPI 和其他类似的东西来公开你的 API 意味着，如果有人说，‘嗯，我们用 Rust 编写所有的软件，我们希望能够连接到你的服务并使用你的数据，所以你能提供一个技术无关的描述，我们可以用它来连接到你的 API 吗？"

诺顿说，作为 OpenAPI 倡议的新成员，彭博希望“强调我们正在使用真正的行业标准”。“这不是一种公开数据并与彭博公开的不同类型的数据进行交互的专有方式。我认为像彭博这样的球员成为金融行业的领导者是件好事。诺顿说:“我们处理的金融工具比大多数其他机构都多，我们还在资本市场上与大多数其他银行和投资公司进行互动。”。“因此，有像彭博这样的(机构)在那里帮助确保——因为该倡议塑造并在某种程度上定义了未来——可以真正帮助并代表这个行业。所以，我觉得这是我们如何受益以及 OpenAPI 作为一项倡议如何受益的一个非常重要的部分。”
