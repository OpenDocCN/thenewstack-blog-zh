# 播客探讨在科技行业工作的心理健康挑战

> 原文：<https://thenewstack.io/podcast-explores-mental-health-challenges-of-working-in-tech/>

这条消息取代了之前发表的一篇关于技术领域心理健康问题的文章，新堆栈已经删除了这篇文章，以回应文章主题之一的个人情况。

如果您正在与自己的精神健康问题作斗争，或者知道有人正在与精神健康问题作斗争，请拨打药物滥用和精神健康服务管理局的全国帮助热线，1-800-662-4357(帮助)(也称为治疗转诊服务)，或 TTY: 1-800-487-4889。更多资源可以在 samhsa.gov 找到。
