# 从 Web 1.0 到博客先驱诺亚·格雷的社交媒体

> 原文：<https://thenewstack.io/from-web-1-0-to-social-media-with-blogging-pioneer-noah-grey/>

做一个博客先驱并不总是容易的，但它确实给了你一个独特而深远的视角。问问诺亚·格雷就知道了，他是 2000 年有影响力的早期博客平台 [Greymatter](https://en.wikipedia.org/wiki/Greymatter_(software)) 的创始人。

威尔·惠顿的新回忆录[认为](https://twitter.com/manmadeghost/status/1519850331416547329)博客看起来确实如此，主要内容在中间，区块在两边，因为是诺亚·格雷先做的然而就在两周前，诺亚·格雷出人意料地重返网络世界，参加了一场名为“我将在四天后失去我的房子”的 [GoFundMe 活动](https://www.gofundme.com/f/i-am-losing-my-home-in-five-days-may-3rd)，除非格雷能筹集到 35000 美元来偿还他的抵押贷款。然后互联网让他大吃一惊，捐赠了超过 112，000 美元。

满怀感激之情的 Grey 同意向我们介绍他的生活，并分享他对技术进步、“旧互联网”和当今博客世界的看法。

**新的堆栈:我不得不问，你现在使用的是什么博客技术？我知道你一直在 ManMadeGhost.com 的个人博客上分享你的摄影和艺术。那是 WordPress 博客吗？**

**诺亚格雷:**的确是！我确实打算在我有力气和时间的时候重做它。现在存在的是 1-2 年前的一个休眠的遗留问题，当时我带着我当时的新作品[……]从“从头开始”的心态中回来。但所有这一切都给了我对自己历史的信心，我希望我迟早能展示一幅更完整的画面。

**那么*你对当今社交媒体的总体格局有什么想法？***

经历了我刚刚经历的，我听到了对“旧互联网”的巨大的爱，我想我以某种微小的方式代表了它。但我认为这不仅仅是怀旧，而是对一种去中心化的、自下而上的、以人为中心的精神气质的渴望——我热切希望这种渴望能够慢慢回归。[…]

我确实认为可以肯定地说，像大多数其他“web 1.0”的人一样，我对社交媒体的感觉充其量是非常复杂的。在网络出现之前，我就已经上网了(从 80 年代的量子链接开始，后来变成了美国在线)，在有一个词来形容它之前，我也写过博客。因此，我就像你所能看到的那样，完全出生在网络的整个历史中，并受其影响，我无法将关心网络的福祉与关心我自己的福祉分开。

我讨厌脸书，但不会公开批评它太多，部分原因是如果有人需要听到它(和 Twitter 等)的所有问题，他们会很难从*我*那里听到——他们可以求助于我的 web-1.0 同志们的卓越口才，他们一直留在舞台上，如果他们真的需要在游戏的这个阶段为他们做这个案子。[……]我确实认为，任何一个把自己想象成我们的主人的人物或团体——你最好相信我是在看其他几个特定的方向——至少必须始终坚定地接受最严格的审查和怀疑。

但是，我喜欢的许多人使用脸书和所有其他人，他们从中获得了各种价值和满足感，这绝对不是我可以否认的。我现在坐在这里，刚刚从 Twitter 上获益匪浅，这是任何一个人都能接受的。它们绝对服务于真正的需求，但我不认为它们——目前在顶部的特定公司/商店——将是它们的最终决定。

> “我认为记住这一点是有帮助的，首先，从长远来看，网络本身以及我们在它基础上建立的一切仍处于初级阶段。”

当我看到像 WordPress 的创造者马特·莫楞威格这样的人时，我有无尽的希望，WordPress 目前为整个互联网的*提供超过 40%的能量——扎克和马斯克甚至能梦想与之比肩吗？但是他并没有控制将近一半的网络:他已经把帮助网络更好地增强自身能力作为他一生的工作。想想它的广阔性和根源，以及它是多么的被需要、被需要，并且正在工作。这是一个我绝对会站在城垛上的未来。*

**现在有什么有趣的博客技术是你一直关注的吗？**

在这一点上，我对“web 3”的了解可能并不比普通的极客门外汉多多少，当然也不知道它会如何发展。不过，我要说的是:首先，我认为记住网络本身以及我们在它之上所建立的一切在长远来看仍处于婴儿期是有帮助的。

10-20 年前对我们的数字再训练大脑来说似乎是古老的历史，但社会技术和社会本身对彼此的塑造的真实轨迹只有在大背景下才能看到。在表面之下，人类的潮流是巨大的、缓慢的、全能的。(想想电话本身有多“古老”,想想 20 年前我们大多数人会怎么想，电话已经或多或少地完成了对社会的塑造。)总的来说，我认为我们正处于历史的巨变之中，我们无法确切地衡量这一变化，就像 20 世纪 30 年代中期的大多数人无法衡量 20 世纪 40 年代末的情况一样。

每一个闪光的时刻都站在一座死气沉沉和被遗忘的技术和平台的山上。尽管他们获得了金钱和权力，尽管他们试图完全按照自己的形象重塑我们，但他们被权衡后发现有所欠缺，于是迅速离开。今天的权力掮客们可以向他们喜欢的任何新领域投入尽可能多的钱，并尽可能努力地控制一切走向，并按照他们自己的形象塑造它。他们可能会暂时按自己的方式弯曲，但从长远来看，人类的心脏会将一切弯曲回自己的形状。我*有*相信这一点。

**那么你还记得创建最初的 Greymatter 博客平台的过程吗？**

我是严格为自己写的，只用了一周多一点的时间(这是我一生中最危险的狂躁的一周)。我基本上是边写边自学编码，上帝知道这是显而易见的。(从那次经历中，我确实成为了一名更好的程序员，但我无法表现出来，除了我想给它的全部重写之外，我最终无法表现出来。)Greymatter 从一个我和几个超级极客朋友分享的东西变成了一个我没有力量维持很久的东西，它的崩溃和最终从网络的记忆中抹去(我是这么想的！)给我留下的创伤让我花了很多很多年才接受。[…]

这显然进入了高度个人化的领域，但我真的不介意——在我近 30 年的博客生涯中，我已经写了很多次了。反击围绕这一问题和相关问题的污名(例如作为一名男性强奸幸存者)对我来说非常重要，甚至在我做 Greymatter 的几年前，我一直希望这是我一生中做的一件积极的小事。我患有双相情感障碍、自闭症、强迫症、创伤后应激障碍、AD(H)D 和慢性疲劳综合征。[……]除了在加利福尼亚和爱尔兰生活过几年之外，我大部分时间都待在德克萨斯州，与世隔绝。所以我很自然地把上网当成一种逃避，一种离开我的世界的方式，当我处于最佳精神状态时，我可以用语言和思想来评判我。

而且——最重要的是——我发现我并不孤单，因为我所生活的一切，我所想的一切，都有一个人——事实上是许多许多了不起的人——知道并生活在同样的事情中，他们理解、倾听和倾听，他们需要和我一样被倾听和倾听……包括都柏林的一个名叫巴里的男人， 当我跨越半个地球去和他在一起并和他结婚时，他成了我一生中最幸运的事，而他的突然死亡成了最糟糕的事。 (他是一名患有癫痫&多发性硬化症的截瘫患者，在睡梦中死于癫痫发作[……])

从那以后的 15 年里，我一直与世隔绝。几年没有本地朋友，这几年除了医疗原因，一直到现在都没有离开过家。[……]我显然从一开始就有一个“受损”的大脑，但在八年前一次自杀未遂后(我试图上吊，套索滑落；我不记得它，也不知道我在上面呆了多久)，我一直在经历一种缓慢的、不同的精神退化。我大脑中能够处理代码的那部分已经…不再真正“工作”了。现在对我来说，这些都只是字母数字的汤，甚至看起来都很痛苦。(这是一件非常苦乐参半的事情，要为我再也不能做的事情庆祝。[…] )

自从 Greymatter 之后，我再也没有写过任何完整的软件，我根本无法告诉你我最后一次写代码是什么时候，我的大脑损伤已经影响到了我，以至于(除了许多其他事情之外)即使我尝试过，我也不能再写了。但它促使我将自己完全交给摄影和艺术，而这些即使不那么有影响力，也是更有成就感的途径——我最终的自我治疗。我没有选择绕道，但从长远来看，我在那里结束得更好。(我认为 WordPress 给互联网带来的好处比我给互联网带来的好处要多得多)。

现在我拥有了这一切，这给了我如此多的疗愈和与自己过去的和解。对于我生活和内心的这个方面，我想我终于可以说，我不愿意和它在一起，也不愿意和我现在在的地方在一起。[……]我在这里的小世界是我的基地，我试图从这里保持我的头脑和保持创造性活动，这一直是我最大的自我治疗。[……]而且……现在我的家安全了，我真的想找到一种方法回到外面的世界，一步一步地(或者可以说是转动轮椅)。[…]

我想，在我上网这么久之后，互联网不会再给我带来惊喜了……！

在我们的采访中，格雷强调了他无比的感激之情，但同时也将他的思绪转向了“无数挣扎和弱势的人们，他们的处境和我一样糟糕甚至更糟。”】

没有人应该*需要* GoFundMe，或者做了“重要的事情”或者得到令人敬畏的名人的支持。每一个人都值得爱、同情和尊重，除了那些想从其他人那里夺走这些的人。

就在此刻——正如我在上面一遍又一遍说过的——我忍不住相信人类的善良，相信我们能够并且必须彼此相爱。没有它，这个世界不会成功——就像我今天不会在这里成功一样。
