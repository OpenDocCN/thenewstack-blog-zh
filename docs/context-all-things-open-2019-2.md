# 上下文:KubeCon+CloudNativeCon 北美 2019

> 原文：<https://thenewstack.io/context-all-things-open-2019-2/>

Portworx 赞助了 New Stack 对 KubeCon+CloudNativeCon 北美 2019 的报道。

我的 KubeCon(现为“KubeCon+CloudNativeCon”)是如何成长的！我们记得四年前，当会议在西雅图一家酒店的二楼举行时，你可以在几分钟内参观完所有的展位。今年，超过 12，000 名与会者挤满了圣地亚哥会议中心，几十个摊位横跨三个巨大的房间。你需要一张地图来找到你要找的地图。我们在 2016 年遇到的许多公司已经成长为成熟的企业。多亏了 Portworx 的赞助，我们才能够派遣一队优秀的记者去捕捉展会上首次亮相的许多产品更新。

在本周的 [The New Stack Context](https://thenewstack.io/podcasts/context) 播客中，我们将讨论其中的一些故事以及事件本身。在这一集里，我们邀请到了 Alex Williams，The New Stack 的创始人兼主编，以及 TNS 的记者 Emily Ormier 和 TNS 的执行主编 Joab Jackson。TNS 编辑和营销总监 Libby Clark 担任主持人。

我们谈到的话题包括将 Kubernetes 集成到企业中的新问题、保护容器的持续挑战，以及服务网格是否可以投入生产使用。我们还分享了我们在活动中的个人亮点，讨论了大会为儿童提供的免费日托(竖起大拇指！)和凯尔西·海托华苦乐参半的告别基调。
