# DigitalOcean 首次推出采用英特尔至强可扩展处理器的高 CPU 水滴

> 原文：<https://thenewstack.io/digitaloceans-4x-high-cpu-servers/>

游戏玩家、数据分析师、应用服务器开发人员和持续集成维护人员有什么共同点？对速度的需求。总部位于纽约的云服务提供商数字海洋(DigitalOcean)今天通过其[新型高 CPU 服务器](https://blog.digitalocean.com/introducing-high-cpu-droplets)发挥了这一潜力。

根据产品副总裁 [Shiven Ramji 的说法，为了响应大量开发者的请求，数字海洋设计了这些新的 droplet(数字海洋虚拟服务器)，在 CPU 性能方面比标准 droplet 快 4 倍。中央处理器(CPU)负责执行计算机程序的指令，解释和执行程序命令。](https://www.linkedin.com/in/shivenramji/)

“这是对需要更新、更高性能 CPU 的开发者的回应，”DigitalOcean 的首席技术官 Julia Austin 说。"这些微滴由英特尔最强大的处理器支持，以提供最高的 CPU 性能."

在英特尔最新一代处理器的驱动下，[英特尔至强可扩展处理器](https://thenewstack.io/intel-updates-xeon-processor-artificial-intelligence-work/)与 DigitalOcean 自己的软件和配置代码相结合，提供了速度。今年早些时候，奥斯汀宣布了一项雄心勃勃的为期一年的升级时间表，旨在使数字海洋产品更易于大规模使用，此后一组工程师开始研究该产品。

Ramji 说，内部团队建立在核心 Droplet 产品的简单性上。他们的承诺是，高 CPU Droplet so 将连接到所有其他 DigitalOcean 产品，并保持其当前 Droplet 的 55 秒供应窗口。

Ramji 说，今年早些时候，他们推出了一款[高内存 Droplet](https://blog.digitalocean.com/high-memory-droplets/) ，这是下一步旨在使 CPU 密集型工作流更加高效的举措。虽然标准 droplet 在 RAM、CPU 和基于本地 SSD 的存储之间保持了健康的平衡，但高内存 droplet 旨在运行大规模数据库或分布式内存缓存。

但他解释说，一些工作负载只是需要更多的 CPU 能力。例如，游戏服务器、广告交换、密集队列/批处理、高度活跃的前端应用服务器和密集的数据处理，都可以使用多个专用的高 CPU 微滴来提高性能。

Ramji 说，应用服务器在商业中日益普遍，需要大量的 CPU 能力。额外的功率也使得持续集成流水线更高效。某些数据分析也是一个新兴领域，需要更多的计算能力。

英特尔数据中心产品管理高级总监 Jennifer Huffstetler 表示:“借助全新的英特尔至强可扩展处理器，DigitalOcean 能够为其客户的关键工作负载提供行业领先的性能。“新的英特尔至强可扩展处理器提供了更多的内核数量和内存容量，提高了云数据中心的密度，同时提供了可扩展性和优化的性能/总拥有成本。”

DigitalOcean 有一个简单的定价计划，有五种选择，起价为每月 40 美元或每小时 0.060 美元，可从 2 个 vCPUs 扩展到 32 个 vCPUs，从 3 个扩展到 48 GB RAM，同时保持 20 GB 的本地 SSD 存储。

Ramji 说:“我们正在建设的主题是，我们希望为开发者消除任何和所有的摩擦，以便开始开发。”“我们的开发人员会很高兴在这个新环境中与数字海洋合作，”他说，称这是一次“愉快的经历”

这一最新功能是一年来发布的最新功能，包括[可扩展防火墙](https://thenewstack.io/digitalocean-adds-scalable-firewall-service-cloud-feature-portfolio/)、[负载平衡](https://thenewstack.io/digitalocean-upgrades-load-balancing/)、[水滴监控](https://thenewstack.io/digitalocean-now-provides-droplet-monitoring/)，所有这些功能都旨在使 DigitalOcean droplets 在大规模开发中对开发者有用。
