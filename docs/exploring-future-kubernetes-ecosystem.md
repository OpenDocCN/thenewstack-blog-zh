# 探索库伯内特生态系统的未来

> 原文：<https://thenewstack.io/exploring-future-kubernetes-ecosystem/>

[库伯内特生态系统的未来](https://thenewstack.simplecast.com/episodes/the-future-of-the-kubernetes-ecosystem)

在今天的《新栈制造者》节目中，TNS 创始人 Alex Williams 在 CloudNativeCon/KubeCon EU 采访了 Google 开发者倡导者 Kelsey Hightower 和 Deis 软件工程师 Michelle Noorali。Hightower 讨论了“永远不要停止学习”的必要性，并补充说他目前正专注于学习如何与使用 Kubernetes 平台的人产生共鸣。与去年的活动形成鲜明对比的是，Hightower 和 Noorali 都指出，2017 年 CloudNativeCon/KubeCon 欧盟会议不太关注运行 Kubernetes 集群的新奇性，而是更多地关注第二天的问题，如管理长期的集群安全，构建自己的工具包以与 Kubernetes 和运营商合作。
