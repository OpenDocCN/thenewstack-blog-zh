# 视频:开始使用 Pivotal 的云铸造堆栈

> 原文：<https://thenewstack.io/video-get-started-pivotals-cloud-foundry-stack/>

这是关于使用 Pivotal 的 Cloud Foundry 构建和部署云原生应用程序的四部分视频教程的第一部分，介绍性的。请在接下来的几天中再次查看其他部分…

开发和部署企业软件的旧的内部方法既复杂又缓慢。它们逐渐被一组新的自动化程度更高的服务所取代，这些服务通常位于云中。但是，如果你愿意的话，开发人员如何跟上这个新堆栈的速度呢？

别担心，我们会让你看到云的闪烁光标。

下面嵌入的视频是一个四部分系列的第一部分，该系列将于下周初播出，将指导用户如何开始使用 Pivotal 基于 [Cloud Foundry](https://www.cloudfoundry.org/) 的云服务。Cloud Foundry 可以运行亚马逊 Web 服务、微软 Azure、OpenStack 云、VMware 基于 vSphere 的云。

在这篇介绍性文章中，[Pivotal 技术营销总监 Michael Coté](https://twitter.com/cote) 提供了云服务架构的高级概述。Cloud Foundry 及其支持工具为开发人员提供了接口，因此他们不必担心配置数据库、中间件或在云中运行应用程序所需的其他服务。

作为一名开发者， [Coté](https://twitter.com/cote) 说，“你不需要担心管理你的应用程序下面的一切”。
