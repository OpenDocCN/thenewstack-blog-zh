# Linux 基金会推出 CommunityBridge，这是一个开源融资平台

> 原文：<https://thenewstack.io/linux-foundation-unveils-communitybridge-an-open-source-funding-platform/>

Linux 基金会推出了一个名为 [CommunityBridge](https://communitybridge.org/) 的[新平台](https://www.linuxfoundation.org/press-release/2019/03/the-linux-foundation-launches-new-communitybridge-platform-to-help-sustain-open-source-communities/)，旨在帮助开源项目获得开发、维护和保养所需的资金。

CommunityBridge 最初将向选定的项目、成员和组织提供早期访问，并且将免费向维护者和开发人员提供访问。Linux 基金会还表示，它将“承担任何平台的成本和通过该平台筹集的第一笔 1000 万美元的支付处理器费用”，以及“匹配任何向 CommunityBridge 项目捐款的组织…所有捐款组织的累计总额高达 50 万美元。”这一消息发布后，立即有[新闻](https://www.linuxfoundation.org/press-release/2019/03/linux-foundation-announces-funding-with-github-for-new-communitybridge-platform-for-developers/)报道称 GitHub 已经捐赠了 10 万美元作为启动资金。

该基金会表示，它将在 2019 年和 2020 年继续开发该产品，但它将立即推出一些基本产品:CommunityBridge Security 将提供“潜在漏洞和修复的透明度”，CommunityBridge People 将实现“导师和有兴趣参与项目和促进多样性的潜在学员之间的轻松联系。”

Linux 基金会的最新努力在 Twitter 上引起了开发者 Jessie Frazelle 和其他人的不同反应。Frazelle 首先承认 CommunityBridge 是一个“良好的开端”，但随后表示，该基金会本可以帮助 Open Collective，而不是“重新发明轮子”。

https://twitter.com/jessfraz/status/1105596780954243075

自称“OSS 麻烦制造者”的 Krzysztof cielak 同样对该基金会的最新声明吹毛求疵，而 Open Collective 欢迎 Linux 基金会参加派对，并在推特上说“我们不参与竞争，而是愿意提供帮助。我们需要每个人让开源社区可持续发展。”

与此同时，红帽首席架构师 [Bilgin Ibryam](https://twitter.com/bibryam) 在[的一条推特](https://twitter.com/bibryam/status/1105808853692792832)中对 Community Bridge 表示欢迎，并表示他将把它“添加到 [oss.fund](http://oss.fund/) 的开源可持续发展平台的长长列表中”，这是一个共享的谷歌文档，列出了货币化平台、商业模式资源和开源项目的资助计划。

最后，[软件自由保护组织](https://sfconservancy.org)也在自己的博客上[发表了自己的想法](https://sfconservancy.org/blog/2019/mar/13/lf-community-bridge/)。这家非营利组织指出了自己和 CommunityBridge 等组织之间的三个主要差异，它称之为“一个雄心勃勃的平台，承诺一个自助服务系统来处理财务，解决安全问题，管理 CLAs 和许可证合规性，并为项目带来指导。”

“第一个巨大的差异(也是整个自由/开源软件社区最大的失望)是，LF 的 CommunityBridge 是一个专有的软件系统，”博客文章写道，并指出 Linux 基金会“不是一个慈善机构，而是一个贸易协会”，Community Bridge“类似于 Patreon 和其他平台，因为它是一个不干涉的系统，收取一部分钱，并提供最少的金融服务。”

Linux 基金会本周正忙于在加州半月湾举办开源领袖峰会。除了引入这个平台，该组织还推出了[新合作](https://www.linuxfoundation.org/press-release/2019/03/the-graphql-foundation-announces-collaboration-with-the-joint-development-foundation-to-drive-open-source-and-open-standards/)、[新](https://thenewstack.io/continuous-delivery-foundation-launches-with-jenkins-jenkins-x-spinnaker-tekton/)和[合并](https://www.linuxfoundation.org/press-release/2019/03/node-js-foundation-and-js-foundation-merge-to-form-openjs-foundation/)基金会。

Linux 基金会和 Red Hat 是新堆栈的赞助商。
