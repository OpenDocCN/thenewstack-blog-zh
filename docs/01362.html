背景:万物开放2019

原文：https://thenewstack.io/context-all-things-open-2019/#0001-01-01

VMware 赞助了这次播客。

在本周的 The New Stack Context 播客中，我们将讨论我们最喜欢的年度活动之一，即本月早些时候在北卡罗来纳州罗利举行的 All Things Open 开源会议。

我们本周的嘉宾有 Matthew Broberg ，红帽OpenSource.com的技术编辑，以及Anais dot is-乔治乌，他是 InfluxData 的开发者倡导者，也是会议的发言人。我们讨论了让所有事情开放如此令人愉快的草根独立性，以及我们在今年的展会上捕捉到的开源趋势。


上下文:万物开启2019

然后，在节目的稍后部分，我们讨论了本周的一些热门帖子和播客，包括对Pivotal的Cornelia Davis的广泛采访，关于如何充分利用Ubuntu上的ZFS文件系统的教程，微软如何让开发人员更容易使用Kubernetes ，以及在最近一波基于CPU的漏洞之后Linux内核开源的状态。

TNS的编辑和营销总监Libby Clark主持了这次播客，一同主持的还有TNS创始人兼总编辑Alex Williams和TNS执行主编Joab Jackson。

InfluxData、Linux基金会、Pivotal和Red Hat是新堆栈的赞助商。
