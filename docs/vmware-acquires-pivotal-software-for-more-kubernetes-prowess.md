# VMware 收购 Pivotal 软件以增强 Kubernetes 的实力

> 原文：<https://thenewstack.io/vmware-acquires-pivotal-software-for-more-kubernetes-prowess/>

虚拟化技术先驱 [VMware](https://www.vmware.com/) 正在收购 [Pivotal](https://pivotal.io/) ，后者为专注于敏捷软件开发的内部编程团队提供云平台托管、应用程序设计环境和咨询。

为什么会发生这种合并？一句话… Kubernetes。

“Kubernetes 正在成为多云现代应用的事实上的标准。VMware 首席执行官帕特·基尔辛格表示:“我们很高兴将 Pivotal 的开发平台、工具和服务与 VMware 的基础设施能力相结合，提供全面的 Kubernetes 产品组合来构建、运行和管理现代应用。

在过去几年中，VMware(像许多公司一样)一直非常重视他们的 Kubernetes 足迹。2018 年 2 月，VMware 和 Pivotal 共同推出了企业 Kubernetes 平台[VMware Pivotal Container Service](https://pivotal.io/platform/pivotal-container-service)(PKS)。此次收购源于依赖 Pivotal Cloud Foundry 及其平台即服务以及 VMware 的客户。这两家公司的协同作用是显而易见的，尽管并非没有挑战。今年 6 月，该公司被投资者起诉，原因是未能在该公司 2017 年首次公开募股中披露该公司的平台“与 Kubernetes 不兼容”，[报道称。](https://www.theregister.co.uk/2019/07/25/pivotal_ford_write_down/)

在收购 Pivotal 之前，VMware 收购了由两位最初的 Kubernetes 开发人员创建的 [Bitnami](https://bitnami.com/) (一个用于 web 应用程序和开发堆栈的安装程序库)和 [Heptio](https://heptio.cloud.vmware.com/) 。Heptio 是另一家专注于帮助企业级公司接受 Kubernetes 的创业公司。根据 VMware 的说法，收购 Bitnami 是为了“加速 multicloud 和 Kubernetes 的应用交付，并扩大 Bitnami 在企业中的采用。”

全球 2000 强公司(全球最大的上市公司)都在使用 Pivotal 的技术，其目标是在 DevOps 和 IT 运营方面实现战略优势。“我们两家公司已经在我们成功的 VMware PKS 合作基础上建立了坚实的基础。Pivotal 首席执行官 Rob Mee 谈到此次合并时说:“我们期待与 VMware 继续合作，为构建现代应用的客户提供更多价值。截至 2 月 1 日，Pivotal 共有 2，949 名员工。该公司成立于 2012 年，是 EMC 和 VMware(由 EMC 共同拥有)的子公司。

交易详情如下:

*   VMware 将支付 8 亿美元现金。
*   戴尔(VMware 的大股东)现在将拥有 VMware 81.09%的股份(作为交易的结果)。
*   每股混合价格为 11.71 美元。
*   公众股东每股 15 美元现金。
*   企业价值 27 亿美元。
*   Pivotal 的 B 类普通股股东戴尔科技公司将获得约 720 万股 VMware 类普通股。
*   VMware 的预期净现金支出为 8 亿美元。
*   声明发布后，Pivotal 的股价上涨了近 8%。
*   声明发布后，VMware 股价下跌了 7%。
*   自上市以来，Pivotal 上一财季亏损 3170 万美元。

VMware 周四还宣布以 19 亿美元收购云安全公司 Carbon Black。

Pivotal 和 VMware 是新体系的赞助商。
