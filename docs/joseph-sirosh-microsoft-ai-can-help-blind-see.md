# 微软的约瑟夫·西罗什:人工智能如何帮助盲人看见

> 原文：<https://thenewstack.io/joseph-sirosh-microsoft-ai-can-help-blind-see/>

[微软的约瑟夫·西罗什:人工智能如何帮助盲人看清](https://thenewstack.simplecast.com/episodes/joseph-sirosh-of-microsoft-how-ai-can-help-the-blind-see)

微软正在使其客户更容易利用其人工智能能力。

在本月早些时候的 Microsoft Connect 上，我们与微软[云人工智能平台](https://www.microsoft.com/en-us/ai)的公司副总裁 [Joseph Sirosh](https://www.linkedin.com/in/joseph-sirosh-39803b1/) 坐在一起，了解更多关于微软人工智能技术的信息，包括机器学习、认知服务、机器人框架，以及该公司如何为开发人员带来所有这些能力，以通过人工智能在云中构建应用程序。

在这一集的 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客中，西罗什解释了机器学习、人工智能和深度学习在我们在科幻电影中看到的反乌托邦描述之外意味着什么。

行业中一个有趣的发展是处理和数据正在向边缘发展，以更好地服务于物联网设备。在活动期间，Sirosh 展示了一项技术，该技术将皮肤癌模型部署在移动设备上，可以使用移动设备扫描皮肤。

“你将看到云被用于聚合大量数据，然后使用大量 GPU 和云计算进行学习，然后学习的模型被投入到低延迟的手机、物联网边缘设备的生产中，”西罗什说。

他还给出几个 AI 帮助解决现实世界问题的真实生活用例。他举了一个人工智能应用程序的例子，该应用程序可以帮助盲人识别他们的环境，识别物体，并通过人工智能的眼睛识别他们周围的世界。他还谈到了一个帮助自闭症儿童交流的应用程序。

人工智能在医疗保健领域有巨大的机会。微软的一个合作伙伴可以通过电子病历数据提前两年预测高血压。

我们也谈到了 AI 是否会反人类，毁灭我们所有人。观看整个采访，听听西罗什对人工智能的看法。
