# 林登实验室的 Sansar 项目将虚拟现实带入了第二人生

> 原文：<https://thenewstack.io/linden-labs-project-sansar-brings-virtual-reality-second-life/>

[项目 Sansar 将 VR 带入第二人生](https://thenewstack.simplecast.com/episodes/project-sansar-brings-vr-to-second-life)

在《新堆栈制作者》播客的最新一集里，[塔拉·埃尔南德斯](https://www.linkedin.com/in/tara-hernandez-755b263/)，[林登实验室的](https://www.lindenlab.com/)高级工程总监，[陈箓](https://www.linkedin.com/in/luchengw/)，林登实验室数据分析师与主持人 T.C .柯里一起谈论目前处于测试阶段的第二生命的下一代项目 Sansar。

埃尔南德斯解释说，虽然“第二人生”是一种身临其境的体验，但它不是像使用真正的虚拟现实(VR)那样的完全身临其境的体验。 [Project Sansar](https://www.sansar.com/) 更专注于社交虚拟现实，这意味着你与其他人的互动更多，而不是复杂的内容。她最喜欢的体验，刚刚向公众发布，是好莱坞体验，其中包括一个早期星球大战材料的收集，你可以拿起并与之互动。

了解 Sansar 背后的堆栈中有什么，奇迹发生在哪里，一旦 Sansar 退出测试，开发团队如何计划扩展大量的预期数据，以及多元化的团队如何使产品变得更好。
