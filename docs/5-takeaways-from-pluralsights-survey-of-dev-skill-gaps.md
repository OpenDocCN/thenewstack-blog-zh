# 从 Pluralsight 的开发人员技能差距调查中获得的 5 点启示

> 原文：<https://thenewstack.io/5-takeaways-from-pluralsights-survey-of-dev-skill-gaps/>

根据一项新的调查，在新冠肺炎疫情的推动下，2020 年远程工作的突然转移揭示了开发人员技能的严重缺口。

科技教育公司 [Pluralsight](https://www.pluralsight.com/) 的年度报告衡量了科技劳动力的技能提升状况。2021 年的研究着眼于在家工作对开发人员技能和对这些能力的信心的影响，以及缩小技能差距的可用资源。

2021 年的研究汇编了来自 7 个国家的 600 多名技术高管和从业者的回复。一半的调查参与者在北美；总体而言，三分之一的受访者在其组织中处于主任或主任以上级别。

其中包括:

**1。网络安全和云计算技能是最大的障碍。**

39%的调查参与者表示，他们感觉在网络安全和云计算领域缺乏足够的技能。几乎同样多的受访者表示缺乏数据存储(35%)和网络基础设施(34%)方面的能力。

**2。现在很少有技术专家对自己的工作有信心。**

在目前的报告中，对当前工作有信心的 IT 员工比例从 2020 年的 84%降至 71%。(在美国受访者中，下降幅度较小——87%至 80%——但仍然很大。)

当被问及他们对三年后拥有工作技能是否有信心时，差距较小:2020 年，82%的人表示他们会有信心，新报告中这一比例降至 76%。(在美国的调查参与者中，2020 年有 85%的人表示他们会感到自信，2021 年仅下降了两个百分点。)

**3。技术专家对他们的培训资源的满意度远低于他们的老板。**

88%的 C 级高管表示，他们对所在组织的教育和培训计划感到满意，而只有 56%的个人参与者表示满意。

报告指出，这种巨大的差距应该成为高管们的一个警告:“技术领导者需要明白，他们的许多技术专家和低层管理人员面临着比最高管理层更具挑战性的时期。为团队成员提供有意义的技能发展支持有助于改善技术人员的工作环境。”

**4。少数组织允许将工作时间用于提高技能。**

尽管 96%的受访者表示他们的公司提供培训资源，但只有 42%的人允许技术人员将工作时间用于学习。在过去的 12 个月里，只有 31%的受访者利用工作时间进行培训。

**5。从下一份工作开始，科技工作者寻求职业发展。**

60%为科技公司工作的从业者表示，他们希望在下一份工作中获得职业发展和学习的机会。相当一部分受访者(54%)还表示，运用新兴技术的能力将是选择新工作的关键因素。相比之下，更少的人(46%)表示，有竞争力的薪酬和福利是选择下一份工作的关键因素。

在非科技公司的 IT 从业者中，职业发展仍然是他们下一份工作的高度期望(47%)，但几乎同样多的人优先考虑薪酬(44%)。只有 37%的人表示，与新兴技术打交道将是他们选择下一份工作的一个因素。

“如果你想吸引和留住人才，这些人才需要看到围绕职业发展的正式努力和对职业发展的承诺，”Pluralsight 开发人员技能副总裁唐·琼斯(Don Jones)说。“在不景气时期，公司往往会削减职业发展之类的事情，但老实说，继续职业发展比招聘和吸纳新人才成本更低。”
