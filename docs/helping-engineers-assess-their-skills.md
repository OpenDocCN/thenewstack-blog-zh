# CodeFights:一个测试编码技能、准备面试和申请工作的平台

> 原文：<https://thenewstack.io/helping-engineers-assess-their-skills/>

找一份工程师的工作从来都不是一件容易的事，其中一部分是与公司面试的严格且有时脱节的过程。您并不总是知道从编码练习中可以期待什么，或者他们将试图覆盖什么样的主题。CodeFights 已经成为程序员和工程师评估他们的技能、展示他们的编程价值以及实际上直接应用于一些不可思议的软件公司的最佳和唯一的解决方案之一。

[CodeFights 的联合创始人兼首席执行官 Tigran Sloyan](https://www.linkedin.com/in/tigransloyan) 在本期 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客中向我们讲述了基于 code fights[游戏化](http://www.bunchball.com/gamification)的过程如何工作，它如何帮助工程师提高他们的技能，以及 code fights 如何将用户与渴望从他们的玩家群中招聘的公司直接联系起来。
