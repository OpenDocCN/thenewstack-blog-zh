# 上周的新堆栈:DockerCon，无服务器业务

> 原文：<https://thenewstack.io/last-week-on-the-new-stack-dockercon-serverless-business/>

[本周新上架:DockerCon，无服务器业务](https://thenewstack.simplecast.com/episodes/this-week-on-the-new-stack-dockercon-serverless-business)

大家好，欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)，在这个播客中，我们回顾了本周云原生技术/大规模应用程序开发的最热门新闻，并展望了我们预计将在未来几周获得更多关注的主题。

上周在 Context 中，我们采访了 New Stack 记者 Mark Boyd，他在本周写了关于业务流程和文化决策的文章，这些在构建无服务器应用架构中扮演了重要角色。

包括功能即服务在内的无服务器计算是一项快速发展的技术，在公共云中运行其应用程序的公司中，企业采用该技术的程度与容器不相上下。分布式系统的专家和远见者认为，无服务器是应用基础设施的未来。

TNS 的读者和听众也在 12 月份的调查中告诉我们，无服务器是您最近最感兴趣的话题之一。因此，TNS 最近推出了一个新的功能系列[关注每周一运行的无服务器技术。马克·博伊德正在为我们写几篇这样的文章——第一篇上周发表了。](/category/serverless/)

此外，TNS 创始人亚历克斯·威廉姆斯正在旧金山参加 DockerCon。他给了我们一些那次活动的新闻和亮点。TNS 执行主编 Joab Jackson 讨论了我们网站上的一些其他有趣的故事。
