# EBay:押注开源对你的生意有好处

> 原文：<https://thenewstack.io/ebay-betting-on-open-source-is-great-for-your-business/>

EBay 受益于大量使用开源代码，我们并不孤单。据估计，现代应用程序 90%以上都是用开源代码构建的。但是开源的合作不止于此。我们有多个组织(包括我们的内部开发工具组织以及我们的数据和云基础设施团队)将开源代码作为他们工作的核心职能。

我们的贡献不是无私的。押注开源是一个伟大的商业决策。我们已经看到，当使用开源工具时，由于已经接触了 React、Kubernetes、Docker 等工具，开发人员可以更快地上手。我们还可以从社区贡献中获益，社区贡献可以为我们发现漏洞和增加功能。

## 易贝在开源贡献中的位置

早在 2011 年，EBay 就通过致力于面向服务的架构框架姜黄开始构建开源工具。自从易贝首席技术官 [Mazen Rawashdeh](https://www.linkedin.com/in/mazenrawashdeh/) 在 2016 年加入以来，我们增加了对开源的采用。

这发生在堆栈的所有层，包括最近迁移到开源网络硬件。2022 年，我们开源了至少 15 个新的存储库，跨越了多个感兴趣的领域，从 [UI 组件](https://github.com/eBay/nice-modal-react)到[高性能 C++数据结构](https://github.com/eBay/sisl)到[复杂的 Java 工作流引擎](https://github.com/eBay/task-graph-toolkit)。

这不仅仅是贡献新代码，还包括改进现有生态系统的代码。今年早些时候，我们改进了 [Apache Maven 的](https://maven.apache.org/)依赖关系解析算法，使得构建时间减少了 70%。这些节省不仅局限于我们的应用程序，还获得了阿里巴巴集团的积极反馈。

我们还是“OpenFeature”的核心贡献者，这是一个旨在为非常繁忙的供应商空间提供公共客户端的库。

EBay 在 2020 年做了一些分析，围绕购买哪种功能标记解决方案，我们最终决定建立自己的解决方案。为了消除这一赌注的风险，我们与多家其他公司合作建立了一个开放标准，名为 [OpenFeature](https://openfeature.dev/) 。在这个过程中，我们编写了大部分的 Java 客户端，并且我们正在积极地在内部部署它以用于生产工作负载。该项目位于[云本地计算基金会](https://cncf.io/?utm_content=inline-mention)之下，该基金会为多个供应商提供了一个“安全地带”，无需争夺所有权。

这些回馈社区的贡献对我们公司的成功至关重要。在易贝，我们喜欢说，我们把人们联系起来，建立社区，为所有人创造经济机会。

这是我们经营业务的核心要素。免费为奇妙的开源产品做贡献让我们能够与在我们的领域做着杰出工作的人们建立联系。除此之外，易贝的规模让我们看到了其他公司可能还没有遇到的问题。我们无疑受益于其他超大规模公司的知识和软件，如谷歌、脸书等！

在易贝，我们将开源视为我们公司的一个增长领域。我们致力于增加我们对现有开源解决方案的贡献，并一直直接资助我们认为对生态系统至关重要的开源事业。我们期待加深与更广泛的生态系统的关系，并渴望看到更广泛的企业参与。
