# OpenJS 基金会为电子桌面 JavaScript 框架做了一个家

> 原文：<https://thenewstack.io/openjs-foundation-makes-a-home-for-the-electron-desktop-javascript-framework/>

本周，在蒙特利尔举行的 [Node+JS Interactive](https://events19.linuxfoundation.org/events/nodejs-interactive-2019/) 会议上，OpenJS 基金会进一步扩展了其技术清单，增加了[电子](https://github.com/electron/electron)，这是一个基于 Node.js 和 Chromium 的开源框架，将这些技术带到了桌面，允许使用 JavaScript、HTML 和 CSS 开发桌面应用。

“当我们宣布 Electron 将加入时，人群变得狂热起来。在会议上，每个人都爆发出热烈的掌声，”[OpenJS 基金会的执行董事罗宾·银](https://twitter.com/rginn206)告诉 New Stack。“乡亲们都很激动。”

GitHub 于 2013 年首次开发了 Electron，旨在使用单一语言 JavaScript 轻松构建可在 Windows、Mac 和 Linux 电脑上运行的桌面应用程序。虽然该项目是在 GitHub 的指导下开始的，但银表示，最近它已经采取措施提供更开放的治理，其对 OpenJS 的捐赠显然是下一步。

“它确实变成了一种由许多开发人员广泛维护的项目。它从去年开始转向开放式管理，”银说。“这真的有助于他们正式制定决策，使其不仅仅是一个由单一实体拥有的项目。对他们来说，进入基金会是很自然的一步。”

Electron 作为一个相当成熟的项目以孵化器的形式加入了基金会，有许多广为人知的采用者，如 Discord、Microsoft Teams、OpenFin、Skype、Slack、Trello 和 Visual Studio Code。正如银所指出的，作为一个孵化项目，时间可能是有限的，与入职相比，它必须做更多的事情。

“加入基金会的每个项目都是从孵化开始的，这实际上只是简单的日常管理。当你加入一个基金会时，会有一些事情，比如把你的 IP 从网站上移走。只是需要一点时间。这并不意味着他们是一个小项目，”银解释说。"一旦他们经历了孵化过程，他们将成为一个有影响力的项目."

银还解释说，成员项目受益于 OpenJS 基金会的各种服务，其理念是项目可以更容易地协同工作，这是成员的核心原则。

“我们建立了自己的模型，为我们的项目提供了强有力的声音。银说:“OpenJS 基金会的工作人员从营销、法律、项目管理和活动角度支持这个项目。“我们真的希望这些项目能够独立运作，有自己的治理，同时作为我们家庭的一部分。我们消除了这种摩擦，这样他们就可以专注于发展他们的项目，增加他们的贡献者。”

Linux 基金会是新堆栈的赞助商。
