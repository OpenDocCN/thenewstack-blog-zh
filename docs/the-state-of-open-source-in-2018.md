# 2018 年企业开源状况

> 原文：<https://thenewstack.io/the-state-of-open-source-in-2018/>

[# 170:Pivotal 董事、RedMonk 分析师讨论 2018 年开源状态](https://thenewstack.simplecast.com/episodes/170-a-pivotal-director-redmonk-analyst-discuss-the-state-of-open-source-in-2018)

很难预测开放源码在今天的平台中的作用有多大，以及由于它的广泛采用，在今天的计算世界中提供的选择会有多大。在行业方面，IBM 今年以惊人的 340 亿美元收购了 Linux 巨头 Red Hat，这是一个更大的惊喜。

在本播客中，我们与 [RedMonk](https://redmonk.com/) 的分析师 [Rachel Stephens](https://www.linkedin.com/in/rachelstephens/) 和 [Pivotal Software](https://pivotal.io/) 的营销总监 [Michael Coté](https://www.linkedin.com/in/michaelcote) 讨论了开源在 2018 年的状况，尤其是 IBM 对 Red Hat 的收购，该播客由编辑总监 Libby Clark 和 New Stack 的创始人兼主编 Alex Williams 主持。

斯蒂芬斯说，事实上，2018 年甚至在许多圈子里被吹捧为“开源年”。“大规模的收购往往会真正验证开源是未来的构建方法，也是构建堆栈的可行方法。我认为，与此同时，我们将这与地平线上的某种云进行了对比，因为运营开源业务的能力与面对云提供商的能力之间的紧张关系越来越大。”

Stephens 表示，Red Hat 是一个公认的收购目标，但 IBM 为该公司支付的 340 亿美元的价格令许多人感到惊讶。

“我认为 IBM 付出的代价是巨大的，我认为看到它如何走到一起会很有趣。斯蒂芬斯说:“如果你将开源精神和走向市场的云机会结合起来，就像红帽与 OpenShift 和 IBM 的销售机器结合起来一样，我认为这可能是件好事。”。“还有相当大的风险，例如投资组合的重叠，这肯定是一些问题，特别是在文化契合度方面。所以，我认为观察它如何发展会很有趣。”

同时注意到 Pivotal 与 Red Hat 竞争，并且“尽量不要有偏见；Coté表示，Red Hat“似乎一直是一笔不错的收购，但它们对任何人来说都太大了，没有理由收购。”

“他们有一套广受尊重的赚钱工具……我想，在 Pivotal 工作时，我总是非常关心一大堆新式的 Kubernetes 云。但我认为，从 IBM 的角度来看，如果有一家公司知道如何在几十年内从操作系统中赚很多钱，”Coté说。“所以，回想起来，这确实是一种协同效应，就像 IBM 销售操作系统一样。红帽公司的大部分收入来自一个难以置信的大规模安装环境中的操作系统，它需要长期的支持和其他东西。”
