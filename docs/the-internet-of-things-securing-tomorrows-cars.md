# 物联网:保护明天的汽车

> 原文：<https://thenewstack.io/the-internet-of-things-securing-tomorrows-cars/>

[获得物联网世界最快超级汽车](https://thenewstack.simplecast.com/episodes/securing-iot-of-the-worlds-fastest-super-car)

Brandon Shibley 是一家制造基于 Arm 的系统级模块(SoM)的公司的创新副总裁。SoM 用于工业自动化、测试和测量、数字标牌、汽车、医疗等专业市场。Toradex SoM 为世界上最快的电动汽车 Rimac Concept One 提供动力，该车采用了 [Apalis iMX6](https://developer.toradex.com/products/apalis-imx6) 。

“我们专注于提供具有长生命周期支持的工业级硬件解决方案。我们还为这些模块系统提供软件支持。

当我们谈论这种物联网设备时，安全成为一个严肃的话题；这不再是事后的想法。Toradex 不仅仅提供 SoM 解决方案，他们倾向于解决随之而来的所有需求，包括提高安全性。

但是安全性并不像看起来那么简单，因为有不同的部分粘在一起——在这些模块上运行的操作系统、硬件组件和应用程序。随着您添加更多的特性和功能，复杂性也会增加。

“以前，不定期更新的设备更容易被接受，但我们现在看到，绝对需要在现场进行更新，以解决安全问题或其他漏洞，”Shibley 说。
