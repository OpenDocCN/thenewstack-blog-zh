# GopherCon India:我们加入了

> 原文：<https://thenewstack.io/gophercon-india-were-in/>

很高兴地说，我们是明年二月在班加罗尔举行的印度 GopherCon 的媒体赞助商。为什么要支持印度的一个 Go 编程语言赛事？很简单——新的堆栈技术及其发展中的文化不仅仅存在于美国西海岸。更重要的是，开发和管理后端系统的新堆栈方式是一场全球性的运动，我们希望成为其中的一员。

更进一步。Go 正在成为一种重要的编程语言，我们乐于追随它。这是一个有趣且包容的社区，我们希望更好地了解它。

一个好的演讲者阵容正在为这次活动而发展。例如，CoreOS 的 Blake Mizerany 正在发言。Heroku 的首批雇员之一 Blake 最近为新堆栈写了一篇关于分布式键值存储的文章。

所以拿到旅游签证吧，也许我们会在今年二月在班加罗尔见到你。
