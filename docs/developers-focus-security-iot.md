# 为什么物联网开发者应该关注安全

> 原文：<https://thenewstack.io/developers-focus-security-iot/>

在本期[新堆栈制造商](https://thenewstack.io/podcasts/)播客中，我们将探索物联网如何影响航空业。前 [M2Mi](http://www.m2mi.com/) 首席运营官[莎拉·库珀](https://www.linkedin.com/in/drsarahcooper)(她后来成为亚马逊[物联网解决方案总经理](https://aws.amazon.com/iot/how-it-works/))解释了航空业如何发展和转变，部分归功于物联网运动带来的革命，以及行李的 WiFi 信号传输传感器等新技术。

主持人[凯尔·麦克唐纳](https://twitter.com/kylemacdonald)和[李·卡尔科特](https://twitter.com/lcalcote)在 2016 年云铸造峰会期间采访了库珀，听听她的想法。

[为什么开发者要关注物联网的安全性](https://thenewstack.simplecast.com/episodes/why-developers-should-focus-on-security-in-iot)

采访也可以在 YouTube 上听到[。](https://www.youtube.com/watch?v=cadcL6xCH8w)

十多年来，M2Mi 一直在开发软件，为用户提供各种工具和平台，使开发人员不仅可以加强现有的基础设施，还可以专注于保护基础设施。

在最近与美国国土安全部的合作中，M2Mi 开发了一种被称为 [SPECK suite](http://www.m2mi.com/about/blog/entry/the-speck-suite-the-m2mi-department-of-homeland-security-s-t-iot-security-suite) 的产品，这是一种开源的物联网分组密码，旨在超越更传统的 AES 加密方法，同时又足够紧凑，适用于当今更纤薄的设备。当 MacDonald 问今天的物联网开发人员应该关注什么时，Cooper 恳求道，“如果我能站在讲台上，就关心安全性。”

虽然物联网正在迅速占据主导地位，但批评者很快提醒我们物联网的最糟糕之处，特别是当谈到家庭中的物联网时。“除非你了解决策的背景，否则很难实时做出决策。五是好还是不好？这取决于是室外的温度还是你冰箱里的温度。

其核心是，M2Mi 与工业航空合作伙伴合作，支持物联网如何从整体上改善航空体验。库珀解释说，通过将强大的数据纳入其平台，M2Mi 不仅可以更清楚地了解机场本身的健康状况，而且最终可能会帮助航空公司获得从乘客那里收集的数据并采取行动。

“我们在 M2Mi 模型中采用的是甲骨文的航空公司数据模型，并结合整个机场的信息，以确定谁的包是谁的，以及你有过什么样的飞行体验，”库珀说。
