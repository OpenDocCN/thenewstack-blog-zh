# 保罗·维谢:互联网英雄的故事

> 原文：<https://thenewstack.io/paul-vixie-story-of-an-internet-hero/>

保罗·维谢在旧金山长大。他在 1980 年从高中辍学。他在 DEC 开发了第一批互联网网关，并在那里创建了[互联网软件联盟](https://www.isc.org/) (ISC)，建立互联网协议，特别是[域名系统](https://thenewstack.io/researcher-hijacked-several-io-tld-nameservers/) (DNS)。

[保罗·维谢:一个互联网英雄的故事](https://thenewstack.simplecast.com/episodes/paul-vixie-story-of-an-internet-hero)

今天，Vixie 是技术世界中少数几个拥有“杰出工程师”头衔的人之一，他在亚马逊网络服务公司担任安全副总裁，他相信他可以让互联网成为一个更安全的地方。和互联网出现之前一样安全。

“我担心我们在互联网时代的安全性会比以前下降多少，”Vixie 在本月早些时候在都柏林举行的开源峰会上接受采访时说。“一切都是相互关联的，但我们却知之甚少。因此，在过去的 20 年里，我的使命就是将人类安全恢复到互联网出现之前的水平。大规模地做到这一点是一个巨大的挑战。这将花费我一生的时间。”

那么为什么要加入 AWS 呢？他花了几十年时间建立 ISC。他在 ISC 创办了一家名为远视的公司。去年 11 月，当与 AWS 开始对话时，他卖掉了 Farsight。

当 AWS 问了一个问题，改变了谈话并引导他到他的新角色时，Vixie 想到了他的使命，即更好地将人类安全恢复到互联网之前的水平。

“他们问我，现在回想起来，这是一个明显的问题，AWS 主机，可能是你试图保护的数字经济的最大份额，’”维谢说。难道你不认为你可以通过帮助保护 AWS 来完成你的使命吗？“答案是肯定的。事实上，我觉得我会得到更多的关注，因为我可以专注于战略和技术，而不是同时经营一家公司。所以对我来说这是一场非常好的胜利，我希望他们也是如此。”

采访维谢是我的荣幸。正是像保罗这样的人为任何使用互联网的人创造了如此多的可能性。请想一想——任何使用互联网的人都要感谢像保罗这样的人。

谢谢保罗——你是很多人的英雄。为你下次在 AWS 的表现干杯。
