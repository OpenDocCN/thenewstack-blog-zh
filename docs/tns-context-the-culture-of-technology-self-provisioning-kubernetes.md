[TNS 语境:技术的文化，自我供给 Kubernetes](https://thenewstack.simplecast.com/episodes/tns-context-the-culture-of-technology-self-provisioning-kubernetes)

本周在我们的每周新闻总结播客 [TNS Context](https://thenewstack.io/podcasts/context) 中，我们采访了我们的科技文化记者大卫·卡塞尔，他每个周末都在[撰写关于科技和互联网更奇怪的文化边缘的故事。如果你读过杰里·埃尔斯沃斯自制芯片的故事，上帝的孤独程序员特里·戴维斯的麻烦遗产，或者 grep 的起源，那么你已经加入了卡塞尔的网络之旅。](https://thenewstack.io/author/destiny/)

然后，在节目的稍后部分，我们将讨论 Kubernetes kubeadm 工具的[可用性，开发者希望该工具将有助于 Kubernetes 供应集群的方式标准化。如果管理员手动完成这项工作，将会涉及大量的权限设置和证书创建。有一些软件包，如警察或牧场主，可以缓解这种推出。Kubeadm 不会与这样的产品竞争。在最好的 Unix 风格中，kubeadm 的范围非常狭窄，只做一件事:集群部署和升级。](https://thenewstack.io/kubernetes-now-does-self-hosting-with-kubeadm/)

TNS 编辑部主任 Libby Clark 主持了这一集，TNS 创始人 Alex Williams 和 TNS 执行主编 Joab Jackson 提供了帮助。

## 故事

*   [**Flux: InfluxData 针对时序数据的新语言**](https://thenewstack.io/flux-influxdatas-new-language-for-time-series-data/) :我们使用 SQL 进行数据库查询已经有几十年了，但是流数据需要一种新的语言。这就是为什么流入数据发布了一个新的声明式查询语言，只是为了流数据。这种语言叫做 Flux，除了标准的查询之外，它还允许你做更复杂的分析、数据修改和 ETL 任务。
*   [**特使代理在 CNCF 找到一个家，亚马逊网络服务**](https://thenewstack.io/the-envoy-proxy-finds-a-home-at-the-cncf-amazon-web-services/) :由 Lyft 开发并开源，特使是一个服务网格底层，提供常见的实用程序，如服务发现、负载平衡、电路中断、日志记录和跟踪异构应用程序架构。它可以用作服务代理，在服务之间路由请求，或者用作边缘代理，处理外部流量。这是继 Kubernetes 和 Prometheus 之后，第三个从云计算原生计算基金会孵化器毕业的项目。
*   [**行为驱动设计(BDD)的两种入门方式**](https://thenewstack.io/two-ways-to-get-started-with-behavior-driven-design-bdd/) :行为驱动设计或 BDD 使用简单的语法——给定、何时、然后——向 IT 和业务解释流程，然后成为可测试的规范。在这里找到 BDD 如何帮助你的软件开发。
*   [**【Buoyant CEO 谈 Linkerd 是如何从 Twitter 令人兴奋的早期发展起来的**](/buoyant-ceo-on-how-linkerd-sprang-from-twitters-heady-early-days/)】:Linkerd 是云计算原生计算基金会(CNCF)的一个项目，“向世界介绍了服务网格的概念和术语，”Buoyant CEO 威廉·摩根在 TNS Makers 播客中说道。本播客深入探讨了 Linkerd 的早期，以及 early 如何使用 Rust 来提高这个服务网格的性能。

云本地计算基金会和 InfluxData 是新堆栈的赞助商。
