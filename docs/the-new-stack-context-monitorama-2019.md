# 新的堆栈环境:Monitorama 2019

> 原文：<https://thenewstack.io/the-new-stack-context-monitorama-2019/>

[Monitorama 2019](https://thenewstack.simplecast.com/episodes/monitorama-2019)

欢迎来到[新的 Stack Context](/podcasts/context) ，这是本周所有新闻和事件的每周总结播客。本周，我们来自本周在波特兰举行的 [Monitorama](https://monitorama.com/) 会议，在那里我们采访了 observability 服务提供商 [Honeycomb.io 的开发者倡导者](https://www.honeycomb.io/) [Liz Fong Jones](https://twitter.com/lizthegrey) 和 ELK as a Service provider[logz . io](https://logz.io/)的开发者倡导者 [Quintessence Anx](https://twitter.com/quintessenceanx) 。

与 Jones 和 Anx 一起，我们讨论了 IT 行业的倦怠文化，以及公司如何更好地考虑设计和运行他们系统的人。我们还讨论了会议本身的收获。然后在节目的后期，节目主持人 Libby Clark，The New Stack 的编辑和营销总监，与 Alex Williams，The New Stack 的创始人和出版商，以及 Joab Jackson，TNS 的执行主编讨论了该网站的头条新闻。我们讨论了对 [Honeycomb.io 的可观察性平台](/honeycomb-making-debugging-collaboration-easier/)的更新，以及 Palo Alto Networks 有意收购容器安全提供商 Twistlock 和无服务器安全提供商 PureSec。哦，我们还讨论了 Pivotal 新的[基于 Spring 的 OpenJDK 发行版](https://thenewstack.io/pivotal-throws-its-weight-behind-openjdk-with-spring-runtime/)，以及[关于“深度伪造”视频的惊人新研究](/deep-learning-ai-generates-convincing-deepfake-videos-of-mona-lisa/)。

### Monitorama 亮点:

Pivotal 是新堆栈的赞助商。
