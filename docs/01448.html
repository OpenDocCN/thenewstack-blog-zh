指导是扩大IT多样性的关键

原文：https://thenewstack.io/changing-culture-openstack-mentoring/#0001-01-01

在今天的 The New Stack Makers 播客中，我们听到两位女性帮助公司成功增加其多样性，她们是HPE开发商倡导者和社区架构师 Lisa-Marie Namphy 和OpenStack生态系统技术负责人 Ildiko Vancsa 。Vancsa和Namphy在巴塞罗那OpenStack峰会上与TNS创始人Alex Williams一起探讨如何创建更加多样化和开放的社区。


通过增加多样性来改变OpenStack的文化

 

虽然行业内的多样性仍处于低谷，但OpenStack社区已经经历了多年的发展。Bitergia的Daniel Izquierdo 报告称，2011年女性仅占OpenStack总人数的10.5%，尽管2015年女性人数略有增加，达到11%。

Nampy解释说，解决持续存在的多样性问题不完全是一个管道问题。特别是，这是OpenStack和tech整体文化的问题。“文化是一件大事。坦率地说，如果你是女性，在IRC上进行的对话并不令人愉快。这是一个由男性主导的世界，谈话可以很好地反映这一点。人们忘记了会议中也有女性。我希望看到的一件事是文化上的清理。”

在基础层面，OpenStack专注于指导和社区建设，将其作为成功的支柱，希望在社区中培养多样性。“我们支持OpenStack 团队的女性，以及旅行和指导项目。我们正努力更多地关注打开大门，支持那些愿意加入的女性，以及那些致力于为女性创造更舒适环境的女性。在过去的两天里，我们几乎所有的技术演示都有女性登台，这非常令人惊讶，”Vancsa说。

采访接近尾声时，Namphy重申了一个事实，即指导不仅是增加多样性的关键因素，也是留住OpenStack社区中有才华的女性的关键因素。“在很多情况下，是那种接触，那种舒适度。如果你走进一个房间，没有人看起来像你，谈话与你无关，事实上这很令人不快，你会走出房间。女人来了又不留下，IRC也一样。那么如何挽留呢？你建立了这种联系。这只是迈出了第一步。”
