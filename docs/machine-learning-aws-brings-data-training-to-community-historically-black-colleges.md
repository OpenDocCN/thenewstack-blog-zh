# AWS 将人工智能/人工智能培训带到了社区，历史上的黑人学院

> 原文：<https://thenewstack.io/machine-learning-aws-brings-data-training-to-community-historically-black-colleges/>

拉斯维加斯— [亚马逊网络服务](https://aws.amazon.com/?utm_content=inline-mention)“机器学习大学”推出了一系列围绕数据库和机器学习的教育培训材料，旨在为社区学院、少数族裔服务机构(MSIs)和历史上的黑人学院和大学(HBCUs)提供课程。

本周在 AWS Re:Invent 大会上宣布的这一举措是云巨头[向 2900 万人提供信息技术贸易和科学教育的承诺](https://aws.amazon.com/blogs/aws/aws-machine-learning-university-new-educator-enablement-program-to-build-diverse-talent-for-ml-ai-jobs/)的一部分。

在数据库、人工智能(AI)和 ML 方面，AWS 希望帮助缩小精英四年制大学和社区大学、MSIs 和 HBCUs 之间的课程差距。

“我们希望在明年的课程中接触到 330 名教育工作者，”人工智能/人工智能设备 AWS 主管[麦克·米勒](https://www.linkedin.com/in/camikemiller/)谈到新的堆栈时说。学习课程的教育者越多，可供学生使用的数据库、人工智能/人工智能程序就越多。

## 缩小差距

该计划旨在弥合历史上黑人学院和大学的技能差距，重点是教育教育者，然后由他们为学生创造课程。

教育家培训，称为教育家支持训练营，是建立在机器学习大学的课程之上的，但将包括一个基于 22 名教育家试点小组反馈的迭代。该计划还将包括计算访问以及对相同型号的访问，以及免费的 [AWS SageMaker](https://thenewstack.io/tutorial-deploying-tensorflow-models-with-amazon-sagemaker-serverless-inference/) 、 [SageMaker 笔记本电脑](https://thenewstack.io/turn-aws-lambda-functions-stateful-with-amazon-elastic-file-system/)和其他客户功能。

课程的第一次迭代本质上是 [AWS 的机器学习大学](https://aws.amazon.com/blogs/aws/aws-machine-learning-university-new-educator-enablement-program-to-build-diverse-talent-for-ml-ai-jobs/) (MLU)。MLU 目前是免费的，是一个自助服务平台，面向对提高他们的人工智能和人工智能技能感兴趣的开发者。它由讲座、动手练习、Jupyter 笔记本和幻灯片组成。这些材料可以在 GitHub 和 YouTube 上找到。

MLU 的课程被发送给参与今年秋天开始的试点项目的 25 个机构的 25 名教育工作者。这些教育者在小团体训练营式的队列中学习。AWS 将把他们的反馈整合到更广泛发布计划的课程中。

课程并不仅限于免费的教材。教育工作者也可以免费使用计算机。教师和学生通过亚马逊 SageMaker 工作室实验室获得教学材料，米勒称这是一种“很好的、简单的获取课程的方式”

## 机器学习大学

“AWS 机器学习大学背后的整个想法是为学生提供与我们的 AWS 客户相同的访问类型，但免费，以便他们可以运行相同的模型，所有相同的功能和技术，他们将在工业中使用，”米勒说。可以通过注册 AWS Academy 来访问额外的资源，这将为一个完整的 AI/ML 项目解锁所有的 AWS 服务。

![](img/e6dee363db37e100d148f6c4917d2500.png)

“精英四年制学院和大学在每个学生身上的花费是美国资源较少的两年制学院或社区学院的两倍至近五倍，”他说。

AWS 不仅注意到资源的缺乏，还注意到接受服务的学生的人口统计数据。关于这一点，米勒说，他们在学习这些新技术的技能和可及性方面存在差距，这些新技术对世界各地的创新增长至关重要。

这是教育家支持训练营背后的灵感，以帮助弥合差距。
