所有发布会:GitHub Universe、Cloud Foundry、Node+JS互动

本周在我们的每周新闻总结播客 TNS Context 中，编辑团队介绍了我们最近参加的一些活动。TNS通讯员 TC Currie 来说说本周举行的 GitHub Universe 。我们还有TNS创始人兼主编 Alex Williams 来谈谈Cloud Foundry Summit Europe和TNS执行主编 Joab Jackson 来聊聊温哥华的 Node+JS Interactive 。这两项活动都是上周举行的。

上周在播客中，我们与IBM的杰森·麦基谈论了云铸造大会上的一些云新闻。但我们没听到太多关于其他事情的消息。威廉姆斯向我们展示了展会上发生的其他一些亮点，包括一些关于 Cloud Foundry BOSH 生命周期管理工具的精彩对话。Currie讨论了自6月份微软宣布收购以来GitHub的首次大型用户会议，包括围绕该存储库的新工作流工具 GitHub Actions 的所有讨论。接下来，杰克逊将向我们讲述他从Node+JS Interactive收集的一些TNS故事。
