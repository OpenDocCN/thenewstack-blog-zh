# Docker 瞄准红杉资本 4000 万美元投资的企业

> 原文：<https://thenewstack.io/docker-sets-sights-on-enterprises-with-40-million-investment/>

有了 4000 万美元的新资金，Docker 希望进一步开发计划来满足企业客户的需求。Docker 企业营销副总裁大卫·梅西纳(David Messina)表示，如此多的企业对 Docker 感兴趣的事实表明，应用程序对各种各样的企业变得多么重要。

该公司今天宣布了由红杉资本(Sequoia Capital)牵头的新投资，并表示将利用这笔资金进行国际扩张，以及更好地服务于企业用户。

“许多世界上最大的金融机构和政府已经在使用 Docker，”Messina 说。"他们非常感兴趣的一件事是与 Docker 建立正式的支持关系."

此外，他说，随着 Docker 在企业中的使用越来越多，许多组织都在寻求培训支持。

Docker 还计划继续建立一个平台，允许组织内的开发人员在防火墙后共享 docker 化的应用程序，只与同事共享。该公司已经支持 [Docker Hub Registry](https://registry.hub.docker.com/ "Docker Hub Registry") ，在这里人们已经分享了 35000 个 Docker 化的应用。但私人提供将允许企业为 Dockerized 应用程序建立一个内部“目录”，类似于许多公司建立的私人应用程序商店，与员工分享电话或云应用程序。

事实证明，为企业提供更多支持是有回报的。当 Docker 1.0，该技术的第一个支持版本，在 6 月份发布时，Docker 有 300 万次下载。梅西纳说，现在，仅仅三个月后，就有 2100 万次下载。虽然一些企业使用实验性技术，但许多企业更喜欢等待受支持的版本，这样下载量的增加至少可以部分归因于企业的加入。

Messina 表示，企业正在学习 Docker 可以帮助他们更快地开发新的产品，这有助于推动他们的业务。他说，例如，设计师购物网站 Gilt 有 300 个分类组件。他说，这使得开发者每天都能给网站带来大量的变化。“这种由 Dockerized 应用程序驱动的下一代分布式应用程序允许公司实时创新，”他说。“随着我们与越来越多的企业交流，这正是他们感到兴奋的地方。他们认为应用是创新业务的最快方式，Docker 正在成为他们实时创新的标准。”
