# 代码 n00b: Hocus，集中注意力

> 原文：<https://thenewstack.io/code-n00b-hocus-focus/>

现代生活的一个普遍真理是时间永远不够。我们在软件生产生活中也有同样的事实，伴随着不合理的截止日期和永无止境的吉拉门票的额外压力。不要让我开始讨论 scrum sprints 这个话题。我的开发伙伴们在大口喝着他们喜欢的含咖啡因的饮料时喃喃自语着这个事实:“我们需要更多的时间来做好这件事，但我们几乎没有时间去做。我希望我能做得更多，但没人有时间做这些。

我们这个勇敢的新的碎片化世界，充斥着无尽的电子邮件和不断的垃圾信息，现在已经催生了一种可以被委婉地称为“聚焦经济”的东西。每个人都希望我们更加专注，这样我们可以更快地交付，哦，顺便说一下，他们有一个很好的新想法，我们可以做到这一点，你刚刚被邀请到 Slack 频道！[番茄钟](https://pomodoro-tracker.com/)，四小时工作周(哈，好像是)，“深度思维冲刺”——Jeebus，谈谈你的矛盾，焦点经济真正缺乏的一件事是讽刺——生产力黑客到处涌现。它们对我们这些在软件开发工业联合体中辛苦工作的人来说最有吸引力，因为我们感到不断有压力要做得更多、更快。我现在工作的公司在休息室有一个霓虹灯招牌，上面轻轻地闪烁着“快睡吧”的字样。就在咖啡机的正上方，咖啡机会自动提示双份浓缩咖啡。

所以压力是毫无疑问的。忠实的 [Code N00b](https://www.thenewstack.io/tag/Code-N00b) 读者们，摆在我们面前的真正问题是，这些更少分心/更高效/更省时的技巧真的能为我们节省时间吗？帮我们黑掉自己的习惯，让我们成为更高产的肉食机器？利用每周四小时睡眠的力量？优化我们的大脑以获得最高效率？

老实说，我很喜欢那些“如果你尝试一下，你会好 X 倍/更快/更专注”的文章。我总是乐于浪费十分钟的工作时间来阅读另一种更好地利用时间的方法。上帝知道我会尝试实际应用它们:我是一个强迫性的列表制造者，每天都在收件箱里浏览，并有一个特雷罗板来组织我的特雷罗板。

我开发的第一个 JavaScript 应用程序是一个番茄计数器，它总是在我屏幕的左上角打开(虽然几周后我又添加了一个贪睡按钮功能，伙计，25 分钟过得真快)。我在买菜或开车时会听蒂姆·菲利斯的播客，收集世界级表演者的工具、策略和技巧。但是这些真的有帮助吗？

老实说，可能不会。我不得不说，我已经真诚地、真正地尝试了其中的大部分，除了番茄红素——在这一点上它只是提醒我起来走动走动，因为[整天坐在我们的办公桌前正在杀死我们](https://www.marketwatch.com/story/sitting-at-your-desk-all-day-is-killing-you-heres-how-to-get-moving-again-2018-04-20)——就像无效或无用一样迅速地抛弃了它们。这就把我们带到了“焦点经济”热潮和促成它的生产力工具的终极谎言:你最终会浪费大量的时间去阅读、下载和实现它们。当真正的黑客只是坐下来专注于手头的工作并完成它时，我每周能读到多少关于减少分心的新方法的文章？我花了多少时间为自己的工作效率比世界认为的要低而感到难过？

真正的问题是:当我很少能从核对完成的项目中获得深深的满足感时，我又浪费了多少时间来做我的每日激光目标清单呢？毕竟，一旦有了一张新的 JIRA 机票，上面标有紧急字样并标有你的名字，一丝不苟的待办事项清单就会被抛到九霄云外。

你不能黑掉你不能预测的东西。

当你以构建东西为生时，尤其是软件，这是一个创造性的过程，比我们通常认识到的更加模糊。更少关于稳定的可预测的进步，更多关于拥抱混乱。所有这些黑客都试图控制这种混乱，将含糊不清的东西锤炼成可交付的东西，但我做这件事的时间越长，就越清楚混乱是过程中必不可少的一部分。它是内置的。做出好的工作并保持理智的真正途径在于接受这一点，并学会做出相应的反应，而不是无休止地尝试——并失败——去驯服它。组织和工具很重要。然而，理智是至关重要的。

因此，我决定在 2019 年尝试的新方法是满足于每天完成我能做的任何事情。不要再纠结于优化和生产力，只要，你知道，做好我的工作。考虑到有些日子是钻石，有些日子是石头，不要为此自责。

我决定在 2019 年试用的新生产力工具是免费的:对自己放松点。
