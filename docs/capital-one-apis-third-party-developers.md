# Capital One 向第三方开发者开放了它的 APIs 这是他们学到的东西

> 原文：<https://thenewstack.io/capital-one-apis-third-party-developers/>

大约一年前，Capital One 推出了[Capital One deve exchange](https://developer.capitalone.com/)，这是一个面向外部 API 的平台，也是了解其开源项目的目的地。当时，Capital One 向第三方公开了三个内部 API，以增强他们自己的产品和服务。API 已经成为银行公司向外部开发人员扩展服务的一种方式。Capital One 的 API 战略展示了该公司如何成为一家既像金融服务机构又像科技公司的公司。

该公司已经学到了很多关于开发、部署和管理 API 的知识。在 2017 年 SXSW 期间，Capital One 开发者营销和平台副总裁 Amy Heidersbach 在 Antone's 的 Capital One House 接受 TNS Makers 采访时解释道:[公司的进展表明该公司如何将 API 视为产品，并继续发展其对开源技术的投资。](https://www.linkedin.com/in/amy-heidersbach-4300831/)
