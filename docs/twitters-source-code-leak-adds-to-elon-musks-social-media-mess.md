# Twitter 的源代码泄露加剧了埃隆·马斯克的社交媒体混乱

> 原文：<https://thenewstack.io/twitters-source-code-leak-adds-to-elon-musks-social-media-mess/>

据报道，Twitter T1 的一些源代码已经在网上泄露。根据美国加州北区地方法院的一份法律文件，一名被称为“自由发言人”的用户被指控在 [GitHub](https://github.com/) 上分享 Twitter 的部分代码。Twitter 现在正在寻求法院命令，迫使 GitHub 披露发帖人的身份和任何下载代码的人，声称侵犯了版权。

作为回应， [GitHub](https://thenewstack.io/github-copilot-a-powerful-controversial-autocomplete-for-developers/) 的发言人在一份电子邮件声明中证实，该公司遵守了 Twitter 的要求，删除了代码，但拒绝进一步置评。Twitter 没有回应置评请求。值得注意的是，在新老板兼首席执行官[埃隆·马斯克](https://thenewstack.io/elon-musk-twitter-and-the-weaponization-of-open-source/)的领导下，Twitter 不再回复任何媒体请求。

知情人士称，代码泄露似乎来自一名前 Twitter 员工。自马斯克接管以来，超过 5000 名员工被解雇，剩下不到 2000 名员工。

这一代码发布发生在 1 月零日 API 违规事件之后，该事件发布了 2.35 亿条 Twitter 用户记录。此外，Twitter 在 3 月 6 日搞砸了自己对 API 的使用。这导致了 [Twitter 的链接、图片和视频在一段时间内全部失效](https://www.theregister.com/2023/03/06/twitter_features_outage/)。马斯克回应:“[这个平台好脆(叹气)。马斯克继续说道，代码“最终需要完全重写。”](https://twitter.com/elonmusk/status/1632794008060567552?s=20)

2 月 21 日，马斯克声称，“我们的算法将于下周开源。”他补充说，一开始效果不会很好，“但它会迅速改善！”他没有遵守诺言。自从马斯克解雇了大部分 Twitter 开发人员以来，谁将修复代码并开放源代码仍然没有答案。

新的泄露给马斯克增加了越来越多的问题，因为他试图通过削减成本和政策变化的积极组合来纠正 Twitter 的路线。这也引发了对网站安全性的质疑。

很难确切知道这次代码泄露会造成多大的麻烦，因为我们不确定里面有什么。我们只知道 Twitter 声称它包含了“[Twitter 的平台和内部工具的专有源代码。](https://github.com/github/dmca/blame/master/2023/03/2023-03-24-twitter.md#L25)“我们也不知道代码是什么时候放的。如果它是在 2023 年 1 月 3 日开始使用 [FreeSpeechEnthusiast 的 GitHub 账户时上传的，那么这段代码可以使用将近 12 周。](https://github.com/FreeSpeechEnthusiast)
