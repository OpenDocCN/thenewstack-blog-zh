## Python 的历史

Python 是一种动态类型的编程语言，它不要求应用程序在运行前首先被编译。仅仅这一点就有助于简化语言，消除了耗时耗力的编译过程。因为 Python 背后的哲学就是代码可靠性，所以你可以肯定学习这种语言比学习其他语言要简单得多。

Python 始于 20 世纪 80 年代，由吉多·范·罗苏姆创建，作为 ABC 语言的继承者。最初的目标是创造一种语言，能够处理异常并与现在已经不存在的 Amoeba 操作系统接口。这项工作始于 1989 年 12 月，Rossum 担任主要开发人员，直到 2018 年 7 月 12 日，他宣布他将作为 Python 的“终身慈善独裁者”(Python 社区给他的头衔)进行“永久休假”。

2019 年 1 月，剩余的 Python 核心开发人员选举了一个由五名成员组成的指导委员会，该委员会将负责领导该项目走向未来。

Python 时间表中的下一个重要里程碑包括:

*   2000 年 10 月 16 日–Python 2.0 的发布
*   2008 年 12 月 3 日–Python 3 的发布
*   2020 年 1 月 1 日 Python 2.0 的生命终结

随着 Python 3 的发布，出现了一个名为 2to3 的实用程序，它可以自动将 Python 2 代码翻译成 Python 3 代码。考虑到许多项目和用户仍在使用 Python 2，这个自动化工具就成了绝对必要的。

对我来说，我认为 Python 如此受欢迎的主要原因是它是学习软件工程技术的理想语言。Python 具有非常容易理解的语法、广泛的支持、跨平台的可用性以及无需编译，因此很容易上手。考虑到这种语言的灵活性，你能让它做的事情是无限的。

## Python 和当今的需求

得益于机器学习和人工智能的飞速发展，Python 最近有了一点复兴。通过访问几个[开源库](https://thenewstack.io/the-ultimate-guide-to-machine-learning-frameworks/)——比如 Scikit-learn、Tensorflow、PyTorch、Apache MXNet、XGBoost、Paddle Paddle 和 ONNX——Python 是机器学习的理想伙伴。

机器学习已经被绝大多数企业投入使用，例如

*   欺诈检测
*   算法交易
*   金融和投资组合管理
*   电子邮件监控
*   客户旅程优化
*   自动驾驶汽车
*   行为预测
*   卫生保健
*   工序自动化

[机器学习](https://thenewstack.io/category/machine-learning/)对于许多希望在高度竞争的世界中保持相关性的企业来说已经变得至关重要。Python 是其中的核心，这要归功于如此多的可用库和工具，使得将机器学习集成到您的流程和工作流中比您想象的更容易。
