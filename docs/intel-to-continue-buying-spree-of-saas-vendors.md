# 英特尔将继续大举收购 SaaS 厂商

> 原文：<https://thenewstack.io/intel-to-continue-buying-spree-of-saas-vendors/>

芯片巨头英特尔并不满足于只卖芯片，所以它正准备出售与芯片配套的软件。在这种程度上，该公司正在积极寻求对软件即服务供应商进行更具战略性的收购。

英特尔首席执行官[帕特·基尔辛格](https://www.intel.com/content/www/us/en/newsroom/biographies/biography-patrick-gelsinger.html#gs.0p3e9s)最近在得克萨斯州达拉斯附近的[愿景会议](https://www.intel.com/content/www/us/en/events/on-event-series/vision.html)上分享其软件即服务战略的一些细节时，宣布了收购 SaaS 公司的意图。

![](img/976af5c4f678b159c500797c5c95ea7e.png)

“你会看到我们在 SaaS 推出芯片差异化产品，并广泛提供给业界。我的简单公式是硅加软件，”格尔辛格说，并补充说，“我们也将进行更多的 SaaS 收购。”

英特尔一直在抢购软件公司[，其中包括提供基于人工智能的云优化服务的 granite](https://techcrunch.com/2022/03/31/intel-confirms-acquisition-of-granulate-reportedly-for-up-to-650m-bringing-ai-based-workload-optimization-into-its-fold/)、[和专注于 Linux 内核实时处理能力的开源开发商店 Linutronix](https://www.datacenterknowledge.com/intel/intel-acquires-ai-powered-cloud-optimization-startup-granulate) 。该公司的目标是增加软件收入，2021 年其总收入为 790 亿美元，仅略高于 1 亿美元。

“在我们过去 40 年的行业生涯中，我们从硬件变成了软件收入的两倍。现在是三比一软件收入的硬件。对于整个行业来说，软件的作用变得越来越重要。

英特尔在展会上宣布了一些将在其芯片上提供的软件即服务功能。这些服务将在订阅的基础上提供。

一个是[项目琥珀](https://www.intel.com/content/www/us/en/newsroom/news/vision-2022-project-amber-security.html)，这是一个证明服务，将为组织提供客户资产的独立验证和可信度，无论它们在哪里运行。基于软件的服务将验证数据和端点的完整性，并建立一个安全层，公司可以在其上运行软件或人工智能模型。

Amber 项目中的认证机构没有链接到基础设施提供商，因此它可以在多云或边缘环境中工作。你可能不需要依赖三大云提供商之一进行验证，但你需要英特尔特有的功能，包括 SGX(软件保护扩展)和基于 Linux 的中间件 Gramine，该中间件基于芯片制造商领导的开发工作。

英特尔首席技术官 [Greg Lavender](https://www.linkedin.com/in/greglavender/) 在会议期间表示:“[Gramine]层使 Linux 内核仿真器能够在称为英特尔 SGX 飞地的受限环境中运行未经修改的 Linux 应用……在那里，我们使用硬件来保护应用代码和数据的安全。”

今年晚些时候，英特尔将提供一项名为 [Project Endgame](https://www.pcgamer.com/what-is-intels-project-endgame/) 的云游戏服务，在这项服务中，图形将在该芯片制造商安装在数据中心的新 GPU 上呈现。

Gelsinger 之前是 VMware 的首席执行官，拥有创建软件公司的经验。英特尔雇佣了大约 17，000 名软件工程师，Gelsinger 说这比他在 VMware 任职期间的任何时候都多。

真实世界计算公司的分析师大卫·坎特说，包括微软的 Azure、[亚马逊网络服务](https://aws.amazon.com/?utm_content=inline-mention)和谷歌云在内的云提供商已经吞噬了 SaaS 市场的大部分份额，但英特尔基本上没有与之竞争

“这不仅仅是‘这是硬件和一些软件优化的驱动因素’坎特说:“关键是要弄清楚哪些东西确实能带来价值，以及如何将它货币化。”。

云服务提供商为工作负载提供英特尔 CPU 实例，这有助于英特尔更好地了解真实世界的工作负载。英特尔将通过超大规模和基础设施提供商覆盖其 SaaS 产品，尽管在展会上没有宣布与亚马逊、谷歌或微软在 SaaS 的特定合作伙伴关系。

英特尔数据中心和人工智能集团执行副总裁兼总经理 Sandra Rivera 表示:“在该基础设施上运行的第三方软件是我们的一大关注点，因为它真正考虑了客户的客户。

Rivera 说，英特尔还将直接将其 SaaS 产品覆盖到大型科技和非科技公司，这些公司使用该公司的服务器芯片部署自己的数据中心基础设施。

亚马逊等公司正在内部开发自己的基于 ARM 的芯片，并在此基础上推出 SaaS 产品，这给英特尔带来了更大的压力，要求其通过部署裸机数据中心基础设施，直接向客户提供 SaaS 服务。

坎特说，英特尔刚刚起步，但 SaaS 的产品将有助于更快的硬件和软件部署。

一个名为 [OneAPI](https://thenewstack.io/intel-pushes-cross-architecture-support-in-oneapi-toolkits/) 的软件开发工具包是将所有软件产品结合在一起的粘合剂。英特尔提供了一系列包括 FPGAs 在内的 CPU、GPU 和 AI 加速器，OneAPI 消除了对特定芯片编程所涉及的复杂性。该框架是开源的，内置了并行机制来利用芯片的处理能力。

“我们高度关注开放安全平台，这意味着开放接口和开放软件体。我们可能拥有这些开放接口的专有实现，这是我们正在提供的特殊调料，”Gelsinger 说。

他提到了几个使用 [OpenVINO](https://thenewstack.io/intel-openvino-brings-ai-inferencing-to-the-desktop/) AI 推理工具包的开源方法的例子，以及用于通过分散和边缘硬件基础设施进行通信的开源 FlexRAN 参考架构。

这家芯片制造商过去也经历过软件方面的挫折，最著名的是 2010 年以 76 亿美元收购迈克菲(McAfee)的不明智交易。2010 年，英特尔以 42 亿美元的价格将迈克菲的多数股权出售给了风险投资公司 TPG，后者去年又以 140 亿美元的价格将其转让给了另一家风险投资集团。
