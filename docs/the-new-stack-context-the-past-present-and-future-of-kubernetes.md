在过去的一年里，Kubernetes 和 cloud native 社区发生了哪些最重要的事件？在瘦身和稳定核心方面做了很多工作。在过去的一年中，操作符是一个增长的趋势——操作符是一种机制，可以扩展你可以在 Kubernetes 上构建的东西的数量。费尔南德斯说:我们也看到 Kubernetes 扩展到新的工作负载。费尔南德斯说，展望未来，我们可能会看到 Kubernetes 的开发人员体验将越来越不同，这取决于具有不同需求的应用程序和工作负载。在核心有标准很重要，但是在边缘，开发者想要更多的选择。
