# 派对赛道:激动人心的浪潮席卷 KubeCon + CloudNativeCon 北美 2019

> 原文：<https://thenewstack.io/party-circuit-waves-of-excitement-wash-over-kubecon-cloudnativecon-north-america-2019/>

Portworx 赞助了 New Stack 对 Kubecon+CloudNativeCon 北美 2019 的报道。

尽管圣地亚哥很少出现雨天，但 KubeCon+CloudNativeCon 还是见证了许多海边社交活动。这当然有助于会议地点，圣地亚哥会议中心，位于邻近城市的主要娱乐区，历史悠久的煤气灯区。会议的赞助商展示也提供了许多交流和学习新技术的机会。

CloudBees、云原生计算基金会、KubeCon+CloudNativeCon、Palo Alto Networks、Pivotal 和 Portworx 是新堆栈的赞助商。

特写图片:来自[波特沃斯](https://portworx.com/)的团队正在冲浪。(从左到右)[艾普丽尔·巴卡罗](https://twitter.com/a_bacarro)，[卡特林娜·伊瓦先科](https://twitter.com/kivashch)，[安德鲁·哈特菲尔德](https://github.com/andrewhatfield)，[格兰特·格里菲思](https://github.com/ggriffiths)，[路易斯·帕邦](https://github.com/lpabon)和瑞安·沃纳。
