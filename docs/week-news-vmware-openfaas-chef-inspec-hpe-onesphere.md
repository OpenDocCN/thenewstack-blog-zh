# 本周新闻:VMware 和 OpenFaaS，厨师 InSpec，HPE OneSphere

> 原文：<https://thenewstack.io/week-news-vmware-openfaas-chef-inspec-hpe-onesphere/>

[本周新闻:VMware 和 OpenFaaS、Chef InSpec、HPE OneSphere](https://thenewstack.simplecast.com/episodes/this-week-in-news-vmware-and-openfaas-chef-inspec-hpe-onesphere)

大家好，欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)，在这个播客中，我们回顾了本周云原生技术/大规模应用程序开发的最热门新闻，并展望了我们预计将在未来几周获得更多关注的主题。

本周，在多雪的俄勒冈州波特兰，我们采访了 OpenFaaS 的创始人亚历克斯·埃利斯。埃利斯受聘于 VMware，全职开发 OpenFaaS 无服务器软件，作为 VMware 新的开源技术中心的一部分。到目前为止，Ellis 一直在业余时间从事 OpenFaaS 方面的工作，同时还在 ADP 担任应用程序开发人员。我们与他讨论了他为什么决定加入 VMware，以及这对 OpenFaaS 社区意味着什么。

我们还讨论了[Chef](https://www.chef.io/)对其安全合规性和自动化工具 [InSpec](https://www.chef.io/products/chef-inspec/) 的最新更新，并与新的 Stack 主编 [Alex Williams](https://twitter.com/alexwilliams) 讨论了我们刚刚与 [HPE](https://www.hpe.com/us/en/greenlake.html) 一起结束的由三部分组成的[新 Stack Makers](https://thenewstack.io/podcasts/makers) 播客系列，该系列专注于企业 IT 和开发人员在迁移到新的云原生架构时面临的一些挑战。

新的 Stack 编辑部主任 [Libby Clark](https://twitter.com/LibbyMClark) 和新的 Stack 执行主编 [Joab Jackson](https://twitter.com/Joab_Jackson) 一起主持了本周的 Context。
