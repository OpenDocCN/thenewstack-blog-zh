# 开源如何推动 Capital One 的云创新

> 原文：<https://thenewstack.io/how-open-source-drives-cloud-innovation-at-capital-one/>

[首都一号](https://developer.capitalone.com/)赞助本帖。

约翰是第一资本公司开源项目办公室的主任。在加入 Capital One 之前，John 是开源协作的长期实践者和倡导者，在 Red Hat、EMC、Gluster、Hyperic 和 SourceForge 领导开源工作。](https://www.linkedin.com/in/johnmarkwalker/) 

八年前，Capital One 开始了我们的技术转型，包括将我们的技术团队大规模发展到近 11，000 人(85%是工程师)。我们还采用了敏捷、RESTful APIs、微服务、DevOps、CI/CD 和开源。今年秋天，我们退出了最后三个数据中心，这使我们成为第一家报告退出传统数据中心并全力投入公共云的美国银行。

在云技术出现之前，企业 IT 基础设施是由计算机和网络设备、服务器机架和冗余电源组成的物理事物，所有这些都位于专门建造的温控建筑物中。这些数据中心需要大批人员来管理错综复杂的设备和专有软件。引入新产品或创新需要时间，需要一系列的截止日期，并且通常需要为新的服务器空间和专有软件做预算。进展缓慢。

然后云技术改变了一切。诞生在云时代的创业公司能够推出新产品，然后根据市场需求进行调整。公司可以以创意的速度前进。借助云，用户可以即时获得基础架构。如果他们需要大型数据中心的强大功能，他们不会等待构建—他们只需订购他们需要的东西，就可以立即投入使用。

过去几年的数字革命改变了我们沟通和联系的方式。它彻底改变了我们购物、旅行、管理健康等的方式。新冠肺炎只是加速了数字化的采用，因为我们更多的是呆在家里。这使得大型企业的领导者更仔细地审视云的优势。

对于进行这种转变的公司来说，这可能是一个分水岭。在云中，可以在更大的规模上管理数据，这可以释放机器学习的力量，提供实时的数字客户体验。这也与更多地参与开源软件社区相关。因为在云中增加了灵活性和上市速度，公司可以更频繁地为开源社区做出贡献。

按照[德勤](https://www2.deloitte.com/us/en/insights/industry/technology/how-open-source-software-is-turbocharging-digital-transformation.html)的说法，开源软件就是“涡轮增压转型”。极具竞争力的公司也在支持它——主要原因是它缩短了开发时间。软件工程师可以从现有的库中获取已建立并经过测试的代码，而不是从头开始创建，这样可以加快完成速度。没有人被迫不断地重新发明轮子，也没有人必须等待供应商经历他们的开发周期。

## **开源创新**

根据来自 Red Hat 的 [2020 年企业开源报告](https://www.redhat.com/en/enterprise-open-source-report/2020)，95%的 IT 领导者认为企业开源对于他们的企业基础架构软件战略至关重要。会计师们意识到许可证费用的缺乏，但是成本并不是一个有远见的公司选择开源软件的唯一因素。

开源社区的大多数成员都对检查代码感兴趣。他们通过与他人分享解决方案来解决问题。价值是通过独创性和实用性来交换的。红帽报告发现，53%的受访者将开源用于安全，51%的受访者将其用于云管理。但它在其他地方也很受欢迎:60%的 it 领导者说他们为 IT 基础设施现代化部署开源软件。

工程师也从对社区的贡献中受益。德勤发现，当公司使用开源软件时，工作满意度和创造力往往会提高。此外，他们还注意到工程学科、产品质量和效率方面的改进。

## **云中转换**

亚马逊网络服务(AWS)提供给我们的云中的现代架构，让我们能够通过不断推出的新产品来融入世界创新。通过我们的开源优先承诺，通过参与协作软件构建方法，交付已经加速。事实上，我们的团队从 2016 年进行季度或月度应用发布，到 2019 年每天发布多次代码。

自 2014 年宣布“开源优先”以来，Capital One 团队已经发布了超过 25 个开源项目，并为大约 100 个不同的项目做出了 1500 多项贡献。这些贡献包括 [Hygieia](https://www.capitalone.com/tech/solutions/hygieia/) 和 [Critical Stack，](https://www.capitalone.com/tech/solutions/container-orchestration/)一个面向高度监管企业的容器管理平台。

我们的技术变革改变了第一资本的文化。它引领我们走向开源，在这里，我们的工程师可以在建设未来银行的过程中创造性地设计解决方案。在云中，我们可以摆脱基础架构管理，将资源集中于提供以客户为中心的创新，并为行业带来积极的变化。

Red Hat 和 Amazon Web Services 是新堆栈的赞助商。
