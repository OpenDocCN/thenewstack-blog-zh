# IBM 收购红帽对开源意味着什么

> 原文：<https://thenewstack.io/what-ibms-purchase-of-red-hat-will-mean-for-open-source/>

[IBM 收购红帽对开源将意味着什么](https://thenewstack.simplecast.com/episodes/what-ibms-purchase-of-red-hat-will-mean-for-open-source)

欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)，这是一个每周一次的播客，我们在这里讨论云原生软件开发和大规模部署领域的最新消息。

在本周的节目中，我们将讨论 IBM 以 340 亿美元收购 Red Hat 的新闻，特邀嘉宾有:Nirmata 公司联合创始人 Ritesh Patel 和 Benjamin Ball。

每隔一段时间，一项收购就会带来彻底改变商业格局的潜力。回想一下甲骨文收购太阳微系统公司，或者最近微软收购 GitHub。IBM 对 Red Hat 的收购很可能属于这一类。IBM 将以 340 亿美元收购 Red Hat，较 Red Hats 上周五的收盘价溢价 64%。这一事件在几个不同的方面都很重要，更不用说这是迄今为止最大的纯软件收购。在本期播客中，我们将讨论此次收购对开源生态系统、竞争激烈的云服务市场以及 Red Hat 和 IBM 员工的影响。

我们的固定主持人，TNS 编辑 Libby Clark 本周不在，所以 New Stack 的创始人兼主编 Alex Williams 在 TNS 执行主编 Joab Jackson 的协助下主持了这一集。
