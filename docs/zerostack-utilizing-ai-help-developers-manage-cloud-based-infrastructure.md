# ZeroStack 如何使用人工智能来管理基于云的基础设施

> 原文：<https://thenewstack.io/zerostack-utilizing-ai-help-developers-manage-cloud-based-infrastructure/>

许多 IT 经理喜欢运行私有云的想法，尽管担心它可能需要太多的维护。为此， [ZeroStack](https://www.zerostack.com/) 有一个新颖的想法:为什么不使用人工智能来自动化一些云管理呢？

ZeroStack 的 [ZBrain](https://www.zerostack.com/product/z-brain-cloud-management/) 通过使用人工智能和机器学习，帮助其客户更好地了解他们的基础设施。在最新一集的[新堆栈制作者](https://thenewstack.io/podcasts/Makers)中，TNS 助理播客制作人 Kiran Oliver 与 ZeroStack 营销副总裁 [Steve Garrison](https://www.linkedin.com/in/steve-garrison-6656a5) 谈论了 ZBrain 的工作方式，以及客户如何更好地应对公共云和私有云基础架构之间的鸿沟。

[zero stack 如何利用人工智能帮助开发人员管理其基于云的基础设施](https://thenewstack.simplecast.com/episodes/how-zerostack-is-utilizing-ai-to-help-developers-manage-their-cloud-based-infrastructure)

ZBrain 利用来自公司运营和遥测的数据来帮助向其团队更好地推荐基础设施改进。加里森说:“我们可以帮助你在三个层面上规划容量:用户、实际集群，然后我们会根据你的行为建议你应该购买什么类型的金属来实现性能最大化。”

ZeroStack 实现可组合基础设施的方法显然是不干涉的。Garrison 将其服务捆绑到所谓的“交钥匙、单一堆栈”包中，他提到，“我们认为市场对构建云更感兴趣，而不是试图构建和维护云。”Garrison 还指出，开发者可以使用[open API](https://www.openapis.org/)来更深入地挖掘 ZeroStack 平台。

ZeroStack 围绕免提自动化推出了第一期 AI 套件，到 2017 年底，它将提供第二期，重点是异常检测、故障排除和性能维护。

然而，加里森指出，在机器能够真正复制人类能够复制的复杂思想和过程之前，人工智能还有很长的路要走。“关于人工智能的好消息是，谈论它很酷，每个人都很高兴听到更多。坏消息是，大约 50 年来一直有很多炒作。我们仍然相信，有一天我们不仅仅能够下棋和开车，而且实际上，能够自动化非常复杂的系统和程序。没有人相信这会在一夜之间发生。”
