# Marten Mickos 前往惠普收购 Eucalyptus

> 原文：<https://thenewstack.io/marten-mickos-heads-to-hp-with-eucalyptus-acquisition/>

如果说 Eucalyptus 首席执行官 Marten Mickos 最近站出来支持 OpenStack 看起来很奇怪，那么现在它开始变得有意义了。惠普今天宣布了收购 Eucalyptus 的计划。Mickos 将运营惠普的云业务，包括构建惠普的 OpenStack 云产品。

Eucalyptus 旨在让企业构建与亚马逊网络服务兼容的私有云。有一段时间，这似乎是一个冒险的提议，因为不确定 AWS 对 Eucalyptus 依赖其 API 的感觉如何。然而，在 2012 年， [AWS 给了 Eucalyptus 它的祝福](http://www.infoworld.com/d/cloud-computing/aws-deal-bolsters-eucalyptus-enterprise-appeal-189279 "InfoWorld")。

与占主导地位的云提供商结盟似乎会给 Eucalyptus 带来巨大的帮助。然而，该公司已经被 OpenStack 超越了一点，open stack 由大型供应商组成的庞大生态系统提供支持。

在宣布这笔交易的新闻稿中，惠普没有提到 Eucalyptus 客户会遇到什么情况。Eucalyptus 网站也没有对这个问题给出多少解释。Eucalyptus 没有立即回应置评请求。鉴于信息的缺乏，这项技术似乎不太可能继续得到惠普的支持。

Mickos 可能会为惠普的云计划提供所需的推动力。他在科技界很出名。在执掌 Eucalyptus 之前，他是 MySQL 的首席执行官，并且一直是开源技术的直言不讳的倡导者。

虽然惠普已经谈论参与 OpenStack 有一段时间了，但今年早些时候，它发布了一项重大公告，宣布向 OpenStack 投资 10 亿美元，并重新制定了构建 OpenStack 公共云的计划。

惠普需要这样的投资来参与竞争。AWS 目前是云计算领域的市场领导者，微软 Azure 和谷歌紧随其后。接下来是一大堆公共云服务，大部分市场份额很小。虽然惠普与企业有着长期的合作关系，但它仍然面临着当前市场领导者的激烈竞争。

就在上个月，Mickos 宣布他同意在即将到来的 OpenStack 大会上发言。考虑到 Eucalyptus 与 OpenStack 云的竞争，这是一个奇怪的举动。当时，他写道:

“我希望 OpenStack 取得成功。当这种情况发生时，桉树也能成功。OpenStack(依我拙见)是一个巨大现象的名称。Eucalyptus 是一款服务于独特用例的紧密聚焦软件的名称。

事后看来，他的评论更有意义，因为他当时肯定是在和惠普对话。

这些公司没有透露收购价格，但是 [Recode](http://recode.net/2014/09/11/hp-tests-the-ma-waters-with-acquisition-of-cloud-startup-eucalyptus/ "Recode") 报道称价格不到 1 亿美元。
