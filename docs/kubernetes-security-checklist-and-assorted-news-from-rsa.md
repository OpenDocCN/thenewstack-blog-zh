欢迎来到新的 Stack Context 播客，在这里我们将讨论最新的新闻和新的 Stack 赞助商活动。本周，在本周举行的 RSA 会议上，我们采访了 [Nirmata](https://nirmata.com/) 的首席执行官 [Jim Bugwadia](https://www.linkedin.com/in/jimbugwadia/) ，讨论您的 Kubernetes 安全清单可能缺少的内容。

Bugwadia 为新的堆栈写了一篇文章，他在文章中分享了 Nirmata 用来指导其企业客户采用 Kubernetes 的安全模型。Nirmata 是一个用于容器化应用程序部署的 DevOps 平台，也是新堆栈的赞助商。在节目的后半部分，我们回顾了本周的其他播客和故事，包括来自 RSA 的一些新闻，如 Twistlock 的新版本[承诺为虚拟机提供容器原生安全性](/twistlock-brings-container-native-security-to-virtual-machines/)以及 Scytale 的[新的基于 SPIFFE 的服务身份管理工具](https://thenewstack.io/scytale-launches-spiffe-based-service-identity-management/)。

《新书库》编辑部主任利比·克拉克主持了这一集，TNS 执行主编约阿布·杰克逊(Joab Jackson)和《新书库》联合创始人兼主编亚历克斯·威廉姆斯(Alex Williams)提供了协助。
