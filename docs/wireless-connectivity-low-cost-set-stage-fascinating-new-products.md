# Kickstarter 乐园中微控制器的魔力

> 原文：<https://thenewstack.io/wireless-connectivity-low-cost-set-stage-fascinating-new-products/>

通过 Kickstarter 的魔力，开发人员、黑客和企业家似乎可以看到无穷无尽的令人兴奋的新微控制器和纳米计算机。

通过 Kickstarter 融资，以及开放软件/开放硬件运动，让我们处于一个有趣的转折点。现成的连接变得如此容易，成本如此之低，以至于任何人只要有一台笔记本电脑，一个互联网连接，一些开发工作和少到 100 美元就可以开始应用微控制器和小尺寸 Linux 技术来解决现实世界的问题。

今天，我将谈论定义这一趋势的三种新设备。这些公司都是久经考验的 Kickstarter 老手。

Oak 是 Digistump 的一款基于云、支持 WiFi 的微控制器。它兼容 Arduino，采用 32 位处理器，时钟频率为 80 MHz。它通过了 FCC 认证，具有 11 个 I/O 引脚，以及 SPI、UART 和 I2C 接口。该主板的尺寸仅为 0.92 x 1.18 英寸。定价为 13 美元，在活动结束前的几天里，他们已经完成了五次目标。

几年前，Digistump 发现他们的微型 Digispark 取得了很大的成功，这是一种基本的、几乎一次性的微控制器设备。现在，他们正加紧提供基于云的 WiFi 设备，通过他们的 RootCloud IDE(集成开发环境)进行无线编程。

Photon 是 Particle 的第二代 Arduino 兼容、基于云、支持 WiFi 的微控制器，以前称为 Spark.io。他们最初的 Spark 核心设备有 5，549 个支持者，资金超过 567，000 美元。

光子采用 STM32F205 ARM Cortex M3 处理器和 Broadcom BCM43362 WiFi 芯片。它还通过了 FCC 认证，具有 18 个混合信号 GPIO 和外设连接，并向后兼容 Spark 内核。这款设备正在生产中，定价 19 美元，计划于 2015 年 6 月交付。

Particle 是基于云的微控制器游戏的首批玩家之一。他们有一个相当完善的流程，将电路板连接到火花云并进行编程。

C.H.I.P .是一个成熟的 9 美元 Linux 主板(加上在美国的 5 美元运费)。

它采用主频为 1 GHz 的 Allwinner R8 处理器，内存为 512 KB，存储容量为 4 GB。C.H.I.P .内置 802.11 b/g/n WiFi，蓝牙 4.0，复合视频/音频。有外部 VGA 和 HDMI 屏蔽选项。

作为一台 Nanolinux 机器，你可以运行 LibreOffice、Chrome 浏览器、Scratch 编程语言、各种游戏以及一整套实用程序和其他应用程序。

C.H.I.P .跨越微控制器/Linux 领域，很像 Raspberry Pi 或 BeagleBone Black，具有 8 个数字 GPIOs 和 1 个 PWM 引脚，耐受 3.3 伏电压。还有常见的 SPI、I2C、UART、USB 和并行 LCD 输出。

惠普的制造商 Next Thing Co .之前在 Kickstarter 上提供了 Otto——一款可破解的 GIF 相机。他们成功地得到了 414 名支持者的资助，资助金额超过 71，000 美元。

C.H.I.P .目前有超过 25000 名支持者，他们在竞选活动还剩三周的时候提供了 130 万美元的资金(超过了他们 5 万美元的目标)。

## 总结

物理计算正在快速变化，不断出现新的、越来越强大的硬件平台。开发人员、发明家和小企业人士有一个黄金机会，可以抓住时机，快速掌握这些物联网技术。
