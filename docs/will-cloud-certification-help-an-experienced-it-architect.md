# 云认证对有经验的 IT 架构师有帮助吗？

> 原文：<https://thenewstack.io/will-cloud-certification-help-an-experienced-it-architect/>

 [詹姆斯贝尔顿

詹姆斯拥有超过 25 年的 IT 行业经验，自 2015 年以来一直直接与 IBM 云及其用户合作。James 参与创建 IBM 的云认证考试已有数年，今年早些时候，他成为了 IBM 云培训中心的技术负责人，带来了他在系统设计、开发、管理和咨询方面的实践经验。](https://uk.linkedin.com/in/jamesbelton) [](https://uk.linkedin.com/in/jamesbelton)

我在 IT 行业工作了 25 年多，在过去的 17 年里我一直是一名 IT 架构师。在这段时间里，我看到认证在行业和工作场所变得越来越普遍。自 2016 年以来，我一直参与为 IBM Cloud 创建专业认证考试的过程，我认为我最常被问及的一个问题是:“人们/我的雇主/我的客户知道我可以做什么，为什么我应该获得认证？”

这是一个公平的问题。说到底，仅仅拥有云认证并不能证明你是一名出色的云架构师。我记得很多年前，有人在一个在线论坛上问，如果他们通过了一家大型数据库公司的考试，预计能挣多少钱。(当时我自己也是 DBA。)问他们有多少经验，结果发现他们大学毕业没多久，在这个行业工作的时间也不长。我想他们听到他们通过考试后的工资不会是百万富翁时会很失望，因为归根结底，尽管他们可能已经获得了认证，但他们缺乏实际经验。然而，让这个人和有相似经历并通过考试的人竞争，意味着他们更有可能找到一份收入更高的工作。

这是天平的一端。但是，对于我们这些在设计真正的 IT 系统的工作场所工作了几年，有着更令人印象深刻的履历的人来说呢？对我们来说，认证也变得越来越重要。

首先，认证的实际过程意味着需要一定程度的学习。大多数人的工作并不完全符合认证考试的广度和深度，这使得认证成为一种以建设性的方式扩展知识的好方法——也许还能学到一些新技巧。IBM 云培训中心为所有考试提供了详细的学习路径，包括那些针对架构师的考试，因此这是扩展您的技能组合的一个很好的方式。

其次，或许也是最重要的一点，拥有证书是一种提供独立、铁证的方式，证明你的知识、技能和能力水平。这让客户和雇主对你更有信心，因为你已经证明了你的知识水平可以获得认证。据招聘网站 Indeed.com 称，“在你的职业中获得认证的最大优势包括竞争优势、效率、收入潜力、扩展的知识和技能以及职业信誉。”

当你在 It 架构师的职业生涯中变得更加成熟时，这种职业信誉可能是最重要的——或者至少，这是我的经验。虽然我已经能够描述我多年来参与的展示我工作经验的某些项目，但我最近合作过的公司都希望确保我的云知识既深入又可信。毕竟，我今天参与的大多数项目都交付到了云环境中。这就是能够展示我的 IBM 云认证有用的地方——也许让我比那些口袋里没有证书的人更有优势。
