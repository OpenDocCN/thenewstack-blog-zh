找到适合数据驱动型应用的产品，并连接到云铸造

数据驱动的应用如何在 Cloud Foundry 的环境中工作？这是一个开始浮出水面的问题，因为像 TensorFlow 这样的技术越来越受到开发者的关注，他们渴望探索如何利用数据来驱动新一代应用和服务的边界。在今天的 The New Stack Makers 播客中，Pivotal首席产品经理 Scott Truitt 在Cloud Foundry Summit Europe上与我们进行了交谈，以了解更多有关数据驱动型应用程序如何在当今企业中得到应用的信息。

Truitt举了他与谷歌合作的一个项目的例子，这是一个使用谷歌云平台和TensorFlow的零售演示，它将一条推文转化为个性化报价。因此，Truitt指出，数据驱动的个性化不再局限于一刀切的应用程序，而是一个动态的、总是在学习和进化的应用程序。

“我们开始看到这种模式的出现，其中数据作为一个围墙花园将不再发挥作用，”特鲁特说。“让这些界限变得越来越模糊，并在一个敏捷的工作流程中将它们整合在一起，这让我非常感兴趣。”
