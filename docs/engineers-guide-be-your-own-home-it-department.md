# 工程师指南:成为你自己的家庭 IT 部门

> 原文：<https://thenewstack.io/engineers-guide-be-your-own-home-it-department/>

工程师、程序员和开发人员现在都争先恐后地在家工作。

回到我在洛克希德-马丁公司和美国电话电报公司公司做工程师/分析师的时候，我总是得到公司的 IT 支持。对于一个问题，我会打电话给服务台，有人会过来，解决它，我会很快恢复和运行。如今，你可能不得不建立自己的基础设施，并学会如何让工作等式的“家”端运转起来，这可能需要减少公司的帮助。

好的一面是，工程师、开发人员和项目人员通常组织良好，系统化，善于提问，最重要的是好奇心强。这是采用“我们如何让它工作起来？”态度。那就继续搞清楚。与此同时，你可能会学到新的技能，比如进入 Linux 或者学习安装电缆调制解调器、路由器、电缆、台式机和显示器的细节。

如果你不是系统/网络管理员或者动手操作硬件的类型，那么在家工作可能会令人望而生畏。今天，我们将开始分析您的需求，检查您的资产，并就您现在可以采取的行动提供一些建议。

全速前进，该死的鱼雷。

## 你需要什么来做你的工作？

一个很好的起点是回顾你的日常工作和流程，并将它们与你的办公室计算能力联系起来。如果你还在办公室，现在就这么做。大多数人都会有一台装有应用程序(既可以在本地访问，也可以通过浏览器访问)、某种互联网接入和电话的台式机或笔记本电脑。布局典型工作流程图，显示您在工作中使用的流程、人员和系统。在家工作将需要大致相同的基础设施。

接下来，列出你一直使用的程序。也列出偶尔使用的程序。将它们按照重要性、复杂性和使用频率进行分类。记下你在申请中遇到的任何困难或问题。开发人员可以使用版本控制工具来管理他们的代码。面向硬件的人可以使用测试应用程序和诊断程序。经理和项目负责人严重依赖电子邮件和演示软件。支持人员需要访问客户管理系统和问题跟踪。作家和文档人员使用内容管理器。Web 开发人员需要……嗯，web 服务器。

记下您的应用程序所需的运行环境。软件是如何授权的？这个软件只能在 Windows 上运行吗？或者，在 Mac 上？有在多种操作系统上运行的程序吗？

应用程序是基于浏览器的吗？如果是基于浏览器的，您需要高带宽的互联网接入吗？像 [Zoom](https://zoom.us/) 这样的视频会议软件，在廉价或不稳定的 WiFi 连接上可能是微不足道的。

尽你所能去了解你的办公基础设施，以及它是如何让你的工作变得可能的。如果你仍然在你的办公室环境中，去拜访你的常驻业务分析师或项目经理以更好地理解你的业务流程可能是有意义的。当你搬到“异地”时，问一些有助于你预测你需要什么的问题你可以打赌，当每个人都不在办公室时，他们会动员起来努力完成工作。当你学习远程操作时，你的兴趣和洞察力可以帮助你找到完成工作的方法。毫无疑问，这对每个人来说都是一条不确定的学习曲线。

尽管面临挑战，我认为灵活、富有想象力、积极和适应性是度过这些困难时期并在未来取得成功的关键。“能做”和团队努力大有帮助。

当你以新的眼光探索你的“工作”时，让我们也来看看我们在家里必须做些什么。

## 什么家庭基础设施？

几天前，我女儿打电话问我们家里有哪家网络供应商。我告诉她光谱。她说她就是这么想的。当然，你知道我的下一个问题。“为什么？”

她在当地的呼叫中心工作，担任客户支持助理，公司提出了让员工在家工作的想法。他们还没有决定关闭他们的办公室，还有许多问题没有答案。他们会提供笔记本电脑还是带回家的台式机？互联网和电话服务将如何工作？谁将做什么？将如何处理 IT 支持？

她肯定没有准备好接受我对她办公室基础设施的大量提问。你知道，我在本文前面谈到的问题类型。

当然，作为一名独立顾问，我已经改进了我的家庭基础设施，以适应我自己的计算需求。我在 WordPress 上工作，通过电子邮件/电话/即时消息进行交流，经常下载大型程序，制作演示文稿，并不断在网上搜索技术信息。我已经很多年没用过 Windows 了，更愿意做一个 Linux 的店，用免费/开源的软件。

我女儿能在我家工作吗？大概吧。这是我的资料。

*   100 Mb/s 下载，10 Mb/s 上传有线互联网接入，通过 Arris TM1602 电缆调制解调器经由频谱。
*   NetTrend TEW711BR 150 150 Mb/s 无线 N 家用路由器。
*   运行频率高达 2.27GHz 的华硕双核英特尔型号 X83VM-X1 笔记本电脑。它有 4GB 的内存和 750GB 的 7200 RPM 硬盘。它还具有非触摸屏 14 英寸 1280×800 彩色液晶显示屏，三个 USB 2.0 端口，无线 N wifi，100/1000mb 有线以太网和 HDMI。这是一台 10 多年前的机器。我在 XFCE 桌面上运行的是 Ubuntu Linux 版本 18.04.04 LTS。
*   联想四核 IdeaPad，U430 型触控笔记本，主频高达 2.30GHz。它有 4GB 内存和 500GB，5400 RPM 硬盘。它还拥有一个触摸屏 14 英寸 1600×900 彩色液晶显示屏，三个 USB 2.0 端口，无线 N wifi，100/1000 Mbs 有线以太网和 HDMI。这台机器已经有几年的历史了，运行的是 XFCE 桌面的 Ubuntu Linux。
*   树莓 Pi 4 四核 ARM 处理器 DIY 蒸汽朋克笔记本运行 800 MHz。它有 4GB 的内存和 64GB 的 SanDisk Extreme 微型 SD 卡。有一个带显示板的非触摸屏 11.5 英寸 1200×800 彩色 LCD，通过 HDMI、无线交流 wifi、一个千兆以太网端口、两个 USB 3.0 和两个 USB 2.0 端口连接。Pi 还有两个 HDMI 端口。它在 LXDE 桌面上运行 Raspbian Linux。
*   三星 S8+智能手机，30GB/月。家庭数据计划和无限的文本和通话。它运行安卓系统，可以作为一个 WiFi 接入点连接到手机网络。
*   19 英寸三星 HDMI 电视/显示器。
*   1606 型激光打印机设置有一个固定的本地 IP 地址，通过 Cat 5 电缆连接到防火墙后面的路由器。
*   美国电话电报公司与互联网和一部目前无法使用的座机连接。连线就在那里，我们只是没有使用他们的硬件或服务。

## 你现在能做什么？

收集你的清单和图表，把它们放在一个文件夹里。我喜欢在我的项目中保持一个经常更新的工程笔记本，以记录重要的信息、配置、相关的规格和各种一次性的想法和笔记。

如果你还在办公室，安排时间和其他团队成员在一起，表达你对在家远程工作的兴趣。即使你已经不在办公室，也要这样做。公司领导肯定在想办法让公司大门敞开，员工健康，业务正常运转。这是一个所有人参与的挑战，每个人都在探索新的领域。一头扎进去，自己想清楚你需要什么。在适应其他工作方式的同时，有很多机会去提升和学习新事物。谁知道呢，你可能只是为自己发明了一份新工作。

测试您的家庭基础设施，让您拥有的一切顺利运行。如果你的网络连接很慢，找出原因并解决它。也许你需要一个更好的计划？您使用的是快速域服务器吗？你在一个清晰的 WiFi 频道上吗？你应该把你的路由器搬到你家更中心的地方吗？是时候买一台新电脑了吗？寻找更好或更大的数据计划(电缆和电话)。例如，如果你在呼叫中心工作，调查一下是否可以接通固定电话。别忘了把笔记记在你的工程笔记本上。

你在不同的浏览器、办公套件和电子邮件客户端(包括笔记本和智能手机)上是最新的吗？如果你现在发现自己有额外的休息时间，也许可以做一些研究和探索。开源软件，如 Firefox、LibreOffice 和 Thunderbird(电子邮件)都是非常强大的应用程序，可以从网上下载。抓住他们，给他们一个旋转。如果您需要使用自己的设备，并且不再有可用的商业应用程序，它们可能是一个选项。

最后，花点时间把你在办公室里做的事情和你在家里可能做的事情联系起来。弄清楚你过去在哪里，你想去哪里，然后用你找到的解决方案填补空白。
