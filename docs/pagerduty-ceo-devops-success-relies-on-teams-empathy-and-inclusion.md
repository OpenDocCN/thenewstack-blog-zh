# page duty CEO:devo PS 的成功依赖于团队、同理心和包容

> 原文：<https://thenewstack.io/pagerduty-ceo-devops-success-relies-on-teams-empathy-and-inclusion/>

[首席执行官詹妮弗·特哈达:DevOps 的成功依赖于团队、同理心和包容](https://thenewstack.simplecast.com/episodes/jennifer-tejada-ceo-at-pagerduty-devops-success-relies-on-teams-empathy-and-inclusion)

上周在旧金山举行的 page duty 峰会上，[page duty](https://www.pagerduty.com/)的首席执行官詹妮弗·特哈达来到 New Stack 播客展台，与亚历克斯·威廉姆斯和 TC·柯里谈论团队。  峰会的主题是运营和机遇，特哈达在主题演讲中谈到了以团队为中心和信任的重要性。  但是怎么去呢？

“在有秩序之前往往会有混乱，”她承认道。人们的工作方式正在改变。  由于分秒必争，工作人员分布在各地，他们需要协同工作。她说，实时运营与技术有关，但更重要的是帮助人们最大限度地利用他们拥有的时间、投资和工具，为他们的客户和业务带来最佳结果。

如今，大多数公司都处于向持续交付和开发运维文化转变的某个阶段。但是你如何训练管理人员来管理这些分散的团队呢？

PagerDuty 在领导力发展和管理培训上投入了大量资金来解决这个问题。  “在我们的行业中，我看不出有多少公司像 80 年代的 IBM、GE 和宝洁那样培养下一代总经理。我们面临着未来缺乏足够的训练有素的经理的危险。”

她说，必须在最接近行动的时候做出决定。当你给员工尽可能多的信息时，他们会做出最好的决定。

PagerDuty 的领导管理培训反映了这一新方向。他们希望自己的经理更像一个仆人式的领导者，带领团队成员成为最好的自己，而不是自上而下。“这是为每个人的成功奠定基础，”她说。
