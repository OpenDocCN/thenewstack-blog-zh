# 马克斯·奥格登的留胡子节在波特兰很奇怪

> 原文：<https://thenewstack.io/tns-analysts-code-gui-keep-node-weird-portland/>

那是 12 月一个下着蒙蒙细雨的日子，New Stack 创始人亚历克斯·威廉姆斯(Alex Williams)快速穿过希尔顿酒店的玻璃门，虽然速度不够快，无法阻止俄勒冈州 T2 市轻轨镶嵌街道上的静电灰尘与他一起进入。

大厅里弥漫着被雨水打湿的泊车垫和[萨沃伊卷心菜卷](https://twitter.com/aviarypdx/status/457255222293716992)的味道，里面有茄子、西葫芦和墨西哥豆瓣菜，配有柠檬香膏肉汤。在走廊的一端，一张宣传煎饼早餐的彩色海报——已经端上来并被分解了——被翻过来，正面对着墙。随着 9 月份 Node.js v4.0.0 的[通用版](https://nodejs.org/en/blog/release/v4.0.0/)的发布，几十台抛光的自动扶梯挤满了数百名 [Node.js Interactive 2015](http://events.linuxfoundation.org/events/node-interactive/program/schedule) 与会者中的许多人，他们代谢着那些短栈碳水化合物，并感激 Node.js 和 io.js 项目解决了他们的分歧。

在一个分隔开的会议室里，一个猫一样的声音正在读出用开源模块新挖掘的数据。在外面，即使透过厚厚的黑木镶板，也能听到镇民们庆祝木材大联盟冠军的声音。在闭会期间休息的高峰时段，威廉姆斯在一个优雅的门厅中心会见了他的主题:留着金发大胡子的俄勒冈州本地人[马克斯·奥格登](https://thenewstack.io/the-new-stack-makers-dats-max-ogden-building-a-github-for-data/)，华纳罗宾斯、佐治亚州 Lords 的 GitHub 开发者和设计师[杰西卡·洛德](http://jlord.us/about.html)，以及 Voicebox 卡拉 ok 休息室的技术副总裁[卡特·萨斯顿](https://www.linkedin.com/in/carterthaxton)。

在远处，沿着一条长满青苔的混凝土小路，环绕着一个看似巨大的仍可负担得起的工匠之家，一只东南邻居的猫不知不觉地走过一台高清视频摄像机，走向一碗工匠食物。

一个可怕的笑容爬过威廉姆斯的脸。“波特诺迪亚，”他想，他自己；很快他就会大声说出他的发明，也许会考虑为他在[新城市](https://neocities.org/)的想法创建一个网站。但现在，这位新生的新词专家以一种沉默而隆重的方式，揭开了一个滑稽的超大尺寸播客麦克风，并将其展示给大家，依次与他敏锐的博主凝视相遇。随着他的鼠标手指的刷动，他启动了一些固态存储并点击了“记录”按钮，以便捕捉这一集的新堆栈分析师。听着:
