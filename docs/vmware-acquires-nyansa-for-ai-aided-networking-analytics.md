# VMware 收购 Nyansa 进行人工智能辅助网络分析

> 原文：<https://thenewstack.io/vmware-acquires-nyansa-for-ai-aided-networking-analytics/>

过去一年，VMware 一直在大举收购，其最新的计划收购是本周宣布的总部位于加州帕洛阿尔托的 Nyansa，收购金额未披露。

Nyansa 是一家基于人工智能的网络分析公司，专注于有线和无线网络应用程序和客户端设备的用户体验，如特定分支机构的问题。

“收购 Nyansa 将加速 VMware 在我们业界领先的 SD-WAN 解决方案中为 LAN/WAN 部署提供端到端监控和故障排除功能，”VMware 的, [Sanjay Uppal](https://www.linkedin.com/in/sanjayuppal) 在收购公告中表示。

首席执行官兼联合创始人 [Abe Ankumah](https://www.linkedin.com/in/abeankumah/) 在 VMware 2018 年用户大会上告诉我们。“你已经看到了设备端和应用端的创新，但中间连接它们的东西并不多，”他说。

“用户体验不仅仅是连接到网络，而是‘我对应用程序的体验如何？’我们为 IT 运营部门提供工具，让他们成为网络中更有效的医生。"

Nyansa [使用网络上的爬虫或探针](/nyansa-analytics-for-diving-into-network-health/)从多个来源收集数据，并将低比特率元数据流发送到云后端。它采用了一个推荐引擎，该引擎使用 Cassandra、Apache Spark 和 Elasticsearch 等组件进行监督机器学习。

该系统在每台设备上建立基准性能，并为整个环境建立总体基准。用户可以查看特定时间段的性能，或者调用特定设备来解决特定用户的问题。

“云外包”功能使客户能够匿名地将其性能与运行类似网络基础设施的其他用户进行比较

Nyansa 的独立于供应商的 AIOps 平台 Voyance 可以将来自不同硬件供应商的网络遥测整合到一个仪表板中。Nyansa 表示，它分析了来自优步、通用医疗保健、旧金山国际机场和斯坦福大学等公司数千个客户站点的 2000 多万台客户端设备的网络流量。

这对 VMware 客户意味着什么？Uppal 在一篇博客文章中写道:

首先，Nyansa 可以主动预测客户端问题，优化其网络，更好地支持关键物联网设备的行为，并根据实际用户、网络和应用数据来调整基础设施的变化。第二，您将能够利用 Nyansa 的数据接收和分析的广度和深度，包括通过跨多供应商有线和无线局域网环境的 API 进行数据包分析和度量。最后，Nyansa 的 AI/ML 功能与 VMware 现有的分析、可见性和补救功能相结合，将使您能够更轻松地对虚拟云网络进行操作和故障排除，并加快自我修复网络的实现。

据《硅谷商业杂志》报道，Nyansa 在两年前的最近一次融资后估值约为 6500 万美元，并筹集了约 2650 万美元。

在过去的一年里，VMware 收购了 [Heptio](https://thenewstack.io/vmware-to-acquire-heptio-to-boost-enterprise-kubernetes-expertise/) ，为其在去年 8 月 VMworld 2019 上宣布[将全力以赴 Kubernetes](/with-heptio-and-pivotal-vmware-doubles-down-on-kubernetes/) 做好了准备。它还收购了提供应用开发工具、数据管理产品和分析智能平台的 Pivotal。 [Bitnami](/vmware-to-acquire-bitnami-the-app-marketplace-platform-and-container-packager/) ，打包组件运行在多个云服务上；以及端点安全软件厂商[炭黑](https://www.networkworld.com/article/3445383/vmware-builds-security-unit-around-carbon-black-tech.html)。

该交易预计将在未来几个月内完成，取决于惯例成交条件。
