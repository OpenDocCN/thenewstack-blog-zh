房地产网站Trulia如何使用人工智能定制用户体验

原文：https://thenewstack.io/real-estate-platform-trulia-utilizing-machine-learning-apache-kafka/#0001-01-01

这一集的 The New Stack Makers 播客探讨了房地产信息提供商 Trulia 如何利用人工智能和计算机视觉为其用户提供更加个性化的体验。在这次采访中，Trulia的数据工程副总裁 Deep Varma 采访了TNS执行主编 Joab Jackson :


房地产平台Trulia如何利用机器学习与Apache Kafka

几乎实时地接收大量数据是当今许多组织不得不考虑的事情，尤其是那些专注于使用客户指标来塑造其产品和服务的组织。在讨论中，Varma注意到Trulia利用 Apache Kafka 为其个性化平台提供支持。

“我们将事件标记到基于Kafka构建的实时消息层中。我们已经连接了我们的机器学习系统，以查看那些实时系统，并使用那些模型来预测客户感兴趣的内容。最后一层是实时API，允许桌面、移动和网络浏览器实时为客户提供个性化体验，”Varma说。

该公司还建立了一个强大的图像识别库，允许该服务识别房地产清单照片中的特征，如硬木地板和凸窗，这些都是清单本身忽略的。Varma解释说，Trulia还利用了针对Python和GPU的 Cython 静态编译器，而不是CPU来满足其处理密集型数据流的要求。
