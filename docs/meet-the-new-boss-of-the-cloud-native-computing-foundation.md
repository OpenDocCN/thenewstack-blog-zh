# 见见云计算原生计算基金会的新老板

> 原文：<https://thenewstack.io/meet-the-new-boss-of-the-cloud-native-computing-foundation/>

欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)，在这个播客中，我们将讨论云计算领域的最新新闻和观点。在本周的节目中，我们采访了[普里扬卡·夏尔马](https://www.linkedin.com/in/pritianka/)，她是[云本地计算基金会](https://www.cncf.io/)的新任总经理，讲述了她丰富的工作经历以及她推动 CNCF 前进的愿景和战略。CNCF 首席技术官 Chris Aniszczyk 也加入了会议。

TNS 编辑和营销总监[利比·克拉克](/author/libby/)主持这一集，旁边还有 TNS 高级编辑[理查德·麦克马努斯](/author/richard/)和 TNS 执行主编[约阿布·杰克逊](/author/joab/)。

[第 120 集:普里扬卡·夏尔马——云原生计算基金会的新老板](https://thenewstack.simplecast.com/episodes/episode-120-priyanka-sharma-the-new-boss-of-the-cloud-native-computing-foundation)

本周，CNCF 宣布 Sharma [现在将领导](https://thenewstack.io/priyanka-sharma-assumes-leadership-of-the-cloud-native-computing-foundation/)云计算原生计算基金会，接替前执行董事 [Dan Kohn](https://www.dankohn.com/) 的职位。

Sharma 之前是 GitLab 的云原生联盟总监，他为这一领导职位带来了丰富的经验，包括参与了几个 CNCF 项目，并且是 CNCF 董事会的成员。她还担任各种社区最终用户的联络人。实际上，她给 CNCF 带来了全方位的经验，与最终用户、项目贡献者和董事会成员一起工作。

为什么要研究像分布式计算这样棘手的技术呢？夏尔马回答道:

真正打动我的是相关人员的体贴。当你在处理非常大规模的系统时，你必须有一定程度的谦逊。我们会尽力而为，但我们知道，我们不是这里的神。它画的那类人真的很酷。我和他们相处得很好。无论是计算机科学还是社会科学，最终都是关于你的同事，你的学习对象，我和他们很合得来。

我们还向 Aniszczyk(自 2015 年该组织成立以来一直在 CNCF 工作)介绍了 CNCF 和云原生技术的各个方面，包括他对这个新的后 COVID 时代的初创公司的展望，当然，还有云原生技术的广阔的 [CNCF 互动前景](https://landscape.cncf.io/):

“地图只是反映了现实的情况。有太多太多的技术了，否则我们会对我们的终端用户和会员撒谎。我认为这反映了空间是复杂的。这很好。有人在想办法简化事情，这有助于我们的社区意识到有哪些选择，”他说。

然后，在节目的后期，我们讨论了本周的其他几个热门播客和帖子:

[云计算本地计算基金会](https://www.cncf.io/)和 [GitLab](https://about.gitlab.com/) 是新堆栈的赞助商。
