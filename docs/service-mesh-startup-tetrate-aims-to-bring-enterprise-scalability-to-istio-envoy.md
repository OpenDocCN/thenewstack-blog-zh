# 服务网格初创公司 Tetrate 旨在为 Istio 带来企业可扩展性，Envoy

> 原文：<https://thenewstack.io/service-mesh-startup-tetrate-aims-to-bring-enterprise-scalability-to-istio-envoy/>

企业服务网格初创公司 [Tetrate](https://www.tetrate.io/) 今天秘密启动，获得了 1250 万美元的资金，并发布了一些开源工具，尽管其主要产品在今年第三季度发布测试版之前仍然不可用。

然而，该公司的发布也伴随着这样的消息:它已经作为银会员加入了[云本地计算基金会](https://www.cncf.io/) (CNCF)，并将与谷歌合作，通过与最近发布的[云服务平台整合，将其产品带给混合云用户。Tetrate 还首次发布了将于今年春天晚些时候举行的行业会议](https://thenewstack.io/googles-cloud-services-platform-brings-managed-kubernetes-to-hybrid-cloud/) [Service Mesh Day 2019](https://www.servicemeshday.com/) 的消息。

“我们使用 Envoy 和 Istio 作为构建模块。如您所知，采用 Istio 存在复杂性。我们想简化它。我们还希望让它在任何地方都能工作——在虚拟机和裸机上，而不仅仅是在 Kubernetes 上。“Istio 目前与 Kubernetes 合作，但企业客户有许多遗留工作负载。我们的第一款产品将有助于提供工作负载之间的安全无缝连接，并帮助公司向容器和公共云过渡。”

根据公司声明，Tetrate“通过企业级可扩展性、性能和生态系统适配器来增强这些项目，同时还将它们扩展到公共云和私有云的棕色区域虚拟机环境中。”Talwar 进一步解释说，Tetrate 正在“构建一个平台，该平台将在裸机、数据中心、私有或公共云中运行的虚拟机或容器中运行的所有工作负载(单片、微服务、功能)之间提供无缝、安全的互连。”

虽然主要产品目前仍不可用，但 Tetrate 还发布了 GetEnvoy、 [Apache Skywalking](https://www.tetrate.io/blog/category/observability/) 和 Tetrate Istio Cloud Map Operator，这三款开源工具旨在加速组织向微服务的过渡。

根据新推出的 Tetrate 网站，GetEnvoy 为 CentOS、Ubuntu 和 Mac 提供经过测试和认证的 Envoy 版本，Talwar 表示，这“消除了开始使用 Envoy 的一些痛苦”，这是一项目前非常耗时的任务。与此同时，Tetrate 的 Istio 云地图运营商“与 AWS 云地图同步，自动将 AWS 服务发现信息填充到 Istio 服务条目中”，并且“允许在 Kubernetes 集群中运行的服务与 AWS 云地图中注册的任何服务进行通信，无论它们是在 EKS、ECS 还是与云地图集成的任何其他服务中运行。”

最后，Apache Skywalking 是一个可观察性分析和应用性能管理平台，它“为云原生工作负载提供分布式跟踪、服务网格遥测分析、指标聚合和可视化”，并由 Tetrate 团队成员[吴声](https://github.com/wu-sheng)构建。

这最后一点是 Tetrate 团队的骄傲之一，因为 Talwar 说他们都非常熟悉他们正在研究的技术，并指出“所有早期团队都是 Istio 和 Skywalking 和 Envoy 的贡献者和维护者。”他还谈到了这样一个事实，即这个由 20 名成员组成的团队分布在 8 个国家，他们“试图保持一种围绕技术和文化开放的核心价值观。”

“Tetrate 的团队推动了微服务网络堆栈的一些基本创新，从而赢得了 CNCF、谷歌云和特使创造者 Matt Klein 的信任，”戴尔技术资本董事总经理 [Deepak Jeevankumar](https://www.linkedin.com/in/deepakjeevankumar/) 在一份公司声明中说。“我们期待帮助 Tetrate 将他们的创新带给正在经历数字化转型之旅的财富 500 强企业。”

Talwar 还指出，该公司将于今年春天晚些时候在 2019 年的[服务网格日发布另一项重大公告，该活动由 Tetrate 主办，将于 2019 年 3 月 29 日在三藩市举行，由 CNCF 和谷歌提供支持。](https://www.servicemeshday.com/)

“这需要一段时间，这将是一个漫长的旅程。我们将在这个过程中开发一系列开放工具，”塔尔瓦说。“我们希望企业采用 Envoy 和 Istio，我们希望帮助他们做到这一点。”

云计算原生计算基金会是新堆栈的赞助商。
