# SXSW 2017:Capital One 如何通过开源成为技术第一

> 原文：<https://thenewstack.io/sxsw-2017-capital-one-became-technology-first-open-source/>

人们希望银行和其他企业能够更加实时，数字优先，并能够预测客户需求。这就是为什么该公司越来越多地构建和开源自己的软件，并授权工程师使用现代架构快速创新。在这一集的《新堆栈制造商》中，我们采访了[Capital One deve exchange](https://developer.capitalone.com/)的开发者营销总监 [Lorinda Brandon](https://www.linkedin.com/in/lorindabrandon/) ，了解他们如何成为一家技术领先的公司。
