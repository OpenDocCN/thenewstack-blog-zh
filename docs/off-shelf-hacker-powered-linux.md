# 现成的黑客:由 Linux 驱动

> 原文：<https://thenewstack.io/off-shelf-hacker-powered-linux/>

Linux 最近庆祝了它的 25 岁生日。

我几乎在 Linux 上做所有的事情。

我是一个讨厌微软的人吗？没有。该公司已经建立了一个有利可图的业务，雇佣了数千人，每天都让数百万台计算机高效地为用户服务。在很长一段时间里，MS-DOS 和 Windows 是 PC 领域唯一的游戏。但是 Windows 根本不符合我的需求。

苹果电脑很有趣。我女儿有一台又快又好用的 MacBook Pro。有一个漂亮的屏幕，一个漂亮的箱子，我说过它很快吗？不过，拥有一辆车需要很快的钱。我女儿几年前花了 2100 美元。

1986 年，2100 美元为我买了第一台电脑。这是一台全新的[东芝 T-1100 Plus 笔记本电脑](https://en.wikipedia.org/wiki/Toshiba_T1100)，配有令人尖叫的 4.7 MHz 80C88 处理器、640K 内存、双 720K 软盘和 MS-DOS 2.1。

有一次，我在那个古老的遗物上短暂地尝试了一下 [Minix](https://en.wikipedia.org/wiki/MINIX) 。我已经为这个小型的 Unix 发行版做好了准备，它最初是由 Andrew Tanenbaum 为他的学生设计的。当托瓦尔兹在开发 Linux 的时候，他和 Tanenbaum[在操作系统设计上有过争执](https://www.ibm.com/developerworks/community/blogs/6e6f6d1b-95c3-46df-8a26-b7efd8ee4b57/entry/linux_is_obsolete_a_must_read_debate_between_andrew_s_tanenbaum_and_linus_torvalds34?lang=en)。几年前，也就是 1981 年，当我在普渡大学的时候，我被介绍认识了 Unix。

所以你可能会说，Linux 和我都“经验丰富”，至少在技术方面是这样。但是今天的情况与过去的好时光大不相同。

## 在 Linux 之前，有 Unix

所有古老神秘的 Unix 传说都是真的。

在普渡大学，不幸的新手用户不得不向胡子拉碴、脾气暴躁的研究生导师寻求任何帮助，这位导师在凌晨 3 点管理电气工程或 Ag 大楼的终端室。通常有几十个航站楼，挤在一个巨大的高等教育砖砌纪念碑的地下室里。房间总是光线昏暗，除了阴极射线管屏幕发出令人安心的绿光。Unix 运行在强大的“大铁”大型机之上，当您在命令行中键入时，您几乎可以实际感受到它的强大。

即使在今天，Linux 新用户也经常被终端的神秘和命令、文件名和系统进程的迷宫所惊吓。

在 90 年代早期，作为一名制造工程师，我管理了几台 [Sun 的“披萨盒”机器](https://en.wikipedia.org/wiki/SPARCstation_1)来支持工具设计组。sun 运行的是 Unix 版本，通过同轴电缆联网，内置 SCSI 硬盘。通过 Unix 和网络访问，系统管理变得简单，允许使用脚本和 **cron** 轻松实现自动化备份和维护工作。

谁对当前的 Raspberry Pi、CHIP 或 Pine64 板有什么想法？您在项目中利用这些能力了吗？当 Linux 完全侵入单板计算机(SBC)市场时，这是值得思考的事情。大量未开发的资源正等着雄心勃勃的个人在 2×3 的主板上释放 Linux 的力量。

最近的一篇读者评论总结了这一切，“Linux…大胆地去了 Windows 不能去的地方。”

在太阳齿轮之后的几年，当我在美国电话电报公司计费集团工作的时候，我有幸参与了一些贝尔实验室的移植。

你听说过的关于贝尔实验室人员的[故事](http://business.time.com/2012/03/21/how-bell-labs-invented-the-world-we-live-in-today/)？都是真的。他们是臭名昭著的科技怪兽。现实生活中的超级开发人员，非常专注，和他们一起出去玩总是很有趣。即使是商业类型的人，比如我的朋友 Ray，在召集 Unix 和编程专家并保护他们不受外界影响方面也是绝对的高手。

Unix 无疑将各种各样聪明、积极的人聚集在一起。

## 那是过去，这是现在

几年前，Unix 毫不客气地把接力棒传给了 Linux。回过头来看，我看不出 Unix 和 Linux 在系统方面或思维方式上有什么不同。

但是今天的硬件确实有点不同。是吗？

拿[单片机](https://getchip.com/pages/chip)来说。这是黑客的梦想。你可以黑软件。可以黑硬件。你可以黑掉文件。你可以黑掉印刷电路板布局本身。向通用输入/输出引脚添加输入和输出是显而易见的。马力是惊人的，能源消耗几乎为零。Linux 一如既往地强大。

只要在网站上输入信用卡号码，一个芯片就会神奇地出现在你的邮箱里。见鬼，你甚至不需要成为一个博士候选人来使用这个该死的东西。

你最近参加过 Linux 会议吗？做好准备，因为他们都是非常聪明、积极的人，对黑客和完成工作充满热情。

Linux 用户组也是如此。开始出现，开始认识人。你会惊讶于他们知识的深度和多样性。请注意，你可能仍然会发现一些胡子拉碴、脾气暴躁的权威人士，他们给人的感觉是直率和对抗性的。随遇而安，建立真诚的关系，会后冷淡一点，向最好的人学习。

这个月早些时候，我在一台 9 美元的 Linux 电脑上写了一个关于 T2 的故事。本周，我又回到了我的老爷车上，那是一台用了 7 年的 64 位华硕笔记本，配有一个老化版本的 Xubuntu。

是的，Linux 完美地适应了。

我的沃克和西梅汁呢？
