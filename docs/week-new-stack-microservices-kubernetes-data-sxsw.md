# 本周的新堆栈:微服务、Kubernetes 数据和 SXSW

> 原文：<https://thenewstack.io/week-new-stack-microservices-kubernetes-data-sxsw/>

[本周新闻:微服务、Kubernetes 数据、SXSW](https://thenewstack.simplecast.com/episodes/this-week-in-news-microservices-kubernetes-data-and-sxsw)

大家好，欢迎来到 New Stack Context，在这个播客中，我们回顾了本周在云原生技术/大规模应用程序开发方面最热门的新闻，并展望了我们预计在未来几周将获得更多关注的主题。

本周，我们邀请到了 TNS 记者[米歇尔·吉诺](https://thenewstack.io/author/michelle_gienow/)，TNS 研究总监[劳伦斯·赫特](https://thenewstack.io/author/lawrence-hecht/)和 TNS 创始人[亚历克斯·威廉姆斯](https://thenewstack.io/author/alex)。本周，Gienow 在我们新的关注微服务的每周专题系列中发表了第二篇文章。在接下来的几个月中，我们将从头到尾介绍云原生微服务，首先介绍一些组织在权衡迁移到微服务架构的决定时应该考虑的事情。在这一集里，Michelle 分享了一些她到目前为止所了解到的关于公司为什么采用微服务架构的信息。

赫克特为我们写了一个每周数据专栏，出现在周六，以更简短的形式，出现在我们的周五简讯中[。上周六，他从 CNCF 最近的 Kubernetes 社区调查中提取数据，并询问云客户是否选择大型云提供商的新 Kubernetes 即服务产品。](https://us11.campaign-archive.com/home/?u=ab6c02b160780b8e6569144f8&id=bbb31f1504)

此外，本周还有一个额外的环节:TNS 创始人亚历克斯·威廉姆斯(Alex Williams)本周从奥斯汀的 SXSW 打来电话，他在那里与 Capital One 和 Atlassian 等公司就价值、数据流和分析以及应用程序架构等主题进行了多次精彩的对话。他分享了他在那次活动中的一些收获。

TNS 编辑部主任[利比·克拉克](https://twitter.com/LibbyMClark)和 TNS 总编辑[约阿布·杰克逊](https://twitter.com/Joab_Jackson)一起主持了这一集。
