# 本周新闻:企业无服务器、语音驱动的 IT 管理

> 原文：<https://thenewstack.io/week-news-enterprise-serverless-voice-driven-management/>

[本周新闻:企业无服务器和语音驱动的 IT 管理](https://thenewstack.simplecast.com/episodes/this-week-in-news-enterprise-serverless-and-voice-driven-it-management)

在本周的[新堆栈环境](https://thenewstack.io/podcasts/context)播客中，我们采访了 [Bessemer Venture Partners](https://www.bvp.com/) 的投资人 Jenny Gao，讨论在企业中采用[无服务器技术](/category/serverless/)。我们探索无服务器的未来，以及对于公司来说，完全跳过容器，直接采用无服务器架构是否可行，甚至可能。高本周与人合作为《新堆栈》撰写了一篇文章，总结了该投资公司就利用无服务器技术采访数十家企业的情况。

我们还将讨论语音命令形式的“即时操作”的演变。运营团队能否使用类似 Siri 的虚拟助理来执行日常任务、协调事件响应并接收状态更新？PagerDuty 认为是这样的，他们为下一波数字操作起了一个名字:“语音操作”本周在 New Stack 上，page duty[贡献了一篇关于其正在进行的一个项目的文章](https://thenewstack.io/voiceops-virtual-assistants-ops-environments/)，该项目旨在为其事故响应管理平台创建一个虚拟助手。我们在这里讨论其含义。

TNS 编辑部主任利比·克拉克主持了这一集，TNS 总编辑约布·杰克逊也参加了。
