# 新的栈环境:Covid 时代的可观测性

> 原文：<https://thenewstack.io/the-new-stack-context-observability-in-the-time-of-covid/>

欢迎来到[新的堆栈环境](https://thenewstack.io/podcasts/context)，在这个播客中，我们将讨论云计算领域的最新新闻和观点。在本周的节目中，我们采访了可观察性平台供应商 [Honeycomb.io](https://www.honeycomb.io/) 的首席执行官 [Christine Yen](https://www.linkedin.com/in/christineyen/) ，讨论了该公司最近的定价变化，以及随着越来越多的公司转向云计算，可观察性实践和工具如何在更广泛的范围内做出改变，以应对新冠肺炎。

TNS 编辑和营销总监[利比·克拉克](https://thenewstack.io/author/libby/)主持这一集，旁边还有 TNS 高级编辑[理查德·麦克马努斯](https://thenewstack.io/author/richard/)和 TNS 执行主编[约阿布·杰克逊](https://thenewstack.io/author/joab/)。

[第 119 集:Covid](https://thenewstack.simplecast.com/episodes/episode-119-observability-in-the-time-of-covid) 时的可观测性

Honeycomb 本周改变了定价结构，以反映企业的成本现实和新冠肺炎的长期影响。该公司最近还发布了一项调查结果，显示一半接受调查的开发人员目前没有使用 observability，但 75%的人计划在未来两年内这样做。4 月，该公司[发布了 OpenTelemetry 的开源导出器](/opentelemetry-steps-up-to-manage-the-mayhem-of-microservices/)，允许团队将开源项目的遥测数据导入任何可观测性平台，包括他们自己的平台，也包括他们的竞争对手。

日元在谈到价格变化时表示:

*我们以前的定价是，你购买一定量的存储和千兆字节，并在一段时间内为一定量的数据接收付费，也是千兆字节。我们觉得这对于人们来说更难映射到他们现有的工作流程，更难预测。因此，我们转向了每月事件摄取模型，一个轴，一种方式来扩展您的使用。*
