# 现成的黑客:Arduino 还是 Raspberry Pi？

> 原文：<https://thenewstack.io/off-shelf-hacker-arduino-raspberry-pi/>

在最近几周的[现成黑客](/tag/off-the-shelf-hacker/)中，我已经详细讨论了在我的一些项目中使用[树莓 Pi](https://www.raspberrypi.org/) 的问题。我喜欢称 Pi 为纳米 Linux 设备。它仍然是一个“微控制器”

一个 [Arduino](https://www.arduino.cc/) 也是一个微控制器。

怎么回事？如果它们都是微控制器，为什么你会在一种情况下使用 Raspberry Pi，而在另一种情况下使用 Arduino？显然，它们不是一回事。

我在我的会议技术演讲中得到这个问题。当您将一些“大脑”引入现成的物理计算项目时，了解每个微控制器的相似性、差异、功能和能力无疑是非常重要的。请记住“物理计算”这个术语。

探索在哪里使用这两种方法是本周专栏的主题。

## **苹果和橘子**

总结相似之处是一个很好的起点。

Arduino 和 Raspberry Pi 都运行在微处理器内核上，具有一定量的板载内存和存储，当然是可编程的，并且具有通用输入/输出(GPIO)引脚，可供现成的黑客使用。它们都可以很容易地在网上买到。这两个模块都有巨大的开发者社区，大量的支持和可靠的跟踪记录。

根据描述，你会认为这两者非常相似，可能会做许多相同的工作。

你是对的。

但是他们也做一些事情，特别是物理计算任务，以相当明显不同的方式。

看看这张 Pi 与 Arduino 特性和功能的对比图。

function model 行给出了第一个提示，说明为什么您可能使用 Arduino 而不是 Raspberry Pi，反之亦然。

物理计算的基本概念是读取一些输入，进行一些计算，然后根据输入和计算结果调整一些输出。

常识表明，一心一意的设备，如 Arduino，其唯一的焦点是执行这些任务，在执行这些任务时，将比多任务、多用户操作环境设备(如 Raspberry Pi)快得多。

使用下面的提示，让你开始如何选择电路板，然后随着你获得原型项目的经验，扩展你的知识。

## **选择技巧 1——速度需求**

如果您的项目需要读取传感器、比较输入值或计算一些结果并实时或接近实时地改变输出，Arduino 型微控制器是该项目的理想选择。在无人机中使用 Arduino 是一种可能的应用。你可能会从加速度计或全球定位系统感应数据，然后改变转子的速度，以保持飞机稳定飞行。

> 如果您的项目需要读取传感器、比较输入值或计算一些结果并实时或接近实时地改变输出，Arduino 型微控制器是该项目的理想选择。

这份工作不太适合树莓派。Pi 的操作环境(Linux)的多任务性质并不适合于监视加速度计如何对运动做出反应，然后立即调整转子速度。当然，通用输入和输出对 Pi 很重要，可以针对速度进行优化。今天，他们只是不能像 Arduino 那样快速反应。

## **选择提示#2 —复杂的界面**

假设你想在你的可穿戴设备上播放视频，以便在你的会议展台上与潜在客户交谈时吸引注意力。

虽然 Arduino 可以点亮 TFT 彩色显示器，并在存储在其 micro-SD 卡上的位图图像中循环播放，但它永远无法在那个小小的 LCD 屏幕上以每秒 30 帧的速度播放你定制的全色 5 MB mp4 视频。

私家侦探可以不费吹灰之力完成这项壮举。

您不仅可以设置 Pi 来显示视频，还可以通过脉宽调制(PWM)使用 GPIO 引脚来读取开关或点亮 led 并运行电机。添加一个 [USB WiFi 模块](http://www.walmart.com/ip/17419471?wmlspartner=wlpa&selectedSellerId=0&adid=22222222227014437374&wl0=&wl1=g&wl2=c&wl3=40882078712&wl4=&wl5=pla&wl6=78811760912&veh=sem)或 esp8266 模块，你的可穿戴设备就可以通过互联网消费数据并与其他程序互动。

您也可以运行连接到 HDMI 显示器的 Pi，将无线键盘/鼠标垫连接到一个 USB 端口。有相当多的桌面将运行在 Raspbian Linux 的标准负载上，这是典型的默认操作系统。Arduino 没有视频输出，更不用说在 HDMI 大屏幕上显示酷酷的桌面了。

底线是，如果你的项目需要一个复杂的用户界面，Pi 就是你的设备。

## **选择技巧 3——克隆支持最小可行项目(MVP)**

假设你想让一个喜庆的圣诞小挂件上的几个发光二极管闪烁。Pi 和 Arduino 都可以轻松处理这种琐碎的任务。

读者可能已经看到了最近关于 [$5 树莓 Pi 零板](https://www.raspberrypi.org/blog/raspberry-pi-zero/)的新闻。它具有全尺寸 Pi 的许多功能，大小大约是信用卡的一半。对于金钱来说这是很大的权力。

对于一个简单的 LED 闪光项目来说，即使 Pi 为零也是多余的。一个 Arduino 的克隆版本，如 [Digispark](http://digistump.com/products/1) ，将是合适的选择。是的，我知道一台 Digispark 大约要 8 美元。在这件事上跟我一起。

Digispark 的耗电量甚至比 Pi Zero 还要低，因此一个微型电池就可以运行很长时间。当您移除 USB 印刷电路板接头时，它约为 1/2 英寸 x 1/2 英寸，使您的吊坠大小和重量便于挂在衬衫上。最后，还有几个模拟引脚。Pi 只有数字引脚，没有模拟引脚。如果需要任何类型的模拟输入，如电位计或应变计，Digispark 都能满足您的需求。你需要额外的电路来实现这一点。

不仅如此，你还必须等待 Pi 供电的挂件启动，Arduino 克隆会在几秒钟后开始闪烁 led。

这里的信息是使微控制器与工作的复杂性相匹配，而不是相反。

## **Arduino vs . Pi——由你决定**

我已经给了你一些关于如何为你的下一个物理计算项目选择“微控制器”的想法。

在过去的几年中，我们已经看到了新微控制器的大量出现。你猜怎么着…它一点也没有慢下来。新的模式一直在出现。检查一下 [ESP8266](https://en.wikipedia.org/wiki/ESP8266) 和 [Pine64](http://pine64.com/) 。哇，那些板子会放在哪里？一定要分享你的想法。

如今，现成的黑客有大量的硬件选择。而且，情况越来越好。

为你的项目选择一个合适的主板，然后做一些很酷的东西。

顺便说一句，我通常会从爱人那里得到一两个微控制器作为圣诞礼物。给自己一些暗示还不算太晚。
