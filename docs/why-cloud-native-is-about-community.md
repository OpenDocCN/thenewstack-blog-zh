# 为什么云原生是关于社区的

> 原文：<https://thenewstack.io/why-cloud-native-is-about-community/>

Cloud native 实际上只取决于社区提供的支持和输入。正是本着这种精神，[云原生计算基金会(CNCF)](https://cncf.io/?utm_content=inline-mention) 继续对社区进行大量投资，以支持新的和现有的项目，包括 Kubernetes、Prometheus 和 Envoy，这些都是当今云原生计算的基石。

在上个月在 [KubeCon + CloudNativeCon](https://www.cncf.io/kubecon-cloudnativecon-events/?utm_content=inline-mention) 现场举行的最新一集 New Stack Makers 播客中，CNCF 营销经理 [Bill Mulligan](https://de.linkedin.com/in/bamulligan) 和 CNCF 开发者倡导者 [Ihor Dvoretskyi](https://ua.linkedin.com/in/idvoretskyi/en) 谈到了 CNCF 的云原生信用和 Kubernetes 社区日计划，以及为什么这些和其他计划对于构建当今和未来的云原生工具和基础架构至关重要。

《新书库》的创始人兼发行人亚历克斯·威廉姆斯主持了这一集。

在 Equinix Metal、谷歌、亚马逊网络服务(AWS)和 GitHub 的支持下，云信用计划旨在帮助那些以信用形式为 CNCF 项目捐赠资源的公司建立一个框架。

[为什么云原生是关于社区的](https://thenewstack.simplecast.com/episodes/why-cloud-native-is-about-community)

“我们已经这样做了很多年，但是，最后，我们需要将它作为一个程序化的解决方案，每个项目都可以申请使用这些解决方案和信用，每个可以为我们提供一些东西的公司都不需要直接联系我们，问‘我们能做什么？’”德沃雷茨基说他们现在在我们的网站上有一个神奇的按钮，他们只需填写一份简短的表格，并为我们提供一些数据，这些数据将在以后持续存在，并找出该公司如何有用。"

创建于 2019 年的 Kubernetes 社区日是在世界各地举行的社区组织的活动。CNCF 组织这些活动来帮助促进开源和云原生社区中的教学、协作和联网，以支持 Kubernetes 的采用。

Kubernetes 社区日的一个主要方面是这些活动是如何面向国际参与者的，并且是多语言的。今年计划举办 20 场，这些活动已经或即将在萨尔瓦多、中国、印度、西班牙、韩国和澳大利亚举办。

“Kubernetes 社区日计划是一件非常令人兴奋的事情，它使人们能够在自己的本地建立云原生社区。因此，你可以把它们想象成在特定城市或特定国家组织起来的小型 KubeCons，它们确实是当地社区感兴趣的东西，甚至可能是当地语言，”Mulligan 说。“KubeCon 很棒，但每年只有三场，而且由于会议场地的限制，只能在某些大城市举行。这些案例社区日计划让我们能够扩展到新的地理区域，去不同的地区，甚至使用不同的语言。”
