# 云计算原生计算基础的未来

> 原文：<https://thenewstack.io/future-cloud-native-computing-foundation/>

[云计算的未来](https://thenewstack.simplecast.com/episodes/the-future-of-the-cloud-native-computing-foundation)

在最新一集的 [The New Stack Makers](https://thenewstack.io/podcasts/makers) 播客中，[Cloud Native Computing Foundation](https://www.cncf.io/)首席运营官 [Chris Aniszczyk](https://www.linkedin.com/in/caniszczyk) 与 [Weaveworks](https://www.weave.works/) 首席执行官 [Alexis Richardson](https://www.linkedin.com/in/richardsonalexis/) 一起讨论 CNCF 的所有事情，包括其 2018 年及以后的路线图。

当考虑在 CNCF 或其初始项目中采用新项目时，Aniszczyk 和 Richardson 指出，项目更新的开放性和频率至关重要。引用 CNCF 参与重组 containerd 的治理模式，从“仁慈的终身独裁者”走向公开透明的治理，这在过去和现在都至关重要。

“arc 走向开放治理。我认为这是公平做事的唯一方式，”Aniszczyk 说。

Aniszczyk 谈到了 CNCF 如何将 linkerd 作为一个初始项目。初始项目每年评审一次，根据项目的开发和生产状态升级，最终达到毕业。

kube con/CloudNativeCon China 2018 将于今年晚些时候在中国上海举行，Aniszczyk 和 Richardson 指出，阿里巴巴等企业已成为合作伙伴。Aniszczyk 继续解释说，许多人跳过容器，直接使用虚拟机。

“丹·科恩和我(CNCF 负责人)今年都去过几次中国。就库伯内特斯而言，那里的增长速度是惊人的。我们在那里有一个很大的社区，现在是我们在那里举办活动的时候了。”
