# 注重安全的文化如何改变坏习惯

> 原文：<https://thenewstack.io/how-a-security-minded-culture-can-change-bad-habits/>

如何建立安全意识文化是我们与 Okta 合作的新系列[新堆栈制造商](/podcasts/makers)的最新一集“安全@规模”的主题。该系列探索了现代环境中的安全问题，讲述了来自战壕的故事，包括安全恐怖故事和离奇的失败。

在这一集里，共同主持人 [Alex Williams](/author/alex/) ，New Stack 的创始人兼发行人，和 [Randall Degges](https://www.linkedin.com/in/rdegges) ，安全服务提供商 [Okta](https://developer.okta.com/?utm_content=inline-mention) 的开发者权益负责人，讨论与建立安全意识文化相关的挑战，什么可行，什么不可行。

[Okta 系列——注重安全的文化如何改变坏习惯](https://thenewstack.simplecast.com/episodes/okta-series-how-a-security-minded-culture-can-change-bad-habits)

该节目的特邀嘉宾有 [Deidre Diamond](https://www.linkedin.com/in/deidrediamond/) ，网络安全招聘公司 [CyberSN](https://www.cybersn.com/) 的创始人兼首席执行官，以及致力于为所有性别的人提供招聘解决方案的多元化招聘公司 [Secure Diversity](https://securediversity.org/) 的创始人；安全解决方案提供商 [Lookout](https://www.lookout.com/fr) 的联邦销售工程师 [Victoria Mosby](https://www.linkedin.com/in/victoria-mosby-343a8a24/) 。

莫斯比曾为许多安全提供商工作过，她提到她有“太多可怕的故事可以告诉你”，这是由低劣的安全政策和文化造成的。

“一些主要的事情是，我们制定了大量的政策，但没有人遵循它们——但每个人都想要一个捷径或豁免，”莫斯比说。“因此，简化政策可能是我们能做的最简单的事情。然后是一般的安全意识:每个人都希望他们的工具做他们想让他们做的事情，或者他们想要新奇的东西，而不考虑安全如何发挥作用，然后当安全真的介入时，我们现在是坏人。”

问题的根源在于，为 DevOps 招聘团队成员的组织需要如何更好地沟通机会，特别是以急需的方式促进多元化，以包括代表性不足的群体，如技术领域的女性。

可悲的是，许多有才华的人仍然没有意识到机会，而许多安全方面的关键职位仍然没有实现。当然，安全团队的人才缺口也意味着保护网络和数据的人员减少了。

戴蒙德指出，人员严重不足的团队会导致高度的倦怠和其他问题。“如果因为预算不存在而没有真正的继任计划，那么这本身怎么不是一个漏洞呢？”钻石说。"看着它，我总是激动不已."

沟通和外联是促进变革的方式。为此，安全日会议(Mosby 和 Diamond 都在其中扮演重要角色)和其他类似的会议可以帮助抵消安全团队中过于普遍的多样性缺乏。

戴蒙德说，网络安全日“是为了向任何 18 岁及以上背景的女性宣传网络安全领域的所有工作”。“因此，通过这种方式，我们正在传播关于网络安全的意识，最终这一切都归结为:人类和我们所有人都有超级意识。
