# 随着牧场主的收购，SUSE 将在 Kubernetes 生态系统中获得一席之地

> 原文：<https://thenewstack.io/suse-will-gain-a-foothold-in-the-kubernetes-ecosystem-with-rancher-purchase/>

开源软件提供商 SUSE 正在收购 T2 Rancher Labs，这是第一批提供商业云原生平台的公司之一，基于当时新的 Docker 容器技术。两家公司没有披露交易的价值，交易预计将于 10 月份完成。在此之前，Rancher 将继续作为一家独立公司运营，之后，该产品线将由 SUSE 全球销售团队提供。

此次收购是 SUSE 高管称之为公司“无机增长战略”的第一次，该公司于 2019 年从微焦点[剥离出来，成为一家独立公司。它计划通过继续构建其开源软件组合来继续其为企业服务的轨迹。](https://www.microfocus.com/about/press-room/article/2019/micro-focus-completes-suse-sale/)

此次收购还将帮助 SUSE 在 Kubernetes 市场获得更稳固的立足点。来自[云本地计算基金会](https://www.cncf.io/)的一项最新调查发现，16%的 Kubernetes 用户部署了 Rancher 软件，只有 AWS、谷歌和 Azure 获得了更大的用户群。调查还发现，1197 名 Kubernetes 用户中有 10 人部署了 SUSE 软件来管理集装箱，其中 2 人还部署了 Rancher Kubernetes 或 Rancher Kubernetes 引擎(RKE)。

“SUSE 强大的生态系统将大大加快 Rancher 正在进行的转变组织采用云原生技术的努力，”Rancher 首席执行官[盛亮](https://www.linkedin.com/in/shengliang/)在宣布收购的博客文章[中表示。梁将继续领导 SUSE 的工程和创新项目。](https://rancher.com/blog/2020/suse-to-acquire-rancher/)

Rancher 于 2014 年推出，是首批通过由企业级支持订阅资助的开源商业模式提供基于商业容器的平台的公司之一。公司的成长很大程度上是通过基层营销努力实现的。Rancher 是一家私有的开源公司，总部位于加州的 Cupertino。

今天，该公司提供了许多不同的支持软件包，包括以其[命名的托管 Kubernetes 服务](https://rancher.com/products/rancher/)、 [RKE](https://rancher.com/products/rke/) 集装箱化的 Kubernetes 发行版、 [Longhorn](https://rancher.com/products/longhorn) Kubernetes [块存储软件](https://thenewstack.io/rancher-donates-its-longhorn-kubernetes-persistent-storage-software-to-cncf/)，以及轻量级的 [K3s Kubernetes 发行版](https://k3s.io/)。

SUSE 成立于 1994 年，是第一家商业 Linux 提供商，此后围绕 [SUSE Enterprise Linux](https://www.suse.com/products/server/) 和 [OpenSUSE](https://www.opensuse.org/) 发行版构建了一系列以企业和物联网(IoT)为重点的开源技术。2018 年，SUSE 发布了自己的面向开发者的 Kubernetes 发行版，[通过并入云代工厂 PaaS 软件](https://thenewstack.io/suse-cloud-application-platform-bundles-kubernetes-with-cloud-foundry/)。

自从 2018 年 IBM [收购红帽](https://thenewstack.io/turning-blue-ibm-to-acquire-red-hat/)以来，SUSE 现在号称是全球最大的独立开源公司。该公司报告称，2020 年第二财季(截至 4 月 30 日)的年同比增长率为 30%，全球云收入年同比增长率为 70%。

“只有 SUSE 和 Rancher 的结合才能拥有全球支持的深度和 100%真正的开源组合，包括云原生技术，”SUSE 首席执行官梅丽莎·迪·多纳托在一份声明中自豪地说。
