# Kinvey 的移动应用程序 Focus and Fit in the New Stack 帮助 BaaS 平台筹集了 1080 万美元

> 原文：<https://thenewstack.io/kinveys-mobile-app-focus-and-fit-in-the-new-stack-helps-the-baas-platform-raise-10-8m/>

今年早些时候，当亚马逊网络服务进入后端即服务市场(BaaS)时，它发现了一个巨大的竞争对手。有一些常见的声明称，AWS 将通过新的 AWS 移动服务埋葬竞争对手。AWS 看起来确实令人生畏，但今天 Kinvey 的消息称，它刚刚筹集了 1080 万美元的资金，这似乎反驳了 AWS 将为 BaaS 开发者工具抢走蓬勃发展的移动应用市场的所有战利品的观点。

相反，Kinvey news 反映了一个多样化的新堆栈，它与一个工具生态系统一起出现，施耐德电气等公司正在使用这些工具来简化和加速他们的移动应用程序开发周期。

上周，金维与施耐德电气签署了一项七位数的交易，该公司创建了一个新的工具链，他们称之为“后”堆栈:[自举](https://getbootstrap.com/)、 [Angular、](https://angularjs.org/) [Cordova](http://cordova.apache.org/) 和金维。它附带了一个开发人员 SDK，为他们的 4000 名开发人员简化开发和品牌一致性。该方法旨在帮助 web 开发者成为移动开发者，并消除为 iOS、Android 和移动 Web 开发单独应用版本的需求。

企业公司面临着一场竞赛。他们开发的每个应用程序都代表着更好的优化或增量收入的增加。为了构建这些应用，公司需要一个应用 frmaeworks、协作环境、版本控制、持续交付和其他服务的生态系统。Kinvey 提供了最佳的 BaaS 来填充这个新兴的工具链。

Kinvey 从 NTT DOCOMO Ventures 和威瑞森 Ventures 获得了 B 轮融资。现有投资者 Avalon Ventures 和 Atlas Ventures 也加入了他们的行列。这笔融资使 Kinvey 迄今筹集的资金总额达到 1780 万美元。

NTT 将金威加速进入亚洲市场视为其投资的推动力，金威首席执行官兼创始人斯拉维什·斯里达尔说。现在有更多的运营商提供 iPhone，推动了对更好的移动企业开发工具的需求。威瑞森正在提供自己的云服务，包括移动开发工具。

尽管如此，AWS 在市场上仍扮演着重要角色。正如 Nancy Ghoring 在新的 Stack 上写的那样，新的 AWS BaaS 环境拥有 Cognito 等服务，可以管理用户身份、同步和安全。该服务允许开发者通过脸书、谷歌和亚马逊登录来验证用户身份。它还可以在设备之间同步用户偏好和应用状态等数据。亚马逊还宣布了一项新的移动分析服务，并更新了其移动 SDK，使开发者更容易访问 S3 和 DynamoDB 等 AWS 服务。

正如戈林指出的:

> 独立的 BaaS 提供商提供的功能远不止这些。但是一些开发者可能会发现选择这些 AWS 移动工具比注册第三方更容易。

BaaS 市场一直在整合。红帽上周收购了 Feed Henry，整合到 OpenShift 中。在过去的两年里，脸书收购了 Parse，而 PayPal 收购了 StackMob，并迅速关闭了这项服务。

Kinvey 集成了 AWS、Google 和现在的 VMware。它专注于企业移动市场。凭借在亚洲的业务和与其他新堆栈参与者的合作，该公司看起来可以通过提供一流的功能来面对 AWS 等竞争对手，这些功能可以简化企业市场的开发，目前企业市场正在扩大其应用程序开发。

金维是新堆栈的赞助商。
