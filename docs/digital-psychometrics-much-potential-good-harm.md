# 数字心理测量:好的和坏的潜力如此之大

> 原文：<https://thenewstack.io/digital-psychometrics-much-potential-good-harm/>

很难确定流传的故事的真实性，当时美国总统候选人唐纳德·特朗普的竞选团队利用社交媒体简介向潜在选民投放广告，最终左右了美国总统大选。

但是，每天都在收集更多的关于人们的数据，这些信息可以用来更有效地进行营销和个性化医疗保健和教育，[剑桥大学的计算社会科学家 Sandra Metz](http://www.psychometrics.cam.ac.uk/about-us/directory/sandra-phd) 在本周举行的 ApacheCon 北美会议上表示。

然而，她说，这带来了更大程度的责任，以确保这些信息用于改善人们的生活，而不是针对他们进行剥削，她指的是一个关于青少年使用社交媒体的研究如何有助于识别那些有抑郁风险的人的故事。

她描述了始于脸书的 [myPersonality](http://mypersonality.org/wiki/doku.php) 应用程序的工作，该程序允许用户进行真正的心理测试。超过 700 万人这样做了，其中大约一半的人同意分享他们的研究成果，包括脸书·李克斯。

这些测试探究了诸如个性、价值观、生活满意度和冲动性等问题。

它已经发展成为一个名为[ApplyMagicSauce.com](https://applymagicsauce.com/)的个性化引擎，来自[剑桥大学心理测量中心](http://www.psychometrics.cam.ac.uk/)作为预测用户未来行为的工具。它使用算法将响应与指定的组进行比较。

梅斯展示了外向者的喜好比较，提到聚会、外出、成群结队，与内向者的喜好，如电脑、电脑游戏和动漫。类似地，来自被认为令人愉快的人的词云以“精彩”和“令人兴奋”为特色，而不太令人愉快的人的词云则倾向于脏话——这引起了人群的笑声。

了解一个人的偏好有助于定制更有效的广告，例如向外向的人强调派对，而向内向的人推销一些更安静的东西，比如“美丽不必大喊大叫”。她说，剑桥大学的研究发现，基于这些偏好使用广告的转化率高出 1.5 倍。

进一步的研究正在进行中，图像分析可以从照片中判断外向/内向。并且正在收集更多的数据来实时了解人们。

英国网站 Metro 报道称，各政党在个性化脸书广告上花费了数百万英镑，这一策略预计将在下个月的英国大选前广泛使用。它所谓的“黑暗广告”不同于常规的政治广告，因为不同的用户得到不同的推销。通过使用 Chrome 扩展，一个名为“[的组织瞄准了我](https://whotargets.me/)？”正试图监控这些广告的使用，这些广告根据地点、年龄和性别锁定目标人群。

然而，例如，Matz 指出，仅仅跟踪手机数据就可以洞察一个人的活动、睡眠和与他人相处的时间。她说，这可能有助于识别处于抑郁症风险中的人，但也开辟了它可以用于针对脆弱人群的可能性。

“权力越大，责任越大，”她说，并补充说，使用这些数据的人必须更好地向公众传达数据的可能性，以及数据的使用将如何影响他们的生活。

“[数据]提供了如此多的潜力来帮助我们做出更好的决策，但我们需要确保这些数据被用来改善他们的生活。”
