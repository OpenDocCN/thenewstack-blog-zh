Cloud Foundry Summit Pancake播客:云原生时代的通信

原文：https://thenewstack.io/cloud-foundry-summit-pancake-podcast-communications-cloud-native-era/#0001-01-01

Stackie是新Stack的旅行煎饼机器人，本周冒险前往加州圣克拉拉，为2017年硅谷云铸造峰会举办煎饼早餐和播客录制。当蒸汽朋克黑客Torq博士倾向于挑剔的3-D煎饼制造商时，TNS创始人Alex Williams主持了一场关于 Cloud Foundry 、开放服务代理APIKubo的生动的小组讨论，Kubo是Pivotal和Google正在开发的一个新的开源项目，它使用 Google Bosh 来打包和部署Kubernetes 。小组还讨论了地理上分散的开发团队最喜欢的沟通形式，提到了Twitter、Slack、telepathy和conferences。

本期新堆栈分析播客的主讲人是:

  • Dieu Cao :产品管理关键总监，
  • 霍尔格·穆勒:星座研究副总裁兼首席分析师，
  • 艾比·卡恩斯 : Cloud Foundry执行董事，
  • 莎拉·诺沃特尼:谷歌Kubernetes社区的项目经理。

查看本播客视频的结尾，可以看到Torq博士解释硬件黑客方法并展示他的蒸汽朋克眼球和他的电子会议个性识别设备的特别片段
