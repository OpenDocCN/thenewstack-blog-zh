# 背景:监测和可观测性趋势，KubeCon + CloudNativeCon 2019

> 原文：<https://thenewstack.io/context-monitoring-and-observability-trends-kubecon-cloudnativecon-2019/>

[背景:监测与可观测性趋势，kube con+CloudNativeCon 2019](https://thenewstack.simplecast.com/episodes/context-monitoring-and-observability-trends-kubecon-cloudnativecon-2019)

本周在[新的堆栈环境](https://thenewstack.io/podcasts/context)播客上，从[kube con+CloudNativeCon 2019](https://events.linuxfoundation.org/events/kubecon-cloudnativecon-europe-2019/schedule/)现场录制，我们谈论的都是监控和可观测性。我们的嘉宾是[克里斯滕·克拉布·托鲁普](https://www.linkedin.com/in/krestenkrabthorup/)，胡密欧[的首席技术官](https://www.humio.com/)和[科林·费尔南德斯](https://www.sumologic.com/resource/blog/author/cfernandes/)，相扑逻辑[的产品营销总监](https://www.sumologic.com/)。

Sumo Logic 是一家机器数据分析公司，刚刚宣布了一轮额外的 1.1 亿美元融资，使其价值超过 10 亿美元。Humio 展示了仅在 25 个节点上每天接收 100 太字节的数据，同时提供数据的实时可观察性。两人都站在理解我们可以从机器的运行条件中收集什么情报的最前沿。

我们与他们讨论了他们在数据管理和日志记录方面看到的趋势，随着最终用户收集越来越多的数据，同时希望看到实时分析，这两种做法都发生了巨大的变化。我们还讨论了云原生监控和日志记录方面的变化，包括最近将 OpenTracing 和 OpenCensus [合并为一个项目，称为 Open Telemetry](https://thenewstack.io/opentracing-opencensus-merge-into-a-single-new-project-opentelemetry/) 。

在节目的后半部分，我们提供了我们的顶级播客和故事精选，包括通过一个名为 KubeMove 的新项目免费提供一些专有的 Kubernetes 扩展。我们还讨论了我们最近的 [@Scale 播客](/can-an-open-source-foundation-fix-continuous-delivery-pipelines/)，它面临着新推出的 CD 基金会在标准化大量云原生工具以持续交付方面的挑战，

新的 Stack 编辑和营销总监 Libby Clark 在 TNS 创始人兼出版商 Alex Williams 和执行主编 Joab Jackson 的帮助下主持了这一集。

KubeCon + CloudNativeCon 是新堆栈的赞助商。
