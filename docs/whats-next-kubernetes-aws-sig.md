# Kubernetes AWS SIG 的下一步是什么

> 原文：<https://thenewstack.io/whats-next-kubernetes-aws-sig/>

在今天的《新堆栈制造商》节目中，我们采访了 [Deis](http://deis.io/) 高级软件工程师 [Kris Nova](https://www.linkedin.com/in/kris-nova) 来一睹 KubeCon EU 期间 Kubernetes AWS SIG 的幕后风采。虽然一些特殊利益集团可能有不干涉的组织，但 Nova 提到 AWS 在帮助 SIG 组织将 Kubernetes AMI 引入亚马逊市场方面发挥了重要作用。

由[weaver works](https://www.weave.works/)开创的用于在 AWS 上启动 Kubernetes 的 AMI 正引起许多商业和非商业 Kubernetes 开源开发者越来越大的兴趣。在获得 CNCF 对该图像的赞助后，SIG 的目标是在 2017 年继续其在 AMI 方面的工作。
