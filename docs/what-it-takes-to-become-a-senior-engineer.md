# 成为高级工程师需要什么

> 原文：<https://thenewstack.io/what-it-takes-to-become-a-senior-engineer/>

成为一名高级软件工程师需要的不仅仅是几年的经验——先决条件包括良好的营销意识、面试技巧以及成为一名高级软件工程师所需的其他个人素质。

在这个新的 Stack Makers 播客中，嘉宾 [Swizec Teller](https://www.linkedin.com/in/swizec/) ，作者和医疗保健公司 [Tia](https://www.asktia.com/) 的高级软件工程师，以及微服务编排平台提供商 [Temporal.io](https://temporal.io/) 的开发者体验主管[王自如](https://twitter.com/swyx)，描述了成为高级工程师所需的心态和其他属性。

TNS 新闻编辑 Darryl Taft 主持了这个播客。

[成为高级工程师需要什么](https://thenewstack.simplecast.com/episodes/what-it-takes-to-become-a-senior-engineer)

当你试图获得下一次晋升或一份有机会的新工作时，制作你的简历非常重要。泰勒说，简历“实际上只是一种营销工具——它表明一份好简历的很大一部分是将它视为一种营销工具，然后对它进行定位，以便它能与你想去的公司类型和你想做的角色类型相适应”。

许多工程师可能没有意识到的是，他们也必须交流，并找到记录他们所做工作的方法。王说，那些不愿意展示他们所做的工作的工程师——他们相信工作应该是不言自明的——正在损害他们的职业生涯，尽管他们的意图是好的。

“工程师在告诉人们他们所做的事情时往往过于害羞……你可以把它看作是 y 轴在建设，x 轴在告诉人们，”王说。“你必须做到这两点，才能拥有最大可能的幸运表面积，让事情发生在你身上。”

要想在工程师的职业生涯中取得进步，不仅要看你是一名多么有才华的软件工程师或程序员，还要看你对雇主的价值。这种价值是可以衡量的——或者至少应该衡量，因为有些雇主可能缺乏敏锐度，无法察觉你对他们的价值。

Teller 说:“脱颖而出的一个简单方法是成为考虑他们所做的事情的商业影响的人，而不是只关注代码和代码的人。”

在当今公司合并和收购的世界中，整个部门都可能以残酷的速度被裁员，许多软件工程师可能会觉得他们的职业缺乏发展机会。换句话说，他们已经停滞不前，但可能会在对公司的忠诚和对自己职业生涯最有利的东西之间产生分歧。

泰勒认为，在这些情况下，是时候继续前进了。

“如果你觉得自己处于停滞状态，你可能在意识到这一点之前就已经开始停滞了。特勒说:“尤其是在这个市场，只要换工作就行了，因为——尤其是一旦你有了几年的工作经验——我认为现在以及未来几年，换工作非常容易。”。“每个人都在寻找这样的人，他们在职业生涯中已经工作了三年、三到五年或更长时间，并且已经准备好迎接更大、更好的挑战。”
