## 下载

### Docker

```
docker pull apachecn0/thenewstack-blog-zh
docker run -tid -p <port>:80 apachecn0/thenewstack-blog-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install thenewstack-blog-zh
thenewstack-blog-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g thenewstack-blog-zh
thenewstack-blog-zh <port>
# 访问 http://localhost:{port} 查看文档
```
